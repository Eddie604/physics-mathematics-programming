## 🧮 数学前置：必须搞懂的三个"度"

> 这一节专门帮你补数学。如果你已经懂散度、旋度、梯度的直观含义，可以跳过。

### 梯度 ∇φ（标量→矢量）
**白话**：一座山的高度分布是 φ(x,y)，梯度指向"上山最陡的方向"，大小等于坡度。

**在电磁学中**：电势 φ 的负梯度就是电场 E：$\vec{E}=-\nabla\varphi$。意思是"电场线指向电势下降最快的方向，下降越快电场越强"。

### 散度 ∇·E（矢量→标量）
**白话**：在一个小体积元上，看"有多少东西流出来"。正散度 = 有源（水龙头向外冒水）；负散度 = 有汇（下水道吸水）；零散度 = 流进等于流出。

**在电磁学中**：电场散度 ∇·E = ρ/ε₀——电荷就是电场的"源"。磁场散度 ∇·B = 0——没有"磁荷"，磁感线永远是闭合的。

### 旋度 ∇×E（矢量→矢量）
**白话**：在一个小面积上，看"场绕圈转的强度"。旋度为零 = 场是"保守的"（像重力——爬多高就积累多少势能，与路径无关）；旋度非零 = 场像漩涡——推着你转圈。

**在电磁学中**：静电场旋度为零（可以定义电势）；变化的磁场会产生有旋的电场（涡旋电场——这就是法拉第定律的核心）。

> **一句话总结**：散度管"从哪里来"，旋度管"怎么转"，梯度管"往哪降"。麦克斯韦方程组就是给电场和磁场各写了一个散度方程和一个旋度方程——四个方程完整描述电磁场。

## 🔧 预备公式（不需要背，但要看得懂）

- **拉普拉斯算子 ∇²**：是"散度+梯度"的组合 ∇²φ = ∇·(∇φ)，等于三个方向二阶导数之和。
- **矢量恒等式****：∇×(∇×A) = ∇(∇·A) - ∇²A。这条是推导电磁波的关键，考试可能用上。
- **高斯公式（散度定理）**：把体积分和面积分互相转换——∫体积 ∇·F dV = ∮表面 F·dS。
- **斯托克斯公式（旋度定理）**：把面积分和线积分互相转换——∫面积 (∇×F)·dS = ∮边界 F·dl。

> 这些公式的名字考试常出现，但实际解题你不是在推导它们，而是在用结论。知道它们是"体积↔面积"和"面积↔线"的转换工具就够了。

# 第一章 电磁现象的普遍规律

## 1.1 电荷怎么产生电场

### 核心公式

**库仑定律**：一个点电荷 Q 对距离 r 处的单位试探电荷施加的力：
$$F = \frac{1}{4\pi\varepsilon_0}\frac{Q}{r^2}$$

> **通俗理解**：离电荷越远，力按 r² 衰减（距离翻倍，力变四分之一）。ε₀ ≈ 8.85×10⁻¹² 是"真空介电常数"——真空"软硬程度"的度量，越软（ε₀越小）同样电荷产生的力越大。

**电场强度 E**：单位正电荷在某点受的力。E = F/q。

**叠加原理**：多个电荷产生的总电场 = 每个电荷单独产生的电场矢量相加。这也意味着——如果你算出了单个点电荷的场，多个点电荷的场就是逐个算然后加起来。

---

### 📖 深入理解——高斯定理到底在说什么

**积分形式**：$\oint_S \vec{E}\cdot d\vec{S} = \frac{Q_{\text{内}}}{\varepsilon_0}$

翻译成白话：想象你用一个闭合的"气球"（高斯面）包住一堆电荷。电场线穿过气球表面的总"通量"（流入流出相抵消后的净值）只与气球内部包了多少电荷有关，与球外电荷无关，也与电荷在球内的位置无关。

**为什么这很强大？**
- 对于高度对称的情况（球对称、无限长圆柱对称、无限大平面对称），你可以选一个与对称性匹配的高斯面，几行就算出电场——完全绕过复杂的库仑积分。
- 例如：一个均匀带电球体外的电场，和把所有电荷压缩到球心一个点产生的电场完全一样。

**微分形式**：$\nabla\cdot\vec{E} = \frac{\rho}{\varepsilon_0}$

每一点处电场线的"发散量"正比于该点的电荷密度。这是局部描述；积分形式是全局描述。

**静电场的无旋性**：$\nabla\times\vec{E}=0$ → 电场力做功与路径无关 → 可以定义电势 φ，且 E = -∇φ。

> ⚠️ **学生常犯错误**：① 误以为高斯面外的电荷不影响面上的 E——它影响！只是不影响总通量（进来的等于出去的）。② 忘了对称性条件——高斯定理只能直接算出高度对称情况下的 E，一般分布还是要用库仑积分或分离变量法。

## 1.2 电流怎么产生磁场

### 核心公式

**毕奥-萨伐尔定律**：一小段通电导线 Idl 在距离 r 处产生的磁场微元：
$$dB = \frac{\mu_0}{4\pi}\frac{I\,dl \times \hat{r}}{r^2}$$

> **与库仑定律对比**：两者都是"r²反比"，但磁场方向由"叉乘"决定——垂直于电流方向和径向方向所决定的平面，用手定则判定。μ₀ = 4π×10⁻⁷ 是真空磁导率。

**磁场高斯定理**：$\nabla\cdot\vec{B}=0$。与电场高斯定理形成鲜明对比：
- 电场有源：正电荷是源，负电荷是汇
- 磁场无源：没有磁单极子，磁感线永远是闭合环

**安培环路定理（稳恒情况）**：$\nabla\times\vec{B} = \mu_0\vec{J}$。翻译：磁场的"旋度"由电流密度决定——电流是磁场的"漩涡源"。

> ⚠️ **注意**：这个版本只在**电流不随时间变化**时成立。麦克斯韦后来修正了它（见1.4节）。

---

## 1.3 变化的磁场产生电场——法拉第定律

### 核心公式

$$\mathcal{E} = -\frac{d\Phi_B}{dt}, \quad \nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t}$$

**用白话解释**：穿过一个线圈的磁通量如果发生变化，线圈两端就会出现电压（电动势）。

**物理图像——两类情况**：

1. **磁场变化，线圈不动**（感生电动势）：变化的磁场在空间中产生**涡旋电场**——电场线呈闭合圆环状。这个电场不是保守场，不能用标量电势描述！
2. **线圈在恒定磁场中运动**（动生电动势）：实质是洛伦兹力推动导线中的电子运动。

**负号（楞次定律）的含义**：感应电流的方向总是"反对引起它的变化"——磁铁靠近线圈时感应电流产生排斥磁场；磁铁远离时感应电流产生吸引磁场。这是能量守恒的体现——如果反过来（磁铁靠近时感应电流吸引磁铁），系统会自发加速，违反能量守恒。

> 🔑 **关键理解**：这是麦克斯韦方程组中最具革命性的一条！它首次揭示了电和磁不是孤立的——变化的磁场能产生电场。再加上麦克斯韦引入的位移电流（变化的电场产生磁场），电磁波的传播就成为了数学必然。

## 1.4 位移电流——麦克斯韦的天才修正

### 问题出在哪？

考虑一个正在充电的平行板电容器：
- 导线中有传导电流 I
- 但两个极板之间是真空——没有传导电流

如果只用 ∇×B = μ₀J（旧版安培定律），在同一个回路上取不同的积分面会得到矛盾的结果（一个面穿过导线，通量是 I；另一个面穿过极板间空隙，通量是 0）。

**麦克斯韦发现**：极板之间有**变化的电场** ∂E/∂t，把它乘以 ε₀ 当作一种"等效电流"：
$$J_d = \varepsilon_0\frac{\partial E}{\partial t}$$

**修正后的安培-麦克斯韦定律**：
$$\nabla\times\vec{B} = \mu_0\vec{J} + \mu_0\varepsilon_0\frac{\partial\vec{E}}{\partial t}$$

| | 传导电流 J | 位移电流 J_d |
|---|---|---|
| 是什么 | 电荷真的在流动 | 电场在变化 |
| 是否产生磁场 | ✓ | ✓ |
| 是否发热 | ✓ | ✗（真空） |
| 需要介质吗 | 需要导体 | 不需要 |

**历史影响**：麦克斯韦加了这个"位移电流"项之后，数学上推出波动方程，预言电磁波以光速传播，并大胆断言光就是电磁波！20多年后赫兹实验证实——这是理论物理的经典案例。

## 🔥 麦克斯韦方程组——全书灵魂（必须背！）

### 真空中的微分形式（最重要）

1. $\nabla\cdot\vec{E} = \dfrac{\rho}{\varepsilon_0}$ —— 电荷是电场的源
2. $\nabla\cdot\vec{B} = 0$ —— 没有磁单极子
3. $\nabla\times\vec{E} = -\dfrac{\partial\vec{B}}{\partial t}$ —— 变化的磁场产生电场
4. $\nabla\times\vec{B} = \mu_0\vec{J} + \mu_0\varepsilon_0\dfrac{\partial\vec{E}}{\partial t}$ —— 电流和变化的电场都产生磁场

### 积分形式

1. $\oint_S \vec{E}\cdot d\vec{S} = Q/\varepsilon_0$
2. $\oint_S \vec{B}\cdot d\vec{S} = 0$
3. $\oint_L \vec{E}\cdot d\vec{l} = -\frac{d}{dt}\int_S \vec{B}\cdot d\vec{S}$
4. $\oint_L \vec{B}\cdot d\vec{l} = \mu_0 I + \mu_0\varepsilon_0\frac{d}{dt}\int_S \vec{E}\cdot d\vec{S}$

### 📖 为什么这四个方程是"全书灵魂"？

你从这四个方程出发，能推导出：
- 静电场和静磁场的全部结论（让 ∂/∂t = 0）
- 电磁波的存在（对3和4取旋度互相代入）
- 光就是电磁波（波速 c = 1/√(μ₀ε₀)）
- 电磁场具有能量和动量

从第一章到第六章，本质上就是这四个方程在不同场景下的展开——这就是为什么第一章最重要。

### 介质中的麦克斯韦方程组（知道即可，考试不常考推导）

介质（非真空）中，物质会对电场和磁场作出响应，引入两个辅助量：

$$\vec{D} = \varepsilon_0\vec{E} + \vec{P} = \varepsilon\vec{E}$$
$$\vec{H} = \frac{\vec{B}}{\mu_0} - \vec{M} = \frac{\vec{B}}{\mu}$$

介质方程就是把真空方程中的 ε₀→ε，μ₀→μ，ρ→ρ_f，J→J_f。

> **理解就好**：D 和 H 只是为了让自由电荷和自由电流单独显出来，方便计算。本质还是 E 和 B。

## 1.5 洛伦兹力——场如何作用于电荷

$$\vec{F} = q(\vec{E} + \vec{v}\times\vec{B})$$

**电场力部分 qE**：沿电场（或反电场）方向推电荷。

**磁场力部分 qv×B**：始终垂直于速度——所以**磁场力不做功**！它只改变电荷运动方向，不改变速率。

**记住五个公式的重要性**：四个麦克斯韦方程 + 洛伦兹力公式 = 经典电动力学的"五条公理"，原则上可以解释一切经典电磁现象。

## 1.6 电磁场的能量——坡印廷定理

### 核心概念

电磁场中储存着能量，而且能量会在空间中流动。

**能量密度（单位体积储存的能量）**：
$$w = \frac{1}{2}(\varepsilon_0 E^2 + \frac{1}{\mu_0}B^2)$$

> 和弹簧势能 ½kx² 类似——场是"线性系统"，½因子自然出现。

**能流密度（坡印廷矢量）**：
$$\vec{S} = \vec{E}\times\vec{H}$$

**物理含义**：S 的方向是电磁能量流动的方向（E 叉乘 H 的右手方向），大小是单位时间穿过单位面积的能量。对于平面电磁波，S 指向波的传播方向。

**坡印廷定理**：
$$\frac{\partial w}{\partial t} + \nabla\cdot\vec{S} = -\vec{J}\cdot\vec{E}$$

翻译：空间某点电磁场能量的增加 + 从该点流出的能量 = 电场对电荷做功的功率（取负号表示"电磁场失去能量→电荷获得能量"）。

> 💡 **实际应用**：为什么太阳能加热地球？因为太阳发出的电磁波携带着坡印廷矢量所指的能量，穿过太空到达地球。为什么手机无线充电？因为变化的磁场产生的电场携带能量。

# 第二章 静电场（∂/∂t = 0 的特殊情况）

## 2.1 电势与泊松方程

### 为什么要引入电势？

因为静电场旋度为零（∇×E=0），可以写成梯度的形式：
$$\vec{E} = -\nabla\varphi$$

其中 φ 是电势（单位：伏特）。两点间电势差 = 单位电荷从一点移到另一点，电场力做的功。

**电势满足的微分方程——泊松方程**：
$$\nabla^2\varphi = -\frac{\rho}{\varepsilon_0}$$

在有电荷的地方，电势满足泊松方程；无电荷的地方（ρ=0），退化为拉普拉斯方程：∇²φ=0。

> **通俗理解**：∇²φ 度量的是"某点电势与周围电势平均值的差异"。如果没有电荷（∇²φ=0），该点电势是周围电势的平均值——所以调和函数的极值只能在边界取到。

---

## 2.2 唯一性定理——为什么镜像法能用？

**定理内容**：给定边界条件（边界上的电势值或电势的法向导数），泊松方程有且仅有一个解。

这意味着：解边值问题时，不管你用什么方法（猜测、硬算、镜像法），只要找到了一个解，并且它满足方程和边界条件——那就是正确答案，不存在另一个可能的答案。

> **唯一性定理是镜像法的数学保障**。镜像法本质上是"猜"一个由假想电荷组成的解——只要这个解在物理区域内满足方程且在边界上满足条件，它就是对的。

## 2.3 分离变量法——把偏微分方程拆成常微分方程

### 核心思想

拉普拉斯方程 ∇²φ=0 是一个偏微分方程。分离变量法假设解可以写成 φ(x,y,z)=X(x)Y(y)Z(z)——把三个变量的偏微分方程拆成三个**各只含一个变量**的常微分方程，每个都好解。

### 球坐标下的通解（轴对称情况——期末考试高频考点！）

当问题对 z 轴旋转对称时（φ 与 φ 角无关）：
$$\varphi(r,\theta) = \sum_{n=0}^{\infty}\left(A_n r^n + \frac{B_n}{r^{n+1}}\right)P_n(\cos\theta)$$

**每一部分的含义**：
- $A_n r^n$：描述内部区域，在原点有限，远处发散
- $B_n/r^{n+1}$：描述外部区域，远处趋于零，原点发散
- $P_n(\cos\theta)$：描述角度分布——勒让德多项式

**勒让德多项式速查（前四个就够用）**：
- $P_0(x)=1$——球对称项
- $P_1(x)=x$——偶极子项（cosθ 分布）
- $P_2(x)=(3x^2-1)/2$——四极子项
- $P_3(x)=(5x^3-3x)/2$

### 标准解题流程（四步走）

1. **分区**：把空间分成不同区域（如球内/球外）
2. **写通解并去发散项**：含 r=0 的区域去掉 B_n/r^{n+1}（那会在原点爆炸）；含 r→∞ 的区域去掉 A_n r^n（那会在无穷远爆炸）——除非有外加均匀电场
3. **用边界衔接条件**：界面上电势连续；界面上电位移法向分量连续
4. **逐项比较系数**：利用勒让德多项式的正交性，逐项定出 A_n, B_n

> **常考经典题**：均匀外电场中放一个介质球（或导体球）。结果：球内是均匀电场（只有 n=1 项非零），球外是原均匀场 + 一个偶极子场。

> ⚠️ **容易出错的点**：① 忘了去掉发散项；② 混淆 P_1(x)=x（不是 1）；③ 导体球直接设球面为等势面，不需要用介质球的 ε→∞ 极限。

## 2.4 镜像法——用假想电荷骗过边界条件

### 核心思路

导体表面的感应电荷分布是**未知的**，但导体表面是等势面（或接地后电势为零）。我们**在导体内部放一些假想电荷**（像电荷），使得原物理区域内的电势恰好在导体表面处满足边界条件。唯一性定理保证——只要边界条件满足，这个解就是对的。

### 两种常考情况

**① 点电荷 q 对无限大接地导体平面（距平面 d）**
- 像电荷：q' = -q，放在平面另一侧距平面 d 处
- 原区域电势 = q 和 -q 两个点电荷的电势叠加
- 平面上任意点电势为零（q 和 -q 的贡献抵消）

**② 点电荷 q 对接地导体球（球半径 R，点电荷距球心 a）**
- 像电荷大小：$q' = -\frac{R}{a}q$（比 q 小）
- 像电荷位置：距球心 $b = \frac{R^2}{a}$（在球内）
- 球面上任意点电势为零

> 💡 **记忆技巧**：「平面镜像——对称翻个、电荷变号；球面镜像——大小缩 R/a，位置缩 R²/a」

### 为什么镜像法不能乱用？

只适用于几何简单、对称性高的导体（平面、球、圆柱）。像电荷必须在求解区域之外（否则会引入虚假的奇点）。不能直接用于介质分界面问题（需要推广）。

---

## 2.5 电多极展开——远处的电荷分布近似

### 核心问题

一个复杂的电荷分布，在很远的地方看，电势长什么样？

**答案**：可以展开成一系列越来越小的项：
$$\varphi \approx \frac{1}{4\pi\varepsilon_0}\left[\frac{Q}{r} + \frac{\vec{p}\cdot\hat{r}}{r^2} + \cdots\right]$$

- **第一项 Q/r**：把所有电荷当成一个点电荷——最粗糙的近似，足够远时的主要贡献
- **第二项 (p·r̂)/r²**：考虑正负电荷分离——电偶极子的贡献。如果总电荷 Q=0（中性系统），这就是主导项
- **第三项及以后**：电四极矩、八极矩……

**电偶极矩定义**：$\vec{p} = \int \vec{r}\,\rho\,dV$。对于两个等量异号点电荷 ±q 相距 d：p = qd，方向从负电荷指向正电荷。

> **关键**：当总电荷为零时，远场由偶极矩决定；偶极矩为零时，由四极矩决定。

---

## 2.6 静电场的能量

$$W = \frac{1}{2}\int\rho\varphi\,dV = \frac{1}{2}\int\vec{D}\cdot\vec{E}\,dV$$

**两种形式的含义**：
- 第一种：电荷在电势中具有的总能量（方便已知电荷分布时使用）
- 第二种：能量储存在电场中（更深刻——电磁波可以脱离电荷传播能量）

**½因子来源**：不是把所有电荷同时放进去的，而是一点一点搬来的——每搬来一点，它感受到的电势是已有电荷产生的，逐步增加，积分后自然出现½。

**电容器的储能**：$W = \frac{1}{2}CV^2 = \frac{Q^2}{2C}$。这是计算电容储能的最实用公式。

---

# 第三章 静磁场

## 3.1 磁矢势 A——磁场的"电势"

### 为什么需要矢势？

静电场可以定义标量电势 φ（因为 ∇×E=0）。但磁场一般不满足 ∇×B=0，不能用一个标量函数来表示 B。

**但是**：∇·B=0（磁场无源），这意味着 B 可以写成另一个矢量场的旋度：
$$\vec{B} = \nabla\times\vec{A}$$

A 就是**磁矢势**。类比：φ 是电场的"标量势"，A 是磁场的"矢量势"。

### 规范自由度（常见考点）

给定 B，A 不是唯一的——可以加上任何标量函数的梯度：
$$\vec{A}' = \vec{A} + \nabla\psi$$

两者产生相同的 B（因为 ∇×∇ψ=0）。

**两种常用规范**：
- **库仑规范**：∇·A = 0（磁静力学中最方便）
- **洛伦兹规范**：$\nabla\cdot\vec{A} + \frac{1}{c^2}\frac{\partial\varphi}{\partial t} = 0$（辐射问题中最方便）

> 🔑 **重要认识**：A 不只是数学工具——阿哈罗诺夫-玻姆效应（1959年）证明，即使 B=0 的区域，A 也能产生可观测的量子效应。A 是真实的物理场！

### A 的微分方程

库仑规范下 A 满足泊松方程：$\nabla^2\vec{A} = -\mu_0\vec{J}$。解的形式与静电势完全类似（把 ε₀→1/μ₀，ρ→J）。

---

## 3.2 磁标势——有条件地借用静电学方法

在**无传导电流**（J=0）的区域，∇×H=0，可以引入磁标势 φ_m：
$$\vec{H} = -\nabla\varphi_m, \quad \nabla^2\varphi_m = 0$$

**限制条件**：只能用在没有电流穿过的"单连通"区域。如果区域中有电流环，绕一圈后 φ_m 会多值。

**实际用途**：把静电场中学到的分离变量法和镜像法搬到磁学中来，只需做形式替换。

---

## 3.3 磁偶极子——小电流环等价于小磁铁

### 磁偶极矩定义

$$\vec{m} = \frac{1}{2}\int\vec{r}\times\vec{J}\,dV$$

对于平面电流环：m = I·S（S 是环路面积，方向用右手定则）。对比电偶极矩 p = qd。

### 磁偶极子产生的场

**矢势（远场）**：$\vec{A} = \dfrac{\mu_0}{4\pi}\dfrac{\vec{m}\times\hat{r}}{r^2}$

**磁场（远场）**：$\vec{B} = \dfrac{\mu_0}{4\pi r^3}\left[3(\vec{m}\cdot\hat{r})\hat{r} - \vec{m}\right]$

> **与电偶极子对比**：电偶极子的 φ ∝ p·r̂/r²（点乘）；磁偶极子的 A ∝ m×r̂/r²（叉乘）。场都按 1/r³ 衰减。

---

## 3.4 静磁场的能量

$$W = \frac{1}{2}\int\vec{A}\cdot\vec{J}\,dV = \frac{1}{2}\int\vec{B}\cdot\vec{H}\,dV$$

一个线圈的自感磁能：$W = \frac{1}{2}LI^2$。

两个线圈的总磁能：$W = \frac{1}{2}L_1I_1^2 + \frac{1}{2}L_2I_2^2 + MI_1I_2$（M 是互感系数）。

---

# 第四章 电磁波的传播

## 4.1 从麦克斯韦方程组到电磁波

### 推导思路（理解，不需要死记每一步）

1. 在无源真空区域（ρ=0, J=0），对法拉第定律 ∇×E = -∂B/∂t 两边取旋度
2. 代入安培-麦克斯韦定律 ∇×B = μ₀ε₀ ∂E/∂t
3. 用矢量恒等式 ∇×(∇×E) = ∇(∇·E) - ∇²E
4. 利用 ∇·E = 0（无源）得到：

$$\nabla^2\vec{E} = \mu_0\varepsilon_0\frac{\partial^2\vec{E}}{\partial t^2}$$

这就是**波动方程**！波速 c = 1/√(μ₀ε₀) ≈ 3×10⁸ m/s——恰好是光速。

> 🎉 **麦克斯韦最伟大的预言**：从两个实验常数 μ₀ 和 ε₀ 算出光速，断言"光就是电磁波"。这是物理学史上最辉煌的理论预言之一。

---

## 4.2 平面电磁波——最简单的电磁波

### 基本形式

$$\vec{E} = \vec{E}_0 e^{i(\vec{k}\cdot\vec{r} - \omega t)}, \quad \vec{B} = \vec{B}_0 e^{i(\vec{k}\cdot\vec{r} - \omega t)}$$

- k 是波矢：方向 = 传播方向，大小 |k| = 2π/λ
- ω 是角频率：ω = 2πf
- 等相位面 k·r - ωt = 常数——一个垂直于 k 的平面（所以叫"平面波"）

### 核心性质（考试必考）

**① 横波性**：E 和 B 都垂直于传播方向。k·E = 0, k·B = 0。

**② E, B, k 是右手正交系**：$\vec{B} = \frac{1}{\omega}\vec{k}\times\vec{E} = \frac{1}{v}\hat{k}\times\vec{E}$

如果传播方向是 z，E 沿 x，则 B 沿 y。

**③ 振幅关系**：$|\vec{B}| = \frac{1}{v}|\vec{E}|$。在真空中 E = cB——电场强度是磁感应强度的 3 亿倍（值大，单位不同所以不能直接比"强弱"）。

**④ 偏振（极化）**：
- 线偏振：E 始终沿固定方向振荡
- 圆偏振：E 大小不变，方向旋转
- 椭圆偏振：最一般情况

---

## 4.3 电磁波的能量与动量

### 平均能流密度（电磁波的"亮度"）

$$\langle S\rangle = \frac{1}{2}\sqrt{\frac{\varepsilon}{\mu}}E_0^2$$

真空中：⟨S⟩ = ½cε₀E₀²。

**½的来源**：对简谐振荡取时间平均，sin² 或 cos² 的平均值为 ½。

### 电磁波有动量！

动量密度 = S/c²。光照射在物体表面产生压力——"光压"。太阳光对地球的压强大约是 5×10⁻⁶ Pa——极小但可测，太阳帆就是利用光压推进的。

---

## 4.4 反射与折射——菲涅尔公式（必须掌握！）

### Snell 定律

$$n_1\sin\theta_1 = n_2\sin\theta_2$$

> 折射率 n = c/v = √(εμ)/√(ε₀μ₀)。n 越大介质中光速越慢。

### 菲涅尔公式

**S 偏振（E 垂直于入射面——即"横"着振）**：
$$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}$$

**P 偏振（E 在入射面内——即"竖"着振）**：
$$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2}$$

### 记忆诀窍

- **S 波**公式里分子分母都是"n cosθ"——分子是 n₁cosθ₁ - n₂cosθ₂
- **P 波**公式分子是 n₂cosθ₁ - n₁cosθ₂（交叉了）
- 透射系数 t = 1 + r（振幅关系）

### 布儒斯特角（制作偏振光的原理）

当入射角满足 $\tan\theta_B = n_2/n_1$ 时，r_p = 0——反射光中不含 P 分量，变为完全线偏振光。激光器中的布儒斯特窗利用的就是这个原理。

### 全反射（光纤的原理）

当光从密介质（大 n）射向疏介质（小 n），且入射角大于临界角 θ_c = arcsin(n₂/n₁) 时，发生全反射——折射波变为沿界面传播的"倏逝波"（垂直界面方向指数衰减）。

---

## 4.5 导体中的电磁波——趋肤效应

导体中有欧姆定律 J = σE。代入麦克斯韦方程后，可以用**复介电常数** $\tilde{\varepsilon} = \varepsilon + i\sigma/\omega$ 来描述导体——形式上与绝缘介质一样，但介电常数是复数。

### 趋肤效应

电磁波进入导体后，振幅随深度指数衰减：
$$E(z) = E_0 e^{-z/\delta}$$

**趋肤深度**：$\delta = \sqrt{\dfrac{2}{\omega\mu\sigma}}$。

物理含义：
- 频率越高→δ 越小→电磁波更"肤浅"，更难深入金属
- 电导率越大→δ 越小→良导体更能屏蔽电磁波
- 这就是为什么高频电路用多股漆包线（增大表面积），为什么金属外壳能屏蔽电磁干扰

---

## 4.6 波导——用金属管引导电磁波

### 基本概念

矩形波导（截面 a×b）中电磁波不能任意传播。对于给定的频率 ω，只有满足条件的模式才能传播。

### 截止频率

$$(\omega_c)_{mn} = \frac{\pi}{\sqrt{\mu\varepsilon}}\sqrt{\left(\frac{m}{a}\right)^2 + \left(\frac{n}{b}\right)^2}$$

- 当 ω < ω_c：波**不能传播**（纵向指数衰减）
- 当 ω > ω_c：波可以传播
- TE₁₀ 通常是最低模式（a > b 时），截止波长 λ_c = 2a

> **波导就像一个"高通滤波器"**——只有频率高于截止频率的电磁波才能通过。

---

# 第五章 电磁波的辐射

## 5.1 推迟势——电磁作用不是瞬间的！

### 核心思想

电磁扰动不是瞬间传播的，而是以光速传播。所以场点 r 在时刻 t 的势，由源点 r' 在**更早时刻** t - R/c 的状态决定：
$$\varphi(\vec{r},t) = \frac{1}{4\pi\varepsilon_0}\int\frac{\rho(\vec{r}', t-R/c)}{R}dV'$$

R/c 是信号从源传播到场所需的时间——"推迟"了的势。

> **与静电势的对比**：静电势中 φ ∝ ∫ ρ/R dV——假设信号瞬时传播（实际上 c→∞ 的近似）。推迟势才是有限速度下的正确描述。

---

## 5.2 电偶极辐射——天线的基本原理

### 物理图像

电荷在振荡→电偶极矩在变化（加速）→发射电磁波。

**辐射场（远场——距离远大于波长的区域）**：
$$\vec{B} = \frac{\mu_0}{4\pi c r}\ddot{\vec{p}}(t-r/c)\times\hat{r}$$
$$\vec{E} = c\vec{B}\times\hat{r}$$

### 关键特征（考试重点）

**① 场按 1/r 衰减**——不是 1/r² 或 1/r³！这是辐射场与静态场的本质区别。静态场随距离衰减得快，辐射场衰减慢——所以远处接收到的电磁波来自辐射场。

**② 辐射功率 ∝ ω⁴**：$P = \frac{\mu_0\omega^4}{12\pi c}|\vec{p}_0|^2$

频率加倍→辐射功率增加 16 倍！这意味着：
- 高频信号更容易辐射（也更难屏蔽）
- 天空为什么是蓝色的——太阳光中蓝光频率高，大气分子（作为微小偶极子）对蓝光的散射（即辐射）远强于红光（瑞利散射）

**③ 角分布 ∝ sin²θ**：θ 是观测方向与偶极矩方向的夹角。垂直于偶极矩辐射最强，沿偶极矩方向辐射为零。辐射图案像一个甜甜圈。

---

## 5.3 磁偶极辐射和电四极辐射（了解即可）

对于低速运动的电荷系统（v ≪ c），电偶极辐射功率远大于磁偶极辐射和电四极辐射——比值约 (v/c)²。

> **考试常考结论**：一般情况下，电偶极辐射是最重要的辐射方式。

---

## 5.4 天线（定性了解）

- **半波天线**：长度 = 半个波长的天线，方向性比单个电偶极子好
- **天线阵列**：多个振子组合，利用干涉原理实现高方向性（雷达用的就是这原理）
- **阵因子**：$F(\theta) \propto \frac{\sin^2(N\alpha/2)}{\sin^2(\alpha/2)}$——与光学的多缝干涉公式完全一样

---

## 5.5 汤姆逊散射——为什么天空是蓝的

自由电子在入射光作用下来回振荡，然后作为电偶极子向四面八方重新辐射电磁波——这就是散射。

散射截面（等效"挡光面积"）：σ_T ≈ 6.65×10⁻²⁹ m²。散射功率 ∝ ω⁴——蓝光散射远强于红光（瑞利散射）。

---

# 第六章 狭义相对论

## 6.1 两条基本原理

1. **相对性原理**：在匀速运动的惯性系中，物理规律的形式完全相同
2. **光速不变原理**：无论光源和观察者的相对运动如何，测得的光速都是 c

> **爱因斯坦的突破（1905年）**：从这两条原理出发，推出必须修改伽利略变换（x'=x-vt, t'=t），代之以洛伦兹变换。麦克斯韦方程组天生就是洛伦兹协变的——不需要任何修改！这也是爱因斯坦创立相对论的重要动机。

---

## 6.2 洛伦兹变换——像傻瓜一样理解相对论

> 🎯 **一个目标**：搞懂这四条公式，第六章就稳了。

---

### 🚂 背景：伽利略说简单，现实说不

牛顿时代，大家都信伽利略变换——太简单了：

你坐在时速100公里的火车里向前走，速度是每小时5公里。地面的人看你，速度就是 100 + 5 = 105 公里/小时。公式就是 **x' = x - vt**，时间不变 **t' = t**——全世界共用同一口钟。

**这个"朴素"的想法被一个实验打脸了。**

麦克斯韦从自己的方程组算出电磁波速度 = c = 3×10⁸ m/s。按照伽利略逻辑，你追着一束光跑，测得的光速应该小于c——就像你追火车，火车相对于你的速度就变小了。但1887年迈克尔逊和莫雷做了极其精密的实验，结论石破天惊：**不管你跑多快、往哪个方向跑，光速永远是c。纹丝不动。**

1905年，26岁的爱因斯坦说：不是麦克斯韦错了，也不是实验错了——**是你们的"常识"错了**。空间和时间不是分开的、绝对的。它们在高速下会互相"借位"。

---

### 🧠 核心直觉：时空是一张银行卡

经典世界：空间和时间是两张独立的银行卡。花空间不影响时间。

相对论世界：空间和时间是**一张联合信用卡**——空间那边刷爆了，时间那边账单就变少。总"时空额度"恒定。

洛伦兹变换就是这张联合卡的**汇率换算表**：如果你和我之间有相对运动，你用你的尺子和钟量的数，和我用我的尺子和钟量的数，要通过这个换算表互相翻译。

---

### 🔍 γ 从哪冒出来的？——手把手推导

不要被"推导"两个字吓到。这个推导只用初中代数，五分钟就能跟上。

**已知条件（两条公理）**：
1. 光速在任何惯性系中都是 c（光速不变）
2. 物理规律在所有惯性系中形式一样（相对性原理）

**目标**：找到 S 系 (x,t) 和 S' 系 (x',t') 之间的变换关系。S' 系相对于 S 系沿 x 轴以速度 v 运动。

---

#### 第一步：猜一个线性变换

最一般的线性变换（只考虑 x 和 t，y 和 z 垂直于运动方向不变）：

$$x' = A(x - vt)$$
$$t' = Bx + Dt$$

其中 A、B、D 是待定系数（可能和 v 有关，但不能和 x、t 有关，因为时空应该是均匀的）。写 x-vt 而不是单独的 x 和 t，是因为当 x=vt 时（即 S' 系的原点），x' 应该 = 0。

---

#### 第二步：用光速不变定关系

在 t = 0 时刻，两系原点重合，同时发出一道光。

在 S 系中，波前运动方程：x = ct（向正方向走），x = -ct（向负方向走）。
在 S' 系中，波前运动方程：x' = ct'，x' = -ct'（光速还是 c！这就是光速不变）

把变换代入 x' = ct'：

$$A(x - vt) = c(Bx + Dt)$$

再把 x = ct 代入（追踪正方向的光）：

$$A(ct - vt) = c(Bct + Dt)$$
$$At(c-v) = ct(Bc + D)$$
$$A(c-v) = c(Bc + D) \quad \text{①}$$

同理，把 x' = -ct' 和 x = -ct 代入（追踪反方向的光）：

$$A(-ct - vt) = -c(-Bct + Dt)$$
$$-At(c+v) = -c(-Bct + Dt)$$
$$-A(c+v) = -c(-Bc + D)$$
$$A(c+v) = c(-Bc + D) \quad \text{②}$$

---

#### 第三步：解出 A、B、D

① + ②：
$$A(c-v) + A(c+v) = c(Bc + D) + c(-Bc + D)$$
$$2Ac = 2cD$$
$$A = D$$

① - ②：
$$A(c-v) - A(c+v) = c(Bc + D) - c(-Bc + D)$$
$$-2Av = 2c^2B$$
$$B = -\frac{Av}{c^2}$$

现在三个未知数只剩一个 A 了：D = A，B = -Av/c²。变换变成：

$$x' = A(x - vt)$$
$$t' = A(t - \frac{v}{c^2}x)$$

---

#### 第四步：用相对性原理定出 A（γ 出场！）

逆变换应该和正变换形式完全一样，只是把 v 换成 -v（因为 S 系相对于 S' 系的速度是 -v）：

$$x = A(x' + vt')$$

把 x' 和 t' 的表达式代入：

$$x = A[A(x-vt) + vA(t - \frac{v}{c^2}x)]$$
$$x = A^2[x - vt + vt - \frac{v^2}{c^2}x]$$
$$x = A^2x(1 - \frac{v^2}{c^2})$$

两边约掉 x，得到：

$$A^2 = \frac{1}{1 - v^2/c^2} \quad \Rightarrow \quad A = \frac{1}{\sqrt{1 - v^2/c^2}}$$

**这个 A 就是 γ！**

> $$\gamma = \frac{1}{\sqrt{1 - v^2/c^2}}$$

---

#### 最终得到洛伦兹变换

$$x' = \gamma(x - vt)$$
$$t' = \gamma(t - \frac{v}{c^2}x)$$
$$y' = y,\quad z' = z$$

**γ 不是人为定义的，是光速不变 + 相对性原理"逼"出来的必然结果。** v 很小的时候 v²/c² ≈ 0，γ ≈ 1，洛伦兹变换退化为伽利略变换——这说明经典物理是相对论在低速下的近似，不是"错"了，是"不够精确"。

---

### 📝 四条公式逐条拆解——每条在说什么人话

S' 系相对于 S 系沿 x 轴以速度 v 运动。记住：γ ≥ 1（v=0 时 = 1，v→c 时 → ∞）。

---

#### 第一条：$x' = \gamma(x - vt)$ —— 空间坐标怎么变

**结构拆解**：
- **x - vt**：这是伽利略的部分——S' 系原点以速度 v 移动，所以 S 系的 x 扣掉 vt 才是 S' 系看到的"相对位置"
- **× γ**：这是相对论修正——空间被"拉伸"了 γ 倍

**人话**：运动的尺子上的刻度被"拉开"了。同样一把尺子，在 S' 系（相对尺子静止）上刻度间距正常；在 S 系（尺子在运动）看来，每个刻度格子变大了 γ 倍——所以同一段长度包含的刻度数变少了——这就是长度收缩。

> 💡 疑惑解答：公式里乘的是 γ（>1），为什么尺子反而变短？因为"测长度"要求**同时**读两端坐标。同时读在 S 系中意味着 Δt=0，但代入第二条公式得到 Δt' = -γvΔx/c² ≠ 0——在 S' 系中两端不是同时读的。这个"同时性破坏"和 γ 的放大效应共同作用，净结果就是 L = L₀/γ。

---

#### 第二条：$t' = \gamma(t - \frac{v}{c^2}x)$ —— 时间坐标怎么变

**结构拆解**：
- **t**：经典的时间——牛顿认为全宇宙共用这口钟
- **- (v/c²)x**：**位置的修正项**——这是相对论独有的！时间不再独立于空间
- **× γ**：整体缩放

**人话**：你的"现在"不等于我的"现在"。位置不同的人，在同一时刻（S 系中的 t），在 S' 系中对应的是不同的 t'。时间变成了位置的函数——就像北京和纽约的"此刻"差12小时一样，只不过这里不是时区，是由相对运动和空间距离共同决定的"同时性偏移"。

> 💡 为什么日常生活中感觉不到？因为 c² ≈ 9×10¹⁶ m²/s² 在分母——你家客厅两端的 x 差几米，v/c² 项的量级是 10⁻¹⁷ 秒，任何人造钟表都测不出来。但在宇宙尺度（星系间距离）和高速（接近光速）下，这一项变得巨大。

---

#### 第三条 $y' = y$ 和第四条 $z' = z$ —— 横向为什么不收缩？

**结构拆解**：垂直于运动方向，什么都不变。最简单的两条。

**人话**：运动的尺子只在沿着运动的方向变短，横着不变。一把竖直放置的尺子，不管多快，站着不动的人测量它的长度还是原来那么长。

**为什么？** 如果横向也收缩，就会出现矛盾：A 看 B 的横向尺子变短了，B 看 A 的横向尺子也变短了——但相对性原理要求物理规律一样，不能两个人各说各的都对而结论矛盾。唯一自洽的方案：横向不变。

---

### 🔑 γ——相对论的"音量旋钮"

γ = 1/√(1 - v²/c²)。v 越大，γ 越大；γ 越大，相对论效应越强。

| v（相对光速） | γ | 直观感受 |
|:---:|:---:|------|
| 0 | 1 | 退化到经典物理——没啥特别 |
| 0.1c | 1.005 | 偏差0.5%，基本感觉不到 |
| 0.5c | 1.15 | 开始有15%的修正 |
| 0.866c | **2** | 时间慢一倍，长度缩一半 |
| 0.99c | **7** | 时间只有原来的1/7 |
| 0.9999c | ≈70 | μ子寿命延长70倍 |
| → c | → ∞ | 永远到不了——需要无穷大能量 |

**现实证据**：宇宙线产生的μ子，固有寿命只有约2.2微秒。即使以光速飞，也最多走660米——根本到不了海平面。但我们在海平面天天检测到它们！因为在**我们的参考系**中，μ子高速运动，γ把它们的寿命拉长了——就像慢动作镜头一样。这每天真实发生在你头顶的大气层里。

---

### ⏰ 时间膨胀：运动的人老得慢

**Δt = γ · Δt₀**

Δt₀ 叫"固有时间"——在事件发生地点、用同一口钟、静止测出来的时间。它**最短**。任何运动的观察者测到的时间都更长。

> 你带一块表坐飞船以 v = 0.866c（γ = 2）飞出。你的表走1秒，地球上的人觉得你过了2秒。你觉得正常——你的心跳、吃饭、衰老全部一起变慢了。飞船里一切都和谐。但回到地球你会发现，你的双胞胎弟弟老了更多。

> **双生子佯谬一句话**：坐飞船的哥哥加速过，所以不对称。谁加速谁年轻。不是佯谬。

---

### 📏 长度收缩：运动的尺子变短

**L = L₀ / γ**

L₀ 叫"固有长度"——在物体静止参考系里量的长度。它**最长**。运动的尺子沿运动方向变短。

> 你开一辆4米长的车以 0.866c 飙车。警察在路边量你的车——只有2米。但你坐在车里量——还是4米。谁对？你和警察都对。长度不是绝对的，和参考系绑定——就像"A在我左边"在你转个身就变成"A在我右边"一样，"长"和"短"在你换参考系时也会变。

**只有纵向收缩**。横向（y和z方向）完全不变。运动的球看起来像个被压扁的飞碟——但这只是比喻，"看起来"涉及光传播的复杂效应，真正的洛伦兹收缩是物理测量层面的。

---

### ⚡ 同时性的相对性：你的"现在"不是我的"现在"

这是最反直觉的一条，也是最核心的一条。

**Δt' = -γ · (v/c²) · Δx**

如果两个事件在你看来同时发生但地点不同（Δx ≠ 0，Δt = 0），在我（相对你运动）看来，它们就不再同时了——时间差正比于空间距离。

> 🚄 **火车闪光实验（爱因斯坦最爱的例子）**：
> 
> 一列火车高速行驶。车厢正中间天花板上有盏灯，瞬间闪一次光。
> 
> 车厢里的你看到：光向前和向后走的速度都是c，距离一样——同时到达车头和车尾。
> 
> 站台上的我看到：车尾在向前跑"接"光，车头在向前跑"躲"光——光**先到车尾，后到车头**。
> 
> 谁对？**都对。** "同时"需要指定参考系——就像"这里"需要指定你在哪一样。时间和空间一样，都是相对的。

### 🚀 速度变换——为什么c是天花板

$$u' = \frac{u - v}{1 - \frac{uv}{c^2}}$$

经典速度相加：u' = u - v。如果地上的人看你以10 m/s跑，火车以30 m/s跑，地上的人看你就是40 m/s。速度没有上限——这是经典物理。

相对论速度变换多了一个分母。验证一下：
- v和u都很小：分母≈1，退化为经典加法 ✓
- u = c：u' = (c-v)/(1 - v/c) = (c-v)/[(c-v)/c] = c。不管v多大，光速还是c ✓

**c不是"很难超越的速度"，而是"根本不属于速度那个分类的东西"**。有质量的物体永远到不了c，无质量的粒子（如光子）天生就是c。不是被限制，而是时空几何的必然结果。

### 💡 考场生存指南

**1. 记住四条公式**：x' = γ(x - vt)，t' = γ(t - vx/c²)，y'=y，z'=z

**2. 逆变换**：v → -v

**3. 三个效应全从公式推出**：
- 时间膨胀：在S'系中原地不动（Δx' = 0）→ Δx = vΔt → 代入t公式 → Δt = γΔt'（固有时间Δt'最短）
- 长度收缩：在S'系中同时测两端（Δt' = 0）→ Δx' = L₀ → 代入x公式 → L = L₀/γ（固有长度L₀最长）
- 同时性破坏：在S系中同时（Δt = 0）但不同地（Δx ≠ 0）→ Δt' = -γvΔx/c² ≠ 0

**4. 光速验证**：令x = ct代入变换，必得x' = ct'——光速不变自动满足

**5. γ永远≥1**：v→0时γ→1（经典物理），v→c时γ→∞（光速壁垒）

> 📌 洛伦兹变换是第六章的根基。时间膨胀、长度收缩、速度变换、场变换、E=mc²全建在这之上。花一个小时真正搞懂，后面全是推论。

## 6.3 四维时空和质能关系

### 四维坐标与动量

闵可夫斯基将时间和空间统一为四维：
$$x^\mu = (ct, x, y, z)$$

**四维动量**：$p^\mu = (E/c, \vec{p})$

### 质能关系（极其重要！）

$$E^2 = p^2c^2 + m_0^2 c^4$$

**三种特例**：
- 粒子静止（p=0）：E = m₀c²——这就是"质能等价"
- 无质量粒子（m₀=0）：E = pc——光子！
- 低速（v≪c）：E ≈ m₀c² + ½m₀v²——静止能 + 经典动能

> 🎯 **考试技巧**：遇到相对论粒子碰撞/衰变问题，优先找**不变量**——$p^\mu p_\mu = m_0^2 c^2$ 在任何惯性系中相同，利用不变量可以绕开洛伦兹变换。

## 6.4 电磁场的相对论变换

### 核心认识

**电场和磁场是同一物理实体（电磁场张量）在不同参考系中的不同投影！** 在一个参考系中的纯电场，在另一个参考系中可能出现磁场。

### 电磁场张量（把六个分量装进一个 4×4 矩阵）

$$F^{\mu\nu} = \begin{pmatrix}
0 & -E_x/c & -E_y/c & -E_z/c \\
E_x/c & 0 & -B_z & B_y \\
E_y/c & B_z & 0 & -B_x \\
E_z/c & -B_y & B_x & 0
\end{pmatrix}$$

### 场变换公式（只记规则）

- **平行于运动方向的分量不变**：E_x' = E_x, B_x' = B_x
- **垂直于运动方向的分量混合**：E 和 B 会互相转化

$$E_y' = \gamma(E_y - vB_z), \quad E_z' = \gamma(E_z + vB_y)$$
$$B_y' = \gamma(B_y + \frac{v}{c^2}E_z), \quad B_z' = \gamma(B_z - \frac{v}{c^2}E_y)$$

> **记忆技巧**：垂直于运动方向，"E 减 vB"和"B 加 vE/c²"交替出现。

## 6.5 麦克斯韦方程组的四维形式（了解即可）

所有电磁理论之美浓缩在两个张量方程中：

$$\partial_\mu F^{\mu\nu} = \mu_0 J^\nu$$
$$\partial_\lambda F_{\mu\nu} + \partial_\mu F_{\nu\lambda} + \partial_\nu F_{\lambda\mu} = 0$$

四个方程合成了两个——体现了电和磁、时间和空间的统一。

**步骤**：

3. 菲涅尔公式求反射/透射系数

**步骤**：
1. 明确两个惯性系及其相对运动
