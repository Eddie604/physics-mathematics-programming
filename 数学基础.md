## 一、先建立直觉：把电磁场想象成"流动的水"

在学习数学定义之前，先在大脑里建一个画面：

> **把电场 E 想象成空间中每一处都有一根小箭头，它指向水流的方向，箭头的长短代表流速的大小。**

这个"到处都是箭头"的东西，数学上叫**矢量场**。电场、磁场、重力场、风速场——都是矢量场。

现在我们关心三个问题：
1. **梯度**：一个标量（比如温度、高度、电势）在空间中怎么变？哪个方向变的最快？
2. **散度**：在一个小区域里，水流是"往外冒"还是"往里吸"？有没有源头？
3. **旋度**：水流是直着走，还是在打转？转圈有多厉害？

这三个问题的答案，分别就是梯度、散度、旋度。

## 二、梯度——"往哪个方向变化最快？"

### 2.1 从爬山说起

想象你站在一座山上。你所在位置的海拔高度是 h(x, y)——x 和 y 是你在地图上的坐标。

你掏出指南针，想看四个方向：
- 往东走一小步，高度变化多少？（∂h/∂x）
- 往北走一小步，高度变化多少？（∂h/∂y）

**梯度就是把这两个数组合成一个矢量**：

$$\nabla h = \left(\frac{\partial h}{\partial x}, \frac{\partial h}{\partial y}\right)$$

这个矢量指向**上山最陡的方向**。它的长度 |∇h| 就是那个方向的**坡度**。

### 2.2 具体算一个例子

**例子**：一座山的海拔分布是 $h(x, y) = 100 - x^2 - 2y^2$（单位：米）。

在点 (1, 2) 处：
- ∂h/∂x = -2x，在 x=1 处 = **-2**（往东走一米，海拔下降 2 米）
- ∂h/∂y = -4y，在 y=2 处 = **-8**（往北走一米，海拔下降 8 米）
- 梯度 ∇h = **(-2, -8)**

**解读**：
- 梯度方向 ≈ 东偏北（两个分量都是负的，指向西南方向）
- 坡度 = |∇h| = √(4 + 64) = √68 ≈ **8.25 米/米**（每走一米上升 8.25 米——非常陡）

如果你站在 (1, 2)，想最快下山，就往梯度的反方向——即 (2, 8) 方向（东北方）走，每米下降 8.25 米。

### 2.3 梯度在电磁学中的核心应用

**电势 φ 的负梯度就是电场 E**：

$$\vec{E} = -\nabla\varphi$$

**用白话翻译**：
- 电场线指向电势下降最快的方向
- 电势下降越快（等势面越密），电场越强
- 电场线始终垂直于等势面（因为梯度方向垂直于等高线）

**实例——平行板电容器**：

两块大金属板，左边接地（φ=0），右边接 φ=10V，相距 d=1mm。

电势分布：φ(x) = 10x/d（线性变化，从左到右均匀增加）

电场：E = -∇φ = -(∂φ/∂x, 0, 0) = -(10/d, 0, 0) ≈ **-10,000 V/m**

结果：电场指向 x 的负方向（从高电势指向低电势），大小 10,000 V/m。这就是为什么平行板电容器中的电场是均匀的。

**实例——点电荷的电势**：

一个点电荷 Q 在原点，电势 $\varphi = \frac{Q}{4\pi\varepsilon_0}\frac{1}{r}$

梯度 ∇φ = $-\frac{Q}{4\pi\varepsilon_0}\frac{1}{r^2}\hat{r}$（方向沿径向向外）

电场 E = -∇φ = $\frac{Q}{4\pi\varepsilon_0}\frac{1}{r^2}\hat{r}$ —— 正是库仑定律！

> **关键结论**：从电势求电场，就是对电势求梯度再加个负号。这和"水往低处流"一个道理——正电荷沿电场方向运动（从高电势到低电势）。

### 2.4 梯度总结卡片

| 问题 | 回答 |
|------|------|
| 梯度作用在什么上？ | **标量场**（温度、高度、电势……） |
| 梯度得出什么？ | **矢量场**（指向变化最快的方向） |
| 数学写法 | ∇φ = (∂φ/∂x, ∂φ/∂y, ∂φ/∂z) |
| 在电磁学中的核心公式 | E = -∇φ |
| 直观类比 | 等高线最密、最陡的方向 |

## 三、散度——"这里是不是源头？"

### 3.1 从水龙头说起

想象一个装满水的浴缸。在浴缸某一点：
- 如果那里有个**水龙头在喷水**——水从那个点往外流——该点就是"源"
- 如果那里有个**排水口在吸水**——水往那个点流——该点就是"汇"
- 如果水只是平缓流过——流进多少就流出多少——该点既非源也非汇

**散度就是衡量每一个小体积元上"净流出量"的数学工具。**

### 3.2 先理解"通量"

在定义散度之前，需要先理解**通量**。

**通量 = 场矢量穿过一个面的"总量"**：
$$\Phi = \int_S \vec{F}\cdot d\vec{S}$$

- 如果场矢量与面积元方向一致（穿出去），贡献为正
- 如果场矢量与面积元方向相反（穿进来），贡献为负
- 如果平行于面（擦过），贡献为零

**类比**：用一个捕虫网立在风中。穿过网的"风量"取决于：风速大小、网的面积、风是否正对网口。如果风从侧面吹（平行于网面），什么也捕不到。

### 3.3 散度的定义

**散度 = 单位体积的净通量**：

$$\nabla\cdot\vec{F} = \lim_{V\to 0}\frac{1}{V}\oint_S \vec{F}\cdot d\vec{S}$$

用直角坐标计算：
$$\nabla\cdot\vec{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}$$

> 通俗理解：把三个方向上"流出减流入"的量加起来。

### 3.4 具体算几个例子

#### 例 1：均匀流（散度为零）

水流 $\vec{F} = (3, 0, 0)$（每点都是水平向右，速度 3 m/s）。

- ∂F_x/∂x = ∂(3)/∂x = **0**（x 方向没有变化）
- ∂F_y/∂y = ∂(0)/∂y = **0**
- ∂F_z/∂z = ∂(0)/∂z = **0**

散度 ∇·F = **0**。

**为什么？** 每一点流进多少就流出多少——没有"多出来"的水。均匀流动是无源的。

#### 例 2：发散场（正散度）

$\vec{F} = (x, y, z)$（从原点向外辐射，越远越快）。

- ∂F_x/∂x = ∂(x)/∂x = **1**
- ∂F_y/∂y = ∂(y)/∂y = **1**
- ∂F_z/∂z = ∂(z)/∂z = **1**

散度 ∇·F = **3**（每一点都是正值！）。

**为什么？** 想象一个小立方体，右边出去的比左边进来的多（因为 F_x = x 在增大），上面出去的比下面进来的多——净结果是每个小体积元都有水"凭空"冒出来。

#### 例 3：收缩场（负散度）

$\vec{F} = (-x, -y, -z)$（从外向内汇聚到原点）。

- 散度 ∇·F = -1 - 1 - 1 = **-3**（每一点都是负的——"汇"）

### 3.5 散度在电磁学中的核心应用

#### 电场的高斯定理：∇·E = ρ/ε₀

**白话翻译**：电场在某点的散度，正比于该点的电荷密度。

| 散度值 | 物理含义 |
|--------|----------|
| ∇·E > 0 | 该点有**正电荷**——电场线从这里"发射"出去 |
| ∇·E < 0 | 该点有**负电荷**——电场线向这里"汇聚" |
| ∇·E = 0 | 该点没有电荷——电场线只是"路过" |

**实例——点电荷的电场**：

$$E = \frac{Q}{4\pi\varepsilon_0}\frac{\hat{r}}{r^2}$$

- 在 r ≠ 0 处：∇·E = **0**（没有电荷，电场线只是穿过）
- 在 r = 0 处：∇·E = **∞**（电荷集中在一个点上，散度是 δ 函数——数学上用 ρ/ε₀ 表示）

#### 磁场的高斯定理：∇·B = 0

**白话翻译**：磁场的散度处处为零——没有磁单极子！

这意味着：磁感线永远是闭合的环，没有"起点"也没有"终点"。你永远找不到一个单独的 N 极或 S 极——切一块磁铁，断面又出现新的 N 和 S。

> 对比电场：正电荷是电场线的起点，负电荷是终点。所以 ∇·E 在电荷处非零，∇·B 处处为零。

### 3.6 散度总结卡片

| 问题 | 回答 |
|------|------|
| 散度作用在什么上？ | **矢量场** |
| 散度得出什么？ | **标量场**（每点一个数值） |
| 正散度代表什么？ | **源**——东西从这里往外冒 |
| 负散度代表什么？ | **汇**——东西往这里吸 |
| 零散度代表什么？ | **无源**——流进等于流出 |
| 电磁学核心公式 | ∇·E = ρ/ε₀（电荷是电场之源）；∇·B = 0（没有磁单极子） |

## 四、旋度——"这里在转圈圈吗？"

### 4.1 从漩涡说起

观察一条河：如果河水平直地流，放一片树叶，它只会平移；如果河中有漩涡，树叶会**打转**。

**旋度衡量的是矢量场在某点"旋转"的强度和方向。**

### 4.2 先理解"环量"

环量 = 场矢量沿着一个闭合回路的"绕圈总量"：
$$\Gamma = \oint_L \vec{F}\cdot d\vec{l}$$

- 如果场与回路走向一致，贡献为正
- 如果场与回路走向相反，贡献为负

**类比**：在操场上跑步，如果顺风跑（场顺着你的方向），环量为正；逆风跑，环量为负。

### 4.3 旋度的定义

**旋度 = 单位面积的环量**（取极限到点）：
$$\nabla\times\vec{F} = \lim_{S\to 0}\frac{1}{S}\oint_L \vec{F}\cdot d\vec{l}$$

用直角坐标计算（最实用的公式）：
$$\nabla\times\vec{F} = \begin{vmatrix}
\hat{x} & \hat{y} & \hat{z} \\
\frac{\partial}{\partial x} & \frac{\partial}{\partial y} & \frac{\partial}{\partial z} \\
F_x & F_y & F_z
\end{vmatrix}$$

分量形式：
- $(\nabla\times\vec{F})_x = \frac{\partial F_z}{\partial y} - \frac{\partial F_y}{\partial z}$
- $(\nabla\times\vec{F})_y = \frac{\partial F_x}{\partial z} - \frac{\partial F_z}{\partial x}$
- $(\nabla\times\vec{F})_z = \frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y}$

> **记忆规律**：每个分量 = 另外两个方向的"交叉偏导之差"（像三阶行列式的展开）

### 4.4 具体算几个例子

#### 例 1：均匀场（旋度为零）

$\vec{F} = (3, 0, 0)$（每点都是向右）。显然没有旋转——旋度 = **(0, 0, 0)**。

用公式验证：F_y = 0, F_x = 3（常数），所有偏导数都是 0。

#### 例 2：纯旋转场（有旋度）

$\vec{F} = (-y, x, 0)$。画出来看——场的方向是绕原点逆时针转圈。

- $(\nabla\times\vec{F})_z = \frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y} = \frac{\partial(x)}{\partial x} - \frac{\partial(-y)}{\partial y} = 1 - (-1) = \mathbf{2}$
- x 分量和 y 分量 = 0

旋度 ∇×F = **(0, 0, 2)**。z 方向非零——场的旋转轴是 z 轴（逆时针）。

> 如果把一片树叶放在这个场中，它会在原地打转——这就是有旋场的特征。

#### 例 3：辐射场（旋度为零——重要！）

点电荷的电场 $\vec{E} = \frac{Q}{4\pi\varepsilon_0}\frac{\hat{r}}{r^2}$（沿径向向外）。

直观上，这个场没有"转圈"的成分——全是直线辐射。可以验证 ∇×E = **0**。

**这就是为什么静电场可以定义电势 φ**：只有旋度为零的场才能写成梯度的形式！

### 4.5 旋度在电磁学中的核心应用

#### 法拉第定律：∇×E = -∂B/∂t

**白话翻译**：电场在某点的旋度，等于该点磁场变化率的负值。

| 旋度值 | 物理含义 |
|--------|----------|
| ∇×E = 0 | 静电场——可以定义电势，电场力做功与路径无关 |
| ∇×E ≠ 0 | 有变化的磁场——产生了**涡旋电场**，不能定义标量电势 |

**实例——变化的磁场产生涡旋电场**：

一个圆柱形区域内有均匀磁场 B(t) 在变化（比如电磁铁中交变电流产生的）。在磁场区域外，虽然 B = 0，但 ∇×E = -∂B/∂t ≠ 0。

这意味着：**即使在没有磁场的地方，只要磁场在变化，就会产生环形的涡旋电场！** 这就是变压器的原理——铁芯中的交变磁场在次级线圈中感应出环形电场，驱动电流。

**实例——静电场（无旋场）**：

点电荷的电场全是径向的，没有旋转成分，∇×E = 0。这就是为什么：
- 电场力是保守力（类似于重力）
- 可以定义电势能
- 可以用电势 φ 来简化计算
- 两个固定点之间，电场力做的功与路径无关

#### 安培-麦克斯韦定律：∇×B = μ₀J + μ₀ε₀ ∂E/∂t

磁场的旋度由两样东西产生：传导电流 J + 变化的电场 ∂E/∂t（位移电流）。

| 旋度值 | 物理含义 |
|--------|----------|
| ∇×B ≠ 0 | 有电流穿过，或有变化的电场 |
| ∇×B = 0 | 没有电流也没有变化的电场（可以用磁标势） |

**实例——直导线周围的磁场**：

无限长直导线载流 I，产生的磁场是围绕导线的同心圆。B ∝ 1/r，方向由右手定则决定。

这种"绕圈"的场，旋度集中在导线所在位置（∇×B = μ₀J），导线外旋度为零（虽然磁场非零！）。

> ⚠️ **重要区分**：**磁场强度非零 ≠ 旋度非零！** 直导线外的磁场是"有旋度为零的非零场"——就像水流可以绕柱子转圈，但每个水分子本身可以不旋转（这种叫做"无旋环流"）。

### 4.6 旋度总结卡片

| 问题 | 回答 |
|------|------|
| 旋度作用在什么上？ | **矢量场** |
| 旋度得出什么？ | **矢量场**（转轴方向 + 转的强度） |
| 非零旋度代表什么？ | 场在**转圈**——像漩涡 |
| 零旋度代表什么？ | **无旋场**——可以写成梯度的形式 |
| 电磁学核心公式 | ∇×E = -∂B/∂t；∇×B = μ₀J + μ₀ε₀∂E/∂t |
| 直观类比 | 树叶会不会打转 |

## 五、拉普拉斯算子——梯度与散度的合体

### 5.1 定义

拉普拉斯算子 ∇² = ∇·∇，是"先求梯度再求散度"的复合操作：

$$\nabla^2\varphi = \nabla\cdot(\nabla\varphi) = \frac{\partial^2\varphi}{\partial x^2} + \frac{\partial^2\varphi}{\partial y^2} + \frac{\partial^2\varphi}{\partial z^2}$$

### 5.2 物理含义

∇²φ 衡量的是"某点的值与周围平均值差多少"。

- ∇²φ > 0：该点低于周围平均值（像山谷底部——周围都比它高）
- ∇²φ < 0：该点高于周围平均值（像山峰顶部——周围都比它低）
- ∇²φ = 0：该点恰好等于周围平均值（调和函数）

### 5.3 在电磁学中的核心地位

**泊松方程**：∇²φ = -ρ/ε₀

翻译：电势的拉普拉斯 = -电荷密度/ε₀。有电荷的地方，电势不是周围电势的平均值——源项在修改它。

**拉普拉斯方程**：∇²φ = 0（无电荷区域）

翻译：没有电荷的区域，每一点的电势都是周围电势的平均值。这就是为什么电势的极值只能出现在边界上（内部不会"自动"出现山顶或山谷）。

**波动方程**：$\nabla^2\vec{E} = \frac{1}{c^2}\frac{\partial^2\vec{E}}{\partial t^2}$

翻译：电场的空间弯曲 = 电场的时间加速度/光速²。这就是电磁波的波动方程——拉普拉斯算子描述空间传播，∂²/∂t² 描述时间振荡。

## 六、三个"度"如何组成麦克斯韦方程组

现在回头看麦克斯韦方程组的微分形式，一切豁然开朗：

| 方程 | 用"度"的语言翻译 |
|------|-------------------|
| ∇·E = ρ/ε₀ | 电场的**散度** = 电荷密度/ε₀ → 电荷是电场之源 |
| ∇·B = 0 | 磁场的**散度** = 0 → 没有磁单极子，磁感线永远闭合 |
| ∇×E = -∂B/∂t | 电场的**旋度** = -磁场变化率 → 变化的磁场产生涡旋电场 |
| ∇×B = μ₀J + μ₀ε₀∂E/∂t | 磁场的**旋度** = μ₀×(传导电流 + 位移电流) → 电流和变化的电场产生磁场 |

**记忆顺口溜**：
> 电散源，磁散零；  
> 电旋感磁变，磁旋源流变。

### 两个"度"的关系：亥姆霍兹定理

**任何一个足够光滑的矢量场，可以唯一地分解为无旋部分（由散度决定）+ 无散部分（由旋度决定）。**

这意味着：给定了 ∇·F（散度）和 ∇×F（旋度），再加上边界条件——整个矢量场 F 就完全确定了。

**麦克斯韦方程组的深层含义**：它用四个方程给出了 E 和 B 各自的散度和旋度——因此（加上适当的边界条件）唯一地确定了电磁场。这就是为什么四个方程就够了！

## 七、不同坐标系下的写法（速查）

### 直角坐标 (x, y, z)

- 梯度：$\nabla f = (\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z})$
- 散度：$\nabla\cdot\vec{F} = \frac{\partial F_x}{\partial x} + \frac{\partial F_y}{\partial y} + \frac{\partial F_z}{\partial z}$
- 旋度：$\nabla\times\vec{F} = (\frac{\partial F_z}{\partial y} - \frac{\partial F_y}{\partial z}, \frac{\partial F_x}{\partial z} - \frac{\partial F_z}{\partial x}, \frac{\partial F_y}{\partial x} - \frac{\partial F_x}{\partial y})$

### 柱坐标 (ρ, φ, z)

- 梯度：$\nabla f = \frac{\partial f}{\partial\rho}\hat{\rho} + \frac{1}{\rho}\frac{\partial f}{\partial\varphi}\hat{\varphi} + \frac{\partial f}{\partial z}\hat{z}$
- 散度：$\nabla\cdot\vec{F} = \frac{1}{\rho}\frac{\partial(\rho F_\rho)}{\partial\rho} + \frac{1}{\rho}\frac{\partial F_\varphi}{\partial\varphi} + \frac{\partial F_z}{\partial z}$
- 旋度：$\nabla\times\vec{F} = \left(\frac{1}{\rho}\frac{\partial F_z}{\partial\varphi} - \frac{\partial F_\varphi}{\partial z}\right)\hat{\rho} + \left(\frac{\partial F_\rho}{\partial z} - \frac{\partial F_z}{\partial\rho}\right)\hat{\varphi} + \frac{1}{\rho}\left(\frac{\partial(\rho F_\varphi)}{\partial\rho} - \frac{\partial F_\rho}{\partial\varphi}\right)\hat{z}$

### 球坐标 (r, θ, φ)

- 梯度：$\nabla f = \frac{\partial f}{\partial r}\hat{r} + \frac{1}{r}\frac{\partial f}{\partial\theta}\hat{\theta} + \frac{1}{r\sin\theta}\frac{\partial f}{\partial\varphi}\hat{\varphi}$
- 散度：$\nabla\cdot\vec{F} = \frac{1}{r^2}\frac{\partial(r^2 F_r)}{\partial r} + \frac{1}{r\sin\theta}\frac{\partial(\sin\theta F_\theta)}{\partial\theta} + \frac{1}{r\sin\theta}\frac{\partial F_\varphi}{\partial\varphi}$
- 旋度：$\nabla\times\vec{F} = \frac{1}{r\sin\theta}\left[\frac{\partial(\sin\theta F_\varphi)}{\partial\theta} - \frac{\partial F_\theta}{\partial\varphi}\right]\hat{r} + \frac{1}{r}\left[\frac{1}{\sin\theta}\frac{\partial F_r}{\partial\varphi} - \frac{\partial(r F_\varphi)}{\partial r}\right]\hat{\theta} + \frac{1}{r}\left[\frac{\partial(r F_\theta)}{\partial r} - \frac{\partial F_r}{\partial\theta}\right]\hat{\varphi}$

> 💡 **实用建议**：直角坐标的公式必须滚瓜烂熟（最常用）。柱坐标和球坐标的公式考试时可以推导或查阅，但要对形式有印象——特别是球坐标散度中那个 r² 因子。

## 八、柱坐标系完全攻略

电动力学中，凡是涉及**无限长圆柱、长直导线、螺线管、同轴电缆**的问题，柱坐标是你的首选武器。

### 8.1 先认识坐标系

**柱坐标三兄弟** (ρ, φ, z)：

| 坐标 | 范围 | 直观理解 |
|------|------|----------|
| ρ | 0 → ∞ | 到 z 轴的**垂直距离**（径向距离） |
| φ | 0 → 2π | 绕 z 轴的**方位角**（从 x 轴正方向逆时针量） |
| z | -∞ → ∞ | **高度**（和直角坐标的 z 一样） |

**与直角坐标的转换：**
$$x = \rho\cos\varphi, \quad y = \rho\sin\varphi, \quad z = z$$
$$\rho = \sqrt{x^2 + y^2}, \quad \varphi = \arctan\frac{y}{x}$$

**单位矢量方向：**
- $\hat{\rho}$：从 z 轴垂直向外的方向（径向向外）
- $\hat{\varphi}$：绕 z 轴逆时针的切线方向
- $\hat{z}$：竖直向上（固定不变）

> ⚠️ **与其他坐标系最关键的区别**：$\hat{\rho}$ 和 $\hat{\varphi}$ **不是固定的**！它们取决于你在空间中的位置。在点 (1, 0, 0) 处，$\hat{\rho} = \hat{x}$；在点 (0, 1, 0) 处，$\hat{\rho} = \hat{y}$。这一点是计算柱坐标导数时最容易出错的地方。

### 8.2 梯度在柱坐标下

$$\nabla f = \frac{\partial f}{\partial\rho}\hat{\rho} + \frac{1}{\rho}\frac{\partial f}{\partial\varphi}\hat{\varphi} + \frac{\partial f}{\partial z}\hat{z}$$

**逐项解读：**

| 项 | 含义 | 为什么有这个系数 |
|----|------|-----------------|
| $\frac{\partial f}{\partial\rho}\hat{\rho}$ | ρ 方向变化率 | 沿径向走一小步 dρ，高度变化率就是 ∂f/∂ρ——没有额外系数，因为 ρ 是"平直"的 |
| $\frac{1}{\rho}\frac{\partial f}{\partial\varphi}\hat{\varphi}$ | φ 方向变化率 | 沿角度走一小步 dφ，实际走的路程是 **ρ·dφ**（弧长 = 半径 × 半径）。所以要除以 ρ 才能得到"单位弧长的变化率" |
| $\frac{\partial f}{\partial z}\hat{z}$ | z 方向变化率 | z 方向是平直的，没有修正系数 |

**实例——长直导线的电势：**

无限长均匀带电直线的电势（取参考点在 ρ₀）：
$$\varphi(\rho) = -\frac{\lambda}{2\pi\varepsilon_0}\ln\frac{\rho}{\rho_0}$$

这里 φ 只依赖于 ρ，与 φ 和 z 无关。在柱坐标下求梯度：

$$\vec{E} = -\nabla\varphi = -\frac{\partial\varphi}{\partial\rho}\hat{\rho} = -\left(-\frac{\lambda}{2\pi\varepsilon_0}\cdot\frac{1}{\rho}\right)\hat{\rho} = \frac{\lambda}{2\pi\varepsilon_0\rho}\hat{\rho}$$

结果：电场沿径向向外，大小 ∝ 1/ρ——这正是我们滚瓜烂熟的结果。如果用直角坐标算同样的东西会痛苦得多。

**实例——螺线管中的磁标势：**

长直螺线管内部磁场均匀（B = μ₀nI 沿 z 方向），在柱坐标下磁标势为 φₘ = -Bz（类比匀强电场中的电势 φ = -Ez）。

### 8.3 散度在柱坐标下

$$\nabla\cdot\vec{F} = \frac{1}{\rho}\frac{\partial(\rho F_\rho)}{\partial\rho} + \frac{1}{\rho}\frac{\partial F_\varphi}{\partial\varphi} + \frac{\partial F_z}{\partial z}$$

**逐项解读：**

| 项 | 含义 |
|----|------|
| $\frac{1}{\rho}\frac{\partial(\rho F_\rho)}{\partial\rho}$ | 径向的"流入流出差"。一定要先乘 ρ 再求导再除以 ρ——因为柱坐标下，离轴越远面积越大 |
| $\frac{1}{\rho}\frac{\partial F_\varphi}{\partial\varphi}$ | 角向的变化。同样除以 ρ 把角度变化换算成弧长变化 |
| $\frac{\partial F_z}{\partial z}$ | z 方向的流入流出差——和直角坐标一样 |

**为什么第一项有那个复杂的 1/ρ 和 ρ 互相搭配？**

考虑一个小体积元：在径向，内表面面积是 ρ·dφ·dz，外表面面积是 (ρ+dρ)·dφ·dz。两个面不一样大！散度关心的是**通量差**，而通量 = 场 × 面积。所以必须把面积因子 ρ 拿进去一起求导。

**实例——验证 ∇·E = ρ/ε₀：**

长直导线电场 $\vec{E} = \frac{\lambda}{2\pi\varepsilon_0\rho}\hat{\rho}$（只有 ρ 分量）。

散度：$\nabla\cdot\vec{E} = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\cdot\frac{\lambda}{2\pi\varepsilon_0\rho}\right) = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\frac{\lambda}{2\pi\varepsilon_0}\right) = \frac{1}{\rho}\cdot 0 = 0$（ρ > 0 时）。

在 ρ = 0 处（导线所在位置），散度是 δ 函数——对应导线上的线电荷密度。

### 8.4 旋度在柱坐标下

$$\nabla\times\vec{F} = 
\left(\frac{1}{\rho}\frac{\partial F_z}{\partial\varphi} - \frac{\partial F_\varphi}{\partial z}\right)\hat{\rho}
+ \left(\frac{\partial F_\rho}{\partial z} - \frac{\partial F_z}{\partial\rho}\right)\hat{\varphi}
+ \frac{1}{\rho}\left(\frac{\partial(\rho F_\varphi)}{\partial\rho} - \frac{\partial F_\rho}{\partial\varphi}\right)\hat{z}$$

**行列式形式（便于记忆和快速查阅）：**

$$\nabla\times\vec{F} = \frac{1}{\rho}
\begin{vmatrix}
\hat{\rho} & \rho\hat{\varphi} & \hat{z} \\
\frac{\partial}{\partial\rho} & \frac{\partial}{\partial\varphi} & \frac{\partial}{\partial z} \\
F_\rho & \rho F_\varphi & F_z
\end{vmatrix}$$

> 🔑 **行列式怎么会多出 ρ 和 1/ρ？** 柱坐标不是正交直角坐标——第二行的 ∂/∂φ 对应的实际"步长"是 ρ·dφ，不是 dφ。行列式展开时必须把这道"曲度修正"含进去：第二列基底写成 ρ·φ̂、第三行对 φ 的"场分量"写成 ρF_φ。展开后自然会冒出 1/ρ 的各种系数——这个行列式是帮你一次记忆全部公式的捷径。

**逐项解读：**

| 分量 | 公式 | 物理图像 | 系数由来 |
|------|------|----------|----------|
| $\hat{\rho}$ 分量 | $\frac{1}{\rho}\frac{\partial F_z}{\partial\varphi} - \frac{\partial F_\varphi}{\partial z}$ | 在 **(φ, z)** 平面上看旋转——看 z 方向场随 φ 的变化 和 φ 方向场随 z 的变化 | $\frac{1}{\rho}$ 原因：沿 φ 方向走 dφ，实际距离是 ρ·dφ，所以单位距离的变化率要除以 ρ |
| $\hat{\varphi}$ 分量 | $\frac{\partial F_\rho}{\partial z} - \frac{\partial F_z}{\partial\rho}$ | 在 **(z, ρ)** 平面上看旋转——看 ρ 方向场随 z 的变化 和 z 方向场随 ρ 的变化 | 无额外系数！因为 z 和 ρ 方向都是"平直"的 |
| $\hat{z}$ 分量 | $\frac{1}{\rho}\frac{\partial(\rho F_\varphi)}{\partial\rho} - \frac{1}{\rho}\frac{\partial F_\rho}{\partial\varphi}$ | 在 **(ρ, φ)** 平面上看旋转——看 φ 方向场随 ρ 的变化 和 ρ 方向场随 φ 的变化 | $\frac{1}{\rho}$ 原因同上（φ 方向弧长修正）；**特别注意**：导的是 $\rho F_\varphi$ 而不是 $F_\varphi$——因为闭合回路在不同 ρ 处周长不同 |

> 📌 **与其他"度"的联动**：z 分量的旋度 = 0 意味着场在 (ρ, φ) 平面是无旋的——此时 ρ 和 φ 方向的分量可以写成一个标量的梯度（类似于静电场中 E = -∇φ）。这正是引入磁标势时需要的条件。

**记忆技巧**：把三个分量想象成直角坐标旋度在三个"局部平面"上的投影——ρ 分量看 (φ→z) 的旋转；φ 分量看 (z→ρ) 的旋转；z 分量看 (ρ→φ) 的旋转。每个非 z 分量都附带 1/ρ 系数。

**实例——直导线周围的磁场：**

磁场 $\vec{B} = \frac{\mu_0 I}{2\pi\rho}\hat{\varphi}$（只有 φ 分量，绕导线转圈）。

在导线外（ρ > 0）求旋度：
- ρ 分量：F_z = 0 且 ∂F_φ/∂z = 0 → **0**
- φ 分量：F_ρ = 0 且 ∂F_z/∂ρ = 0 → **0**
- z 分量：$\frac{1}{\rho}\frac{\partial(\rho B_\varphi)}{\partial\rho} = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\cdot\frac{\mu_0 I}{2\pi\rho}\right) = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\frac{\mu_0 I}{2\pi}\right) = 0$

旋度在 ρ > 0 处为零——磁场绕圈不等于旋度非零！旋度只集中在 ρ = 0（导线所在位置），那里有电流 J。

### 8.5 拉普拉斯算子 ∇² 在柱坐标下

**定义：拉普拉斯 = 梯度的散度**
$$\nabla^2 f = \nabla\cdot(\nabla f)$$

将柱坐标梯度代入柱坐标散度，即得上式。这是最安全的推导方式——不需要死记硬背。

$$\nabla^2 f = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\frac{\partial f}{\partial\rho}\right) + \frac{1}{\rho^2}\frac{\partial^2 f}{\partial\varphi^2} + \frac{1}{\rho^0}\frac{\partial^2 f}{\partial z^2}$$

**结构分解（Lamé 系数形式）：**

柱坐标的三个 Lamé 系数（各方向的"步长缩放因子"）：
$$h_\rho = 1,\quad h_\varphi = \rho,\quad h_z = 1$$

用 Lamé 系数写 ∇²（正交曲线坐标通用公式）：
$$\nabla^2 f = \frac{1}{h_\rho h_\varphi h_z}\left[\frac{\partial}{\partial\rho}\left(\frac{h_\varphi h_z}{h_\rho}\frac{\partial f}{\partial\rho}\right) + \frac{\partial}{\partial\varphi}\left(\frac{h_\rho h_z}{h_\varphi}\frac{\partial f}{\partial\varphi}\right) + \frac{\partial}{\partial z}\left(\frac{h_\rho h_\varphi}{h_z}\frac{\partial f}{\partial z}\right)\right]$$

代入 hρ=1, hφ=ρ, hz=1：
$$\nabla^2 f = \frac{1}{\rho}\left[\frac{\partial}{\partial\rho}\left(\rho\frac{\partial f}{\partial\rho}\right) + \frac{\partial}{\partial\varphi}\left(\frac{1}{\rho}\frac{\partial f}{\partial\varphi}\right) + \frac{\partial}{\partial z}\left(\rho\frac{\partial f}{\partial z}\right)\right]$$

整理后即得标准形式。**这个 Lamé 系数公式是三种坐标系的统一"母公式"——记住了它，直角/柱/球三种 ∇² 全部自动推出。** 考试时如果忘了某个坐标系的 ∇²，从梯度 + 散度两步复合推导也能还原。

**逐项解读：**

| 项 | 公式 | 含义 | 系数由来 |
|----|------|------|----------|
| 径向项 | $\frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\frac{\partial f}{\partial\rho}\right)$ | f 沿径向的"弯曲程度" | 外层的 1/ρ 来自散度的面积修正（柱面面积 ∝ ρ）；内层的 ρ 来自梯度通量在不同 ρ 处的面积差异 |
| 角向项 | $\frac{1}{\rho^2}\frac{\partial^2 f}{\partial\varphi^2}$ | f 沿角向的"弯曲程度" | 分母 ρ² 的来历：梯度中 ∂f/∂φ 自带 1/ρ；散度再对 φ 求导时又出一个 1/ρ——两个 1/ρ 合起来就是 1/ρ² |
| z 向项 | $\frac{\partial^2 f}{\partial z^2}$ | f 沿 z 方向的"弯曲程度" | 无修正——z 方向是平直的，和直角坐标一样 |

> 🔑 **核心理解**：Laplacian 衡量的是"某点的值偏离周围平均值的程度"。柱坐标下，这个"周围"包括径向的圆柱面、角向的弧段、z 向的直线段——三种几何各有各的尺度修正。

**实例——无限长均匀带电直线的电势：**

电势 $\varphi(\rho) = -\frac{\lambda}{2\pi\varepsilon_0}\ln\rho$（与 φ 和 z 无关）。

拉普拉斯：$\nabla^2\varphi = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\frac{\partial\varphi}{\partial\rho}\right) = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\cdot\left(-\frac{\lambda}{2\pi\varepsilon_0}\cdot\frac{1}{\rho}\right)\right) = \frac{1}{\rho}\frac{\partial}{\partial\rho}\left(-\frac{\lambda}{2\pi\varepsilon_0}\right) = 0$

在 ρ > 0 处 ∇²φ = 0——满足拉普拉斯方程（无电荷区域）。在 ρ = 0 处对应线电荷的 δ 函数源。

**实例——圆柱形波导中的电磁波（TM 模式）：**

柱坐标下波动方程 $\nabla^2 E_z + k^2 E_z = 0$ 展开后，z 分量满足：
$$\frac{1}{\rho}\frac{\partial}{\partial\rho}\left(\rho\frac{\partial E_z}{\partial\rho}\right) + \frac{1}{\rho^2}\frac{\partial^2 E_z}{\partial\varphi^2} + \frac{\partial^2 E_z}{\partial z^2} + k^2 E_z = 0$$

分离变量后径向部分给出**贝塞尔方程**：
$$\rho^2\frac{d^2 R}{d\rho^2} + \rho\frac{dR}{d\rho} + (k_c^2\rho^2 - n^2)R = 0$$

解为 $R(\rho) = A J_n(k_c\rho) + B N_n(k_c\rho)$——这就是为什么贝塞尔函数在柱对称问题中无处不在。

**常见简并情况：**

如果 f 与 φ 和 z 都无关（轴对称 + 无限长均匀），拉普拉斯方程简化为：
$$\frac{1}{\rho}\frac{d}{d\rho}\left(\rho\frac{df}{d\rho}\right) = 0$$

解为 $f = A\ln\rho + B$——这就是柱对称下对数电势的数学来源。

> 📌 **与其他"度"的联动**：柱坐标 ∇²φ = -ρ/ε₀ 就是**泊松方程**的柱坐标形式。当电荷分布具有柱对称性时（如无限长圆柱均匀带电），用这个方程求解电势比用库仑定律积分方便得多。

### 8.6 柱坐标总结卡片

| 问题 | 回答 |
|------|------|
| 什么时候用柱坐标？ | 问题有**绕某条轴线的旋转对称性**（无限长圆柱、直导线、螺线管、同轴电缆） |
| 最关键的区别是什么？ | $\hat{\rho}$ 和 $\hat{\varphi}$ 随位置改变方向！导数要小心 |
| 为什么散度多出 1/ρ？ | 面积元与 ρ 成正比——离轴越远，面积越大 |
| 为什么梯度角向分量有 1/ρ？ | 角度增量对应的实际距离是 ρ·dφ |
| ∇² 的解长什么样？ | 径向部分涉及 Bessel 函数或对数 |

## 九、球坐标系完全攻略

电动力学中，凡是涉及**点电荷、带电球体、球面边界问题、偶极辐射、原子的电磁场**，球坐标是最自然的选择。

### 9.1 先认识坐标系

**球坐标三兄弟** (r, θ, φ)：

| 坐标 | 范围 | 直观理解 |
|------|------|----------|
| r | 0 → ∞ | 到原点的**径向距离** |
| θ | 0 → π | **极角**：从 z 轴正方向往下的倾角（θ=0 是北极，θ=π/2 是赤道，θ=π 是南极） |
| φ | 0 → 2π | **方位角**：绕 z 轴的角度（从 x 轴正方向逆时针量，和柱坐标的 φ 一样） |

**与直角坐标的转换：**
$$x = r\sin\theta\cos\varphi, \quad y = r\sin\theta\sin\varphi, \quad z = r\cos\theta$$
$$r = \sqrt{x^2 + y^2 + z^2}, \quad \theta = \arccos\frac{z}{r}, \quad \varphi = \arctan\frac{y}{x}$$

**单位矢量方向：**
- $\hat{r}$：径向向外（从原点指向该点）
- $\hat{\theta}$：沿经线向南的方向（θ 增加的方向，垂直于 $\hat{r}$）
- $\hat{\varphi}$：沿纬线向东的方向（φ 增加的方向，垂直于 $\hat{r}$ 和 $\hat{\theta}$）

> ⚠️ **三个单位矢量都随位置改变**！$\hat{r}$ 指向取决于该点位置，$\hat{\theta}$ 和 $\hat{\varphi}$ 同样。球坐标下求导时，单位矢量的变化不能忽略。

### 9.2 梯度在球坐标下

$$\nabla f = \frac{\partial f}{\partial r}\hat{r} + \frac{1}{r}\frac{\partial f}{\partial\theta}\hat{\theta} + \frac{1}{r\sin\theta}\frac{\partial f}{\partial\varphi}\hat{\varphi}$$

**逐项解读：**

| 项 | 含义 | 为什么有这个系数 |
|----|------|-----------------|
| $\frac{\partial f}{\partial r}\hat{r}$ | 径向变化率 | 径向是"平直"的——走一小步 dr，实际距离就是 dr |
| $\frac{1}{r}\frac{\partial f}{\partial\theta}\hat{\theta}$ | θ 方向变化率 | θ 增加 dθ，在球面上走的弧长是 **r·dθ**（曲率半径 × 角度微元）。所以要除以 r |
| $\frac{1}{r\sin\theta}\frac{\partial f}{\partial\varphi}\hat{\varphi}$ | φ 方向变化率 | φ 增加 dφ，在纬度 θ 处，纬线半径是 **r·sinθ**，走的弧长是 **r·sinθ·dφ**。所以要除以 r·sinθ |

> 🔑 **记忆口诀**：径向 r 无修正；θ 方向除以 r；φ 方向除以 r·sinθ——越靠近极轴（θ→0 或 π），sinθ → 0，φ 方向的梯度趋于无穷大（因为纬线圈收缩为一个点，走同样角度但实际距离为零，单位距离的变化率趋于无穷）。

**实例——点电荷的电势**：

$$\varphi(r) = \frac{Q}{4\pi\varepsilon_0}\frac{1}{r}$$

电势只依赖于 r。在球坐标下：
$$\vec{E} = -\nabla\varphi = -\frac{\partial\varphi}{\partial r}\hat{r} = -\left(-\frac{Q}{4\pi\varepsilon_0}\frac{1}{r^2}\right)\hat{r} = \frac{Q}{4\pi\varepsilon_0}\frac{\hat{r}}{r^2}$$

恰好退化为库仑定律——球坐标系让球对称问题变得极其简单。

**实例——电偶极子的电势**：

原点处沿 z 轴的电偶极子 $\vec{p} = p\hat{z}$，电势为：
$$\varphi(r,\theta) = \frac{p\cos\theta}{4\pi\varepsilon_0 r^2}$$

在球坐标下，φ 同时依赖于 r 和 θ（但与 φ 无关——轴对称）。电场有两个分量：
$$E_r = -\frac{\partial\varphi}{\partial r} = \frac{2p\cos\theta}{4\pi\varepsilon_0 r^3}, \quad E_\theta = -\frac{1}{r}\frac{\partial\varphi}{\partial\theta} = \frac{p\sin\theta}{4\pi\varepsilon_0 r^3}$$

这就是电偶极场的标准结果——用球坐标一步到位。

### 9.3 散度在球坐标下

$$\nabla\cdot\vec{F} = \frac{1}{r^2}\frac{\partial(r^2 F_r)}{\partial r} + \frac{1}{r\sin\theta}\frac{\partial(\sin\theta F_\theta)}{\partial\theta} + \frac{1}{r\sin\theta}\frac{\partial F_\varphi}{\partial\varphi}$$

**逐项解读：**

| 项 | 含义 |
|----|------|
| $\frac{1}{r^2}\frac{\partial(r^2 F_r)}{\partial r}$ | 径向的流入流出差。球面面积 ∝ r²——必须先乘 r² 再求导再除以 r² |
| $\frac{1}{r\sin\theta}\frac{\partial(\sin\theta F_\theta)}{\partial\theta}$ | θ 方向的流入流出差。注意被导函数是 **sinθ·Fθ**（不是 Fθ！）——因为不同 θ 处环带的面积不同 |
| $\frac{1}{r\sin\theta}\frac{\partial F_\varphi}{\partial\varphi}$ | φ 方向的流入流出差——这个和柱坐标的 φ 分量类似 |

**为什么径向有 r² 而不是柱坐标中的 r？**

柱坐标下，侧面积 ∝ ρ（圆柱侧面积 = 2πρ·h）。球坐标下，球面积 ∝ r²（球面积 = 4πr²）。所以径向的修正因子从 ρ 变成 r²。

**实例——验证 ∇·E = ρ/ε₀：**

点电荷电场 $\vec{E} = \frac{Q}{4\pi\varepsilon_0}\frac{\hat{r}}{r^2}$（只有径向分量）。

球坐标散度（在 r > 0 处）：
$$\nabla\cdot\vec{E} = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\cdot\frac{Q}{4\pi\varepsilon_0 r^2}\right) = \frac{1}{r^2}\frac{\partial}{\partial r}\left(\frac{Q}{4\pi\varepsilon_0}\right) = \frac{1}{r^2}\cdot 0 = 0$$

r² 和 1/r² 完美抵消——在 r ≠ 0 处散度为零。r = 0 处散度无穷大（对应点电荷的 δ 函数）。

### 9.4 旋度在球坐标下

$$\nabla\times\vec{F} = 
\frac{1}{r\sin\theta}\left[\frac{\partial(\sin\theta F_\varphi)}{\partial\theta} - \frac{\partial F_\theta}{\partial\varphi}\right]\hat{r}
+ \frac{1}{r}\left[\frac{1}{\sin\theta}\frac{\partial F_r}{\partial\varphi} - \frac{\partial(r F_\varphi)}{\partial r}\right]\hat{\theta}
+ \frac{1}{r}\left[\frac{\partial(r F_\theta)}{\partial r} - \frac{\partial F_r}{\partial\theta}\right]\hat{\varphi}$$

**行列式形式（便于记忆和快速查阅）：**

$$\nabla\times\vec{F} = \frac{1}{r^2\sin\theta}
\begin{vmatrix}
\hat{r} & r\,\hat{\theta} & r\sin\theta\,\hat{\varphi} \\
\frac{\partial}{\partial r} & \frac{\partial}{\partial\theta} & \frac{\partial}{\partial\varphi} \\
F_r & r F_\theta & r\sin\theta\,F_\varphi
\end{vmatrix}$$

> 🔑 **行列式中 r 和 sinθ 的修正逻辑**：球坐标三个方向的"实际步长"分别是 dr、r·dθ、r·sinθ·dφ。行列式的第二行 ∂/∂θ 和 ∂/∂φ 对应的基底分别要乘以 r 和 r·sinθ；第三行的场分量同样要乘对应因子。最前面除以 r²·sinθ 的体积权重来自"正交曲线坐标下的普遍旋度公式"。展开这个行列式，得到的恰好就是上面分量形式的完整旋度——行列式是推导和记忆的捷径。

**逐项解读：**

| 分量 | 公式 | 物理图像 | 系数由来 |
|------|------|----------|----------|
| $\hat{r}$ 分量 | $\frac{1}{r\sin\theta}\frac{\partial(\sin\theta F_\varphi)}{\partial\theta} - \frac{1}{r\sin\theta}\frac{\partial F_\theta}{\partial\varphi}$ | 在 **(θ, φ)** 平面上看旋转——看 φ 方向场随 θ 的变化 和 θ 方向场随 φ 的变化 | 分母 **r·sinθ** 是关键：小回路沿 θ 走一步的边长 ∝ r，沿 φ 走一步的边长 ∝ r·sinθ；回路面积 ∝ r²·sinθ。但最终结果中，两个偏导各承担了 r 的一部分修正，最终整体系数是 **1/(r·sinθ)**（而非 1/(r²·sinθ)），因为分子内 sinθ 被导数的链式法则部分吸收 |
| $\hat{\theta}$ 分量 | $\frac{1}{r\sin\theta}\frac{\partial F_r}{\partial\varphi} - \frac{1}{r}\frac{\partial(r F_\varphi)}{\partial r}$ | 在 **(φ, r)** 平面上看旋转——看 r 方向场随 φ 的变化 和 φ 方向场随 r 的变化 | 第一项分母 1/(r·sinθ)（沿 φ 方向弧长修正）；第二项分母 1/r 且导的是 **rF_φ** 而非 F_φ（沿 r 方向的回路对边长度不同） |
| $\hat{\varphi}$ 分量 | $\frac{1}{r}\frac{\partial(r F_\theta)}{\partial r} - \frac{1}{r}\frac{\partial F_r}{\partial\theta}$ | 在 **(r, θ)** 平面上看旋转——看 θ 方向场随 r 的变化 和 r 方向场随 θ 的变化 | 分母 **1/r**（θ 方向弧长修正）；注意第一项导的是 **rF_θ** 而非 F_θ——回路在不同 r 处的边长不同 |

> 📌 **与其他"度"的联动**：球坐标旋度中，最常见情况是 F 只有 r 分量且与 φ 无关（球对称径向场）——此时旋度三个分量全部为零。这就是为什么点电荷的电场 ∇×E = 0。如果场同时有 r 和 θ 分量，φ 分量的旋度可能非零——这是电偶极辐射中磁场的来源（∇×E 给出 ∂B/∂t，B 沿 φ 方向）。

**实例——用行列式快速验证点电荷电场无旋：**

$\vec{E} = \frac{Q}{4\pi\varepsilon_0 r^2}\hat{r}$，即 $F_r = Q/(4\pi\varepsilon_0 r^2), F_\theta = 0, F_\varphi = 0$。

代入行列式：第三行只有 F_r 非零，但 F_r 只依赖于 r，∂F_r/∂θ = 0，∂F_r/∂φ = 0。第二列和第三列的基底 rθ̂ 和 r·sinθ·φ̂ 都不为零但第三行对应元素都是零——展开后所有项为零。∇×E = **0**（静电场无旋，可以定义电势）。

**实例——电偶极辐射中 ∇×E：**

电偶极辐射场 $\vec{E}$ 有 r 和 θ 分量（远场近似下）。此时 φ 分量的旋度（即 $\hat{\varphi}$ 分量）通常非零——由 $\frac{1}{r}\frac{\partial(r E_\theta)}{\partial r} - \frac{1}{r}\frac{\partial E_r}{\partial\theta}$ 给出。而这个非零旋度恰好通过 ∇×E = -∂B/∂t 给出辐射磁场 B（沿 φ 方向）。

> 📌 球坐标的旋度公式看起来很吓人，但**静电场中基本上用不到**——因为静电场的旋度处处为零，你不需要去算它。真正用到球坐标旋度是在**电偶极辐射**和**电磁波的球面波展开**中。考试时多半会给出公式或可以只写出最终结果，但掌握行列式套路可以让你不慌。

**球坐标旋度记忆口诀：**
> r 分看 θ φ 转，分母 r·sinθ 记心间；  
> θ 分看 φ r 转，一半 sinθ 一半 r φ 伴；  
> φ 分看 r θ 转，分母 r 来导 r·Fθ 算。

### 9.5 拉普拉斯算子 ∇² 在球坐标下

**定义：拉普拉斯 = 梯度的散度**
$$\nabla^2 f = \nabla\cdot(\nabla f)$$

将球坐标梯度代入球坐标散度，即得上式。这是最安全的记忆方式——梯度里有 1/r 和 1/(r·sinθ)，散度里有 r² 和 sinθ 的面积修正，两层复合自然得出最终形式。

$$\nabla^2 f = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial f}{\partial r}\right) + \frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial f}{\partial\theta}\right) + \frac{1}{r^2\sin^2\theta}\frac{\partial^2 f}{\partial\varphi^2}$$

**结构分解（Lamé 系数形式）：**

球坐标的三个 Lamé 系数：
$$h_r = 1,\quad h_\theta = r,\quad h_\varphi = r\sin\theta$$

代入正交曲线坐标系 ∇² 的通用 Lamé 公式：
$$\nabla^2 f = \frac{1}{h_r h_\theta h_\varphi}\left[\frac{\partial}{\partial r}\left(\frac{h_\theta h_\varphi}{h_r}\frac{\partial f}{\partial r}\right) + \frac{\partial}{\partial\theta}\left(\frac{h_r h_\varphi}{h_\theta}\frac{\partial f}{\partial\theta}\right) + \frac{\partial}{\partial\varphi}\left(\frac{h_r h_\theta}{h_\varphi}\frac{\partial f}{\partial\varphi}\right)\right]$$

代入 hr=1, hθ=r, hφ=r·sinθ：
$$\nabla^2 f = \frac{1}{r^2\sin\theta}\left[\frac{\partial}{\partial r}\left(r^2\sin\theta\frac{\partial f}{\partial r}\right) + \frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial f}{\partial\theta}\right) + \frac{\partial}{\partial\varphi}\left(\frac{1}{\sin\theta}\frac{\partial f}{\partial\varphi}\right)\right]$$

整理后即得标准形式。**和柱坐标一样，这个 Lamé 公式是母公式——直角 (1,1,1)、柱 (1,ρ,1)、球 (1,r,r·sinθ) 三套 Lamé 系数对应三种 ∇²。记住母公式 + 三套 Lamé 系数，考试时永远不会写错。**

**逐项解读：**

| 项 | 公式 | 含义 | 系数由来 |
|----|------|------|----------|
| 径向项 | $\frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial f}{\partial r}\right)$ | f 沿径向的"弯曲程度" | 外层的 1/r² 来自散度的球面积修正（球面积 ∝ r²）；内层的 r² 来自梯度通量在不同 r 处的球面积差异——**柱坐标是第一项 ∝ ρ，球坐标是第一项 ∝ r²，这就是两者的本质区别** |
| 极角项 | $\frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial f}{\partial\theta}\right)$ | f 沿经线（θ 方向）的"弯曲程度" | 外层的 1/(r²·sinθ)来自散度体积元中的面积因子；内层 sinθ 来自不同 θ 处环带的面积差异——赤道 (θ=π/2) 处面积最大，两极 (θ=0,π) 处面积趋零 |
| 方位角项 | $\frac{1}{r^2\sin^2\theta}\frac{\partial^2 f}{\partial\varphi^2}$ | f 沿纬线（φ 方向）的"弯曲程度" | 分母 r²·sin²θ 的来历：梯度中 ∂f/∂φ 自带 1/(r·sinθ)，散度中再出一次 1/(r·sinθ)——两个合起来就是 1/(r²·sin²θ)。在极轴附近 (sinθ→0)，这一项趋于无穷大——物理上是因为纬线圈收缩为点 |

**这是电动力学中最重要的拉普拉斯形式！** 球坐标下拉普拉斯方程 ∇²φ = 0 的解就是**球谐函数展开**——分离变量法的核心工具。

**轴对称情况（f 与 φ 无关，这是最常见的情况）：**
$$\nabla^2 f = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial f}{\partial r}\right) + \frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial f}{\partial\theta}\right)$$

拉普拉斯方程 ∇²φ = 0 的通解为：
$$\varphi(r,\theta) = \sum_{n=0}^{\infty}\left(A_n r^n + \frac{B_n}{r^{n+1}}\right)P_n(\cos\theta)$$

其中 $r^n$ 项在原点处正则，$r^{-(n+1)}$ 项在无穷远处趋于零。这就是**分离变量法**的数学根基——$P_n$ 是勒让德多项式。考试中无论是介质球、导体球还是球面边界问题，都要从这个通解出发。

**实例——点电荷的电势验证 ∇²φ = -ρ/ε₀：**

$\varphi = \frac{Q}{4\pi\varepsilon_0 r}$，在 r > 0 处：
$$\nabla^2\varphi = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\cdot\left(-\frac{Q}{4\pi\varepsilon_0 r^2}\right)\right) = \frac{1}{r^2}\frac{\partial}{\partial r}\left(-\frac{Q}{4\pi\varepsilon_0}\right) = 0$$

在 r ≠ 0 处满足拉普拉斯方程。r = 0 处对应点电荷的 δ 函数源：∇²φ = -Q·δ(r)/ε₀ = -ρ/ε₀。

**实例——均匀带电球体的内部电势：**

半径为 R 的均匀带电球体，内部电荷密度 ρ₀ 为常数。球内电势满足泊松方程：
$$\nabla^2\varphi = -\frac{\rho_0}{\varepsilon_0}$$

由于球对称，φ 只依赖于 r。径向项简化为：
$$\frac{1}{r^2}\frac{d}{dr}\left(r^2\frac{d\varphi}{dr}\right) = -\frac{\rho_0}{\varepsilon_0}$$

积分两次得：$\varphi(r) = -\frac{\rho_0 r^2}{6\varepsilon_0} + \frac{C_1}{r} + C_2$。由自然边界条件（原点处有限 → C₁=0）和边界匹配得到完整解——这是静电学第二章的典型考题。

> 📌 **与其他"度"的联动**：∇²φ = -ρ/ε₀ 是**泊松方程**。φ 的拉普拉斯（即散度与梯度的复合）等于负电荷密度/ε₀——这本质上是 ∇·E = ρ/ε₀ 和 E = -∇φ 两道公式的合并。**拉普拉斯算子是连接电势和电荷的桥梁**——球坐标下的分离变量法（勒让德展开）是求解这道桥的最重要工具。

### 9.6 球坐标总结卡片

| 问题 | 回答 |
|------|------|
| 什么时候用球坐标？ | 问题有**球对称性**（点电荷、带电球体、球面边界、偶极辐射） |
| 最难适应的是什么？ | 三个单位矢量都随位置变化；θ 范围是 [0, π] 而非 [0, 2π] |

| 梯度 φ 分量分母是什么？ | **r·sinθ**——纬度线半径 × 角度微元 |
| ∇² 的解长什么样？ | $\sum (A_n r^n + B_n/r^{n+1})P_n(\cos\theta)$——勒让德多项式展开 |

## 十、三种坐标系对比与选择指南

### 10.1 一图看懂三种坐标系

| 特征 | 直角坐标 (x,y,z) | 柱坐标 (ρ,φ,z) | 球坐标 (r,θ,φ) |
|------|------------------|----------------|----------------|
| 基本微元 | dx, dy, dz | dρ, ρdφ, dz | dr, rdθ, r sinθ dφ |
| 体积元 dV | dxdydz | ρ dρ dφ dz | r² sinθ dr dθ dφ |
| 对称性匹配 | 平移对称 | 绕轴旋转对称 + z 平移 | 球对称/绕轴旋转对称 |
| 单位矢量是否固定 | ✅ 固定 | ❌ ρ̂, φ̂ 变化 | ❌ 全部变化 |
| 散度面积因子 | 无（各面相等） | ρ（侧面积 ∝ ρ） | r²（球面积 ∝ r²） |

### 10.2 选择决策树

```
问题长什么样？
├── 有长方体/平面边界？
│   └── → 直角坐标（镜像法、平行板电容器）
├── 有无限长圆柱/直导线/螺线管？
│   └── → 柱坐标（安培环路定理、高斯定理中的柱对称）
└── 有点电荷/球体/球面/原点辐射？
    └── → 球坐标（库仑定律、偶极子、偶极辐射、分离变量法）
```

### 10.3 最常犯的三个错误

| 错误 | 正确做法 |
|------|----------|
| 柱坐标散度写成 ∂F_ρ/∂ρ | 必须写 **1/ρ · ∂(ρF_ρ)/∂ρ**——面积因子 ρ 不能被遗忘 |
| 球坐标梯度 θ 分量忘了除以 r | 必须加上 **1/r**——θ 增加 dθ 对应的实际距离是 rdθ |
| 球坐标中把 θ 当成 0→2π | θ 的范围是 **[0, π]**——从北极到南极只需半圈 |

### 10.4 解题的通用流程

1. **看清对称性** → 选对坐标系（这是最关键的一步）
2. **写出通解形式**（拉普拉斯方程在对应坐标系下的级数解）
3. **写边界条件**（这是唯一性定理要求的——没有边界条件，解不唯一）
4. **利用边界条件和自然边界条件确定系数**（r→∞ 处不能发散；原点处不能有奇点等等）
5. **代回得到完整场分布**

> 📝 **黄金法则**：选对了坐标系，问题就解决了一半。回到电动力学第二章的分离变量法——**球面边界用球坐标的勒让德展开，柱面边界用柱坐标的贝塞尔展开，平面边界用直角坐标的傅里叶展开**——万变不离其宗。

### Q1
电场 $\vec{E} = (2x, 0, 0)$ 在点 (3, 0, 0) 处的散度是多少？这代表什么？

<details>
<summary>点击查看答案</summary>

散度 = ∂(2x)/∂x + 0 + 0 = **2**（常数，每点都是 2）。这意味着空间中每一点都有电荷密度 ρ = 2ε₀——这是一个均匀带电空间的电场。

</details>

### Q2
电势 $\varphi(x) = 5x^2$，求电场 $\vec{E}$。

<details>
<summary>点击查看答案</summary>

$\vec{E} = -\nabla\varphi = -\frac{\partial}{\partial x}(5x^2)\hat{x} = -10x\,\hat{x}$。在 x=2 处，电场为 -20 V/m，指向 x 负方向。

</details>

### Q3
磁场 $\vec{B} = (0, 0, B_0\cos(kx - \omega t))$（沿 z 方向的平面电磁波），求 ∇·B 和 ∇×E（提示：用 ∇×E = -∂B/∂t）。

<details>
<summary>点击查看答案</summary>

∇·B = ∂B_z/∂z = **0**（B_z 与 z 无关）——满足磁场高斯定理。

由 ∇×E = -∂B/∂t = -(0, 0, ωB₀sin(kx-ωt))，可以反推出 E 沿 y 方向振荡——这是典型的平面电磁波解。

</details>

### Q4
判断：$\vec{F} = (y, -x, 0)$ 是有旋场吗？如果是，旋度方向是什么？

<details>
<summary>点击查看答案</summary>

(∇×F)_z = ∂(-x)/∂x - ∂(y)/∂y = -1 - 1 = **-2**。

旋度 = (0, 0, -2)，沿 z 轴负方向（顺时针旋转）。这个场有旋且 ∇·F = 0——是一个纯旋转的无源场。

</details>

### Q5
在静电场中，为什么 ∇×E = 0 意味着可以定义电势？

<details>

# 第十二章 贝塞尔函数

> **为什么学这个？** 柱坐标下拉普拉斯方程 ∇²φ = 0 分离变量后，径向部分天然就是贝塞尔方程。电磁波在圆柱波导中的传播、圆柱谐振腔的谐振模式、圆形膜的振动——全部归结为贝塞尔函数。

## 12.1 贝塞尔方程从哪里来？

从柱坐标拉普拉斯方程 ∇²φ = 0 出发，设 φ(ρ, φ, z) = R(ρ) · Φ(φ) · Z(z)。

分离变量后，径向部分 R(ρ) 满足：

$$\rho^2\frac{d^2R}{d\rho^2} + \rho\frac{dR}{d\rho} + (k^2\rho^2 - n^2)R = 0$$

其中 $n$ 是角向分离变量引入的整数（$n = 0, 1, 2, \ldots$，来源于 Φ(φ) = e^{±inφ} 的单值性要求）。

这就是 **n 阶贝塞尔方程**。令 $x = k\rho$，变成标准形式：

$$x^2\frac{d^2y}{dx^2} + x\frac{dy}{dx} + (x^2 - n^2)y = 0$$

> 🔑 **核心理解**：贝塞尔方程 = "径向振动方程"。它描述的是柱对称下，物理量沿径向如何"波动"——不是简谐波（那是 sin/cos），而是有衰减振荡特性的特殊函数。

## 12.2 第一类贝塞尔函数 Jₙ(x)

$$J_n(x) = \sum_{m=0}^{\infty}\frac{(-1)^m}{m!(n+m)!}\left(\frac{x}{2}\right)^{n+2m}$$

**前几个常用的：**

| n | Jₙ(x) 近似（小 x 行为） | 第一个零点位置 |
|---|--------------------------|----------------|
| 0 | $J_0(x) \approx 1 - \frac{x^2}{4}$ | $x \approx 2.4048$ |
| 1 | $J_1(x) \approx \frac{x}{2}$ | $x \approx 3.8317$ |
| 2 | $J_2(x) \approx \frac{x^2}{8}$ | $x \approx 5.1356$ |

**关键性质：**
- **原点行为：** $J_0(0) = 1$，$J_n(0) = 0$（$n \ge 1$）
- **振荡衰减：** 当 $x \to \infty$，$J_n(x) \approx \sqrt{\frac{2}{\pi x}}\cos\left(x - \frac{n\pi}{2} - \frac{\pi}{4}\right)$——像 $\frac{1}{\sqrt{x}}$ 衰减的余弦波
- **递推关系：** $\frac{d}{dx}[x^n J_n(x)] = x^n J_{n-1}(x)$

**在电动力学中的出场：**
- 圆柱波导 TM 模的径向电场：$E_z \propto J_n(k_c\rho)$（由导体边界条件 $E_z(a) = 0$ 定出截止波数 $k_c = x_{nm}/a$）
- 圆柱波导 TE 模：$H_z \propto J_n(k_c\rho)$，边界条件为 $J_n'(k_c a) = 0$

## 12.3 第二类贝塞尔函数 Nₙ(x)（诺伊曼函数）

当圆柱区域**包含 ρ = 0** 时，Nₙ(x) 在原点发散——物理上就必须舍弃 Nₙ。

$$N_n(x) = \frac{J_n(x)\cos n\pi - J_{-n}(x)}{\sin n\pi}$$

> 📌 **考试技巧**：题目涉及圆柱内部（含轴线 ρ = 0）的问题时，解只能是 $J_n$，不能有 $N_n$。如果问题区域是环形区域（不含 ρ = 0），两项都保留。

## 12.4 贝塞尔函数的正交性

$$\int_0^a J_n\left(\frac{x_{nm}}{a}\rho\right)J_n\left(\frac{x_{nk}}{a}\rho\right)\rho d\rho = \frac{a^2}{2}[J_{n+1}(x_{nm})]^2\delta_{mk}$$

> 📌 这个正交性用于确定分离变量法中的展开系数——和傅里叶级数中 sin/cos 的正交性作用完全一样，只是权重多了个 ρ。

## 12.5 考试闪电应用——圆柱波导 TM 模

**题目**：半径为 a 的圆柱波导，求 TM 模的截止波长。

**解题模板：**
1. $E_z(\rho,\varphi) = E_0 J_n(k_c\rho)e^{in\varphi}$（柱内含轴线 → 只用 Jₙ）
2. 导体边界条件：$E_z(a,\varphi) = 0 \Rightarrow J_n(k_c a) = 0$
3. $k_c a$ = $J_n$ 的第 m 个零点 $x_{nm}$
4. 截止波长 $\lambda_c = \frac{2\pi}{k_c} = \frac{2\pi a}{x_{nm}}$

| 模式 | 零点 $x_{nm}$ | 截止波长 |
|------|--------------|----------|
| TM₀₁ | 2.4048 | 2.613a |
| TM₁₁ | 3.8317 | 1.640a |
| TM₀₂ | 5.5201 | 1.138a |

> 📌 **TM₀₁ 是圆柱波导的最低阶 TM 模**——因为它对应 J₀ 的第一个零点，截止波长最长。

# 第十三章 勒让德多项式

> **为什么学这个？** 球坐标下拉普拉斯方程 ∇²φ = 0 在轴对称情况下，角向部分的解就是勒让德多项式。导体球、介质球、球面边界问题——全部需要 $P_n(\cos\theta)$。

## 13.1 勒让德方程从哪里来？

从球坐标拉普拉斯方程轴对称情况（∂/∂φ = 0）出发，设 φ(r, θ) = R(r) · Θ(θ)。

令 $x = \cos\theta$，Θ(θ) = y(x)，则角向部分满足：

$$\frac{d}{dx}\left[(1-x^2)\frac{dy}{dx}\right] + n(n+1)y = 0$$

这就是**勒让德方程**。$n$ 是分离变量引入的**非负整数**（$n = 0, 1, 2, \ldots$）。

> 🔑 **为什么 n 必须是整数？** 只有 n 为整数时，解在 $x = ±1$（即 θ = 0, π——南北极）处才有限。这是物理上自然边界条件的要求。

## 13.2 勒让德多项式 Pₙ(x)

$$P_n(x) = \frac{1}{2^n n!}\frac{d^n}{dx^n}(x^2-1)^n \quad \text{（罗德里格公式）}$$

**前 5 个：**

| n | Pₙ(x) | Pₙ(cosθ) | 物理场景 |
|---|-------|-----------|----------|
| 0 | $1$ | $1$ | 球对称（点电荷电势 ∝ 1/r） |
| 1 | $x$ | $\cos\theta$ | 均匀外电场中导体球的感应偶极 |
| 2 | $\frac{3x^2-1}{2}$ | $\frac{3\cos^2\theta-1}{2}$ | 电四极场的角分布 |
| 3 | $\frac{5x^3-3x}{2}$ | $\frac{5\cos^3\theta-3\cos\theta}{2}$ | 更高阶多极展开 |
| 4 | $\frac{35x^4-30x^2+3}{8}$ | — | 更高阶球谐展开 |

**关键性质：**
- $P_n(1) = 1$，$P_n(-1) = (-1)^n$
- $P_n$ 是 n 次多项式，有 n 个零点，对称分布在 $[-1, 1]$ 上
- **递推关系：** $(n+1)P_{n+1}(x) = (2n+1)xP_n(x) - nP_{n-1}(x)$

> 📌 这是考试写通解时必须用到的——知道前几个 Pₙ 的具体形式，代入边界条件展开就能定系数。

## 13.3 正交性

$$\int_{-1}^{1} P_n(x)P_m(x)dx = \frac{2}{2n+1}\delta_{nm}$$

等价于：
$$\int_0^\pi P_n(\cos\theta)P_m(\cos\theta)\sin\theta d\theta = \frac{2}{2n+1}\delta_{nm}$$

> 📌 **用法**：已知边界上 φ(a, θ) = f(θ)，把 f(θ) 用勒让德多项式展开：
> $$f(\theta) = \sum_{n=0}^{\infty} C_n P_n(\cos\theta)$$
> 其中 $C_n = \frac{2n+1}{2}\int_0^\pi f(\theta)P_n(\cos\theta)\sin\theta d\theta$。

## 13.4 轴对称拉普拉斯方程的通解——完整推导

> 🔑 **这是电动力学第二章分离变量法的核心**。考试中遇到导体球、介质球、球面边界问题，最终都要回归到这个通解。下面一步步展示这个通解是怎么来的。

### 第 1 步：写出球坐标拉普拉斯方程（轴对称情况）

球坐标下的拉普拉斯算子（轴对称，∂/∂φ = 0）：

$$\nabla^2\varphi = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\frac{\partial\varphi}{\partial r}\right) + \frac{1}{r^2\sin\theta}\frac{\partial}{\partial\theta}\left(\sin\theta\frac{\partial\varphi}{\partial\theta}\right) = 0$$

### 第 2 步：分离变量

设 $\varphi(r,\theta) = R(r) \cdot \Theta(\theta)$，代入方程：

$$\frac{\Theta}{r^2}\frac{d}{dr}\left(r^2\frac{dR}{dr}\right) + \frac{R}{r^2\sin\theta}\frac{d}{d\theta}\left(\sin\theta\frac{d\Theta}{d\theta}\right) = 0$$

两边同乘 $r^2/(R\Theta)$，把 r 和 θ 的项分开：

$$\frac{1}{R}\frac{d}{dr}\left(r^2\frac{dR}{dr}\right) = -\frac{1}{\Theta\sin\theta}\frac{d}{d\theta}\left(\sin\theta\frac{d\Theta}{d\theta}\right)$$

> 📌 **关键一步**：左边只依赖于 r，右边只依赖于 θ。两个变量独立的函数相等——它们的共同值必须是一个**常数**。设这个常数为 $l(l+1)$（写成这种形式是为了后续求解方便，l 暂时是一个待定参数）：

$$\frac{1}{R}\frac{d}{dr}\left(r^2\frac{dR}{dr}\right) = l(l+1)$$

$$-\frac{1}{\Theta\sin\theta}\frac{d}{d\theta}\left(\sin\theta\frac{d\Theta}{d\theta}\right) = l(l+1)$$

> 📌 **为什么常数写成 $l(l+1)$？** 这样写纯粹是为了让后续解出来的方程形式简洁。如果写成 $k$，后面角向方程会变成 $\sin\theta \cdot \Theta'' + \cos\theta \cdot \Theta' + k\sin\theta \cdot \Theta = 0$——很难看。用 $l(l+1)$ 这个参数化，恰好让勒让德方程以最标准的形式出现。

### 第 3 步：解径向方程

径向方程展开：

$$\frac{d}{dr}\left(r^2\frac{dR}{dr}\right) = l(l+1)R$$

$$r^2 R'' + 2r R' - l(l+1)R = 0$$

这是一个**欧拉型方程**——求解方法是设 $R(r) = r^\alpha$（幂律试探解）：

代入：$r^2 \cdot \alpha(\alpha-1)r^{\alpha-2} + 2r \cdot \alpha r^{\alpha-1} - l(l+1)r^\alpha = 0$

整理：$[\alpha(\alpha-1) + 2\alpha - l(l+1)]r^\alpha = 0$

即：$\alpha^2 + \alpha - l(l+1) = 0$

解这个关于 α 的一元二次方程：

$$\alpha = \frac{-1 \pm \sqrt{1 + 4l(l+1)}}{2} = \frac{-1 \pm \sqrt{4l^2 + 4l + 1}}{2} = \frac{-1 \pm (2l+1)}{2}$$

两个根：
- $\alpha_1 = \frac{-1 + (2l+1)}{2} = l$
- $\alpha_2 = \frac{-1 - (2l+1)}{2} = -(l+1)$

因此径向方程的通解为：

$$R_l(r) = A_l r^l + B_l r^{-(l+1)}$$

> 📌 **两项的物理含义**：
> - $r^l$：当 $r \to 0$ 时趋于零（l ≥ 1）或有限（l = 0），保证原点处物理量不会发散——用于**球内区域**
> - $r^{-(l+1)}$：当 $r \to \infty$ 时趋于零，保证无穷远处电势为零——用于**球外区域**
> - 如果问题涉及球壳，内外区域分别保留不同项，在球面上匹配

### 第 4 步：解角向方程 → 勒让德方程

角向方程（令 $x = \cos\theta$，则 $\sin^2\theta = 1 - x^2$）：

$$-\frac{1}{\Theta\sin\theta}\frac{d}{d\theta}\left(\sin\theta\frac{d\Theta}{d\theta}\right) = l(l+1)$$

用 $x = \cos\theta$ 换元：$\frac{d}{d\theta} = \frac{dx}{d\theta}\frac{d}{dx} = -\sin\theta\frac{d}{dx}$

代入后得到（过程见 13.1 节）：

$$\frac{d}{dx}\left[(1-x^2)\frac{dy}{dx}\right] + l(l+1)y = 0$$

这就是**勒让德方程**。

### 第 5 步：自然边界条件定出 l 必须是整数

勒让德方程在 $x \in [-1, 1]$（即 $\theta \in [0, \pi]$）上的解，只有当 $l$ 是**非负整数**时，解在 $x = \pm 1$（南北极）处才保持有限。

> 🔑 **为什么？** 如果 l 不是整数，勒让德方程的解（称为"勒让德函数"）在 $x = \pm 1$ 处会发散到无穷大。物理上，球的南北极处的电势必须是有限值——这个物理要求把 l 限制为整数 0, 1, 2, …。

当 $l = n$（$n = 0, 1, 2, \ldots$）时，有界的解就是**勒让德多项式** $P_n(x) = P_n(\cos\theta)$。

因此角向解为：

$$\Theta_n(\theta) = P_n(\cos\theta)$$

### 第 6 步：写出完整的通解

将径向解和角向解组合起来。由于拉普拉斯方程是线性的，**任意 n 的线性组合也是解**（叠加原理）：

$$\varphi(r,\theta) = \sum_{n=0}^{\infty} R_n(r) \cdot \Theta_n(\theta) = \sum_{n=0}^{\infty}\left(A_n r^n + \frac{B_n}{r^{n+1}}\right)P_n(\cos\theta)$$

**这就是球坐标下拉普拉斯方程在轴对称情况下的通解！**

### 第 7 步：系数 A_n 和 B_n 怎么定？

系数由**边界条件**唯一确定（唯一性定理）。两类典型场景：

| 场景 | 约束条件 | 保留的项 |
|------|----------|----------|
| **球内问题**（r ≤ a） | 原点处 φ 有限 → 不能有 r^{-(n+1)} | 仅保留 $A_n r^n$ |
| **球外问题**（r ≥ a） | r → ∞ 时 φ 有限 → 不能有 r^n（除 n=0 外） | 保留 $B_n r^{-(n+1)}$（加上可能的外加均匀场 $A_1 r$） |
| **球壳问题**（a ≤ r ≤ b） | 两个边界上给定条件 | 两项都保留，在 r=a 和 r=b 处分别匹配 |

具体操作：
1. 把边界上的已知电势分布用勒让德多项式展开：$f(\theta) = \sum C_n P_n(\cos\theta)$
2. 利用正交性 $\int_{-1}^1 P_n P_m dx = \frac{2}{2n+1}\delta_{nm}$ 提取每个 n 的系数
3. 比较同阶 $P_n$ 的系数，列出关于 $A_n$、$B_n$ 的方程组
4. 解出 $A_n$、$B_n$

### 整个推导的直观总结

```
∇²φ = 0（球坐标，轴对称）
       │
       │ 分离变量 φ(r,θ) = R(r)Θ(θ)
       │
       ├─ 径向：r²R'' + 2rR' - n(n+1)R = 0
       │      ↓ 设 R = r^α，解二次方程
       │      R(r) = A_n r^n + B_n r^{-(n+1)}
       │
       └─ 角向：(1-x²)y'' - 2xy' + n(n+1)y = 0（x = cosθ）
              ↓ 自然边界条件 → n 必须是整数
              Θ(θ) = P_n(cosθ)（勒让德多项式）
       │
       ↓ 叠加原理（线性方程的通解 = 所有特解的线性组合）
       │
φ(r,θ) = Σ [A_n r^n + B_n r^{-(n+1)}] P_n(cosθ)
       │
       ↓ 边界条件 → 正交性 → 定出 A_n, B_n
       │
完整的电势分布 ✅
```

> 📌 **为什么考试必考这个？** 因为从麦克斯韦方程组 → 静电场 → 泊松/拉普拉斯方程 → 分离变量法，这是电动力学第一章到第二章的逻辑主线。而球坐标分离变量法最终的解就是这个通解。掌握了这 7 步推导，考试中所有球对称/轴对称的边值问题都只是在这个通解上套边界条件而已。

## 13.5 考试闪电应用——均匀外电场中的导体球

**题目**：半径为 R 的接地导体球置于均匀外电场 $E_0\hat{z}$ 中，求电势分布。

**解题模板：**
1. 外电场电势：$\varphi_0 = -E_0 z = -E_0 r\cos\theta = -E_0 r P_1(\cos\theta)$
2. 球外通解：$\varphi(r,\theta) = -E_0 r P_1 + \sum B_n r^{-(n+1)} P_n$
3. 接地导体边界 $\varphi(R,\theta) = 0$，代入：
   $$-E_0 R P_1 + \frac{B_0}{R} P_0 + \frac{B_1}{R^2} P_1 + \frac{B_2}{R^3} P_2 + \cdots = 0$$
4. 由 Pₙ 的正交性，每个 n 独立满足：$B_0 = 0$，$B_1 = E_0 R^3$，$B_n = 0$（n ≥ 2）
5. 最终：$\varphi(r,\theta) = -E_0 r\cos\theta + \frac{E_0 R^3}{r^2}\cos\theta$

第二项就是导体球感应的**电偶极场**——$\propto 1/r^2 \cdot \cos\theta$，偶极矩 $p = 4\pi\varepsilon_0 E_0 R^3$。

# 第十四章 傅里叶变换

> **为什么学这个？** 电磁波不是只有单一频率——实际信号是各种频率的叠加。傅里叶变换告诉你任何一个波形都可以拆成 sin/cos 的叠加，这就是频谱分析的数学基础。

## 14.1 从傅里叶级数到傅里叶变换

### 傅里叶级数（周期函数）

周期为 T 的函数 f(t) 可以展开为：

$$f(t) = \frac{a_0}{2} + \sum_{n=1}^{\infty}[a_n\cos(n\omega_0 t) + b_n\sin(n\omega_0 t)]$$

其中 $\omega_0 = 2\pi/T$ 是基频：

$$a_n = \frac{2}{T}\int_{-T/2}^{T/2} f(t)\cos(n\omega_0 t)dt$$
$$b_n = \frac{2}{T}\int_{-T/2}^{T/2} f(t)\sin(n\omega_0 t)dt$$

**复数形式（更简洁）：**
$$f(t) = \sum_{n=-\infty}^{\infty} C_n e^{in\omega_0 t}, \quad C_n = \frac{1}{T}\int_{-T/2}^{T/2} f(t)e^{-in\omega_0 t}dt$$

### 傅里叶变换（非周期函数）

让周期 $T \to \infty$，离散频谱变成连续频谱：

$$F(\omega) = \int_{-\infty}^{\infty} f(t)e^{-i\omega t}dt \quad \text{（正变换）}$$
$$f(t) = \frac{1}{2\pi}\int_{-\infty}^{\infty} F(\omega)e^{i\omega t}d\omega \quad \text{（逆变换）}$$

> 🔑 **直观理解**：$F(\omega)$ 告诉你频率为 ω 的成分有多强。$F(\omega)$ 越大的地方，原信号中该频率的分量越多。

**三维傅里叶变换（电磁波必备）：**
$$F(\vec{k}) = \int f(\vec{r})e^{-i\vec{k}\cdot\vec{r}}d^3r$$
$$f(\vec{r}) = \frac{1}{(2\pi)^3}\int F(\vec{k})e^{i\vec{k}\cdot\vec{r}}d^3k$$

> 📌 在电磁波中，$\vec{k}$ 是**波矢**——方向代表传播方向，大小 $|\vec{k}| = 2\pi/\lambda$ 代表空间频率。

## 14.2 关键性质速查表

| 性质 | 时域 $f(t)$ | 频域 $F(\omega)$ | 物理含义 |
|------|-------------|------------------|----------|
| 线性 | $af + bg$ | $aF + bG$ | 叠加原理 |
| 时移 | $f(t-t_0)$ | $e^{-i\omega t_0}F(\omega)$ | 延迟→相位旋转 |
| 频移 | $e^{i\omega_0 t}f(t)$ | $F(\omega-\omega_0)$ | 调制→频谱搬移 |
| 微分 | $f'(t)$ | $i\omega F(\omega)$ | **把微分变成乘法！** |
| 积分 | $\int f(t)dt$ | $\frac{F(\omega)}{i\omega}$ | **把积分变成除法！** |
| 卷积 | $(f*g)(t)$ | $F(\omega)G(\omega)$ | 时域卷积 = 频域乘积 |

> 🔥 **最重要的性质：$f'(t) \leftrightarrow i\omega F(\omega)$**。这意味着微分方程可以通过傅里叶变换变成代数方程！这就是为什么在电磁波理论中，麦克斯韦方程组的时谐形式（用 $e^{-i\omega t}$ 假设）能把 $\partial/\partial t$ 替换为 $-i\omega$。

## 14.3 常见傅里叶变换对

| f(t) | F(ω) | 在电磁学中的用途 |
|------|------|-----------------|
| $\delta(t)$ | $1$ | 点源、冲激响应 |
| $1$ | $2\pi\delta(\omega)$ | 直流分量 |
| $e^{-\alpha|t|}$ | $\frac{2\alpha}{\alpha^2+\omega^2}$ | 阻尼振荡包络 |
| 矩形脉冲（宽 τ） | $\tau\frac{\sin(\omega\tau/2)}{\omega\tau/2}$ | 有限时长信号 |
| 高斯脉冲 $e^{-t^2/2\sigma^2}$ | $\sqrt{2\pi}\sigma e^{-\sigma^2\omega^2/2}$ | 高斯波包 |

## 14.4 在电动力学中的三大应用

**① 平面波展开——从时域到频域**

麦克斯韦方程组的时谐形式是全书核心。为什么可以假设 $\vec{E} \propto e^{-i\omega t}$？因为任意时间依赖都可以傅里叶变换后叠加：

$$\vec{E}(\vec{r},t) = \frac{1}{2\pi}\int \vec{E}(\vec{r},\omega)e^{-i\omega t}d\omega$$

对每个频率分量，$\partial/\partial t \to -i\omega$，麦克斯韦方程组变成：
$$\nabla\times\vec{E} = i\omega\vec{B}, \quad \nabla\times\vec{B} = \mu_0\vec{J} - i\omega\mu_0\varepsilon_0\vec{E}$$

**② 波包的角谱展开**

有限宽度的波束可以写成不同 $\vec{k}$ 方向平面波的叠加（空间傅里叶变换）：
$$\vec{E}(\vec{r}) = \frac{1}{(2\pi)^3}\int \vec{E}(\vec{k})e^{i\vec{k}\cdot\vec{r}}d^3k$$

**③ 延迟势的频谱形式**

推迟势在频域中特别简洁——延迟变成相位因子 $e^{ikR}$：
$$\varphi(\vec{r},\omega) = \frac{1}{4\pi\varepsilon_0}\int \rho(\vec{r}',\omega)\frac{e^{ikR}}{R}dV'$$

> 📌 **记忆技巧**：傅里叶变换把"推迟"（时间上的平移）变成了"相位"（频率域的旋转）——时域的 $t - R/c$ 变成频域的 $e^{ikR}$。

## 14.5 考试闪电应用——矩形脉冲的频谱

**题目**：宽度为 τ 的矩形脉冲 $f(t) = \begin{cases}1, & |t| < \tau/2 \\ 0, & \text{其他}\end{cases}$，求频谱。

$$F(\omega) = \int_{-\tau/2}^{\tau/2} 1 \cdot e^{-i\omega t}dt = \frac{e^{-i\omega\tau/2} - e^{i\omega\tau/2}}{-i\omega} = \tau\frac{\sin(\omega\tau/2)}{\omega\tau/2} = \tau\text{sinc}\left(\frac{\omega\tau}{2}\right)$$

- 主瓣宽度 $\Delta\omega \approx 2\pi/\tau$——脉冲越短，频谱越宽
- 这就是**不确定性原理**的雏形：$\Delta t \cdot \Delta\omega \sim 2\pi$

| 项目 | 内容 |
|------|------|
| 核心思想 | 任何信号 = 不同频率 sin/cos 的叠加 |
| 正变换 | $F(\omega) = \int f(t)e^{-i\omega t}dt$ |
| 逆变换 | $f(t) = \frac{1}{2\pi}\int F(\omega)e^{i\omega t}d\omega$ |

# 第十五章 拉普拉斯变换

> **为什么学这个？** 拉普拉斯变换是处理**含初始条件**的微分方程的利器。在电路瞬态分析、传输线充放电、电磁场的时域问题中，拉普拉斯变换比傅里叶变换更方便——因为它天然包含初始条件。

## 15.1 从傅里叶到拉普拉斯

傅里叶变换要求 $\int_{-\infty}^{\infty}|f(t)|dt < \infty$——很多信号不满足（如 $e^{at}$ 发散的）。

**修复方案**：给 $f(t)$ 乘上衰减因子 $e^{-\sigma t}$，再做傅里叶变换：

$$\mathcal{L}[f(t)] = F(s) = \int_0^{\infty} f(t)e^{-st}dt$$

其中 $s = \sigma + i\omega$ 是**复频率**。

> 🔑 **核心理解**：拉普拉斯变换 = 给信号"强加衰减"后的傅里叶变换。$s$ 的实部 σ 控制衰减，虚部 ω 是振荡频率。这就像用一个"阻尼包络" $e^{-\sigma t}$ 把发散信号压制住。

**与傅里叶变换的关系**：$F(\omega) = \mathcal{L}[f(t)]|_{s = i\omega}$——令 σ = 0 即恢复纯傅里叶变换。

## 15.2 关键性质速查表

| 性质 | 时域 $f(t)$ | s 域 $F(s)$ | 物理含义 |
|------|-------------|-------------|----------|
| 线性 | $af + bg$ | $aF + bG$ | 叠加原理 |
| 时移 | $f(t-t_0)u(t-t_0)$ | $e^{-st_0}F(s)$ | 延迟→指数衰减因子 |
| **微分（重点！）** | **$f'(t)$** | **$sF(s) - f(0)$** | **自动带上初始条件！** |
| 二阶微分 | $f''(t)$ | $s^2F(s) - sf(0) - f'(0)$ | 二阶微分方程必备 |
| 积分 | $\int_0^t f(\tau)d\tau$ | $\frac{F(s)}{s}$ | 积分→除法 |
| 卷积 | $(f*g)(t)$ | $F(s)G(s)$ | 时域卷积 = s 域乘积 |
| 终值定理 | $\lim_{t\to\infty}f(t)$ | $\lim_{s\to 0}sF(s)$ | 稳态值 |
| 初值定理 | $\lim_{t\to 0^+}f(t)$ | $\lim_{s\to\infty}sF(s)$ | 初始跳变 |

> 🔥 **最重要的性质：$f'(t) \leftrightarrow sF(s) - f(0)$。** 初始条件自动进入代数方程——不需要像傅里叶变换那样手动处理积分常数！

## 15.3 常见拉普拉斯变换对

| f(t) | F(s) | 在电磁学中的用途 |
|------|------|-----------------|
| $\delta(t)$ | $1$ | 点源、冲激响应（格林函数法） |
| $1$（阶跃函数 $u(t)$） | $\frac{1}{s}$ | 直流接通 |
| $t$ | $\frac{1}{s^2}$ | 斜坡信号 |
| $e^{-at}$ | $\frac{1}{s+a}$ | **RC/RL 充放电衰减** |
| $\sin(\omega t)$ | $\frac{\omega}{s^2+\omega^2}$ | 正弦驱动 |
| $\cos(\omega t)$ | $\frac{s}{s^2+\omega^2}$ | 余弦驱动 |
| $te^{-at}$ | $\frac{1}{(s+a)^2}$ | 临界阻尼响应 |

## 15.4 在电动力学中的典型应用

**① RC 电路充电（含初始条件）**

电路方程：$RC\frac{dv}{dt} + v = V_0 u(t)$，初始条件 $v(0) = 0$。

拉普拉斯变换：$RC[sV(s) - 0] + V(s) = \frac{V_0}{s}$

代数学求解：$V(s) = \frac{V_0}{s(RCs + 1)} = \frac{V_0}{s} - \frac{V_0}{s + 1/RC}$

逆变换：$v(t) = V_0(1 - e^{-t/RC})$

> 📌 **三步走**：微分方程 $\xrightarrow{\text{拉普拉斯}}$ 代数方程 $\xrightarrow{\text{求解}}$ s 域解 $\xrightarrow{\text{逆变换}}$ 时域解——比直接解微分方程快、且自动处理初始条件。

**② 传输线的电报方程**

$$\frac{\partial V}{\partial z} = -L\frac{\partial I}{\partial t} - RI$$
$$\frac{\partial I}{\partial z} = -C\frac{\partial V}{\partial t} - GV$$

对 t 做拉普拉斯变换（初始条件为零）：
$$\frac{dV(z,s)}{dz} = -(Ls + R)I(z,s)$$
$$\frac{dI(z,s)}{dz} = -(Cs + G)V(z,s)$$

这样偏微分方程就变成关于 z 的常微分方程——可以像处理稳恒电路一样处理传输线。

**③ 色散介质的介电函数**

在拉普拉斯域（或傅里叶域），介质的响应写成：
$$\vec{D}(s) = \varepsilon(s)\vec{E}(s)$$

$\varepsilon(s)$ 是**复介电函数**——它是时域卷积核 $\varepsilon(t)$ 的拉普拉斯变换。频率依赖 = 时间上的记忆效应。

## 15.5 考试闪电应用——RLC 串联电路

**题目**：RLC 串联电路，$t=0$ 时接通直流电压 $V_0$，初始电流 $i(0)=0$，电容初始电压 $v_C(0)=0$。求电流 $i(t)$。

**解题模板：**
1. 电路方程：$L\frac{di}{dt} + Ri + \frac{1}{C}\int_0^t i(\tau)d\tau = V_0$
2. 拉普拉斯变换：$LsI(s) + RI(s) + \frac{I(s)}{Cs} = \frac{V_0}{s}$
3. 求解：$I(s) = \frac{V_0}{L}\frac{1}{s^2 + \frac{R}{L}s + \frac{1}{LC}}$
4. 令 $\alpha = R/(2L), \omega_0 = 1/\sqrt{LC}$：
   - 欠阻尼 $(\alpha < \omega_0)$：$i(t) = \frac{V_0}{\omega_d L}e^{-\alpha t}\sin(\omega_d t)$，$\omega_d = \sqrt{\omega_0^2 - \alpha^2}$
   - 临界阻尼 $(\alpha = \omega_0)$：$i(t) = \frac{V_0}{L}te^{-\alpha t}$
   - 过阻尼 $(\alpha > \omega_0)$：含双指数衰减

> 📌 拉普拉斯变换一步到位给出所有阻尼情况的解——不需要像经典法那样分情况讨论特解形式。
