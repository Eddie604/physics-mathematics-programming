## 1. 自查项一：2×2 矩阵的特征值与特征向量

### 1.1 为什么要掌握

在仿真中，**特征值 = 系统的固有频率/能级，特征向量 = 对应的振动模式/本征态**。

一个经典的仿真场景：耦合双摆在某个平衡位置附近的小振动分析，最终完全归结为求一个 2×2 矩阵的特征值和特征向量。

### 1.2 从零推导：2×2 一般情况

设矩阵：

```
A = [a  b]
    [c  d]
```

**特征值方程：** `det(A - λI) = 0`

```
det([a-λ  b  ]) = (a-λ)(d-λ) - bc = 0
    [c    d-λ])

展开：λ² - (a+d)λ + (ad - bc) = 0
```

定义：
- **迹 (Trace)：** `tr(A) = a + d`
- **行列式 (Determinant)：** `det(A) = ad - bc`

于是特征方程就是：

```
λ² - tr(A)·λ + det(A) = 0
```

二次方程求根公式直接给出两个特征值：

```
λ₁, λ₂ = [tr(A) ± √(tr(A)² - 4·det(A))] / 2
```

**判别式 `Δ = tr(A)² - 4·det(A)`** 决定了特征值的性质：

| Δ | 特征值类型 | 物理含义 |
|---|-----------|----------|
| Δ > 0 | 两个不同实数 | **过阻尼**，无振荡，指数衰减/增长 |
| Δ = 0 | 一个重根实数 | **临界阻尼**，最快回到平衡 |
| Δ < 0 | 一对共轭复数 | **欠阻尼**，有振荡频率 = √(-Δ)/2 |

### 1.3 求特征向量（λ 已知后）

解 `(A - λI)v = 0`：

```
[a-λ  b  ][x] = [0]
[c    d-λ][y]   [0]
```

这是一个齐次线性方程组，有非零解的充要条件是矩阵奇异（det = 0），这正是 λ 为特征值时的情形。

**实际操作技巧（2×2 专用）：**

因为两行线性相关，选第一行（或第二行，哪行非零用哪行）：

```
(a-λ)x + b·y = 0  →  令 x = b, 则 y = λ-a
```

因此特征向量为：

```
v = [b]
    [λ-a]
```

### 1.4 物理案例：耦合双摆的简正模分析

两个质量 m 相同、摆长 l 相同、用弹簧 k 耦合的单摆，在线性近似下的运动方程矩阵形式：

```
d²/dt² [x₁] = [-ω₀²-κ   κ   ] [x₁]
        [x₂]   [  κ   -ω₀²-κ] [x₂]

其中 ω₀² = g/l，κ = k/m
```

此矩阵的 `a = d = -ω₀²-κ`，`b = c = κ`。

```
tr(A) = -2(ω₀²+κ)
det(A) = (ω₀²+κ)² - κ² = ω₀²(ω₀²+2κ)

Δ = 4(ω₀²+κ)² - 4ω₀²(ω₀²+2κ) = 4κ² > 0  ✓
```

得特征值：

```
λ₁ = -(ω₀²+κ) + κ = -ω₀²      → 对应同相模式（弹簧不伸缩，频率=ω₀）
λ₂ = -(ω₀²+κ) - κ = -(ω₀²+2κ) → 对应反相模式（弹簧伸缩，频率更高）
```

因为 `d²x/dt² = λx`，固有频率 `ω = √(-λ)`，所以：

```
ω₁ = ω₀ (同相模式，慢)
ω₂ = √(ω₀²+2κ) (反相模式，快)
```

**仿真中的操作：** 当你的程序算出这两个频率并对比数值结果时，你就完成了小振动分析→数值验证的闭环。

## 2. 自查项二：Taylor 展开到二阶项的几何含义

### 2.1 Taylor 定理的完整形式

**一维 Taylor 展开（在 x₀ 处）：**

```
f(x) = f(x₀) + f'(x₀)(x-x₀) + f''(x₀)/2! (x-x₀)² + f'''(x₀)/3! (x-x₀)³ + ...
```

**保留到二阶：**

```
f(x₀ + h) ≈ f(x₀) + f'(x₀)·h + f''(x₀)/2 · h²
```

### 2.2 几何含义分层解读

| 阶数 | 数学 | 几何 | 物理类比 |
|------|------|------|----------|
| **0 阶** | f(x₀) | 一个点 | 初始位置 |
| **1 阶** | + f'(x₀)·h | 切线（直线，局部线性近似） | 匀速运动：x = x₀ + vt |
| **2 阶** | + f''(x₀)/2 · h² | 抛物线（曲率，局部二次近似） | 匀加速运动：x = x₀ + vt + ½at² |
| **3 阶** | + f'''(x₀)/6 · h³ | 扭曲（加速的加速度） | 加加速度（jerk） |

**核心直觉：**

- 0 阶告诉我们"函数在这里的值"
- 1 阶告诉我们"函数在这里的走向（斜率）"
- 2 阶告诉我们"斜率的变化率（弯曲程度）"
- 高过 2 阶的项告诉我们"弯曲本身怎么变"

### 2.3 为什么仿真中二阶 Taylor 如此重要

**理由一：物理定律大多是二阶微分方程。**

牛顿第二定律 F = m·d²x/dt²，本身就是一个"只到二阶"的陈述——力决定加速度，不需要三阶导。

**理由二：数值方法的误差分析完全依赖 Taylor 展开。**

以 Euler 法为例：

```
精确解： x(t+h) = x(t) + h·x'(t) + h²/2 · x''(ξ)  （Taylor 定理，余项形式）
Euler 法：x_{n+1} = x_n + h·f(t_n, x_n)

局部截断误差 = 精确解 - Euler 近似 = h²/2 · x''(ξ) = O(h²)
```

如果不理解 Taylor 展开的余项，你就无法理解为什么 Euler 法是一阶精度，RK4 是四阶精度。

**理由三：PDE 的有限差分格式都是从 Taylor 展开出发的。**

```
f(x+h) = f(x) + h·f'(x) + h²/2·f''(x) + h³/6·f'''(x) + O(h⁴)    —— ①
f(x-h) = f(x) - h·f'(x) + h²/2·f''(x) - h³/6·f'''(x) + O(h⁴)    —— ②
```

① - ② 得：

```
f'(x) = [f(x+h) - f(x-h)] / (2h) + O(h²)    ← 中心差分，二阶精度
```

① + ② 得：

```
f''(x) = [f(x+h) - 2f(x) + f(x-h)] / h² + O(h²)    ← 二阶导数中心差分
```

**你看到的每一个数值微分公式，都能从 Taylor 展开用 2 分钟推出来。**

### 2.4 手工验证：用 Taylor 展开分析误差

以 `f(x) = sin(x)` 在 `x₀ = 0` 处展开到二阶（即 `sin(x) ≈ x`）：

```
sin(h) ≈ h              （一阶近似）
sin(h) ≈ h - h³/6        （三阶近似，二阶项为 0 因为 sin''(0) = 0）
```

当 `h = 0.1` 时：
- 一阶近似：sin(0.1) ≈ 0.1，真值 ≈ 0.09983，误差 ≈ 1.7×10⁻⁴
- 三阶近似：sin(0.1) ≈ 0.1 - 0.001/6 = 0.099833，误差 ≈ 8.3×10⁻⁹

**Taylor 展开让你能预判：步长缩小 10 倍，误差缩小多少？**
- 一阶方法：h→h/10 → 误差→误差/10
- 四阶方法：h→h/10 → 误差→误差/10000

## 3. 自查项三：ODE 初值问题 vs 边值问题

### 3.1 本质区别

| 特性 | 初值问题 (IVP) | 边值问题 (BVP) |
|------|---------------|---------------|
| **条件位置** | 全部在起点给出 | 在两端给出 |
| **物理原型** | 抛体运动、轨道演化 | 稳态温度分布、梁的挠度 |
| **时间方向** | 单向（因果，时间箭头） | 双向（空间，无因果方向） |
| **数值方法** | 推进法：Runge-Kutta, Adams | 整体法：打靶法、有限差分、配置法 |
| **求解方式** | 从 t₀ 一步步往前走 | 需要同时解整个区间（通常是边值→线性方程组） |

### 3.2 同一个方程，不同条件 → 不同问题

以最简单的二阶 ODE 为例：

```
y'' + y = 0    （谐振子方程）
```

**初值问题：**

```
y(0) = 1, y'(0) = 0     → y(t) = cos(t)
```

所有条件都在 t=0 给出。程序从 t=0 开始，一步步往前推即可。

**边值问题：**

```
y(0) = 0, y(π/2) = 1    → y(t) = sin(t)
```

条件分居两端。你不能"从 t=0 推出去"因为不知道 y'(0) 是多少。

解法（打靶法思想）：猜一个 y'(0)，解 IVP，看 y(π/2) 是否等于 1。不等就调整，直到射中目标。

### 3.3 仿真中的判断准则

**问自己三个问题即可区分：**

1. **自变量是时间还是空间？** 时间是 IVP，稳态空间是 BVP
2. **问题有"因果方向"吗？** 已知"现在"，求"未来"→ IVP
3. **条件都在一端吗？** 是→ IVP；两端都有→ BVP

**典型例子速查：**

| 方程 | 条件 | IVP/BVP |
|------|------|---------|
| mẍ = -kx | x(0)=1, ẋ(0)=0 | **IVP** |
| d²u/dx² = -ρ(x)/ε₀ | u(0)=V₀, u(L)=0 | **BVP**（泊松方程） |
| ∂u/∂t = α∂²u/∂x² | u(x,0)=f(x), u(0,t)=u(L,t)=0 | **IVP** 在时间上，**BVP** 在空间上 |
| Schrödinger 方程 | 含时→IVP；定态→BVP (本征值问题) | 两种情况都有 |

### 3.4 代码层面的体现

```python
# IVP 解法（推进）
for step in range(n_steps):
    state = rk4_step(state, dt)  # 从当前状态算下一状态

# BVP 解法（打靶法）
def shoot(guess):
    """从初始猜测 y'(0) 打靶"""
    sol = solve_ivp(ode, [0, L], [y0, guess])
    return sol.y[0][-1] - target  # 末端误差

# 用求根法找到正确的 y'(0)
from scipy.optimize import root
result = root(shoot, x0=1.0)
correct_guess = result.x
```

## 4. 自查项四：Fourier 变换的物理意义

### 4.1 一对变换的完整形式

```
f̂(ω) = ∫_{-∞}^{+∞} f(t) · e^{-iωt} dt      —— 正变换（时域→频域）
f(t)  = (1/2π) ∫_{-∞}^{+∞} f̂(ω) · e^{iωt} dω  —— 逆变换（频域→时域）
```

### 4.2 物理意义的三个层次

#### 层次一：分解（最直观）

任何（满足 Dirichlet 条件的）信号，都能**唯一地**拆分成一系列不同频率的正弦波/余弦波的叠加。

```
f(t) = A₀ + Σ A_n sin(ω_n t + φ_n)
```

Fourier 变换做的就是这件事——告诉你每个频率的振幅 A_n 和相位 φ_n。

#### 层次二：基底变换（线性代数视角）

Fourier 变换是一个**基变换**：从"时间基"（δ(t-τ)，Dirca delta）换到"频率基"（e^{iωt}，平面波）。

这和你在线性代数里学到的"把向量从标准基换到特征基"是完全一样的思想，只是维度从有限变成了无限。

```
时域表示：f(t) = Σ f(t_i) · δ(t-t_i)           （信号的"时间成分"）
频域表示：f̂(ω) = Σ f̂(ω_j) · e^{iω_j t}          （信号的"频率成分"）

这是同一个信号在两组不同基下的坐标。
```

#### 层次三：算子的对角化（最高级理解）

**为什么要在仿真中用 Fourier 变换？因为微分在频域变成了乘法！**

时域：d/dt → 乘 iω

```
d/dt f(t)   ──FFT──→  iω · f̂(ω)
d²/dt² f(t) ──FFT──→  -ω² · f̂(ω)
```

这意味着**偏微分方程在频域变成了代数方程**。

**最炸裂的例子——热传导方程的谱方法求解：**

```
时域：∂u/∂t = α·∂²u/∂x²           （需要搭网格、选差分格式、满足 CFL）
频域：dû/dt = -α·k²·û              （对每个 k 只是一个常微分方程！）
```

利用 FFT，你可以**在一个时间步内精确地**（谱精度）求解偏微分方程，而不是用有限差分的低阶近似。

### 4.3 仿真中最常用的三个 Fourier 操作

**应用一：频谱分析（诊断仿真结果）**

```python
from scipy.fft import fft, fftfreq

# 你已经仿真出一个系统的时间序列 x(t)
spectrum = np.abs(fft(x))**2     # 功率谱
freqs = fftfreq(N, dt)           # 频率轴

# 峰值对应的频率 = 系统的特征频率
# 如果峰值位置和理论预测不符，你的仿真有 bug
```

**应用二：滤波/降噪**

```python
spectrum = fft(signal)
spectrum[np.abs(freqs) > cutoff] = 0   # 切掉高频噪声
clean_signal = np.real(ifft(spectrum))  # 逆变换回去
```

**应用三：谱方法解 PDE**

这是最高级的 PDE 解法，利用 FFT 把空间微分变成乘法，然后用 ODE 求解器推进时间。精度可达**指数收敛**（网格加倍，误差不是减半而是指数下降）。

### 4.4 离散傅里叶变换 (DFT) 的实用要点

计算机不处理连续的积分，处理的是离散采样点：

```
DFT: X_k = Σ_{n=0}^{N-1} x_n · e^{-i2πkn/N},   k=0,1,...,N-1
FFT:  计算 DFT 的快速算法，复杂度 O(N log N) vs 暴力 O(N²)
```

**关键参数：**

- **采样定理（Nyquist）：** 采样频率 f_s 至少是信号最高频率的 2 倍，否则发生混叠（aliasing）
- **频率分辨率：** Δf = f_s / N。要分辨两个靠近的频率，增加采样点数 N
- **频谱泄露：** 信号如果不是周期截断，会在频谱中产生旁瓣。解决方案：加窗函数（Hann, Hamming）

## 5. 自查项五：中心极限定理与蒙特卡洛误差

### 5.1 中心极限定理 (CLT) 的精确陈述

设 X₁, X₂, ..., X_N 是**独立同分布**的随机变量，均值为 μ，方差为 σ²（有限）。

定义样本均值：

```
X̄_N = (1/N) Σ X_i
```

则当 N → ∞ 时：

```
√N · (X̄_N - μ) / σ  ──→  N(0, 1)    （标准正态分布）
```

**等价地，对于大 N：**

```
X̄_N ≈ N(μ, σ²/N)
```

两个关键性质：
1. **无论原始分布是什么形状**（指数分布、均匀分布、甚至双峰分布），均值的分布都趋向正态
2. **方差随 N 缩小**：Var(X̄_N) = σ²/N

### 5.2 为什么蒙特卡洛误差 ∝ 1/√N

这是中心极限定理的直接推论。

**蒙特卡洛积分估计：**

```
I = ∫ f(x) dx ≈ Î_N = (1/N) Σ f(X_i),   X_i ~ 均匀分布
```

把 f(X_i) 看作随机变量，其方差 σ_f²。由 CLT：

```
Î_N ≈ N(I, σ_f²/N)
```

因此：
- **标准误差** = σ_f / √N
- **收敛速度** = O(1/√N)

**这个速度与维度无关！** 这是蒙特卡洛方法碾压传统数值积分（高斯求积等）的核心优势。当积分维度 d ≥ 4 时，确定性方法的计算量指数增长，而蒙特卡洛只要 N ∝ 1/ε² 就能达到精度 ε。

### 5.3 数值演示（建议亲自跑一遍）

```python
import numpy as np
import matplotlib.pyplot as plt

# 验证：误差是否真的随 1/√N 下降
true_value = 2.0  # ∫₀¹ 2dx = 2

Ns = [100, 400, 1600, 6400, 25600, 102400]
errors = []

for N in Ns:
    estimates = []
    for trial in range(200):  # 多次试验看分布
        x = np.random.uniform(0, 1, N)
        est = np.mean(2 * np.ones(N))  # 被积函数 f(x)=2
        estimates.append(est)
    errors.append(np.std(estimates))   # 标准误差

# 绘制误差 vs 1/√N
plt.loglog(1/np.sqrt(Ns), errors, 'bo-', label='实测误差')
plt.loglog(1/np.sqrt(Ns), errors[0]*np.sqrt(Ns[0])/np.sqrt(Ns), 'r--', label='理论 ∝ 1/√N')
plt.xlabel('1/√N'); plt.ylabel('标准误差')
plt.legend(); plt.grid(); plt.show()
# 两条线应几乎重合，完美验证 CLT
```

### 5.4 从 CLT 到仿真的实用推论

**推论一：要增加一位有效数字（误差÷10），需要 100 倍的粒子/采样点。**

这意味着蒙特卡洛的"粗精度"（例如 1%）很便宜，但"高精度"（0.01%）非常贵。实际中通常结合重要性采样、控制变量法（control variates）等降方差技术。

**推论二：多跑几次独立仿真取平均值，可以降低误差。**

跑 M 次独立仿真，每次结果 Yᵢ，平均值的方差 = Var(Y)/M。这是仿真科研中最基本的实验设计技巧。

**推论三：做误差棒 (error bar) 时，报告的是 σ/√N，不是 σ。**

σ/√N 反映的是"均值的估计精度"，它可以随 N 增大而无限缩小。σ 反映的是"总体的分散程度"，不随 N 变化。

### 5.5 CLT 不适用时的典型警告

- **长尾分布**（如 Cauchy 分布）：方差不存在，CLT 不适用 → 蒙特卡洛不收敛
- **强相关样本**（如 MCMC 相邻迭代）：独立性不满足 → 有效样本数 < N，需用自相关修正
- **厚尾但有限方差**：CLT 仍然成立，但需要更大的 N 才能接近正态

## 6. 线性代数五大操作深度讲解

### 6.1 矩阵乘法（线性变换）

**操作定义：**

```
C = A B,   其中 C_{ij} = Σ_k A_{ik} B_{kj}
```

**物理意义：** 矩阵乘法 = 复合线性变换。先做 B 变换，再做 A 变换。

**仿真中的核心场景：**

1. **坐标旋转：** 旋转矩阵 R(θ) 作用于位置向量 r，得新坐标系下的坐标
   ```
   R(θ) = [cosθ  -sinθ]
          [sinθ   cosθ]
   r' = R(θ)·r
   ```

2. **量子态演化：** |ψ(t)⟩ = U(t)|ψ(0)⟩，其中 U(t) = exp(-iHt/ħ) 是时间演化算符。矩阵乘法就是态矢的演化。

3. **有限元刚度矩阵：** K·u = f，K 的元素由基函数的积分给出，u 是节点自由度。

**手算训练：** 验证旋转矩阵的复合
```
R(α)·R(β) = R(α+β)    ← 角度的复合等于求和，正对应旋转的群性质
```

### 6.2 特征值分解（简正模分析）

**操作定义：**

```
A v_i = λ_i v_i

矩阵形式：A = V Λ V^{-1}
其中 V 的列是特征向量，Λ = diag(λ₁, λ₂, ...)
```

如果 A 是实对称矩阵（物理系统中绝大多数情况），则 V 是正交矩阵：

```
A = Q Λ Q^T,    其中 Q^T Q = I
```

**物理意义：** 把系统的运动"对角化"——在特征向量基下，每个自由度独立演化，不存在耦合。

**仿真中的"标准流程"：**

1. 写出系统的质量矩阵 M 和刚度矩阵 K（通常来自 Lagrange 力学）
2. 解广义特征值问题：K·v = ω²·M·v
3. ω_i = 各简正模的频率
4. v_i = 各简正模的振型（各质点振幅的相对比例）

```python
import numpy as np

# 示例：三质点弹簧链（两端固定）
K = np.array([[ 2, -1,  0],
              [-1,  2, -1],
              [ 0, -1,  2]])   # 刚度矩阵
M = np.eye(3)                   # 单位质量矩阵

eigenvalues, eigenvectors = np.linalg.eigh(K)  # eigh 专用于对称矩阵
# eigenvalues = [ω₁², ω₂², ω₃²]
# eigenvectors[:, i] = 第 i 个简正模式的振型

omega = np.sqrt(eigenvalues)
print(f"简正频率: {omega}")
```

### 6.3 SVD 分解（奇异值分解）

**操作定义：**

```
A = U Σ V^T

其中：
A : m×n 任意矩阵
U : m×m 正交矩阵（左奇异向量 = A 的列空间正交基）
Σ : m×n 对角矩阵（奇异值 σ₁ ≥ σ₂ ≥ ... ≥ σ_r ≥ 0）
V : n×n 正交矩阵（右奇异向量 = A 的行空间正交基）
```

**关键理解——SVD 的"输入-输出"视角：**

```
A·v_k = σ_k · u_k

输入方向 v_k（右奇异向量），经过 A 变换后，
输出方向 u_k（左奇异向量），放大倍数 σ_k。
```

**SVD vs 特征值分解：**

| 特性 | 特征值分解 | SVD |
|------|----------|-----|
| 适用范围 | 方阵（最好是 Hermite 的） | **任意矩阵** |
| 正交基 | 一个（特征向量） | 两个（左右奇异向量各一） |
| 值 | 特征值，可正可负可复数 | 奇异值，**恒为非负实数** |
| 物理类比 | 固有频率和模式 | 输入→输出的放大因子 |

**仿真中的应用：**

1. **PCA 降噪（从实验数据中提取物理信号）：**
   ```python
   U, s, Vt = np.linalg.svd(data_matrix, full_matrices=False)
   # s 是奇异值，保留前 k 大的，其余置零 → 去噪
   k = 5  # 保留 5 个主成分
   denoised = U[:, :k] @ np.diag(s[:k]) @ Vt[:k, :]
   ```

2. **分析多体系统的集体运动模式：** 将 N 个粒子的 M 帧轨迹排成矩阵（N×3M 或类似形状），SVD 分解后，最大的奇异值对应的模式 = 最主要的集体运动。

3. **求解病态线性方程组：**
   ```python
   # 截断 SVD (TSVD) 求解 Ax = b
   # 把最小奇异值对应的分量去掉，避免被噪声放大
   U, s, Vt = np.linalg.svd(A, full_matrices=False)
   threshold = 1e-10 * s[0]  # 小奇异值的截断阈值
   s_inv = np.where(s > threshold, 1/s, 0)
   x = Vt.T @ np.diag(s_inv) @ U.T @ b
   ```

### 6.4 稀疏矩阵（局部相互作用的数学表达）

**物理直觉：** 大部分物理相互作用是**局域**的——一个网格点的更新只依赖于它最近的几个邻居。这意味着离散化后的矩阵**绝大多数元素是零**。

**实例对比：**

```
一维 Laplace 算子（N=5，Dirichlet 边界）：
        [-2   1   0   0   0]
A = 1/h²[ 1  -2   1   0   0]
        [ 0   1  -2   1   0]
        [ 0   0   1  -2   1]
        [ 0   0   0   1  -2]

非零元素：5 + 4 + 4 = 13 个（对角 + 两侧）
总元素：25 个
密度：13/25 = 52%（小矩阵，密度显得高）

扩展到 N=1000：
非零元素：1000 + 999 + 999 = 2998
总元素：1,000,000
密度：0.3%  ← 极度稀疏！
```

**稀疏存储格式（COO / CSR / CSC）：**

```python
from scipy import sparse

# 构建三对角矩阵（COO 格式 → CSR）
N = 1000
diagonals = [
    -2 * np.ones(N),       # 主对角线
    np.ones(N-1),           # 上对角线
    np.ones(N-1)            # 下对角线
]
A_sparse = sparse.diags(diagonals, [0, -1, 1], shape=(N, N), format='csr')

# 稀疏矩阵-向量乘法：O(非零元数) ∼ 3000 次运算
b = np.random.randn(N)
y = A_sparse @ b

# 稠密矩阵-向量乘法：O(N²) ∼ 1,000,000 次运算
# A_dense = A_sparse.toarray()
# y_dense = A_dense @ b  # 慢 300 倍
```

**仿真中的典型稀疏矩阵：**

| 矩阵类型 | 来源 | 每行非零元数 |
|----------|------|-------------|
| 三对角 | 1D Laplace/Poisson FDM | 3 |
| 五对角 | 2D Laplace FDM (五点差分) | 5 |
| 七对角 | 3D Laplace FDM | 7 |
| 块三对角 | 2D 问题按行排序 | 随网格尺寸 |
| 带状 | 高精度 FDM（多点模板） | = 模板宽度 |
| FEM 刚度矩阵 | 有限元法 | ≈ 单元节点连接数 |

### 6.5 迭代求解（大规模线性系统）

**当矩阵太大，高斯消去法 O(N³) 不可行时，改用迭代法。**

#### 6.5.1 共轭梯度法 (Conjugate Gradient, CG)

**适用范围：** 对称正定矩阵（SPD）。恰好，大多数物理系统的刚度矩阵和 Laplace 矩阵都是 SPD。

**核心思想：** 在 Krylov 子空间中逐步构造近似解，每次迭代选最优的新方向（与前所有方向 A-共轭），使得误差的 A-范数单调递减。

**物理类比：** 在高维空间中找势能函数的最小值点。CG 的每一步都沿着与前面所有步"不矛盾"的方向下降。

```python
from scipy.sparse.linalg import cg

x, info = cg(A_sparse, b, rtol=1e-8, maxiter=1000)
# info = 0 表示收敛成功
```

**收敛速度：** 误差缩小到 ε 所需的迭代次数 ≈ √(κ)·log(1/ε)，其中 κ = λ_max/λ_min 是条件数。

**预处理（Preconditioning）：** 如果条件数 κ 很大，收敛会很慢。预处理 = 找一个 M ≈ A 但 M⁻¹ 易于计算，解 M⁻¹Ax = M⁻¹b，新矩阵条件数更小。

```python
# 不完全 Cholesky 预处理
from scipy.sparse.linalg import cg, spilu, LinearOperator

M = spilu(A)  # 不完全 LU 分解，作为预处理子
M_op = LinearOperator(A.shape, M.solve)

x, info = cg(A, b, M=M_op, rtol=1e-8)  # 大幅加速
```

#### 6.5.2 其他常用迭代法

| 方法 | 适用矩阵类型 | 经典场景 |
|------|------------|----------|
| **CG** | 对称正定 | Poisson 方程 FEM/FDM |
| **GMRES** | 任意非对称 | 对流-扩散方程 |
| **BiCGSTAB** | 非对称、一般 | 流体力学（Navier-Stokes 线性化） |
| **Jacobi / Gauss-Seidel** | 严格对角占优 | 简单热传导，教学用 |
| **多重网格 (Multigrid)** | 椭圆型 PDE | **极其快**，O(N) 可解 |

**多重网格是 PDE 求解的终极武器：** 利用"细网格上的低频误差 = 粗网格上的高频误差"这个原理，在不同网格层级间来回迭代消除误差。对 Poisson 方程，复杂度接近 O(N)（比 CG 的 O(N√κ) 更优）。

## 7. 数值微积分的数学推导

### 7.1 导数的有限差分公式——完整推导

**前向差分（Forward Difference）：**

Taylor 展开：

```
f(x+h) = f(x) + h·f'(x) + h²/2·f''(ξ₁),   ξ₁ ∈ (x, x+h)
```

解出 f'(x)：

```
f'(x) = [f(x+h) - f(x)] / h - h/2·f''(ξ₁)
       = [f(x+h) - f(x)] / h + O(h)
```

结论：前向差分是**一阶精度**。误差正比于 h。

---

**后向差分（Backward Difference）：**

```
f(x-h) = f(x) - h·f'(x) + h²/2·f''(ξ₂)

f'(x) = [f(x) - f(x-h)] / h + O(h)
```

同样一阶精度。

---

**中心差分（Central Difference）：** ⭐ 最重要

```
f(x+h) = f(x) + h·f'(x) + h²/2·f''(x) + h³/6·f'''(ξ₁)    —— (1)
f(x-h) = f(x) - h·f'(x) + h²/2·f''(x) - h³/6·f'''(ξ₂)    —— (2)
```

(1) - (2)：

```
f(x+h) - f(x-h) = 2h·f'(x) + h³/3·f'''(ξ)
```

解出 f'(x)：

```
f'(x) = [f(x+h) - f(x-h)] / (2h) + O(h²)
```

结论：中心差分是**二阶精度**。原因：奇数阶误差项（h¹, h³, ...）在减法中抵消了。

---

**二阶导数中心差分：**

(1) + (2)：

```
f(x+h) + f(x-h) = 2f(x) + h²·f''(x) + h⁴/12·f''''(ξ)
```

解出 f''(x)：

```
f''(x) = [f(x+h) - 2f(x) + f(x-h)] / h² + O(h²)
```

### 7.2 数值积分的 Simpson 法则——推导

**思路：** 用二次抛物线（3 个点决定）代替直线（2 个点），拟合被积函数后在每个小区间上精确积分。

在一个区间 [-h, h] 上（平移后），已知三点 (-h, f₋₁), (0, f₀), (h, f₁)：

Lagrange 插值多项式：

```
p(x) = f₋₁·(x)(x-h)/(2h²) + f₀·(x+h)(h-x)/h² + f₁·(x)(x+h)/(2h²)
```

积分 ∫[-h,h] p(x) dx：

```
∫ p(x)dx = h/3 · (f₋₁ + 4f₀ + f₁)
```

推广到 n 个小区间（n 必须为偶数，总点数 = n+1 为奇数）：

```
∫[a,b] f(x)dx ≈ h/3 · [f(a) + 4Σf(奇数下标) + 2Σf(偶数下标) + f(b)]
```

**误差分析：** Simpson 法则的误差 ∝ h⁴·f⁽⁴⁾(ξ)，因此步长减半，误差缩小 16 倍。

| 方法 | 区间近似 | 精度 | 步长减半效果 |
|------|---------|------|-------------|
| 矩形法则 | 常数 | O(h) | 误差÷2 |
| 梯形法则 | 线性 | O(h²) | 误差÷4 |
| **Simpson** | **二次** | **O(h⁴)** | **误差÷16** |
| Boole 法则 | 四次 | O(h⁶) | 误差÷64 |

**为什么实际中 Simpson 最常用：** 梯形太粗糙，Boole 需要太多点（没有显著更好），Simpson 正好在精度和简洁之间取得最佳平衡。

### 7.3 数值微分的不稳定性——你必须知道的陷阱

**核心矛盾：** 微分是"不适定问题"——输入数据微小误差（如测量噪声）在数值微分中会被放大。

**分析：** 考虑带噪声的信号 f̃(x) = f(x) + ε·sin(ωx)，做中心差分：

```
f̃'(x) = [f̃(x+h) - f̃(x-h)] / (2h)
       ≈ f'(x) + ε·ω·cos(ωx)   ← 噪声被放大了 ω 倍！
```

如果噪声是高频的（ω 大），数值微分会把噪声放大到完全淹没真实信号。

**更精细的分析：** 中心差分的总误差 = 截断误差 + 舍入误差

```
总误差 ≈ h²·|f'''(x)|/6 + ε_machine/h
          ↑ 离散化误差      ↑ 机器精度/h 放大
```

最优 h ≈ (ε_machine)^(1/3) ≈ 6×10⁻⁶（双精度）。更小的 h 反而让舍入误差爆炸。

**仿真经验法则：** 数值导数中 h 取 10⁻⁵ ∼ 10⁻⁶（双精度）。不要取 10⁻¹⁵——计算机会把 f(x+h) 和 f(x-h) 算成完全一样，导致导数为 0。

## 8. 线性代数在仿真中的核心用法（扩展）

### 8.1 完整示例：二维 Poisson 方程的有限差分矩阵

**物理方程：**

```
∂²u/∂x² + ∂²u/∂y² = -ρ(x,y) / ε₀    在 Ω = [0,L]×[0,L]
u|_∂Ω = 0                              (Dirichlet 边界)
```

**离散化：** N×N 网格，总未知数 = (N-1)²（边界已知为 0）。

用五点差分：

```
(u_{i+1,j} + u_{i-1,j} + u_{i,j+1} + u_{i,j-1} - 4u_{i,j}) / h² = -ρ_{i,j}/ε₀
```

**矩阵形式：** 每个未知数 u_{i,j} 是向量中的一个分量。这 (N-1)² 个线性方程写成：

```
A·u = f
```

其中 A 是一个 (N-1)² × (N-1)² 的矩阵：

```
A = (1/h²) × [T  I  0  0  ...]
             [I  T  I  0  ...]
             [0  I  T  I  ...]
             [0  0  I  T  ...]
             [...            ]
```

这里 T 是 (N-1)×(N-1) 的三对角矩阵 [-4, 1, 1]，I 是单位矩阵，0 是零矩阵。

**A 的性质：**
- 对称正定
- 稀疏（每行 5 个非零元）
- 条件数 ∝ N²（网格越密，条件数越大，越难解）

**完整代码（含预处理共轭梯度）：**

```python
import numpy as np
from scipy import sparse
from scipy.sparse.linalg import cg, spilu, LinearOperator

def solve_poisson_2d(N, rho_func):
    """用有限差分 + CG 求解二维 Poisson 方程"""
    h = 1.0 / (N - 1)
    n_unknown = (N - 2) * (N - 2)  # 内部网格点数
    
    # 构建稀疏矩阵
    main_diag = -4 * np.ones(n_unknown) / h**2
    side_diag = np.ones(n_unknown - 1) / h**2
    far_diag = np.ones(n_unknown - (N - 2)) / h**2
    
    # 剔除块边界上的虚假连接
    for i in range(1, N - 2):
        side_diag[(N-2)*i - 1] = 0
    
    diagonals = [main_diag, side_diag, side_diag, far_diag, far_diag]
    offsets = [0, -1, 1, -(N-2), (N-2)]
    A = sparse.diags(diagonals, offsets, format='csr')
    
    # 右端项
    x = np.linspace(0, 1, N)
    y = np.linspace(0, 1, N)
    X, Y = np.meshgrid(x, y)
    f = rho_func(X[1:-1, 1:-1], Y[1:-1, 1:-1]).ravel()
    
    # 预处理共轭梯度求解
    M = spilu(A)  # 不完全 LU
    M_op = LinearOperator(A.shape, M.solve)
    u_flat, info = cg(A, f, M=M_op, rtol=1e-10)
    
    if info != 0:
        print(f"警告：CG 未收敛，info={info}")
    
    # 恢复为二维数组
    u = np.zeros((N, N))
    u[1:-1, 1:-1] = u_flat.reshape(N-2, N-2)
    return u
```

### 8.2 从线性代数角度看数值方法

| 数值方法 | 矩阵的角色 | 线性代数核心操作 |
|----------|-----------|----------------|
| FDM (显式) | 时间推进 A·uⁿ → uⁿ⁺¹ | 矩阵-向量乘法 |
| FDM (隐式) | Auⁿ⁺¹ = uⁿ | 解稀疏线性系统 |
| FEM | Ku = f (刚度方程) | 稀疏矩阵组装 + LU/CG |
| 谱方法 | 对角化微分算子 | FFT（快速酉变换） |
| 量子仿真 | |ψ⟩ = U|ψ₀⟩ | 矩阵指数 e⁻ⁱᴴᵗ 乘以向量 |

**共同模式：** 仿真 = 离散化（构建矩阵 A）+ 求解（matvec 或 solve）。

## 9. 稳定性、刚性方程的数学分析

### 9.1 Von Neumann 稳定性分析——完整推导

**对 FDM 格式做稳定性分析的通用方法：**

**步骤一：** 假设数值解的形式——Fourier 模式

```
u_j^n = ξ^n · e^{ik·jΔx}
```

其中：
- ξ 是**放大因子**（amplification factor），表征该 Fourier 模式从第 n 步到第 n+1 步是被放大还是衰减
- k 是波数

**步骤二：** 代入有限差分格式

以 FTCS（热传导方程，即给显式 Euler + 中心差分）为例：

```
u_j^{n+1} = u_j^n + r(u_{j+1}^n - 2u_j^n + u_{j-1}^n),   r = αΔt/Δx²
```

代入 Fourier 模式：

```
ξ·e^{ikjΔx} = e^{ikjΔx} + r(e^{ik(j+1)Δx} - 2e^{ikjΔx} + e^{ik(j-1)Δx})

两边同除 e^{ikjΔx}：
ξ = 1 + r(e^{ikΔx} - 2 + e^{-ikΔx})
  = 1 + r(2cos(kΔx) - 2)
  = 1 - 4r·sin²(kΔx/2)
```

**步骤三：** 稳定性条件：对所有 k，|ξ| ≤ 1

```
|1 - 4r·sin²(kΔx/2)| ≤ 1,   ∀θ = kΔx/2 ∈ [0, π/2]

→ -1 ≤ 1 - 4r·sin²θ ≤ 1
→ 0 ≤ 4r·sin²θ ≤ 2
→ r ≤ 1 / (2·sin²θ)

sin²θ 最大为 1，所以最严格条件：
r ≤ 1/2  即  α·Δt / Δx² ≤ 1/2
```

这就是 CFL 条件的完整推导。

### 9.2 为什么显式方法对扩散问题如此苛刻

扩散方程中 Δt ∝ Δx²。这意味着：

- 网格加密 2 倍 → Δt 必须缩小 4 倍
- 在 3D 中，总工作量 ∝ N³ × (N²) = N⁵，非常恐怖

**这就是为什么做扩散/热传导仿真必须用隐式方法（Crank-Nicolson 等无条件稳定格式）的原因。**

**Crank-Nicolson 的放大因子：**

```
ξ = [1 - 2r·sin²θ] / [1 + 2r·sin²θ]

对任意 r > 0，|ξ| ≤ 1。无条件稳定。
```

代价：每一步需要解线性系统（三对角矩阵 → Thomas 算法 O(N) 可解）。

### 9.3 刚性方程的数学刻画

**刚性比 (Stiffness Ratio)：**

```
S = |Re(λ_max)| / |Re(λ_min)|

其中 λ 是系统 Jacobian 矩阵的特征值。
```

经典例子（化学动力学中经常出现）：

```
y₁' = -1000·y₁ + 1000·y₂
y₂' =      y₁ -    2·y₂

Jacobian J = [-1000  1000]
             [   1    -2]

λ₁ ≈ -1000,  λ₂ ≈ -2
S = 1000/2 = 500  ← 高度刚性
```

**物理直觉：** λ₁ = -1000 对应的模式在 Δt ≈ 0.001 的尺度上衰减（快模式）；λ₂ = -2 对应的模式在 Δt ≈ 0.5 的尺度上演化（慢模式）。显式 Euler 的稳定条件 Δt < 2/|λ_max| = 0.002，但要看到慢模式的演化需要 T ≈ 50 时间单位。这意味着需要 25000 步——绝大部分步骤在浪费时间追踪已经衰减完的快模式。

**隐式方法（BDF）：** Δt 只由精度要求决定，可以取到 ∼ 0.1 甚至更大，此时只需要 500 步。

**这就是为什么刚性问题必须上隐式方法。**

### 9.4 稳定性分析速查表

| PDE 类型 | 显式格式 | 稳定性条件 | 隐式替代方案 |
|----------|---------|-----------|-------------|
| 热传导 (扩散) | FTCS | Δt ≤ Δx²/(2α) | Crank-Nicolson (无条件稳定) |
| 波动方程 | FTCS (蛙跳) | Δt ≤ Δx/c (CFL=1) | 隐式 (无条件稳定但耗散) |
| 对流方程 | FTFS / FTCS | Δt ≤ Δx/|v| | 迎风格式 / Lax-Wendroff |
| 薛定谔方程 (线性) | FTCS | 条件稳定 | Crank-Nicolson (无条件、幺正) |

**关键规律：**
- **扩散型 PDE（∂u/∂t = ∂²u/∂x²）：** Δt ∝ Δx² —— 对网格加密极度敏感
- **波动型 PDE（∂²u/∂t² = c²∂²u/∂x²）：** Δt ∝ Δx —— 加密网格的影响可以接受
- **对流型 PDE（∂u/∂t = v·∂u/∂x）：** Δt ∝ Δx —— 与波动类似

## 10. MCMC 与重要性采样的概率论基础

### 10.1 蒙特卡洛积分的方差分析

对积分 `I = ∫_Ω f(x)dx`，标准 Monte Carlo 估计：

```
Î = (V/N) · Σ f(X_i),   X_i ~ Uniform(Ω)
```

其中 V = Ω 的体积。

方差：

```
Var(Î) = V²/N · Var(f) = V²/N · [⟨f²⟩ - ⟨f⟩²]
```

其中 ⟨·⟩ 表示在 Ω 上对均匀分布的期望。

**标准误差 = √Var(Î) ∝ 1/√N。**

### 10.2 重要性采样——降低方差的数学原理

引入提议分布 p(x)，并改写积分：

```
I = ∫_Ω f(x) dx = ∫_Ω [f(x)/p(x)] · p(x) dx = E_{x~p}[f(x)/p(x)]
```

Monte Carlo 估计：

```
Î_IS = (1/N) Σ f(X_i)/p(X_i),   X_i ~ p(x)
```

方差：

```
Var(Î_IS) = (1/N) · Var_{x~p}[f(x)/p(x)]
          = (1/N) · [∫ f(x)²/p(x) dx - I²]
```

**命题：** 当 `p(x) ∝ |f(x)|` 时，方差最小（如果 f 不变号，则达到零方差）。

**证明（f(x) ≥ 0 时）：** 设 p(x) = f(x)/I（即按 f 自身采样），则 f(X_i)/p(X_i) = I，方差 = 0。

**直观理解：** 重要性采样的核心是——在被积函数 f(x) 大的地方多采样，在 f(x) 小的地方少采样。这样每个采样点的贡献 f/p 更均匀，方差自然更低。

### 10.3 马尔可夫链的细致平衡条件

在 MCMC 中，我们构建一条马尔可夫链，使得其**平稳分布 = 目标分布 π(x)**。

**细致平衡 (Detailed Balance)：**

```
π(x) · T(x → y) = π(y) · T(y → x)      ∀x, y
```

其中 T(x → y) 是从状态 x 转移到状态 y 的转移概率。

**细致平衡 → 平稳分布：** 如果链满足细致平衡，则 π 是平稳分布：

```
∫ π(x)·T(x → y) dx = ∫ π(y)·T(y → x) dx = π(y)·∫ T(y → x) dx = π(y)
```

**Metropolis-Hastings 如何满足细致平衡：**

MH 将转移概率拆成两步：

1. **提议步：** 从提议分布 q(x → y) 中生成候选状态 y
2. **接受步：** 以概率 α(x → y) = min(1, π(y)q(y→x)/[π(x)q(x→y)]) 接受 y

则实际转移概率：

```
T(x → y) = q(x → y) · α(x → y)
```

验证细致平衡：

```
π(x)·q(x→y)·α(x→y) = π(x)·q(x→y)·min(1, π(y)q(y→x)/[π(x)q(x→y)])
                     = min(π(x)q(x→y), π(y)q(y→x))
                     = π(y)·q(y→x)·min(1, π(x)q(x→y)/[π(y)q(y→x)])
                     = π(y)·q(y→x)·α(y→x)
                     = π(y)·T(y→x)
✓
```

**Metropolis 特例（对称提议分布 q(x→y) = q(y→x)）：**

```
α(x → y) = min(1, π(y)/π(x))

如果 π(x) ∝ exp(-E(x)/kT)（Boltzmann 分布）：
α = min(1, exp(-[E(y) - E(x)]/kT))
  = min(1, exp(-ΔE/kT))
```

这正是"新状态能量更低 → 总是接受；新状态能量更高 → 以 Boltzmann 因子概率接受"的物理规则。

### 10.4 MCMC 收敛诊断的基本方法

**Gelman-Rubin R 统计量：**

同时运行 M 条马尔可夫链（从**不同的、过分散的**初始点出发），每条链采样 N 次。

- 计算**链内方差 W**（within-chain variance）
- 计算**链间方差 B**（between-chain variance）

```
R = √[(N-1)/N · W + B/N) / W]

R ≈ 1：链已混合，平稳分布达成
R > 1.1：链尚未收敛，需要更多采样
```

**有效样本数 (ESS)：**

由于 MCMC 样本存在自相关，实际上包含的信息量少于独立样本。

```
ESS = N / [1 + 2 Σ_{t=1}^{∞} ρ(t)]

其中 ρ(t) 是滞后 t 的自相关系数。
```

常见的规则：ESS > 400 才能可靠地估计分布的均值和 95% 置信区间。

### 10.5 MCMC 在物理仿真中的应用全景

| 物理领域 | 目标分布 π(x) | 物理含义 |
|----------|-------------|----------|
| 统计力学 | exp(-E(config)/kT) | Ising 模型、LJ 流体在热平衡态的构型分布 |
| 量子 Monte Carlo | |Ψ(R)|²（试函数平方） | 基态能量和期望值 |
| 粒子物理 | exp(-S[φ]/ħ)（作用量权重） | 格点 QCD 中的路径积分 |
| 贝叶斯推断 | 后验分布 P(参数|数据) | 从实验数据推算物理参数 |
| 分子构象 | exp(-E(r)/kT)（全原子能量） | 蛋白质折叠、药物分子对接 |
