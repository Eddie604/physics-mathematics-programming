## 第一章 电磁现象的普遍规律

### 习题 1：$\nabla$ 算符对源变量和场变量的求导关系

**题目**：证明下列结果，并体会对源变数求微商（$\nabla'$）与对场变数求微商（$\nabla$）的关系：

(1) $\nabla r = -\nabla' r = \frac{\hat{\mathbf{r}}}{r}$

(2) $\nabla\cdot\left(\frac{\hat{\mathbf{r}}}{r^2}\right) = -\nabla'\cdot\left(\frac{\hat{\mathbf{r}}}{r^2}\right) = 0 \quad (r \neq 0)$

(3) $\nabla\times\left(\frac{\hat{\mathbf{r}}}{r^2}\right) = 0$

(4) $\nabla\cdot\left(\frac{\mathbf{r}}{r^3}\right) = 0 \quad (r \neq 0)$

并求：$\nabla\cdot\mathbf{r}$，$\nabla\times\mathbf{r}$，$(\mathbf{a}\cdot\nabla)\mathbf{r}$，$\nabla(\mathbf{a}\cdot\mathbf{r})$，$\nabla\cdot[\mathbf{E}_0\sin(\mathbf{k}\cdot\mathbf{r})]$ 及 $\nabla\times[\mathbf{E}_0\sin(\mathbf{k}\cdot\mathbf{r})]$，其中 $\mathbf{a},\mathbf{k},\mathbf{E}_0$ 均为常矢量。

---

#### 思路分析

核心在于区分两个梯度算符：
- $\nabla$ = 对**场点坐标** $\mathbf{r} = (x,y,z)$ 求偏导
- $\nabla'$ = 对**源点坐标** $\mathbf{r}' = (x',y',z')$ 求偏导

相对位置矢量 $\mathbf{R} = \mathbf{r} - \mathbf{r}'$，$R = |\mathbf{r} - \mathbf{r}'|$。

关键关系：$\nabla R = \frac{\mathbf{R}}{R} = \hat{\mathbf{R}}$，$\nabla' R = -\frac{\mathbf{R}}{R} = -\hat{\mathbf{R}}$。即对场点求导得 $+\hat{\mathbf{R}}$，对源点求导得 $-\hat{\mathbf{R}}$。

本题中源点在原点，故 $\mathbf{r}' = 0$，$R = r$。

---

#### 详细解算

**（1）证明 $\nabla r = -\nabla' r = \frac{\hat{\mathbf{r}}}{r}$**

$r = \sqrt{x^2 + y^2 + z^2}$。

$$\frac{\partial r}{\partial x} = \frac{x}{\sqrt{x^2+y^2+z^2}} = \frac{x}{r}$$

所以：
$$\nabla r = \left(\frac{\partial r}{\partial x}, \frac{\partial r}{\partial y}, \frac{\partial r}{\partial z}\right) = \left(\frac{x}{r}, \frac{y}{r}, \frac{z}{r}\right) = \frac{\mathbf{r}}{r} = \hat{\mathbf{r}} \neq \frac{\hat{\mathbf{r}}}{r}$$

> ⚠️ **更正**：题目中 $\frac{\hat{\mathbf{r}}}{r}$ 有误。正确结果应为 $\nabla r = \hat{\mathbf{r}}$（单位矢量），不是 $\hat{\mathbf{r}}/r$。但若理解为 $\nabla(\frac{1}{r}) = -\frac{\hat{\mathbf{r}}}{r^2}$，则是另一回事。这里按课本原意，$\nabla r = \hat{\mathbf{r}}$。

类似地，$\nabla' r = -\hat{\mathbf{r}}$（源点求导多一个负号）。

---

**（2）证明 $\nabla\cdot(\hat{\mathbf{r}}/r^2) = 0$（$r \neq 0$）**

在球坐标中，对矢量场 $\mathbf{v} = v_r \hat{\mathbf{r}}$：

$$\nabla\cdot(v_r \hat{\mathbf{r}}) = \frac{1}{r^2}\frac{\partial}{\partial r}(r^2 v_r)$$

这里 $v_r = 1/r^2$：
$$\nabla\cdot\left(\frac{\hat{\mathbf{r}}}{r^2}\right) = \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2\cdot\frac{1}{r^2}\right) = \frac{1}{r^2}\frac{\partial}{\partial r}(1) = 0 \quad (r \neq 0)$$

在 $r=0$ 处该散度是一个 $\delta$ 函数：$\nabla\cdot(\hat{\mathbf{r}}/r^2) = 4\pi\delta^3(\mathbf{r})$。

对源变量：$\nabla' \cdot (\hat{\mathbf{r}}/r^2) = -\nabla\cdot(\hat{\mathbf{r}}/r^2) = 0$。

---

**（3）证明 $\nabla\times(\hat{\mathbf{r}}/r^2) = 0$**

在球坐标中，旋度的径向分量：
$$(\nabla\times\mathbf{v})_r = \frac{1}{r\sin\theta}\left[\frac{\partial}{\partial\theta}(\sin\theta\, v_\phi) - \frac{\partial v_\theta}{\partial\phi}\right]$$

由于 $\hat{\mathbf{r}}/r^2$ 只有径向分量 $v_r = 1/r^2$，$v_\theta = v_\phi = 0$，且 $v_r$ 只与 $r$ 有关：
$$(\nabla\times\mathbf{v})_\theta = \frac{1}{r\sin\theta}\frac{\partial v_r}{\partial\phi} - \frac{1}{r}\frac{\partial}{\partial r}(r v_\phi) = 0$$
$$(\nabla\times\mathbf{v})_\phi = \frac{1}{r}\frac{\partial}{\partial r}(r v_\theta) - \frac{1}{r}\frac{\partial v_r}{\partial\theta} = 0$$

故 $\nabla\times(\hat{\mathbf{r}}/r^2) = \mathbf{0}$。这其实是**库仑场是无旋场**的体现。

---

**（4）证明 $\nabla\cdot(\mathbf{r}/r^3) = 0$（$r \neq 0$）**

$$\nabla\cdot\left(\frac{\mathbf{r}}{r^3}\right) = \frac{\partial}{\partial x}\left(\frac{x}{r^3}\right) + \frac{\partial}{\partial y}\left(\frac{y}{r^3}\right) + \frac{\partial}{\partial z}\left(\frac{z}{r^3}\right)$$

逐项计算：
$$\frac{\partial}{\partial x}\left(\frac{x}{r^3}\right) = \frac{1}{r^3} + x\cdot\left(-\frac{3}{r^4}\right)\cdot\frac{x}{r} = \frac{1}{r^3} - \frac{3x^2}{r^5}$$

三项相加：
$$\nabla\cdot\left(\frac{\mathbf{r}}{r^3}\right) = \frac{3}{r^3} - \frac{3(x^2+y^2+z^2)}{r^5} = \frac{3}{r^3} - \frac{3r^2}{r^5} = 0 \quad (r\neq 0)$$

---

**附加计算：**

- $\nabla\cdot\mathbf{r} = \frac{\partial x}{\partial x} + \frac{\partial y}{\partial y} + \frac{\partial z}{\partial z} = 1+1+1 = 3$

- $\nabla\times\mathbf{r} = \left(\frac{\partial z}{\partial y} - \frac{\partial y}{\partial z}, \frac{\partial x}{\partial z} - \frac{\partial z}{\partial x}, \frac{\partial y}{\partial x} - \frac{\partial x}{\partial y}\right) = (0,0,0) = \mathbf{0}$

- $(\mathbf{a}\cdot\nabla)\mathbf{r} = \left(a_x\frac{\partial}{\partial x} + a_y\frac{\partial}{\partial y} + a_z\frac{\partial}{\partial z}\right)\mathbf{r} = (a_x\cdot 1, a_y\cdot 1, a_z\cdot 1) = \mathbf{a}$

- $\nabla(\mathbf{a}\cdot\mathbf{r}) = \nabla(a_x x + a_y y + a_z z) = (a_x, a_y, a_z) = \mathbf{a}$（因 $\mathbf{a}$ 为常矢量）

- $\nabla\cdot[\mathbf{E}_0\sin(\mathbf{k}\cdot\mathbf{r})] = \mathbf{E}_0\cdot\nabla\sin(\mathbf{k}\cdot\mathbf{r}) = \mathbf{E}_0\cdot[\mathbf{k}\cos(\mathbf{k}\cdot\mathbf{r})] = (\mathbf{E}_0\cdot\mathbf{k})\cos(\mathbf{k}\cdot\mathbf{r})$

- $\nabla\times[\mathbf{E}_0\sin(\mathbf{k}\cdot\mathbf{r})] = \nabla\sin(\mathbf{k}\cdot\mathbf{r})\times\mathbf{E}_0 = \mathbf{k}\cos(\mathbf{k}\cdot\mathbf{r})\times\mathbf{E}_0 = \cos(\mathbf{k}\cdot\mathbf{r})(\mathbf{k}\times\mathbf{E}_0)$

> 这里用了恒等式 $\nabla\times(\phi\mathbf{A}) = \nabla\phi\times\mathbf{A} + \phi\nabla\times\mathbf{A}$。因 $\mathbf{E}_0$ 为常矢量，$\nabla\times\mathbf{E}_0 = \mathbf{0}$。

---

#### 要点总结

| 公式 | 结果 | 记忆技巧 |
|------|------|----------|
| $\nabla r$ | $\hat{\mathbf{r}}$ | 径向单位矢量 |
| $\nabla\cdot\mathbf{r}$ | $3$ | 三维空间维数 |
| $\nabla\times\mathbf{r}$ | $\mathbf{0}$ | 位置矢量无旋 |
| $(\mathbf{a}\cdot\nabla)\mathbf{r}$ | $\mathbf{a}$ | 沿 $\mathbf{a}$ 方向求导得 $\mathbf{a}$ |
| $\nabla(\mathbf{a}\cdot\mathbf{r})$ | $\mathbf{a}$ | 梯度就是 $\mathbf{a}$ 本身 |

核心恒等式：$\nabla f(r) = f'(r)\,\hat{\mathbf{r}}$，$\nabla\cdot[f(r)\hat{\mathbf{r}}] = \frac{1}{r^2}\frac{d}{dr}[r^2 f(r)]$。

### 习题 2：高斯定理与斯托克斯定理的应用

**题目**：
(1) 应用高斯定理证明 $\displaystyle\int_V \nabla\times\mathbf{J}\,dV = \oint_S d\mathbf{S}\times\mathbf{J}$

(2) 应用斯托克斯定理证明 $\displaystyle\int_S d\mathbf{S}\times\nabla\varphi = \oint_L \varphi\,d\mathbf{l}$

---

#### 思路分析

这是两个"非标准形式"的积分定理。标准形式是：
- 高斯定理：$\int_V \nabla\cdot\mathbf{A}\,dV = \oint_S \mathbf{A}\cdot d\mathbf{S}$
- 斯托克斯定理：$\int_S (\nabla\times\mathbf{A})\cdot d\mathbf{S} = \oint_L \mathbf{A}\cdot d\mathbf{l}$

但题目中是 $\int \nabla\times\mathbf{J}\,dV$（被积函数是旋度而非散度）和 $\int d\mathbf{S}\times\nabla\varphi$（矢量形式）。技巧是**引入辅助常矢量 $\mathbf{c}$**，将其转化为标准形式。

---

#### 详细解算

**（1）证明 $\int_V \nabla\times\mathbf{J}\,dV = \oint_S d\mathbf{S}\times\mathbf{J}$**

任取常矢量 $\mathbf{c}$（$\nabla\mathbf{c} = \mathbf{0}$，$\nabla\cdot\mathbf{c} = 0$）。

考虑 $\mathbf{c}\cdot(\nabla\times\mathbf{J})$。利用矢量恒等式：
$$\nabla\cdot(\mathbf{J}\times\mathbf{c}) = \mathbf{c}\cdot(\nabla\times\mathbf{J}) - \mathbf{J}\cdot(\nabla\times\mathbf{c}) = \mathbf{c}\cdot(\nabla\times\mathbf{J})$$

（因为 $\mathbf{c}$ 为常矢量，$\nabla\times\mathbf{c} = \mathbf{0}$）

对 $\nabla\cdot(\mathbf{J}\times\mathbf{c})$ 应用标准高斯定理：
$$\int_V \nabla\cdot(\mathbf{J}\times\mathbf{c})\,dV = \oint_S (\mathbf{J}\times\mathbf{c})\cdot d\mathbf{S}$$

利用混合积的轮换性：$(\mathbf{J}\times\mathbf{c})\cdot d\mathbf{S} = \mathbf{c}\cdot(d\mathbf{S}\times\mathbf{J}) = (d\mathbf{S}\times\mathbf{J})\cdot\mathbf{c}$

而左边等于 $\int_V \mathbf{c}\cdot(\nabla\times\mathbf{J})\,dV = \mathbf{c}\cdot\int_V \nabla\times\mathbf{J}\,dV$。

所以：
$$\mathbf{c}\cdot\int_V \nabla\times\mathbf{J}\,dV = \mathbf{c}\cdot\oint_S d\mathbf{S}\times\mathbf{J}$$

由于 $\mathbf{c}$ 是任意的，必须有：
$$\int_V \nabla\times\mathbf{J}\,dV = \oint_S d\mathbf{S}\times\mathbf{J}$$

证毕。

---

**（2）证明 $\int_S d\mathbf{S}\times\nabla\varphi = \oint_L \varphi\,d\mathbf{l}$**

任取常矢量 $\mathbf{c}$。考虑 $\mathbf{c}\cdot(d\mathbf{S}\times\nabla\varphi)$：
$$\mathbf{c}\cdot(d\mathbf{S}\times\nabla\varphi) = (d\mathbf{S}\times\nabla\varphi)\cdot\mathbf{c} = d\mathbf{S}\cdot(\nabla\varphi\times\mathbf{c})$$

又 $\nabla\times(\varphi\mathbf{c}) = \nabla\varphi\times\mathbf{c} + \varphi\nabla\times\mathbf{c} = \nabla\varphi\times\mathbf{c}$（因为 $\mathbf{c}$ 为常矢量）。

所以：
$$\int_S \mathbf{c}\cdot(d\mathbf{S}\times\nabla\varphi) = \int_S d\mathbf{S}\cdot[\nabla\times(\varphi\mathbf{c})]$$

对右边应用标准斯托克斯定理：
$$\int_S [\nabla\times(\varphi\mathbf{c})]\cdot d\mathbf{S} = \oint_L (\varphi\mathbf{c})\cdot d\mathbf{l} = \mathbf{c}\cdot\oint_L \varphi\,d\mathbf{l}$$

而左边 $= \mathbf{c}\cdot\int_S d\mathbf{S}\times\nabla\varphi$。

由于 $\mathbf{c}$ 任意，得：
$$\int_S d\mathbf{S}\times\nabla\varphi = \oint_L \varphi\,d\mathbf{l}$$

证毕。

---

#### 要点总结

**通用技巧**：当遇到"非标准形式"的积分恒等式时，引入任意常矢量 $\mathbf{c}$ 点乘两边，利用矢量混合积恒等式转化为散度或旋度的标准形式，再应用高斯定理或斯托克斯定理。最后利用 $\mathbf{c}$ 的任意性去掉它。

### 习题 3：电荷系统偶极矩的变化率

**题目**：已知电荷系统的电偶极矩 $\displaystyle\mathbf{p}(t) = \int \rho(\mathbf{x}',t)\,\mathbf{x}'\,dV'$，利用电荷守恒定律 $\displaystyle\nabla'\cdot\mathbf{J} + \frac{\partial\rho}{\partial t} = 0$，证明 $\displaystyle\frac{d\mathbf{p}}{dt} = \int \mathbf{J}(\mathbf{x}',t)\,dV'$。

---

#### 思路分析

对 $\mathbf{p}(t)$ 求时间导数，把 $\partial\rho/\partial t$ 用连续性方程替换为 $-\nabla'\cdot\mathbf{J}$，再用分部积分（高斯定理）消去 $\nabla'$ 作用在 $\mathbf{J}$ 上的效果，最终只留下 $\mathbf{J}$ 的体积分。

---

#### 详细解算

**步骤 1**：对偶极矩求时间导数

$$\frac{d\mathbf{p}}{dt} = \frac{d}{dt}\int \rho(\mathbf{x}',t)\,\mathbf{x}'\,dV' = \int \frac{\partial\rho(\mathbf{x}',t)}{\partial t}\,\mathbf{x}'\,dV'$$

（$\mathbf{x}'$ 是积分变量，不显含时间）

**步骤 2**：利用连续性方程

$$\frac{\partial\rho}{\partial t} = -\nabla'\cdot\mathbf{J}$$

代入：
$$\frac{d\mathbf{p}}{dt} = -\int (\nabla'\cdot\mathbf{J})\,\mathbf{x}'\,dV'$$

**步骤 3**：分部积分（用分量形式更清晰）

取 $x$ 分量（设 $\mathbf{x}' = (x',y',z')$，$\mathbf{J} = (J_x, J_y, J_z)$）：

$$\frac{dp_x}{dt} = -\int (\nabla'\cdot\mathbf{J})\,x'\,dV' = -\int \left(\frac{\partial J_x}{\partial x'} + \frac{\partial J_y}{\partial y'} + \frac{\partial J_z}{\partial z'}\right)x'\,dV'$$

利用 $\frac{\partial}{\partial x'}(x' J_x) = J_x + x'\frac{\partial J_x}{\partial x'}$，即 $x'\frac{\partial J_x}{\partial x'} = \frac{\partial}{\partial x'}(x' J_x) - J_x$。

同理对 $y',z'$ 分量处理：
$$x'\frac{\partial J_y}{\partial y'} = \frac{\partial}{\partial y'}(x' J_y) - J_y\frac{\partial x'}{\partial y'} = \frac{\partial}{\partial y'}(x' J_y)$$
$$x'\frac{\partial J_z}{\partial z'} = \frac{\partial}{\partial z'}(x' J_z)$$

所以：
$$\frac{dp_x}{dt} = -\int\left[\frac{\partial}{\partial x'}(x'J_x) - J_x + \frac{\partial}{\partial y'}(x'J_y) + \frac{\partial}{\partial z'}(x'J_z)\right]dV'$$

$$= -\int\left[\nabla'\cdot(x' \mathbf{J}) - J_x\right]dV' = -\oint_S x' \mathbf{J}\cdot d\mathbf{S}' + \int J_x\,dV'$$

**步骤 4**：边界项为零

电流分布在有限区域内，在无穷远边界 $S_\infty$ 上 $\mathbf{J} = \mathbf{0}$，故 $\oint_S x'\mathbf{J}\cdot d\mathbf{S}' = 0$。

因此 $\displaystyle\frac{dp_x}{dt} = \int J_x\,dV'$。

同理对 $y,z$ 分量有相同结论。故：
$$\boxed{\frac{d\mathbf{p}}{dt} = \int \mathbf{J}(\mathbf{x}',t)\,dV'}$$

---

#### 要点总结

这个结论非常直观：**电偶极矩的时间变化率 = 总电流**。如果有净电流流动，偶极矩就会变化。这是一个重要的物理关系，也是验证模型是否自洽的好方法。

### 习题 4：空心介质球——电场与极化电荷（⭐考试高频）

**题目**：内外半径分别为 $r_1$ 和 $r_2$ 的空心介质球，电容率为 $\varepsilon$，均匀带静止自由电荷密度 $\rho_f$。求：
(1) 空间各点的电场；
(2) 极化体电荷和极化面电荷分布。

---

#### 思路分析

球对称问题 → 用高斯定理 $\oint_S \mathbf{D}\cdot d\mathbf{S} = Q_f$。

分三个区域：
- $r < r_1$（空腔，无自由电荷）
- $r_1 < r < r_2$（介质体内，有均匀 $\rho_f$）
- $r > r_2$（球外真空）

每一区域用球对称高斯定理求出 $\mathbf{D}$，再由 $\mathbf{D} = \varepsilon\mathbf{E}$ 得 $\mathbf{E}$。然后求极化 $\mathbf{P}$，再求极化电荷。

---

#### 详细解算

**（1）空间各点的电场**

由球对称性，$\mathbf{D} = D(r)\,\hat{\mathbf{r}}$。高斯面为半径 $r$ 的同心球面：

$$\oint_S \mathbf{D}\cdot d\mathbf{S} = D(r)\cdot 4\pi r^2 = Q_f(\text{包围的自由电荷})$$

**区域 I：$r < r_1$**

空腔内无自由电荷：$Q_f = 0$。
$$D(r) \cdot 4\pi r^2 = 0 \quad\Rightarrow\quad D(r) = 0,\quad \mathbf{E} = \mathbf{0}$$

**区域 II：$r_1 < r < r_2$（介质内）**

自由电荷：$Q_f = \rho_f \cdot \frac{4\pi}{3}(r^3 - r_1^3)$
$$D(r) = \frac{\rho_f(r^3 - r_1^3)}{3r^2}$$
$$\mathbf{D} = \frac{\rho_f(r^3 - r_1^3)}{3r^2}\,\hat{\mathbf{r}}$$

介质内 $\mathbf{D} = \varepsilon\mathbf{E}$：
$$\boxed{\mathbf{E}_{\text{II}} = \frac{\rho_f(r^3 - r_1^3)}{3\varepsilon r^2}\,\hat{\mathbf{r}}}$$

**区域 III：$r > r_2$（球外真空）**

自由电荷：$Q_f = \rho_f \cdot \frac{4\pi}{3}(r_2^3 - r_1^3)$
$$D(r) = \frac{\rho_f(r_2^3 - r_1^3)}{3r^2}$$
$$\boxed{\mathbf{E}_{\text{III}} = \frac{\rho_f(r_2^3 - r_1^3)}{3\varepsilon_0 r^2}\,\hat{\mathbf{r}}}$$

> 球外电场等效于总自由电荷 $Q_{\text{总}} = \frac{4\pi}{3}\rho_f(r_2^3 - r_1^3)$ 集中在球心产生的电场。

---

**（2）极化电荷分布**

极化强度 $\mathbf{P} = \mathbf{D} - \varepsilon_0\mathbf{E} = (\varepsilon - \varepsilon_0)\mathbf{E}$。

在介质内（区域 II）：
$$\mathbf{P} = (\varepsilon - \varepsilon_0)\mathbf{E}_{\text{II}} = \frac{(\varepsilon - \varepsilon_0)\rho_f(r^3 - r_1^3)}{3\varepsilon r^2}\,\hat{\mathbf{r}}$$

**极化体电荷密度** $\rho_p = -\nabla\cdot\mathbf{P}$：

$$\rho_p = -\frac{1}{r^2}\frac{\partial}{\partial r}(r^2 P_r) = -\frac{1}{r^2}\frac{\partial}{\partial r}\left[\frac{(\varepsilon-\varepsilon_0)\rho_f(r^3 - r_1^3)}{3\varepsilon}\right]$$
$$= -\frac{(\varepsilon-\varepsilon_0)\rho_f}{3\varepsilon}\cdot\frac{1}{r^2}\cdot 3r^2 = -\frac{\varepsilon-\varepsilon_0}{\varepsilon}\rho_f$$
$$\boxed{\rho_p = -\left(1 - \frac{\varepsilon_0}{\varepsilon}\right)\rho_f}$$

（介质内 $r_1 < r < r_2$，体电荷均匀分布）

**极化面电荷密度** $\sigma_p = \mathbf{P}\cdot\hat{\mathbf{n}}$（$\hat{\mathbf{n}}$ 为介质表面外法向）：

- **内表面** $r = r_1$：外法向指向球心 $-\hat{\mathbf{r}}$，$P_r(r_1) = 0$（因为 $r=r_1$ 时 $r^3 - r_1^3 = 0$）。
  $$\sigma_p(r_1) = \mathbf{P}\cdot(-\hat{\mathbf{r}}) = -P_r(r_1) = 0$$

- **外表面** $r = r_2$：外法向 $\hat{\mathbf{r}}$，
  $$P_r(r_2) = \frac{(\varepsilon-\varepsilon_0)\rho_f(r_2^3 - r_1^3)}{3\varepsilon r_2^2}$$
  $$\boxed{\sigma_p(r_2) = \frac{(\varepsilon-\varepsilon_0)\rho_f(r_2^3 - r_1^3)}{3\varepsilon r_2^2}}$$

---

#### 要点总结

| 区域 | $D$ | $E$ |
|------|-----|-----|
| $r<r_1$ | $0$ | $0$ |
| $r_1<r<r_2$ | $\frac{\rho_f(r^3-r_1^3)}{3r^2}$ | $D/\varepsilon$ |
| $r>r_2$ | $\frac{\rho_f(r_2^3-r_1^3)}{3r^2}$ | $D/\varepsilon_0$ |

极化体电荷 $\rho_p = -(1-\varepsilon_0/\varepsilon)\rho_f$（均匀分布）。极化面电荷只在外表面出现。

### 习题 5：中空导体圆柱——磁感应强度与磁化电流

**题目**：内外半径分别为 $r_1$ 和 $r_2$ 的无穷长中空导体圆柱，沿轴向流有恒定均匀自由电流 $\mathbf{J}_f = J_f\hat{\mathbf{z}}$，导体的磁导率为 $\mu$。求磁感应强度和磁化电流。

---

#### 思路分析

柱对称 + 无穷长 → 用安培环路定理 $\oint \mathbf{H}\cdot d\mathbf{l} = I_f$。

分三个区域。磁场只有 $\phi$ 方向分量（用右手定则），由对称性 $H_\phi$ 只是 $r$ 的函数。

---

#### 详细解算

**（1）磁感应强度**

取半径为 $r$ 的圆形安培环路（$z$ 轴为中心轴）：

$$\oint_L \mathbf{H}\cdot d\mathbf{l} = H_\phi(r) \cdot 2\pi r = I_f(\text{包围的自由电流})$$

**区域 I：$r < r_1$（空腔）**

$I_f = 0$，$H_\phi = 0$，$\mathbf{B} = \mathbf{0}$。

**区域 II：$r_1 < r < r_2$（导体内）**

$$I_f = J_f \cdot \pi(r^2 - r_1^2)$$
$$H_\phi = \frac{J_f(r^2 - r_1^2)}{2r}$$

$\mathbf{B} = \mu\mathbf{H}$：
$$\boxed{\mathbf{B}_{\text{II}} = \frac{\mu J_f(r^2 - r_1^2)}{2r}\,\hat{\mathbf{\phi}}}$$

**区域 III：$r > r_2$（真空）**

$$I_f = J_f \cdot \pi(r_2^2 - r_1^2)$$
$$H_\phi = \frac{J_f(r_2^2 - r_1^2)}{2r}$$

$\mathbf{B} = \mu_0\mathbf{H}$：
$$\boxed{\mathbf{B}_{\text{III}} = \frac{\mu_0 J_f(r_2^2 - r_1^2)}{2r}\,\hat{\mathbf{\phi}}}$$

---

**（2）磁化电流**

磁化强度 $\mathbf{M} = \frac{\mathbf{B}}{\mu_0} - \mathbf{H}$。

在导体内（区域 II）：
$$\mathbf{M} = \frac{\mu\mathbf{H}}{\mu_0} - \mathbf{H} = \left(\frac{\mu}{\mu_0} - 1\right)\mathbf{H} = \left(\frac{\mu}{\mu_0} - 1\right)\frac{J_f(r^2 - r_1^2)}{2r}\,\hat{\mathbf{\phi}}$$

**磁化体电流密度** $\mathbf{J}_m = \nabla\times\mathbf{M}$：

在柱坐标中，$\mathbf{M} = M_\phi(r)\hat{\mathbf{\phi}}$（只有 $\phi$ 分量，且只与 $r$ 有关）：
$$\nabla\times\mathbf{M} = \frac{1}{r}\frac{\partial}{\partial r}(r M_\phi)\,\hat{\mathbf{z}}$$

$$r M_\phi = \left(\frac{\mu}{\mu_0} - 1\right)\frac{J_f(r^2 - r_1^2)}{2}$$
$$\frac{\partial}{\partial r}(r M_\phi) = \left(\frac{\mu}{\mu_0} - 1\right)J_f r$$

$$\mathbf{J}_m = \left(\frac{\mu}{\mu_0} - 1\right)J_f\,\hat{\mathbf{z}} \quad (r_1 < r < r_2)$$

均匀分布！

**磁化面电流密度** $\mathbf{\alpha}_m = \mathbf{M}\times\hat{\mathbf{n}}$：

- **内表面** $r = r_1$：$\hat{\mathbf{n}} = -\hat{\mathbf{r}}$（指向空腔），$M_\phi(r_1) = 0$ → $\mathbf{\alpha}_m(r_1) = \mathbf{0}$

- **外表面** $r = r_2$：$\hat{\mathbf{n}} = \hat{\mathbf{r}}$，
  $$M_\phi(r_2) = \left(\frac{\mu}{\mu_0} - 1\right)\frac{J_f(r_2^2 - r_1^2)}{2r_2}$$
  $$\mathbf{\alpha}_m = M_\phi(r_2)\,\hat{\mathbf{\phi}}\times\hat{\mathbf{r}} = -M_\phi(r_2)\,\hat{\mathbf{z}}$$

  即沿 $-\hat{\mathbf{z}}$ 方向（与自由电流方向相反）。

---

#### 要点总结

这是典型的"圆柱 + 轴向电流"对称问题。关键步骤：
1. 安培环路定理求 $H$（只看自由电流 $I_f$）
2. $\mathbf{B} = \mu\mathbf{H}$（各区域内用各区域的 $\mu$）
3. $\mathbf{M} = \mathbf{B}/\mu_0 - \mathbf{H}$，然后 $\mathbf{J}_m = \nabla\times\mathbf{M}$，$\mathbf{\alpha}_m = \mathbf{M}\times\hat{\mathbf{n}}$

### 习题 6：极化电荷与自由电荷的关系

**题目**：证明均匀介质内部的极化电荷体密度 $\rho_p$ 总是等于自由电荷体密度 $\rho_f$ 的 $-(1-\varepsilon_0/\varepsilon)$ 倍。

---

#### 详细解算

这是习题4结论的推广证明。

均匀各向同性介质：$\mathbf{D} = \varepsilon\mathbf{E}$，$\varepsilon$ 为常数。

极化强度：$\mathbf{P} = \mathbf{D} - \varepsilon_0\mathbf{E} = (\varepsilon - \varepsilon_0)\mathbf{E}$

极化体电荷密度：
$$\rho_p = -\nabla\cdot\mathbf{P} = -\nabla\cdot[(\varepsilon - \varepsilon_0)\mathbf{E}] = -(\varepsilon - \varepsilon_0)\nabla\cdot\mathbf{E}$$

（因介质均匀，$\varepsilon$ 为常量，可提出）

由麦克斯韦方程 $\nabla\cdot\mathbf{E} = \frac{\rho_f + \rho_p}{\varepsilon_0}$（总电荷产生 $\mathbf{E}$），但更直接地：

由 $\nabla\cdot\mathbf{D} = \rho_f$，且 $\mathbf{D} = \varepsilon\mathbf{E}$：
$$\nabla\cdot(\varepsilon\mathbf{E}) = \varepsilon\nabla\cdot\mathbf{E} = \rho_f \quad\Rightarrow\quad \nabla\cdot\mathbf{E} = \frac{\rho_f}{\varepsilon}$$

代入：
$$\boxed{\rho_p = -(\varepsilon - \varepsilon_0)\frac{\rho_f}{\varepsilon} = -\left(1 - \frac{\varepsilon_0}{\varepsilon}\right)\rho_f}$$

证毕。

---

#### 要点总结

- 若介质中无自由电荷（$\rho_f = 0$），则 $\rho_p = 0$——均匀极化不会在均匀介质内部产生极化体电荷
- 极化的体电荷只出现在 $\rho_f \neq 0$ 或介质不均匀的地方
- 真空极限 $\varepsilon \to \varepsilon_0$：$\rho_p \to 0$（真空不能极化）

### 习题 7：平行板电容器——两层介质

**题目**：平行板电容器内有两层介质，厚度分别为 $l_1$ 和 $l_2$，电容率为 $\varepsilon_1$ 和 $\varepsilon_2$，接电动势为 $\mathcal{E}$ 的电池。求：
(1) 电容器两板上的自由电荷面密度 $\omega_f$；
(2) 介质分界面上的自由电荷面密度 $\omega_f$。

---

#### 思路分析

这是**一维问题**。电场方向垂直于极板。在两层介质中 $\mathbf{D}$ 连续（分界面上无自由面电荷时），但从一层到另一层 $\mathbf{E}$ 不连续（因为 $\varepsilon$ 不同）。

---

#### 详细解算

设上极板（接正极）自由电荷面密度为 $+\omega_f$，下极板为 $-\omega_f$。

在介质 1 中：$D_1 = \omega_f$（由高斯定理，$D$ 的法向分量等于自由面电荷密度），$E_1 = D_1/\varepsilon_1 = \omega_f/\varepsilon_1$。

在介质 2 中：$D_2 = \omega_f$（同样），$E_2 = \omega_f/\varepsilon_2$。

**（1）求 $\omega_f$**

电池电动势等于电压之和：
$$\mathcal{E} = E_1 l_1 + E_2 l_2 = \omega_f\left(\frac{l_1}{\varepsilon_1} + \frac{l_2}{\varepsilon_2}\right)$$

$$\boxed{\omega_f = \frac{\mathcal{E}}{\frac{l_1}{\varepsilon_1} + \frac{l_2}{\varepsilon_2}} = \frac{\varepsilon_1\varepsilon_2\,\mathcal{E}}{\varepsilon_2 l_1 + \varepsilon_1 l_2}}$$

**（2）介质分界面上的自由电荷面密度**

在分界面上，如果介质中没有自由电荷体密度，则 $\mathbf{D}$ 的法向分量连续：$D_{1n} = D_{2n}$。这要求**分界面上无自由面电荷**。

但如果分界面上有自由面电荷 $\sigma_f$，则：
$$D_{2n} - D_{1n} = \sigma_f$$

在本题中，$D_{1n} = D_{2n} = \omega_f$（两边相等），所以：
$$\boxed{\sigma_f = 0}$$

即介质分界面上无自由面电荷。这是由于两介质都是绝缘体，没有自由电荷移动到分界面上。

---

#### 要点总结

| 物理量 | 介质 1 | 介质 2 |
|--------|--------|--------|
| $D$ | $\omega_f$ | $\omega_f$（连续） |
| $E$ | $\omega_f/\varepsilon_1$ | $\omega_f/\varepsilon_2$ |
| 分压 | $\omega_f l_1/\varepsilon_1$ | $\omega_f l_2/\varepsilon_2$ |

类似于两个串联电容：$C_1 = \varepsilon_1 S/l_1$，$C_2 = \varepsilon_2 S/l_2$，总电容 $C = (C_1^{-1} + C_2^{-1})^{-1}$。

### 习题 8：电场线在界面上的折射

**题目**：证明：
(1) 两种绝缘介质分界面上无自由面电荷时，$\displaystyle\frac{\tan\theta_2}{\tan\theta_1} = \frac{\varepsilon_2}{\varepsilon_1}$
(2) 两种导电介质有恒定电流时，$\displaystyle\frac{\tan\theta_2}{\tan\theta_1} = \frac{\sigma_2}{\sigma_1}$

其中 $\theta$ 为电场线与界面法线的夹角。

---

#### 详细解算

**（1）绝缘介质分界面（无自由面电荷）**

边值关系：
- $D_{1n} = D_{2n}$（无自由面电荷，$\mathbf{D}$ 法向连续）
- $E_{1t} = E_{2t}$（$\mathbf{E}$ 切向连续）

$$\tan\theta_1 = \frac{E_{1t}}{E_{1n}},\quad \tan\theta_2 = \frac{E_{2t}}{E_{2n}}$$

$$\frac{\tan\theta_2}{\tan\theta_1} = \frac{E_{2t}/E_{2n}}{E_{1t}/E_{1n}} = \frac{E_{2t}}{E_{1t}}\cdot\frac{E_{1n}}{E_{2n}} = 1\cdot\frac{D_{1n}/\varepsilon_1}{D_{2n}/\varepsilon_2}$$

由于 $D_{1n} = D_{2n}$：
$$\boxed{\frac{\tan\theta_2}{\tan\theta_1} = \frac{\varepsilon_2}{\varepsilon_1}}$$

---

**（2）导电介质分界面（恒定电流）**

在恒定电流情况下，电荷守恒要求 $\mathbf{J}$ 的法向分量连续（否则电荷会在分界面上积累，破坏稳恒条件）：
$$J_{1n} = J_{2n}$$

而切向分量满足 $E_{1t} = E_{2t}$。由欧姆定律 $\mathbf{J} = \sigma\mathbf{E}$：
$$J_{1n} = \sigma_1 E_{1n},\quad J_{2n} = \sigma_2 E_{2n}$$
$$\sigma_1 E_{1n} = \sigma_2 E_{2n} \quad\Rightarrow\quad \frac{E_{1n}}{E_{2n}} = \frac{\sigma_2}{\sigma_1}$$

$$\frac{\tan\theta_2}{\tan\theta_1} = \frac{E_{2t}/E_{2n}}{E_{1t}/E_{1n}} = \frac{E_{2t}}{E_{1t}}\cdot\frac{E_{1n}}{E_{2n}} = 1\cdot\frac{\sigma_2}{\sigma_1}$$
$$\boxed{\frac{\tan\theta_2}{\tan\theta_1} = \frac{\sigma_2}{\sigma_1}}$$

---

#### 要点总结

| 情况 | 法向连续量 | 结果 |
|------|-----------|------|
| 绝缘介质 | $D_n$（无自由面电荷时） | $\tan\theta_2/\tan\theta_1 = \varepsilon_2/\varepsilon_1$ |
| 导电介质（恒流） | $J_n$（电荷守恒） | $\tan\theta_2/\tan\theta_1 = \sigma_2/\sigma_1$ |

### 习题 9：导体表面电场线方向

**题目**：证明在绝缘介质与导体的分界面上：(1) 静电情况下导体外表面的电场线总是垂直于导体表面；(2) 恒定电流情况下导体内部的电场线总是平行于导体表面。

---

#### 详细解算

**（1）静电情况**

静电平衡时导体是等势体，表面是等势面。电场线必须垂直于等势面。

从边值关系看：导体内部 $\mathbf{E} = \mathbf{0}$（静电平衡）。表面外侧切向电场连续：$E_t^{\text{外}} = E_t^{\text{内}} = 0$。所以外侧电场只有法向分量：
$$\mathbf{E}_{\text{外}} = E_n\hat{\mathbf{n}},\quad E_n = \sigma_f/\varepsilon_0$$

电场线垂直于导体表面。

---

**（2）恒定电流情况（导体内部）**

在恒定电流情况下，导体内部的电流必须沿导体表面流动（因为电流不能流到绝缘介质中去）。即 $\mathbf{J}$ 的法向分量为零：
$$J_n^{\text{内}} = 0$$

由欧姆定律 $\mathbf{J} = \sigma\mathbf{E}$（导体内部），$\mathbf{E}$ 也只有切向分量：
$$\mathbf{E}_{\text{内}} \parallel \text{表面}$$

即导体内部电场线平行于导体表面。

同时，在恒流情况下导体内部可以有电场（推动电流），不再是静电平衡的 $\mathbf{E} = \mathbf{0}$。

---

#### 要点总结

| | 导体内电场 | 表面电场线方向 |
|------|-----------|--------------|
| 静电 | $\mathbf{E} = \mathbf{0}$ | 垂直于表面 |
| 恒流 | $\mathbf{E} = \mathbf{J}/\sigma \neq \mathbf{0}$ | 平行于表面（法向分量为零） |

## 第二章 静电场

### 习题 10：均匀极化介质球——全面的多极分析（⭐考试高频）

**题目**：半径为 $R$ 的电介质球，极化强度 $\mathbf{P} = K\mathbf{r}$（$K$ 为常数），电容率为 $\varepsilon$。
(1) 计算束缚电荷的体密度和面密度；
(2) 计算自由电荷体密度；
(3) 计算球外和球内的电势；
(4) 求该带电介质球产生的静电场总能量。

---

#### 思路分析

这是一个关键题：$\mathbf{P} = K\mathbf{r}$ 意味着极化从球心向外线性增大（类似均匀带电球的电场形式）。需要先分析束缚电荷，再由高斯定理找到自由电荷，最后求电势和能量。

---

#### 详细解算

**（1）束缚电荷**

极化体电荷：
$$\rho_p = -\nabla\cdot\mathbf{P} = -\nabla\cdot(K\mathbf{r}) = -K\nabla\cdot\mathbf{r} = -3K$$

（均匀分布！虽然 $\mathbf{P}$ 随 $r$ 增大，但其散度是常数。）

极化面电荷（外表面 $r = R$）：
$$\sigma_p = \mathbf{P}\cdot\hat{\mathbf{n}}\big|_{r=R} = K\mathbf{r}\cdot\hat{\mathbf{r}}\big|_{r=R} = KR$$


**（2）自由电荷体密度**

介质中的高斯定理：$\nabla\cdot\mathbf{D} = \rho_f$，其中 $\mathbf{D} = \varepsilon_0\mathbf{E} + \mathbf{P} = \varepsilon\mathbf{E}$。

极化强度 $\mathbf{P} = \varepsilon_0\chi_e\mathbf{E}$，其中 $\chi_e$ 为极化率：
$$\mathbf{P} = (\varepsilon - \varepsilon_0)\mathbf{E}$$

由 $\mathbf{P} = K\mathbf{r}$：
$$\mathbf{E} = \frac{\mathbf{P}}{\varepsilon - \varepsilon_0} = \frac{K}{\varepsilon - \varepsilon_0}\mathbf{r}$$

则：
$$\mathbf{D} = \varepsilon\mathbf{E} = \frac{\varepsilon K}{\varepsilon - \varepsilon_0}\mathbf{r}$$

$$\rho_f = \nabla\cdot\mathbf{D} = \nabla\cdot\left(\frac{\varepsilon K}{\varepsilon - \varepsilon_0}\mathbf{r}\right) = \frac{\varepsilon K}{\varepsilon - \varepsilon_0}\nabla\cdot\mathbf{r} = \frac{3\varepsilon K}{\varepsilon - \varepsilon_0}$$

（也是均匀的！）

---

**（3）电势**

总电荷密度：$\rho_{\text{总}} = \rho_f + \rho_p = \frac{3\varepsilon K}{\varepsilon - \varepsilon_0} - 3K = \frac{3\varepsilon_0 K}{\varepsilon - \varepsilon_0}$

球内总电荷：$Q_{\text{总}} = \frac{4\pi}{3}R^3 \cdot \frac{3\varepsilon_0 K}{\varepsilon - \varepsilon_0} = \frac{4\pi\varepsilon_0 K R^3}{\varepsilon - \varepsilon_0}$

由高斯定理，球内电场（$r < R$）：
$$E = \frac{Q_{\text{内}}}{4\pi\varepsilon_0 r^2} = \frac{\rho_{\text{总}}\cdot\frac{4\pi}{3}r^3}{4\pi\varepsilon_0 r^2} = \frac{\rho_{\text{总}} r}{3\varepsilon_0} = \frac{Kr}{\varepsilon - \varepsilon_0}$$

与前面 $\mathbf{E} = \frac{K}{\varepsilon-\varepsilon_0}\mathbf{r}$ 一致。

球外电场（$r > R$）：等效于点电荷 $Q_{\text{总}}$ 在球心的电场：
$$E = \frac{Q_{\text{总}}}{4\pi\varepsilon_0 r^2} = \frac{K R^3}{(\varepsilon - \varepsilon_0)r^2}$$

**球内电势**（$r < R$）：
$$\varphi_{\text{in}} = \int_r^\infty E\,dr = \int_r^R \frac{K r'}{\varepsilon-\varepsilon_0}dr' + \int_R^\infty \frac{K R^3}{(\varepsilon-\varepsilon_0)r'^2}dr'$$
$$= \frac{K}{2(\varepsilon-\varepsilon_0)}(R^2 - r^2) + \frac{K R^2}{\varepsilon-\varepsilon_0}$$
$$= \frac{K}{2(\varepsilon-\varepsilon_0)}(3R^2 - r^2)$$

**球外电势**（$r > R$）：
$$\varphi_{\text{out}} = \frac{K R^3}{(\varepsilon-\varepsilon_0)r}$$

验证：$r = R$ 时：$\varphi_{\text{in}} = \frac{K R^2}{\varepsilon-\varepsilon_0} = \varphi_{\text{out}}$，连续。

---

**（4）静电场总能量**

$$W = \frac{1}{2}\int \mathbf{D}\cdot\mathbf{E}\,dV$$

球内：$\mathbf{D}\cdot\mathbf{E} = \varepsilon E^2 = \varepsilon\left(\frac{Kr}{\varepsilon-\varepsilon_0}\right)^2$
$$W_{\text{in}} = \frac{1}{2}\int_0^R \varepsilon\left(\frac{Kr}{\varepsilon-\varepsilon_0}\right)^2 \cdot 4\pi r^2\,dr = \frac{2\pi\varepsilon K^2}{(\varepsilon-\varepsilon_0)^2}\int_0^R r^4\,dr = \frac{2\pi\varepsilon K^2 R^5}{5(\varepsilon-\varepsilon_0)^2}$$

球外：$\mathbf{D}\cdot\mathbf{E} = \varepsilon_0 E^2 = \varepsilon_0\left(\frac{K R^3}{(\varepsilon-\varepsilon_0)r^2}\right)^2$
$$W_{\text{out}} = \frac{1}{2}\int_R^\infty \varepsilon_0\frac{K^2 R^6}{(\varepsilon-\varepsilon_0)^2 r^4}\cdot 4\pi r^2\,dr = \frac{2\pi\varepsilon_0 K^2 R^6}{(\varepsilon-\varepsilon_0)^2}\int_R^\infty\frac{dr}{r^2} = \frac{2\pi\varepsilon_0 K^2 R^5}{(\varepsilon-\varepsilon_0)^2}$$

$$W = \frac{2\pi K^2 R^5}{(\varepsilon-\varepsilon_0)^2}\left(\frac{\varepsilon}{5} + \varepsilon_0\right)$$

---

#### 要点总结

$\mathbf{P} = K\mathbf{r}$ 意味着：
- 束缚体电荷均匀：$\rho_p = -3K$
- 自由体电荷也均匀：$\rho_f = 3\varepsilon K/(\varepsilon-\varepsilon_0)$
- 球内场与 $r$ 成正比（类似均匀带电球内部）
- 球外场等效于点电荷

### 习题 11：均匀外电场中的导体球——分离变量法（⭐必考）

**题目**：在均匀外电场 $\mathbf{E}_0 = E_0\hat{\mathbf{z}}$ 中置入半径 $R_0$ 的导体球，用分离变量法求两种情况下的电势：(1) 导体球接电池与地保持电势差 $\Phi_0$；(2) 导体球带总电荷 $Q$。

---

#### 思路分析

这是**轴对称分离变量法的标准模板**。外加均匀电场沿 $z$ 轴，电势 $\varphi_0 = -E_0 z = -E_0 r\cos\theta = -E_0 r P_1(\cos\theta)$。

导体球表面是等势面，球外无电荷 → 球外电势满足拉普拉斯方程。通解：
$$\varphi(r,\theta) = \sum_{n=0}^\infty \left(A_n r^n + \frac{B_n}{r^{n+1}}\right)P_n(\cos\theta)$$

利用边界条件定系数。

---

#### 详细解算

**情况 (1)：导体球接电池，与地保持电势差 $\Phi_0$**

边界条件：
- $r \to \infty$：$\varphi \to -E_0 r\cos\theta$（外场主导）
- $r = R_0$：$\varphi = \Phi_0$（等势面）

**$r\to\infty$ 条件分析**：
$$\varphi(r\to\infty,\theta) = \sum A_n r^n P_n(\cos\theta) \quad (\text{高阶项 } r^{-(n+1)}\to 0)$$

必须等于 $-E_0 r P_1(\cos\theta)$。因此：
- $A_1 = -E_0$
- 其余 $A_n = 0$（$n \neq 1$）

**$r = R_0$ 条件分析**：
$$\varphi(R_0,\theta) = -E_0 R_0\cos\theta + \sum_{n=0}^\infty \frac{B_n}{R_0^{n+1}}P_n(\cos\theta) = \Phi_0$$

$\Phi_0$ 是常数 → 必须等于 $P_0(\cos\theta) = 1$ 项的系数。

- $n = 0$：$\frac{B_0}{R_0} = \Phi_0$ → $B_0 = \Phi_0 R_0$
- $n = 1$：$-E_0 R_0 + \frac{B_1}{R_0^2} = 0$ → $B_1 = E_0 R_0^3$
- $n \geq 2$：$B_n = 0$（含 $P_n(\cos\theta)$ 的项在等势面上必须为零）

**最终电势**：
$$\boxed{\varphi(r,\theta) = \frac{\Phi_0 R_0}{r} - E_0 r\cos\theta + \frac{E_0 R_0^3}{r^2}\cos\theta \quad (r \geq R_0)}$$

---

**情况 (2)：导体球带总电荷 $Q$（孤立导体球）**

边界条件：
- $r \to \infty$：$\varphi \to -E_0 r\cos\theta + \frac{Q}{4\pi\varepsilon_0 r}$（远处叠加外场 + 点电荷势）
- $r = R_0$：$\varphi = \text{constant}$（等势面）

由 $r\to\infty$ 条件：
- $A_1 = -E_0$（外场）
- $A_0$ 项对应 $\frac{Q}{4\pi\varepsilon_0 r}$ 的形式 → 其实是 $n=0$ 的 $B_0$ 项！因为通解中 $n=0$ 的 $B_0/r$ 才是 $1/r$ 项。

所以 $B_0$ 由总电荷决定：$\oint_S \mathbf{E}\cdot d\mathbf{S} = Q/\varepsilon_0$。

在通解中，$n=0$ 项的电场：
$$E_r^{(0)} = -\frac{\partial}{\partial r}\left(\frac{B_0}{r}\right) = \frac{B_0}{r^2}$$

$$\oint_S \frac{B_0}{r^2}\cdot r^2 d\Omega = 4\pi B_0 = \frac{Q}{\varepsilon_0} \quad\Rightarrow\quad B_0 = \frac{Q}{4\pi\varepsilon_0}$$

$r=R_0$ 等势面条件：
$$\varphi(R_0,\theta) = \frac{Q}{4\pi\varepsilon_0 R_0} - E_0 R_0\cos\theta + \frac{B_1}{R_0^2}\cos\theta + \cdots = \text{const}$$

含 $\cos\theta$ 的项必须为零：$-E_0 R_0 + B_1/R_0^2 = 0$ → $B_1 = E_0 R_0^3$。

其他 $B_n = 0$。

**最终电势**：
$$\boxed{\varphi(r,\theta) = \frac{Q}{4\pi\varepsilon_0 r} - E_0 r\cos\theta + \frac{E_0 R_0^3}{r^2}\cos\theta \quad (r \geq R_0)}$$

---

#### 要点总结

| 项 | 物理含义 |
|----|----------|
| $-E_0 r\cos\theta$ | 外加均匀电场 |
| $E_0 R_0^3\cos\theta/r^2$ | 导体球感应的**偶极子场** |
| $\Phi_0 R_0/r$ 或 $Q/(4\pi\varepsilon_0 r)$ | 导体球净电荷的贡献 |

导体球在外电场中的感应偶极矩：$\mathbf{p} = 4\pi\varepsilon_0 E_0 R_0^3\hat{\mathbf{z}}$。

### 习题 12：介质球中心的自由电偶极子（⭐综合性）

**题目**：均匀介质球（电容率 $\varepsilon_1$）的中心置一自由电偶极子 $\mathbf{p}_f$，球外充满另一种介质（电容率 $\varepsilon_2$）。求空间各点的电势和极化电荷分布。

---

#### 思路分析

这是含点源的边值问题。在球内，电势 = 偶极子场（特解）+ 拉普拉斯通解（齐次解）；球外只有通解。用 $r=R$ 处的边值关系定系数。

---

#### 详细解算

**设偶极子沿 $z$ 轴**：$\mathbf{p}_f = p_f\hat{\mathbf{z}}$。

偶极子产生的电势（特解）：
$$\varphi_{\text{dipole}} = \frac{\mathbf{p}_f\cdot\mathbf{r}}{4\pi\varepsilon_1 r^3} = \frac{p_f\cos\theta}{4\pi\varepsilon_1 r^2} \quad (\text{在 }\varepsilon_1\text{ 介质中})$$

> 💡 **注意**：偶极子位于 **介质 $\varepsilon_1$** 中，不是真空中！所以分母是 $4\pi\varepsilon_1$。

**通解设 置**：

球内（$r < R$）：
$$\varphi_{\text{in}}(r,\theta) = \frac{p_f\cos\theta}{4\pi\varepsilon_1 r^2} + \sum_{n=0}^\infty A_n r^n P_n(\cos\theta)$$

（注意 $r^n$ 项：因包含原点，不能有 $r^{-(n+1)}$ 发散项）

球外（$r > R$）：
$$\varphi_{\text{out}}(r,\theta) = \sum_{n=0}^\infty \frac{B_n}{r^{n+1}} P_n(\cos\theta)$$

（$r\to\infty$ 时 $\varphi\to 0$，无 $r^n$ 项）

**边界条件**（$r = R$）：

(1) 电势连续：$\varphi_{\text{in}}(R,\theta) = \varphi_{\text{out}}(R,\theta)$

(2) $\mathbf{D}$ 法向连续（无自由面电荷）：
$$\varepsilon_1\frac{\partial\varphi_{\text{in}}}{\partial r}\bigg|_{r=R} = \varepsilon_2\frac{\partial\varphi_{\text{out}}}{\partial r}\bigg|_{r=R}$$

**展开匹配** $P_n(\cos\theta)$ 系数。

偶极子项属于 $\propto P_1(\cos\theta)$。设 $A_1, B_1$ 非零，其余 $n \neq 1$ 的系数为零（由角向正交性，每项独立满足边界条件）。

在 $r = R$：

电势连续：
$$\frac{p_f}{4\pi\varepsilon_1 R^2} + A_1 R = \frac{B_1}{R^2}$$

$\mathbf{D}$ 法向连续：
$$\varepsilon_1\left(-\frac{2p_f}{4\pi\varepsilon_1 R^3} + A_1\right) = \varepsilon_2\left(-\frac{2B_1}{R^3}\right)$$

化简：
$$-\frac{p_f}{2\pi R^3} + \varepsilon_1 A_1 = -\frac{2\varepsilon_2 B_1}{R^3}$$

联立求解：
$$A_1 = \frac{2(\varepsilon_1 - \varepsilon_2)}{2\varepsilon_2 + \varepsilon_1}\cdot\frac{p_f}{4\pi\varepsilon_1 R^3}$$
$$B_1 = \frac{3\varepsilon_1}{2\varepsilon_2 + \varepsilon_1}\cdot\frac{p_f}{4\pi\varepsilon_1}$$

**最终结果**：

球内：
$$\varphi_{\text{in}} = \frac{p_f\cos\theta}{4\pi\varepsilon_1}\left[\frac{1}{r^2} + \frac{2(\varepsilon_1-\varepsilon_2)}{2\varepsilon_2+\varepsilon_1}\cdot\frac{r}{R^3}\right]$$

球外：
$$\varphi_{\text{out}} = \frac{3p_f\cos\theta}{4\pi(2\varepsilon_2+\varepsilon_1)r^2}$$

> 极化电荷分布在球表面，由 $\sigma_p = \mathbf{P}\cdot\hat{\mathbf{n}}$ 计算。

---

#### 要点总结

当 $\varepsilon_2 = \varepsilon_1$（均匀无限介质）时：$A_1 = 0$，$B_1 = p_f/(4\pi\varepsilon_1)$，退化为无限均匀介质中偶极子的场。介质差异导致界面极化电荷，产生反射场（$A_n r^n$ 项）。

### 习题 13：导体球壳中的电偶极子

**题目**：空心导体球壳内外半径为 $R_1$ 和 $R_2$，球中心置一偶极子 $\mathbf{p}$，球壳带电 $Q$。求空间各点电势和电荷分布。

---

#### 详细解算

**区域划分**：
- 区域 I：$r < R_1$（球壳内空腔）
- 区域 II：$R_1 < r < R_2$（导体壳内部）
- 区域 III：$r > R_2$（球壳外）

**区域 II（导体内部）**：$\mathbf{E} = \mathbf{0}$，电势为常数 $\varphi_{\text{壳}}$。

**区域 I（$r < R_1$）**：

偶极子 $\mathbf{p} = p\hat{\mathbf{z}}$ 在球心。空腔内表面有感应电荷（非均匀分布），在腔内产生一个**均匀电场**的响应势。

由分离变量法（轴对称），腔内电势 = 偶极子势 + 由内表面感应电荷产生的调和函数：
$$\varphi_{\text{I}}(r,\theta) = \frac{p\cos\theta}{4\pi\varepsilon_0 r^2} + \sum_{n=0}^\infty A_n r^n P_n(\cos\theta)$$

在 $r = R_1$：$\varphi_{\text{I}} = \varphi_{\text{壳}}$（等势面）。

$\varphi_{\text{壳}}$ 为常数 → 含 $P_n$（$n\geq 1$）的项在 $r=R_1$ 处必须抵消：
$$\frac{p}{4\pi\varepsilon_0 R_1^2} + A_1 R_1 = 0 \quad\Rightarrow\quad A_1 = -\frac{p}{4\pi\varepsilon_0 R_1^3}$$
其余 $A_n = 0$（$n\geq 2$），$A_0 = \varphi_{\text{壳}}$。

所以：
$$\boxed{\varphi_{\text{I}} = \frac{p\cos\theta}{4\pi\varepsilon_0}\left(\frac{1}{r^2} - \frac{r}{R_1^3}\right) + \varphi_{\text{壳}}}$$

**区域 III（$r > R_2$）**：

球壳外表面均匀带电 $Q$，产生球对称场：
$$\boxed{\varphi_{\text{III}} = \frac{Q}{4\pi\varepsilon_0 r}}$$

球壳电势：$\varphi_{\text{壳}} = \varphi_{\text{III}}(R_2) = \dfrac{Q}{4\pi\varepsilon_0 R_2}$。

**电荷分布**：

内表面感应电荷面密度（$r = R_1$）：
$$\sigma_{\text{内}} = -\varepsilon_0\frac{\partial\varphi_{\text{I}}}{\partial r}\bigg|_{r=R_1} = -\frac{3p\cos\theta}{4\pi R_1^3}$$

（总电荷为零，但非均匀分布，形成偶极分布）

外表面电荷（$r = R_2$）：总电荷 $Q$ 均匀分布，$\sigma_{\text{外}} = Q/(4\pi R_2^2)$。

---

#### 要点总结

导体壳的屏蔽效应：腔内的偶极子不影响球壳外部（球壳外只看到总电荷 $Q$），球壳内表面感应出非均匀电荷来"抵消"偶极子的影响。

## 第三章 静磁场

### 习题 16：无穷长线电流在介质分界面的磁场

**题目**：无穷长线电流 $I$ 沿 $z$ 轴，$z<0$ 空间充满磁导率 $\mu$ 的均匀介质，$z>0$ 区域为真空。求 $\mathbf{B}$ 和磁化电流分布。

---

#### 详细解算

**对称性分析**：无穷长直线电流的磁场只有 $\phi$ 方向分量，且由轴对称性 $B_\phi$ 只是 $r$ 的函数（与 $z$ 无关，上下两个半空间分别均匀）。

**用安培环路定理（含 $\mathbf{H}$）**：

$$\oint_L \mathbf{H}\cdot d\mathbf{l} = H_\phi(r)\cdot 2\pi r = I_f$$
$$H_\phi(r) = \frac{I}{2\pi r}\quad (\text{全空间相同！})$$

因为安培环路定理中 $\mathbf{H}$ 的环量只取决于包围的**自由电流**，与介质无关。

**磁感应强度**：

- $z > 0$（真空）：$\mathbf{B} = \mu_0\mathbf{H} = \dfrac{\mu_0 I}{2\pi r}\,\hat{\mathbf{\phi}}$
- $z < 0$（介质）：$\mathbf{B} = \mu\mathbf{H} = \dfrac{\mu I}{2\pi r}\,\hat{\mathbf{\phi}}$

**磁化强度**（$z < 0$ 区）：
$$\mathbf{M} = \frac{\mathbf{B}}{\mu_0} - \mathbf{H} = \left(\frac{\mu}{\mu_0} - 1\right)\frac{I}{2\pi r}\,\hat{\mathbf{\phi}}$$

**磁化体电流**：$\mathbf{J}_m = \nabla\times\mathbf{M}$。

在柱坐标中，$\mathbf{M} = M_\phi(r)\hat{\mathbf{\phi}}$：
$$(\nabla\times\mathbf{M})_z = \frac{1}{r}\frac{\partial}{\partial r}(r M_\phi) = \frac{1}{r}\frac{\partial}{\partial r}\left[\left(\frac{\mu}{\mu_0} - 1\right)\frac{I}{2\pi}\right] = 0 \quad (r \neq 0)$$

（$r M_\phi$ 为常数，导数为零）——**无体磁化电流**。

**磁化面电流**（$z=0$ 平面）：

$\mathbf{\alpha}_m = \mathbf{M}\times\hat{\mathbf{n}}$，$\hat{\mathbf{n}} = \hat{\mathbf{z}}$（从介质指向真空）。

$z=0^-$ 处 $\mathbf{M} = \left(\frac{\mu}{\mu_0}-1\right)\frac{I}{2\pi r}\hat{\mathbf{\phi}}$。

$$\mathbf{\alpha}_m = M_\phi\hat{\mathbf{\phi}}\times\hat{\mathbf{z}} = M_\phi\hat{\mathbf{r}} = \left(\frac{\mu}{\mu_0} - 1\right)\frac{I}{2\pi r}\,\hat{\mathbf{r}}$$

——径向流出的面磁化电流！

---

#### 要点总结

| 区域 | $H_\phi$ | $B_\phi$ |
|------|----------|----------|
| $z>0$ | $I/(2\pi r)$ | $\mu_0 I/(2\pi r)$ |
| $z<0$ | $I/(2\pi r)$ | $\mu I/(2\pi r)$ |

$\mathbf{H}$ 是连续的（因为 $I_f$ 是唯一的自由电流源），但 $\mathbf{B}$ 在分界面上不连续（$\mu$ 跳变）。

## 第四章 电磁波的传播

### 习题 17：拍波——相速度与群速度

**题目**：两列振幅相同、偏振方向相同的线偏振平面波，频率分别为 $\omega+d\omega$ 和 $\omega-d\omega$，沿 $z$ 轴传播。
(1) 求合成波，证明振幅不是常数；
(2) 求相位传播速度和振幅传播速度。

---

#### 详细解算

两列波：
$$E_1 = E_0 e^{i[(k+dk)z - (\omega+d\omega)t]}$$
$$E_2 = E_0 e^{i[(k-dk)z - (\omega-d\omega)t]}$$

合成：
$$E = E_1 + E_2 = E_0 e^{i(kz-\omega t)}\left[e^{i(dk\cdot z - d\omega\cdot t)} + e^{-i(dk\cdot z - d\omega\cdot t)}\right]$$

利用欧拉公式 $e^{i\theta} + e^{-i\theta} = 2\cos\theta$：
$$\boxed{E(z,t) = 2E_0\cos(dk\cdot z - d\omega\cdot t) \cdot e^{i(kz-\omega t)}}$$

**解读**：
- $\cos(dk\cdot z - d\omega\cdot t)$ 是一个**慢变包络**（调制振幅）
- $e^{i(kz-\omega t)}$ 是**快变载波**

**相速度**（载波等相位面的传播速度）：
等相位条件：$kz - \omega t = \text{const}$
$$v_p = \frac{dz}{dt} = \frac{\omega}{k}$$

**群速度**（包络等值面的传播速度）：
等包络条件：$dk\cdot z - d\omega\cdot t = \text{const}$
$$v_g = \frac{dz}{dt} = \frac{d\omega}{dk}$$

在真空中 $\omega = ck$，$v_p = v_g = c$。在色散介质中 $v_p \neq v_g$。

### 习题 18：平面电磁波在介质界面的反射（菲涅尔公式计算）

**题目**：平面电磁波以 $\theta=45^\circ$ 从真空入射到 $\varepsilon_r=2$ 的介质，电场垂直于入射面。求反射系数和折射系数。

---

#### 详细解算

**第一步：判断偏振类型**

"电场垂直于入射面" = **S 偏振（TE 波）**。

**第二步：确定参数**

$n_1 = 1$（真空），$n_2 = \sqrt{\varepsilon_r} = \sqrt{2}$，$\theta_1 = 45^\circ$。

Snell 定律：
$$\sin\theta_2 = \frac{n_1}{n_2}\sin\theta_1 = \frac{1}{\sqrt{2}}\cdot\frac{\sqrt{2}}{2} = \frac{1}{2}$$
$$\theta_2 = 30^\circ$$

**第三步：S 波菲涅尔公式**

$$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2}$$
$$t_s = \frac{2n_1\cos\theta_1}{n_1\cos\theta_1 + n_2\cos\theta_2}$$

代入数值：
$$n_1\cos\theta_1 = 1\cdot\frac{\sqrt{2}}{2} = \frac{\sqrt{2}}{2}$$
$$n_2\cos\theta_2 = \sqrt{2}\cdot\frac{\sqrt{3}}{2} = \frac{\sqrt{6}}{2}$$

$$r_s = \frac{\sqrt{2}/2 - \sqrt{6}/2}{\sqrt{2}/2 + \sqrt{6}/2} = \frac{1-\sqrt{3}}{1+\sqrt{3}} = \frac{1-\sqrt{3}}{1+\sqrt{3}}\cdot\frac{1-\sqrt{3}}{1-\sqrt{3}} = \frac{4-2\sqrt{3}}{-2} = \sqrt{3}-2 \approx -0.268$$

$$t_s = \frac{2\cdot\sqrt{2}/2}{\sqrt{2}/2 + \sqrt{6}/2} = \frac{\sqrt{2}}{(\sqrt{2}+\sqrt{6})/2} = \frac{2\sqrt{2}}{\sqrt{2}+\sqrt{6}} = \frac{2}{1+\sqrt{3}} \approx 0.732$$

**第四步：反射率和透射率**

反射率：$R = |r_s|^2 = (\sqrt{3}-2)^2 = 7-4\sqrt{3} \approx 0.072$

透射率：$T = 1 - R = 4\sqrt{3}-6 \approx 0.928$（无吸收时能量守恒）

---

#### 要点总结

| 项目 | 值 |
|------|-----|
| 入射角 | $45^\circ$ |
| 折射角 | $30^\circ$ |
| 反射系数 $r_s$ | $\sqrt{3}-2 \approx -0.268$ |
| 透射系数 $t_s$ | $2/(1+\sqrt{3}) \approx 0.732$ |
| 反射率 | $\approx 7.2\%$ |
| 透射率 | $\approx 92.8\%$ |

负的反射系数意味着反射波有 $\pi$ 相位跃变（从光疏到光密介质）。

### 习题 19：全反射与倏逝波

**题目**：可见光由水（$n_1=1.33$）入射到空气（$n_2=1$），入射角 $60^\circ$。证明发生全反射，求折射波沿表面相速度和透入深度。

---

#### 详细解算

**判断全反射**：

临界角：$\sin\theta_c = \frac{n_2}{n_1} = \frac{1}{1.33} \approx 0.752$，$\theta_c \approx 48.8^\circ$。

$60^\circ > 48.8^\circ$ → **全反射发生**。

**折射波分析**：

Snell 定律在 $\theta_1 > \theta_c$ 时：$\sin\theta_2 = \frac{n_1}{n_2}\sin\theta_1 > 1$ → $\theta_2$ 无实数解。

但这不意味着第二介质中没有场！折射波是倏逝波（evanescent wave）：

$$\cos\theta_2 = \sqrt{1-\sin^2\theta_2} = i\sqrt{\sin^2\theta_2 - 1} = i\kappa$$

折射波波矢：$\mathbf{k}_2 = k_2(\sin\theta_2\,\hat{\mathbf{x}} + \cos\theta_2\,\hat{\mathbf{z}}) = k_2(\sin\theta_2\,\hat{\mathbf{x}} + i\kappa\,\hat{\mathbf{z}})$

折射波场：$\mathbf{E} \propto e^{i(k_2\sin\theta_2\cdot x - \omega t)} \cdot e^{-k_2\kappa z}$

——沿 $x$ 方向传播，沿 $z$ 方向指数衰减。

**沿表面相速度**：
$$v_{\parallel} = \frac{\omega}{k_2\sin\theta_2} = \frac{c}{n_2\sin\theta_2}$$

由于 $n_1\sin\theta_1 = n_2\sin\theta_2$：
$$v_{\parallel} = \frac{c}{n_1\sin\theta_1} = \frac{c}{1.33\cdot\sin 60^\circ} = \frac{c}{1.33\cdot 0.866}$$

**透入深度**（场衰减到 $1/e$ 的距离）：
$$\delta = \frac{1}{k_2\kappa} = \frac{\lambda_0}{2\pi\sqrt{n_1^2\sin^2\theta_1 - n_2^2}}$$

代入 $\lambda_0 = 6.28\times 10^{-5}\text{cm}$，$n_1=1.33$，$n_2=1$，$\theta_1=60^\circ$：
$$\sqrt{1.33^2\cdot 0.866^2 - 1} = \sqrt{1.33^2\cdot 0.75 - 1} = \sqrt{1.327 - 1} \approx 0.571$$
$$\delta \approx \frac{6.28\times 10^{-5}}{2\pi\cdot 0.571} \approx 1.75\times 10^{-5}\text{ cm}$$

（约一个波长的量级）

---

#### 要点总结

| 概念 | 说明 |
|------|------|
| 全反射条件 | $\theta_1 > \theta_c$ 且 $n_1 > n_2$ |
| 倏逝波 | 第二介质中有场，沿垂直方向指数衰减 |
| 穿透深度 $\delta$ | 约波长量级 |
| 表面相速度 | $v_{\parallel} = c/(n_1\sin\theta_1)$ |

### 习题 20：各向异性介质中的电磁波

**题目**：在各向异性介质中，$\mathbf{D}$ 与 $\mathbf{E}$ 不再平行。单色波 $\propto e^{i(\mathbf{k}\cdot\mathbf{r}-\omega t)}$。证明：
(1) $\mathbf{k}\cdot\mathbf{B} = \mathbf{k}\cdot\mathbf{D} = \mathbf{B}\cdot\mathbf{D} = \mathbf{B}\cdot\mathbf{E} = 0$，但一般 $\mathbf{k}\cdot\mathbf{E} \neq 0$
(2) $\mathbf{D} = \frac{k^2\mathbf{E} - (\mathbf{k}\cdot\mathbf{E})\mathbf{k}}{\mu\omega^2}$
(3) 能流 $\mathbf{S}$ 与 $\mathbf{k}$ 一般不同向

---

#### 详细解算

将麦克斯韦方程组中的 $\nabla$ 替换为 $i\mathbf{k}$，$\partial/\partial t$ 替换为 $-i\omega$。

**无源麦克斯韦方程组**（$\rho_f = 0, \mathbf{J}_f = 0$）：

$$\nabla\cdot\mathbf{D} = i\mathbf{k}\cdot\mathbf{D} = 0 \quad\Rightarrow\quad \boxed{\mathbf{k}\cdot\mathbf{D} = 0}$$
（意味着 $\mathbf{D}$ 垂直于传播方向——横波性对 $\mathbf{D}$ 仍成立）

$$\nabla\cdot\mathbf{B} = i\mathbf{k}\cdot\mathbf{B} = 0 \quad\Rightarrow\quad \boxed{\mathbf{k}\cdot\mathbf{B} = 0}$$

$$\nabla\times\mathbf{E} = -\frac{\partial\mathbf{B}}{\partial t} \quad\Rightarrow\quad i\mathbf{k}\times\mathbf{E} = i\omega\mathbf{B} \quad\Rightarrow\quad \boxed{\mathbf{B} = \frac{\mathbf{k}\times\mathbf{E}}{\omega}}$$

由此 $\mathbf{B}\cdot\mathbf{k} = 0$（自动满足），$\mathbf{B}\cdot\mathbf{E} = 0$（因为 $\mathbf{B} \propto \mathbf{k}\times\mathbf{E}$，与 $\mathbf{E}$ 垂直）。

$$\nabla\times\mathbf{H} = \frac{\partial\mathbf{D}}{\partial t} \quad\Rightarrow\quad i\mathbf{k}\times\mathbf{H} = -i\omega\mathbf{D}$$

在非磁介质中 $\mathbf{B} = \mu_0\mathbf{H}$：
$$\mathbf{k}\times\mathbf{B} = -\mu_0\omega\mathbf{D} \quad\Rightarrow\quad \boxed{\mathbf{D} = -\frac{\mathbf{k}\times\mathbf{B}}{\mu_0\omega}}$$

代入 $\mathbf{B} = \mathbf{k}\times\mathbf{E}/\omega$：
$$\mathbf{D} = -\frac{\mathbf{k}\times(\mathbf{k}\times\mathbf{E})}{\mu_0\omega^2}$$

用矢量恒等式 $\mathbf{k}\times(\mathbf{k}\times\mathbf{E}) = \mathbf{k}(\mathbf{k}\cdot\mathbf{E}) - k^2\mathbf{E}$：
$$\boxed{\mathbf{D} = \frac{k^2\mathbf{E} - (\mathbf{k}\cdot\mathbf{E})\mathbf{k}}{\mu_0\omega^2}}$$

这证明了 (2)。

---

**关于 (1)**：

在各向同性介质中 $\mathbf{D} \propto \mathbf{E}$，$\mathbf{k}\cdot\mathbf{D} = 0$ → $\mathbf{k}\cdot\mathbf{E} = 0$。

但在各向异性介质中 $\mathbf{D}$ 与 $\mathbf{E}$ 方向不同，$\mathbf{k}\cdot\mathbf{D} = 0$ 不意味着 $\mathbf{k}\cdot\mathbf{E} = 0$！即 $\mathbf{E}$ 可以有纵向分量。

---

**关于 (3)**：

坡印廷矢量 $\mathbf{S} = \mathbf{E}\times\mathbf{H}$。在各向同性介质中 $\mathbf{S} \parallel \mathbf{k}$。在各向异性介质中 $\mathbf{D}$ 和 $\mathbf{E}$ 不同向，能量传播方向（$\mathbf{S}$ 的方向）与相位传播方向（$\mathbf{k}$ 的方向）分离——这就是**双折射**的物理根源。

### 习题 21：矩形波导的单向截止

**题目**：无限长矩形波导管，在 $z=0$ 处被垂直插入的理想导体平板完全封闭。求 $z=-\infty$ 到 $z=0$ 段内可能存在的波模。

---

#### 详细解算

波导内场的一般形式（沿 $+z$ 和 $-z$ 传播的波叠加）：
$$\mathbf{E}(x,y,z,t) = \left[\mathbf{E}_+ e^{ik_z z} + \mathbf{E}_- e^{-ik_z z}\right] e^{-i\omega t}\cdot (\text{横向分布函数})$$

在 $z=0$ 理想导体端面：切向电场为零，即 $E_x(z=0) = E_y(z=0) = 0$。

这要求 $z=0$ 处入射波和反射波的切向分量相消耦合：
$$\mathbf{E}_+ + \mathbf{E}_- = \mathbf{0} \quad (\text{切向分量})$$

即形成**驻波**。$z$ 方向的场分布变为 $\sin(k_z z)$ 形式（而非行波的 $e^{ik_z z}$）。

驻波条件：$\sin(k_z \cdot 0) = 0$（自动满足，这是封闭端），同时在 $z=-L$ 处如果也有封闭端则形成**谐振腔**。

对于半无限波导（$z=-\infty$ 无反射），在 $z=0$ 全反射后，$-z$ 方向形成驻波：
$$\mathbf{E} \propto \sin(k_z z)$$

能存在的模式取决于波导截面尺寸 $a\times b$ 确定的截止频率。只有满足 $\omega > \omega_c$ 的模式才能传播（即 $k_z$ 为实数）。低于截止频率的模式 $k_z$ 为虚数 → 倏逝衰减，不能传播。

---

#### 要点总结

封闭端 → 切向电场为零 → 形成驻波 → 模式量子化（类似谐振腔）。

## 第五章 电磁波的辐射

> 本章核心公式，考试以计算偶极辐射功率和角分布为主。

### 核心公式回顾

**推迟势**（洛伦兹规范）：
$$\varphi(\mathbf{r},t) = \frac{1}{4\pi\varepsilon_0}\int\frac{\rho(\mathbf{r}',t - R/c)}{R}\,dV'$$
$$\mathbf{A}(\mathbf{r},t) = \frac{\mu_0}{4\pi}\int\frac{\mathbf{J}(\mathbf{r}',t - R/c)}{R}\,dV'$$

**电偶极辐射场**（远场区 $r \gg \lambda$）：
$$\mathbf{B} = \frac{\mu_0}{4\pi c r}\,\ddot{\mathbf{p}}(t - r/c)\times\hat{\mathbf{r}}$$
$$\mathbf{E} = c\mathbf{B}\times\hat{\mathbf{r}}$$

**辐射功率（拉莫尔公式）**：
$$P = \frac{\mu_0}{6\pi c}\,|\ddot{\mathbf{p}}|^2$$

对简谐振荡 $\mathbf{p} = \mathbf{p}_0 e^{-i\omega t}$：
$$\langle P\rangle = \frac{\mu_0\omega^4}{12\pi c}\,|\mathbf{p}_0|^2$$

**角分布**：
$$\frac{dP}{d\Omega} = \frac{\mu_0}{16\pi^2 c}\,|\ddot{\mathbf{p}}|^2\sin^2\theta$$

### 习题 22：运动车厢中推小球（洛伦兹变换应用）

**题目**：静止长度 $L_0$ 的车厢以速度 $v$ 相对地面运行，车厢后壁以速度 $u_0$ 向前推出一个小球。求地面观察者看到小球从后壁到前壁的运动时间。

---

#### 详细解算

**步骤 1**：在车厢系（$S'$）中分析

在 $S'$ 系：
- 车厢后壁：$x' = 0$
- 车厢前壁：$x' = L_0$
- 小球速度：$u' = u_0$

小球从后壁到前壁所需时间：
$$\Delta t' = \frac{L_0}{u_0}$$

**步骤 2**：洛伦兹变换

$S'$ 系以速度 $v$ 相对 $S$ 系沿 $x$ 轴运动。

洛伦兹变换：
$$t = \gamma\left(t' + \frac{v}{c^2}x'\right)$$
其中 $\gamma = \dfrac{1}{\sqrt{1-v^2/c^2}}$。

在 $S'$ 系中，事件 1（小球离开后壁）：$t_1'=0$，$x_1'=0$。
事件 2（小球到达前壁）：$t_2' = L_0/u_0$，$x_2' = L_0$。

**步骤 3**：变换到 $S$ 系

$$t_1 = \gamma(0 + 0) = 0$$
$$t_2 = \gamma\left(\frac{L_0}{u_0} + \frac{v}{c^2}L_0\right) = \gamma L_0\left(\frac{1}{u_0} + \frac{v}{c^2}\right)$$

$$\boxed{\Delta t = t_2 - t_1 = \gamma L_0\left(\frac{1}{u_0} + \frac{v}{c^2}\right)}$$

---

#### 要点总结

这是"同时性的相对性"和"时间膨胀"的综合体现：$\gamma$ 因子来自时间膨胀，$vL_0/c^2$ 项来自异地同时性的破坏。

### 习题 23：粒子衰变——四维动量守恒

**题目**：粒子 $m$ 衰变成质量为 $m_1$ 和 $m_2$，动量 $\mathbf{p}_1$ 和 $\mathbf{p}_2$（夹角 $\theta$）的两个粒子。求 $m$。

---

#### 详细解算

**四维动量守恒**：

$$p^\mu = p_1^\mu + p_2^\mu$$

其中 $p^\mu = (E/c,\mathbf{p})$，$E = \sqrt{p^2c^2 + m^2c^4}$。

**不变质量**——用四维动量的模方：

$$m^2c^2 = \frac{p^\mu p_\mu}{c^2} = \frac{(p_1^\mu + p_2^\mu)(p_{1\mu} + p_{2\mu})}{c^2}$$
$$= \frac{p_1^\mu p_{1\mu} + p_2^\mu p_{2\mu} + 2p_1^\mu p_{2\mu}}{c^2}$$

$p_1^\mu p_{1\mu} = m_1^2 c^2$，$p_2^\mu p_{2\mu} = m_2^2 c^2$。

$p_1^\mu p_{2\mu} = \frac{E_1 E_2}{c^2} - \mathbf{p}_1\cdot\mathbf{p}_2 = \frac{E_1 E_2}{c^2} - p_1 p_2\cos\theta$。

代入：
$$m^2 = m_1^2 + m_2^2 + \frac{2}{c^2}\left(\frac{E_1 E_2}{c^2} - p_1 p_2\cos\theta\right)$$

$$\boxed{m = \sqrt{m_1^2 + m_2^2 + \frac{2(E_1 E_2 - p_1 p_2 c^2\cos\theta)}{c^4}}}$$

---

#### 要点总结

四维动量守恒 + 不变质量（模方）是处理相对论粒子反应的标准方法。关键在于 $p^\mu p_\mu$ 是洛伦兹不变量。

### 简答 1：什么是位移电流？麦克斯韦为什么要引入它？

**标准答案模板**：

位移电流密度定义为 $\displaystyle\mathbf{J}_d = \varepsilon_0\frac{\partial\mathbf{E}}{\partial t}$。

麦克斯韦引入位移电流的原因：在非稳恒情况下，安培环路定理 $\nabla\times\mathbf{B} = \mu_0\mathbf{J}$ 与电荷守恒定律矛盾。对安培定理取散度：$\nabla\cdot(\nabla\times\mathbf{B}) = \mu_0\nabla\cdot\mathbf{J} = 0$，但在非稳恒时 $\nabla\cdot\mathbf{J} = -\partial\rho/\partial t \neq 0$。

麦克斯韦将 $\varepsilon_0\partial\mathbf{E}/\partial t$ 加入安培定理，使 $\nabla\times\mathbf{B} = \mu_0\mathbf{J} + \mu_0\varepsilon_0\partial\mathbf{E}/\partial t$。取散度后利用高斯定理 $\nabla\cdot\mathbf{E} = \rho/\varepsilon_0$，自动满足电荷守恒。

**物理意义**：变化的电场和传导电流一样，能产生磁场。位移电流直接预言了电磁波的存在。

---

### 简答 2：简述唯一性定理的内容和意义

**标准答案模板**：

唯一性定理：满足泊松方程 $\nabla^2\varphi = -\rho/\varepsilon_0$ 且在边界 $S$ 上满足给定边界条件（狄利克雷条件 $\varphi|_S$ 或诺伊曼条件 $\partial\varphi/\partial n|_S$）的电势解是唯一的。

**考试意义**：只要找到一个解满足方程和边界条件，它就是唯一正确的解。镜像法成立的根本依据就是唯一性定理——像电荷构造的电势在求解区域内满足拉普拉斯方程，且在导体边界上满足等势/接地条件，因此是唯一解。

---

### 简答 3：简述平面电磁波的基本性质

**标准答案模板**：

(1) **横波性**：$\mathbf{k}\cdot\mathbf{E} = 0$，$\mathbf{k}\cdot\mathbf{B} = 0$——电场和磁场都垂直于传播方向。

(2) **右手正交关系**：$\mathbf{B} = \frac{1}{\omega}\mathbf{k}\times\mathbf{E} = \frac{1}{v}\hat{\mathbf{k}}\times\mathbf{E}$。$\mathbf{E},\mathbf{B},\mathbf{k}$ 构成右手正交系。

(3) **振幅关系**：$|\mathbf{B}| = |\mathbf{E}|/v = \sqrt{\mu\varepsilon}\,|\mathbf{E}|$。真空中 $B = E/c$。

(4) **相位同步**：$\mathbf{E}$ 和 $\mathbf{B}$ 同相位振荡，同时达到最大值和零点。

---

### 简答 4：什么是推迟势？为什么是"推迟"的？

**标准答案模板**：

推迟势是时变源的电磁势表达式：
$$\varphi(\mathbf{r},t) = \frac{1}{4\pi\varepsilon_0}\int\frac{\rho(\mathbf{r}',t - R/c)}{R}\,dV'$$
$$\mathbf{A}(\mathbf{r},t) = \frac{\mu_0}{4\pi}\int\frac{\mathbf{J}(\mathbf{r}',t - R/c)}{R}\,dV'$$

其中 $R = |\mathbf{r}-\mathbf{r}'|$。

"推迟"的含义：场点 $\mathbf{r}$ 在 $t$ 时刻的势由源点 $\mathbf{r}'$ 在**更早时刻** $t - R/c$ 的源分布决定。这是因为电磁作用以有限速度 $c$ 传播，从源点传播到场点需要时间 $R/c$。这是因果律的体现：原因（源的扰动）必须早于结果（场的变化）。

---

### 简答 5：洛伦兹变换与伽利略变换的本质区别是什么？

**标准答案模板**：

伽利略变换：$x' = x - vt$，$t' = t$——**时间是绝对的**，与空间无关。

洛伦兹变换：$x' = \gamma(x - vt)$，$t' = \gamma(t - vx/c^2)$——**时间和空间耦合**，"同时"不再是绝对的。

本质区别：(1) 洛伦兹变换保证了光速在任何惯性系中都是 $c$（麦克斯韦方程组协变）；(2) 时间与空间平权，构成四维时空 $(ct,x,y,z)$；(3) 当 $v \ll c$ 时洛伦兹变换退化为伽利略变换（对应原理）。

实验判别：迈克耳孙-莫雷实验证实了光速不变，支持洛伦兹变换。

**第四段（总结）**：

### 计算题 2：接地导体球 + 点电荷——镜像法（15 分）⭐⭐

**题目**：半径 $R=10\text{cm}$ 的接地导体球，在距球心 $a=20\text{cm}$ 处放置点电荷 $q = 5\times 10^{-9}\text{C}$。求：
(1) 像电荷的大小和位置；
(2) 球外任意点的电势表达式；
(3) 导体球表面的最大电场强度；
(4) 导体球上的总感应电荷。

**详细解算**：

**(1) 像电荷**

对接地导体球：
$$q' = -\frac{R}{a}q = -\frac{10}{20}\times 5\times 10^{-9} = -2.5\times 10^{-9}\;\text{C}$$

$$b = \frac{R^2}{a} = \frac{100}{20} = 5\;\text{cm}$$

像电荷位于球心到原电荷连线上，距球心 $5\text{cm}$ 处。

**(2) 球外电势**

取球心为原点，$q$ 在 $z=a$ 处，像电荷 $q'$ 在 $z=b$ 处。球外任意点 $(r,\theta)$：
$$\varphi(r,\theta) = \frac{1}{4\pi\varepsilon_0}\left(\frac{q}{\sqrt{r^2+a^2-2ra\cos\theta}} + \frac{q'}{\sqrt{r^2+b^2-2rb\cos\theta}}\right) \quad (r>R)$$

**(3) 球表面最大电场**

球表面 $r=R$ 处，$\varphi=0$（接地）。表面电荷密度：
$$\sigma(\theta) = -\varepsilon_0\frac{\partial\varphi}{\partial r}\bigg|_{r=R}$$

计算得：
$$\sigma(\theta) = -\frac{q}{4\pi R}\cdot\frac{a^2-R^2}{(R^2+a^2-2Ra\cos\theta)^{3/2}}$$

最大电场在 $\theta=0$（面向 $q$ 的点）：
$$E_{\max} = \frac{|\sigma(0)|}{\varepsilon_0} = \frac{q}{4\pi\varepsilon_0 R}\cdot\frac{a+R}{(a-R)^2}$$

代入：$E_{\max} = \frac{9\times 10^9\times 5\times 10^{-9}}{0.1}\cdot\frac{0.3}{(0.1)^2} = 1.35\times 10^4\;\text{V/m}$

**(4) 总感应电荷**

$$Q_{\text{感应}} = q' = -2.5\times 10^{-9}\;\text{C}$$

（可通过积分表面电荷 $\oint \sigma\,dS = q'$ 验证。）

### 计算题 3：平面电磁波的反射——菲涅尔公式（12 分）⭐⭐

**题目**：一束平面单色光从空气（$n_1=1$）以入射角 $\theta_1=60^\circ$ 入射到玻璃（$n_2=1.5$）表面。分别计算 S 偏振和 P 偏振的反射率和透射率。

**详细解算**：

**折射角**：
$$\sin\theta_2 = \frac{\sin 60^\circ}{1.5} = \frac{0.866}{1.5} = 0.577,\quad \theta_2 = 35.3^\circ$$

**S 偏振（TE）**：

$$r_s = \frac{\cos 60^\circ - 1.5\cos 35.3^\circ}{\cos 60^\circ + 1.5\cos 35.3^\circ} = \frac{0.5 - 1.5\times 0.816}{0.5 + 1.5\times 0.816} = \frac{0.5-1.224}{0.5+1.224} = \frac{-0.724}{1.724} = -0.420$$

$$R_s = |r_s|^2 = 0.176 = 17.6\%$$

$$t_s = \frac{2\cos 60^\circ}{0.5+1.224} = \frac{1}{1.724} = 0.580$$

$$T_s = \frac{n_2\cos\theta_2}{n_1\cos\theta_1}|t_s|^2 = \frac{1.5\times 0.816}{0.5}\times 0.336 = 0.824 = 82.4\%$$

验证：$R_s+T_s = 100\%$ ✓

**P 偏振（TM）**：

$$r_p = \frac{1.5\cos 60^\circ - \cos 35.3^\circ}{1.5\cos 60^\circ + \cos 35.3^\circ} = \frac{0.75-0.816}{0.75+0.816} = \frac{-0.066}{1.566} = -0.0424$$

$$T_p = 1 - R_p = 99.82\%$$

### 计算题 4：电偶极辐射（12 分）⭐

**题目**：一电偶极子以角频率 $\omega = 2\pi\times 10^8\text{ rad/s}$ 振荡，偶极矩振幅 $p_0 = 10^{-12}\text{ C·m}$。求：
(1) 平均辐射总功率；
(2) 在 $\theta = 45^\circ$ 方向距离 $r = 1\text{km}$ 处电场振幅；
(3) 该方向的平均能流密度。

**详细解算**：

**(1) 平均辐射功率**

$$\langle P\rangle = \frac{\mu_0\omega^4}{12\pi c}\,p_0^2$$

$\omega = 2\pi\times 10^8 = 6.28\times 10^8\text{ rad/s}$

$\omega^4 = 1.56\times 10^{35}$

$$\langle P\rangle = \frac{4\pi\times 10^{-7}\times 1.56\times 10^{35}}{12\pi\times 3\times 10^8}\times 10^{-24} = \frac{6.24\times 10^{28}}{3.6\pi\times 10^9}\times 10^{-24} = \frac{5.52\times 10^{18}}{\pi} \approx 1.76\times 10^{18}\times 10^{-24} = 1.76\times 10^{-6}\;\text{W}$$

**(2) 电场振幅**

辐射场（远场）：
$$\mathbf{E}(\mathbf{r},t) = \frac{\mu_0}{4\pi r}(\ddot{\mathbf{p}}\times\hat{\mathbf{r}})\times\hat{\mathbf{r}}$$

振幅：$|\mathbf{E}| = \frac{\mu_0\omega^2 p_0}{4\pi r}\sin\theta$

代入：$\mu_0 = 4\pi\times 10^{-7}$，$\omega^2 = 3.95\times 10^{17}$，$r=10^3\text{m}$，$\sin 45^\circ = 0.707$

$$E_0 = \frac{4\pi\times 10^{-7}\times 3.95\times 10^{17}\times 10^{-12}}{4\pi\times 10^3}\times 0.707 = \frac{3.95\times 10^5\times 10^{-12}}{10^3}\times 0.707 = 2.79\times 10^{-10}\;\text{V/m}$$

**(3) 平均能流密度**

$$\langle S\rangle = \frac{1}{2}\varepsilon_0 c E_0^2 = \frac{1}{2}\times 8.85\times 10^{-12}\times 3\times 10^8\times (2.79\times 10^{-10})^2$$

$$= 1.33\times 10^{-3}\times 7.78\times 10^{-20} = 1.03\times 10^{-22}\;\text{W/m}^2$$

### 计算题 5：洛伦兹变换与相对论能量（10 分）

**题目**：$\pi^0$ 介子（$m_\pi = 135\text{ MeV}/c^2$）衰变为两个光子。在 $\pi^0$ 介子静止系中：
(1) 求每个光子的能量；
(2) 若 $\pi^0$ 介子以 $\beta = 0.8$ 的速度沿 $x$ 轴运动，求实验室系中两个光子的能量范围。

**详细解算**：

**(1) 静止系中**

四维动量守恒：$p^\mu_\pi = p^\mu_{\gamma 1} + p^\mu_{\gamma 2}$

静止系中：$p^\mu_\pi = (m_\pi c, \mathbf{0})$

光子：$E_\gamma = |\mathbf{p}_\gamma|c$，$m_\gamma = 0$。

由动量守恒：$\mathbf{p}_{\gamma 1} = -\mathbf{p}_{\gamma 2}$

由能量守恒：$m_\pi c^2 = E_{\gamma 1} + E_{\gamma 2} = 2E_\gamma$（两者对等）

$$\boxed{E_\gamma = \frac{m_\pi c^2}{2} = 67.5\text{ MeV}}$$

**(2) 在实验室系中（$\beta = 0.8$，$\gamma = \frac{1}{\sqrt{1-0.64}} = 1.667$）**

在 $\pi^0$ 静止系中，光子四维动量：$p_{\gamma 1}^\mu = (E_\gamma/c, E_\gamma/c\cdot\hat{\mathbf{n}}_1)$，其中 $\hat{\mathbf{n}}_1$ 为光子出射方向。

洛伦兹变换到实验室系：
$$E_{\text{lab}} = \gamma(E_\gamma + v\,p_{\gamma x}) = \gamma E_\gamma(1 + \beta\cos\theta')$$

其中 $\theta'$ 是光子出射方向与 $x$ 轴的夹角。

- 当光子向前出射（$\cos\theta' = 1$）：$E_{\max} = 1.667\times 67.5\times(1+0.8) = 1.667\times 67.5\times 1.8 \approx 202.5\text{ MeV}$
- 当光子向后出射（$\cos\theta' = -1$）：$E_{\min} = 1.667\times 67.5\times(1-0.8) = 1.667\times 67.5\times 0.2 \approx 22.5\text{ MeV}$

$$\boxed{22.5\text{ MeV} \leq E_{\text{lab}} \leq 202.5\text{ MeV}}$$

## 二、论述题预测（15分）

### 题目：试论述电磁场能量守恒定律——坡印廷定理的物理意义，并结合实例说明。

**模板范文（精简版 ~1500字）：**

**一、坡印廷定理的数学表述**

坡印廷定理的微分形式为：

$$\frac{\partial w}{\partial t} + \nabla\cdot\mathbf{S} = -\mathbf{J}\cdot\mathbf{E}$$

其中 $w = \frac{1}{2}(\mathbf{E}\cdot\mathbf{D} + \mathbf{B}\cdot\mathbf{H})$ 是电磁场能量密度，$\mathbf{S} = \mathbf{E}\times\mathbf{H}$ 是坡印廷矢量（能流密度）。

积分形式：
$$\frac{d}{dt}\int_V w dV + \oint_A \mathbf{S}\cdot d\mathbf{A} = -\int_V \mathbf{J}\cdot\mathbf{E} dV$$

**二、各物理量的含义**

- 第一项 $\frac{d}{dt}\int w dV$：体积 $V$ 内电磁场能量的时间变化率
- 第二项 $\oint\mathbf{S}\cdot d\mathbf{A}$：单位时间通过包围面 $A$ 流出的电磁场能量
- 右侧 $-\int\mathbf{J}\cdot\mathbf{E} dV$：电场对电荷做功的功率

**三、定理的物理意义**

坡印廷定理是电磁场中的能量守恒定律。它表明：电磁场的能量不是无中生有的——场能的增加要么来自外部流入（坡印廷矢量），要么由电磁场对电荷做功转化而来。它体现了"场"作为独立物理实体的思想——场本身就携带能量和动量。

**四、实例：直流导线中的能量传输**

经典电路理论给人"能量沿导线内部传播"的错觉。实际上，考虑一根载稳定电流的圆柱导线，导线内部电场沿轴向，磁场环绕导线。计算坡印廷矢量：

导线表面：$\mathbf{E}$ 沿轴向（驱动电流），$\mathbf{B}$ 沿切向，$\mathbf{S} = \mathbf{E}\times\mathbf{H}$ 指向径向向内！

这说明电磁能量是从导线外部的空间（电磁场）向着导线径向流入的，能量在导线内通过焦耳热 $\mathbf{J}\cdot\mathbf{E}$ 被耗散。能量并不在导线内部"流动"，而在空间中通过电磁场传输，导线仅起引导电磁场的作用。

## 三、计算题预测（每题12-13分，共60分）

### 第1题：静电场边值问题

> 一接地无限大导体平面位于 $x=0$ 平面。在距平面 $d$ 处的点 $(d, 0, 0)$ 放置一点电荷 $q$。求 $x>0$ 区域的电势分布。

**解法——镜像法**：

接地导体平面 $\varphi|_{x=0} = 0$。将像电荷 $-q$ 置于 $(-d, 0, 0)$，则平面边界条件自动满足。

$x>0$ 区域电势：
$$\varphi(x,y,z) = \frac{1}{4\pi\varepsilon_0}\left[\frac{q}{\sqrt{(x-d)^2 + y^2 + z^2}} - \frac{q}{\sqrt{(x+d)^2 + y^2 + z^2}}\right]$$

导体表面感应电荷面密度：
$$\sigma = -\varepsilon_0\frac{\partial\varphi}{\partial x}\bigg|_{x=0} = -\frac{qd}{2\pi(d^2 + y^2 + z^2)^{3/2}}$$

总感应电荷：
$$Q_{\text{ind}} = \int_{-\infty}^{\infty}\int_{-\infty}^{\infty}\sigma dy dz = -q$$

验证：感应电荷总量等于像电荷。

### 第2题：平面电磁波

> 一平面电磁波在真空中沿 $z$ 轴传播，电场为：
> $$\mathbf{E}(z,t) = E_0\cos(\omega t - kz)\hat{x}$$
> 求：(1) 磁场 $\mathbf{B}(z,t)$；(2) 坡印廷矢量 $\mathbf{S}$ 及其时间平均值。

**解**：

(1) 平面波 $\mathbf{B} = \frac{1}{\omega}\mathbf{k}\times\mathbf{E}$，$\mathbf{k} = k\hat{z}$：

$$\mathbf{B} = \frac{1}{\omega}k\hat{z}\times[E_0\cos(\omega t - kz)\hat{x}] = \frac{kE_0}{\omega}\cos(\omega t - kz)\hat{y}$$

真空中 $k = \omega/c$，故：
$$\mathbf{B}(z,t) = \frac{E_0}{c}\cos(\omega t - kz)\hat{y}$$

(2) 坡印廷矢量：
$$\mathbf{S} = \frac{1}{\mu_0}\mathbf{E}\times\mathbf{B} = \frac{E_0^2}{\mu_0 c}\cos^2(\omega t - kz)\hat{z}$$

时间平均值（$\langle\cos^2(\omega t - kz)\rangle = 1/2$）：
$$\langle\mathbf{S}\rangle = \frac{E_0^2}{2\mu_0 c}\hat{z} = \frac{1}{2}\varepsilon_0 c E_0^2 \hat{z}$$

### 第3题：菲涅尔公式

> 可见光从空气（$n_1 = 1$）正入射到玻璃（$n_2 = 1.5$）表面。求反射率和透射率。

**解**：

正入射 $\theta_1 = \theta_2 = 0$。反射系数（S波和P波结果相同）：
$$r = \frac{n_1 - n_2}{n_1 + n_2} = \frac{1 - 1.5}{1 + 1.5} = \frac{-0.5}{2.5} = -0.2$$

反射率：
$$R = |r|^2 = 0.04 = 4\%$$

透射率（无吸收）：
$$T = 1 - R = 96\%$$

> 💡 反射系数为负表示反射波电场有 $\pi$ 相位跃变（从光疏到光密介质正入射）。

### 第4题：电偶极辐射

> 一短天线长度 $\ell$，载有振荡电流 $I(t) = I_0\cos\omega t$。天线沿 $z$ 轴放置。求：(1) 辐射功率；(2) 在 $\theta = 45^\circ$ 处的辐射角分布密度。

**解**：

(1) 天线电偶极矩：$p(t) = I_0\ell\int\cos\omega t dt = \frac{I_0\ell}{\omega}\sin\omega t = p_0\sin\omega t$

其中 $p_0 = I_0\ell/\omega$。

$$\ddot{p}(t) = -\omega^2 p_0\sin\omega t = -\omega^2 p(t)$$
$$|\ddot{p}|_{\text{max}} = \omega^2 p_0 = \omega I_0\ell$$

辐射功率（拉莫尔公式）：
$$P = \frac{\mu_0}{6\pi c}\langle|\ddot{p}|^2\rangle = \frac{\mu_0(\omega I_0\ell)^2}{12\pi c}$$

代入 $\mu_0 = 4\pi \times 10^{-7}$：
$$P = \frac{(I_0\ell)^2\omega^2}{3c}\times 10^{-7}$$

(2) 辐射角分布：
$$\frac{dP}{d\Omega} = \frac{\mu_0|\ddot{p}|_{\text{max}}^2}{16\pi^2 c}\sin^2\theta = \frac{\mu_0(\omega I_0\ell)^2}{16\pi^2 c}\sin^2\theta$$

在 $\theta = 45^\circ$，$\sin^2 45^\circ = 1/2$：
$$\frac{dP}{d\Omega}\bigg|_{45^\circ} = \frac{\mu_0(\omega I_0\ell)^2}{32\pi^2 c}$$

### 第5题：狭义相对论

> 一粒子静质量为 $m_0$，总能量为 $E = 2m_0c^2$。求该粒子的速度 $v$ 和动量 $p$。

**解**：

由质能关系 $E = \gamma m_0c^2$：
$$\gamma m_0c^2 = 2m_0c^2 \quad \Rightarrow \quad \gamma = 2$$

由 $\gamma = \frac{1}{\sqrt{1 - v^2/c^2}} = 2$：
$$1 - \frac{v^2}{c^2} = \frac{1}{4} \quad \Rightarrow \quad v = \frac{\sqrt{3}}{2}c \approx 0.866c$$

动量：
$$p = \gamma m_0 v = 2m_0\cdot\frac{\sqrt{3}}{2}c = \sqrt{3}m_0c$$

验证：$E^2 = p^2c^2 + m_0^2c^4 = 3m_0^2c^4 + m_0^2c^4 = 4m_0^2c^4 = (2m_0c^2)^2$，一致。

## 二、论述题预测（15分）

### 题目：试对比静电场的标势描述与静磁场的矢势描述，说明规范变换自由度的物理含义。

**模板范文（精简版）：**

**一、静电标势**

静电场满足 $\nabla\times\mathbf{E} = 0$，故可引入标势：
$$\mathbf{E} = -\nabla\varphi$$

将 $\varphi$ 平移常量 $\varphi \to \varphi + C$，$\mathbf{E}$ 不变。这是静电势的规范自由度——标势可以加上任意常数而不改变物理量。

**二、磁矢势**

磁场满足 $\nabla\cdot\mathbf{B} = 0$，故可引入矢势：
$$\mathbf{B} = \nabla\times\mathbf{A}$$

规范变换：
$$\mathbf{A} \to \mathbf{A}' = \mathbf{A} + \nabla\psi$$
则 $\mathbf{B}' = \nabla\times(\mathbf{A} + \nabla\psi) = \nabla\times\mathbf{A} = \mathbf{B}$，磁场不变。

此时标势也需要同步变换以保持电场不变：
$$\varphi \to \varphi' = \varphi - \frac{\partial\psi}{\partial t}$$

**三、规范变换的物理含义**

规范自由度并未引入新的物理——而是反映了用势描述电磁场时的"冗余"。势 $\varphi, \mathbf{A}$ 不是唯一确定的，但由它们导出的可观测物理量 $\mathbf{E}, \mathbf{B}$ 在规范变换下不变。

常见的规范条件：
- **库仑规范**：$\nabla\cdot\mathbf{A} = 0$（对于静磁场问题方便，此时 $\nabla^2\mathbf{A} = -\mu_0\mathbf{J}$）
- **洛伦兹规范**：$\nabla\cdot\mathbf{A} + \frac{1}{c^2}\frac{\partial\varphi}{\partial t} = 0$（使方程协变对称，便于辐射问题）

**四、两者的对比**

| 特性 | 静电场标势 | 静磁场矢势 |
|------|-----------|-----------|
| 场的旋度 | $\nabla\times\mathbf{E}=0$ | $\nabla\times\mathbf{B}\neq0$ |
| 势的类型 | 标量 $\varphi$ | 矢量 $\mathbf{A}$ |
| 基本方程 | $\nabla^2\varphi = -\rho/\varepsilon_0$ | $\nabla^2\mathbf{A} = -\mu_0\mathbf{J}$ |
| 规范自由度 | $\varphi \to \varphi + C$ | $\mathbf{A} \to \mathbf{A} + \nabla\psi$ |
| 物理来源 | 有源无旋 | 有旋无源 |

## 三、计算题预测（每题12-13分，共60分）

### 第1题：分离变量法

> 一无限长矩形槽，截面为 $0 \leq x \leq a, 0 \leq y \leq b$。边界条件：$x=0$ 和 $x=a$ 接地（$\varphi=0$），$y=0$ 接地，$y=b$ 保持电势 $V_0$。求槽内电势分布。

**解**：

拉普拉斯方程 $\nabla^2\varphi = 0$，直角坐标系分离变量。

设 $\varphi(x,y) = X(x)Y(y)$：

$$\frac{X''}{X} + \frac{Y''}{Y} = 0 \quad \Rightarrow \quad X'' + k^2 X = 0, \quad Y'' - k^2 Y = 0$$

由 $X(0) = X(a) = 0$：
$$X_n(x) = \sin\frac{n\pi x}{a}, \quad k_n = \frac{n\pi}{a}$$

$$Y_n(y) = A_n\sinh(k_n y) + B_n\cosh(k_n y)$$

由 $Y(0) = 0 \Rightarrow B_n = 0$。通解：
$$\varphi(x,y) = \sum_{n=1}^{\infty} A_n\sin\frac{n\pi x}{a}\sinh\frac{n\pi y}{a}$$

由 $y=b$ 边界条件 $\varphi(x,b) = V_0$：
$$V_0 = \sum_{n=1}^{\infty} A_n\sinh\frac{n\pi b}{a}\sin\frac{n\pi x}{a}$$

傅里叶展开：
$$A_n\sinh\frac{n\pi b}{a} = \frac{2}{a}\int_0^a V_0\sin\frac{n\pi x}{a}dx = \frac{2V_0}{n\pi}(1 - \cos n\pi)$$

$n$ 偶数时为 0，$n$ 奇数 $=2m-1$ 时为 $\frac{4V_0}{(2m-1)\pi}$。

$$A_{2m-1} = \frac{4V_0}{(2m-1)\pi\sinh\frac{(2m-1)\pi b}{a}}$$

最终：
$$\varphi(x,y) = \frac{4V_0}{\pi}\sum_{m=1}^{\infty}\frac{\sin\frac{(2m-1)\pi x}{a}\sinh\frac{(2m-1)\pi y}{a}}{(2m-1)\sinh\frac{(2m-1)\pi b}{a}}$$

### 第2题：磁矢势与规范

> 已知一均匀磁场 $\mathbf{B} = B_0\hat{z}$，求出其磁矢势 $\mathbf{A}$（取库仑规范），并验证 $\nabla\times\mathbf{A} = \mathbf{B}$。

**解**：

由 $\mathbf{B} = \nabla\times\mathbf{A}$ 和库仑规范 $\nabla\cdot\mathbf{A} = 0$。

尝试 $\mathbf{A} = (-\frac{1}{2}B_0 y, \frac{1}{2}B_0 x, 0)$（对称规范）：

$$\nabla\times\mathbf{A} = \begin{vmatrix} \hat{x} & \hat{y} & \hat{z} \\ \partial_x & \partial_y & \partial_z \\ -B_0y/2 & B_0x/2 & 0 \end{vmatrix} = (0, 0, \frac{B_0}{2} + \frac{B_0}{2}) = B_0\hat{z} = \mathbf{B}$$

验证库仑规范：
$$\nabla\cdot\mathbf{A} = \frac{\partial}{\partial x}\left(-\frac{B_0y}{2}\right) + \frac{\partial}{\partial y}\left(\frac{B_0x}{2}\right) = 0 + 0 = 0$$

注：也可选择朗道规范 $\mathbf{A} = (0, B_0 x, 0)$ 或 $\mathbf{A} = (-B_0 y, 0, 0)$。

### 第3题：反射与折射

> 自然光从空气以 $60^\circ$ 入射角射到折射率为 $\sqrt{3}$ 的玻璃表面。求：(1) 折射角；(2) 反射光的偏振度（反射率中S波和P波各自的分量）。

**解**：

(1) Snell定律 $n_1\sin\theta_1 = n_2\sin\theta_2$：
$$\sin\theta_2 = \frac{1\cdot\sin 60^\circ}{\sqrt{3}} = \frac{\sqrt{3}/2}{\sqrt{3}} = \frac{1}{2}$$
$$\theta_2 = 30^\circ$$

(2) S波反射系数：
$$r_s = \frac{n_1\cos\theta_1 - n_2\cos\theta_2}{n_1\cos\theta_1 + n_2\cos\theta_2} = \frac{\cos 60^\circ - \sqrt{3}\cos 30^\circ}{\cos 60^\circ + \sqrt{3}\cos 30^\circ}$$
$$\cos 60^\circ = 0.5, \quad \cos 30^\circ = \sqrt{3}/2 \approx 0.8660$$
$$r_s = \frac{0.5 - \sqrt{3}\cdot\sqrt{3}/2}{0.5 + \sqrt{3}\cdot\sqrt{3}/2} = \frac{0.5 - 1.5}{0.5 + 1.5} = \frac{-1}{2} = -0.5$$

P波反射系数：
$$r_p = \frac{n_2\cos\theta_1 - n_1\cos\theta_2}{n_2\cos\theta_1 + n_1\cos\theta_2} = \frac{\sqrt{3}\cdot 0.5 - 1\cdot\sqrt{3}/2}{\sqrt{3}\cdot 0.5 + 1\cdot\sqrt{3}/2} = \frac{\sqrt{3}/2 - \sqrt{3}/2}{\sqrt{3}/2 + \sqrt{3}/2} = 0$$

反射率：$R_s = |r_s|^2 = 0.25$，$R_p = 0$。

自然光入射（S:P = 1:1），反射光中S分量占比 = $\frac{R_s}{R_s + R_p} = 100\%$。反射光为完全线偏振光（仅S偏振）。

> 💡 重点：$\theta_1 = 60^\circ, \theta_2 = 30^\circ$，$\theta_1 + \theta_2 = 90^\circ$，此时 $\tan\theta_1 = \sqrt{3} = n_2/n_1$，恰为布儒斯特角！所以P波反射系数为零。

### 第5题：洛伦兹变换与时间膨胀

> 一宇宙飞船以 $0.8c$ 速度飞离地球。地球上测得飞船需要10年飞到某星球。求：(1) 飞船上乘客感受到的飞行时间；(2) 地球参照系中飞船飞过的距离；(3) 飞船上乘客测得的地球距离。

**解**：

已知：$v = 0.8c, \Delta t = 10\text{年}$（地球上的时间坐标差——飞船飞离地球，非固有时）。

> ⚠️ 注意：这10年是地球上两次读数（出发和到达）的时间差，对应的固有时是飞船上的时间。

(1) 飞船上测得的时间（固有时）：
$$\Delta t' = \frac{\Delta t}{\gamma} = \Delta t\sqrt{1 - v^2/c^2}$$
$$\gamma = \frac{1}{\sqrt{1 - 0.64}} = \frac{1}{0.6} \approx 1.667$$
$$\Delta t' = 10\times 0.6 = 6\text{年}$$

(2) 地球参照系中的距离（飞船速度 $0.8c$，飞10年）：
$$L = v\Delta t = 0.8c\times 10\text{年} = 8\text{光年}$$

(3) 飞船上乘客测得的距离（长度收缩）：
$$L' = \frac{L}{\gamma} = 8\times 0.6 = 4.8\text{光年}$$

验证：$L' = v\Delta t' = 0.8c\times 6\text{年} = 4.8\text{光年}$，一致。

## 二、论述题预测（15分）

### 题目：从麦克斯韦方程组出发，推导无源区域中电磁场的波动方程，并论述电磁波的本质是横波。

**模板范文：**

**一、波动方程的推导**

真空中无源区域（$\rho=0, \mathbf{J}=0$）的麦克斯韦方程组：
$$\nabla\cdot\mathbf{E} = 0 \quad\text{(1)}$$
$$\nabla\cdot\mathbf{B} = 0 \quad\text{(2)}$$
$$\nabla\times\mathbf{E} = -\frac{\partial\mathbf{B}}{\partial t} \quad\text{(3)}$$
$$\nabla\times\mathbf{B} = \mu_0\varepsilon_0\frac{\partial\mathbf{E}}{\partial t} \quad\text{(4)}$$

对(3)取旋度：$\nabla\times(\nabla\times\mathbf{E}) = -\nabla\times\frac{\partial\mathbf{B}}{\partial t} = -\frac{\partial}{\partial t}(\nabla\times\mathbf{B})$

左边：$\nabla\times(\nabla\times\mathbf{E}) = \nabla(\nabla\cdot\mathbf{E}) - \nabla^2\mathbf{E} = -\nabla^2\mathbf{E}$（利用了(1)）

右边代入(4)：$-\frac{\partial}{\partial t}(\mu_0\varepsilon_0\frac{\partial\mathbf{E}}{\partial t}) = -\mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2}$

$$\nabla^2\mathbf{E} - \mu_0\varepsilon_0\frac{\partial^2\mathbf{E}}{\partial t^2} = 0$$

同理得磁场波动方程：
$$\nabla^2\mathbf{B} - \mu_0\varepsilon_0\frac{\partial^2\mathbf{B}}{\partial t^2} = 0$$

波速：$c = 1/\sqrt{\mu_0\varepsilon_0}$。

**二、平面波解与横波性**

设 $\mathbf{E}(\mathbf{r},t) = \mathbf{E}_0 e^{i(\mathbf{k}\cdot\mathbf{r} - \omega t)}$，代入波动方程得色散关系：$k = \omega/c$。

由 $\nabla\cdot\mathbf{E} = i\mathbf{k}\cdot\mathbf{E} = 0 \Rightarrow \mathbf{k}\cdot\mathbf{E} = 0$，电场与传播方向垂直。

由 $\nabla\cdot\mathbf{B} = i\mathbf{k}\cdot\mathbf{B} = 0 \Rightarrow \mathbf{k}\cdot\mathbf{B} = 0$，磁场也与传播方向垂直。

由法拉第定律 $\nabla\times\mathbf{E} = i\mathbf{k}\times\mathbf{E} = i\omega\mathbf{B} \Rightarrow \mathbf{B} = \frac{1}{\omega}\mathbf{k}\times\mathbf{E}$

综上：$\mathbf{E} \perp \mathbf{B} \perp \mathbf{k}$，且 $\mathbf{E}\times\mathbf{B}$ 沿 $\mathbf{k}$ 方向。电磁波是横波——电磁场振荡方向与传播方向垂直。

**三、历史与哲学意义**

麦克斯韦从纯粹理论推导预言了电磁波的存在，并算出其速度等于当时测定的光速，由此提出"光是电磁波"的伟大统一。这是理论物理的典范——数学的对称性和自洽性引导出新的物理发现。

- 电荷3 $(0,0,0)$：$r_3^2 = 0$。$\mathcal{D}_{zz}^{(3)} = -q(0) = 0$

$$\mathcal{D}_{zz} = -2qd^2$$

其他对角元：$\mathcal{D}_{xx} = q(3d^2 - d^2) + q(3d^2 - d^2) + 0 = 4qd^2$，$\mathcal{D}_{yy} = q(-d^2) + q(-d^2) + 0 = -2qd^2$。

验证：$\mathcal{D}_{xx} + \mathcal{D}_{yy} + \mathcal{D}_{zz} = 4qd^2 - 2qd^2 - 2qd^2 = 0$，无迹条件满足。

### 第3题：平面波在导体中的衰减

> 一频率 $f = 100\text{MHz}$ 的电磁波垂直入射到铜表面。铜的电导率 $\sigma = 5.8\times 10^7\text{S/m}$。求：(1) 趋肤深度；(2) 波在铜中的波长与真空中波长的比值。

**解**：

(1) 铜是良导体（$\sigma/(\omega\varepsilon_0) \gg 1$），趋肤深度近似：
$$\delta = \sqrt{\frac{2}{\omega\mu_0\sigma}}$$

$\omega = 2\pi f = 2\pi\times 10^8 \approx 6.28\times 10^8\text{rad/s}$

$\mu_0 = 4\pi\times 10^{-7}\text{H/m}$

$$\delta = \sqrt{\frac{2}{6.28\times 10^8 \times 4\pi\times 10^{-7} \times 5.8\times 10^7}}$$
$$\delta = \sqrt{\frac{2}{6.28\times 4\pi\times 5.8\times 10^8}} = \sqrt{\frac{2}{4.57\times 10^{10}}} \approx 6.6\times 10^{-6}\text{m} = 6.6\mu\text{m}$$

(2) 铜中波数 $k = (1 + i)/\delta$，波长 $\lambda_{\text{Cu}} = 2\pi\delta = 2\pi\times 6.6\times 10^{-6} \approx 4.15\times 10^{-5}\text{m}$。

真空中波长：$\lambda_0 = c/f = 3\times 10^8/10^8 = 3\text{m}$。

比值：
$$\frac{\lambda_{\text{Cu}}}{\lambda_0} = \frac{4.15\times 10^{-5}}{3} \approx 1.38\times 10^{-5}$$

> 💡 良导体中波长极短（微米量级），电磁波在铜中几微米就衰减殆尽。

### 第4题：推迟势

> 一电偶极子位于原点，振荡形式为 $\mathbf{p}(t) = p_0\cos\omega t\hat{z}$。写出推迟矢势 $\mathbf{A}(\mathbf{r}, t)$ 的表达式（远场近似）。

**解**：

由推迟势公式（偶极近似，电流元 $\dot{\mathbf{p}}$）：
$$\mathbf{A}(\mathbf{r}, t) = \frac{\mu_0}{4\pi r}\dot{\mathbf{p}}(t - r/c)$$

远场近似：$r \gg \lambda$ 且 $r \gg \ell$（天线长度）。

$$\dot{\mathbf{p}}(t) = -\omega p_0\sin\omega t\hat{z}$$
$$\ddot{\mathbf{p}}(t) = -\omega^2 p_0\cos\omega t\hat{z}$$

代入推迟时间：
$$\mathbf{A}(\mathbf{r}, t) = \frac{\mu_0}{4\pi r}[-\omega p_0\sin(\omega t - kr)]\hat{z}$$
其中 $k = \omega/c$。

由 $\mathbf{B} = \nabla\times\mathbf{A}$ 得辐射磁场：
$$\mathbf{B} = \frac{\mu_0}{4\pi cr}\ddot{\mathbf{p}}(t - r/c)\times\hat{r} = \frac{\mu_0\omega^2 p_0}{4\pi cr}\cos(\omega t - kr)(\hat{z}\times\hat{r})$$

辐射电场：
$$\mathbf{E} = c\mathbf{B}\times\hat{r} = \frac{\mu_0\omega^2 p_0}{4\pi r}\cos(\omega t - kr)(\hat{z}\times\hat{r})\times\hat{r}$$

### 第5题：相对论速度变换

> 在 $S$ 系中，一粒子以速度 $u_x = 0.9c$ 沿 $x$ 轴运动。$S'$ 系相对于 $S$ 系以 $v = 0.6c$ 沿 $x$ 轴同向运动。求 $S'$ 系中观测到的粒子速度。若 $S'$ 系反向运动（$v = -0.6c$），结果如何？

**解**：

洛伦兹速度变换：
$$u_x' = \frac{u_x - v}{1 - u_x v/c^2}$$

**情况1：$v = 0.6c$（同向）**
$$u_x' = \frac{0.9c - 0.6c}{1 - 0.9\times 0.6} = \frac{0.3c}{1 - 0.54} = \frac{0.3c}{0.46} \approx 0.652c$$

$S'$ 系测得粒子速度为 $0.652c$。

**情况2：$v = -0.6c$（反向）**
$$u_x' = \frac{0.9c - (-0.6c)}{1 - 0.9\times (-0.6)} = \frac{1.5c}{1 + 0.54} = \frac{1.5c}{1.54} \approx 0.974c$$

粒子在 $S'$ 系中速度接近 $c$，但仍未超过光速——这是洛伦兹速度合成保证了光速不变原理。
