# 第一章 绪论与数学准备

## 1.1 理论力学在学什么？

牛顿力学用 $\vec{F}=m\vec{a}$ 处理问题。一个质点、几个力——很顺手。但遇到**双摆、陀螺、多体约束系统**，写出所有力和几何关系就变得极其繁琐。

理论力学换了一个思路：**不再纠缠于每一个力矢量，而是用能量（标量）来描述系统**。核心哲学就一句话——**用标量（动能 $T$、势能 $V$）替代矢量（力 $\vec{F}$），用广义坐标替代直角坐标**。

> **一句话**：牛顿力学是"一笔一画描轨迹"，分析力学是"找到系统的DNA，自动生成一切运动"。

## 1.2 需要的数学工具

### 变分法初步

**核心问题**：什么样的函数 $y(x)$ 能让积分 $\int F(x,y,y')dx$ 取极值？

答案——**欧拉-拉格朗日方程**：

$$\frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) - \frac{\partial F}{\partial y} = 0$$

> 类比微积分：$f'(x)=0$ 找函数的极值点；欧拉方程找**泛函**（函数的函数）的极值。

### 约束的分类

> **核心概念**：约束 = 系统中质点坐标不能自由变化的限制条件。自由度 = 坐标总数 - 独立的约束方程数。

---

#### 第一层分类：完整约束 vs 非完整约束

| 类型 | 数学形式 | 物理含义 | 考试要点 |
|------|----------|----------|----------|
| **完整约束** | $f(\vec{r}_1,\vec{r}_2,...,t)=0$（只含坐标，不含速度） | 限制质点的**位置** | 拉格朗日方程**可用** |
| **非完整约束** | $f(\vec{r}_1,...,\dot{\vec{r}}_1,...,t)=0$（含速度且不可积分） | 限制质点的**运动方式** | 拉格朗日方程**不可直接用** |

**完整约束深入讲解**

完整约束是最常见的约束类型，数学上写为：

$$f(\vec{r}_1,\vec{r}_2,...,\vec{r}_N,t)=0$$

这约束方程只涉及质点的**位置坐标**和**时间**（可选），不涉及速度。例如：

- 双原子分子（刚性连接）：$|\vec{r}_1-\vec{r}_2|-l=0$（两个质点距离恒为 $l$）
- 单摆：$x^2+y^2-l^2=0$（摆锤被绳子约束在圆弧上）
- 粒子在曲面上运动：$z-f(x,y)=0$（$z$ 坐标由 $x,y$ 决定）

**为什么"完整"叫这个名字？**

> "完整"的英文是 holonomic，来源于希腊语"holos"（整体）+ "nomos"（法则）。完整约束可以写成一个"完整的整体法则"来限制坐标——你可以把它积分配成位置间的函数关系，从而**减少独立坐标的数量**。

---

**非完整约束深入讲解**

非完整约束**不能**写成纯坐标的函数关系——因为它本质上是速度层面的限制，且**不可积分**。如果有办法把速度关系积分成坐标关系，那它实际上是完整约束（只是写成了微分形式）。

**典型例子——冰刀问题**：

冰刀只能沿刀刃方向滑动，垂直于刀刃方向速度为零：

$$\dot{x}\sin\theta-\dot{y}\cos\theta=0$$

这里的 $\theta$ 是冰刀的朝向角度。这个方程含速度，且**不能积分**成纯坐标关系（因为 $\theta$ 本身会随时间变化，由动力学决定），所以是非完整约束。

**另一个经典例子——滚动的盘子**：

盘子在地面做纯滚动，约束联系着速度、角速度和方向——三个约束方程中有两个是非完整的。

**考试判断技巧**：

1. 看约束方程里有没有**速度** $\dot{q}_i$——没有 → 完整约束
2. 如果含速度，试一下能否**积分**成纯坐标方程——能积分 → 它其实还是完整约束（只是写成了微分形式）
3. 含速度且**不能积分** → 非完整约束

---

#### 第二层分类：定常约束 vs 非定常约束

| 类型 | 数学形式 | 物理含义 | 关键区别 |
|------|----------|----------|----------|
| **定常约束** | $f(\vec{r}_1,\vec{r}_2,...)=0$（不显含时间 $t$） | 约束本身不随时间改变 | 虚位移 = 可能的实位移方向 |
| **非定常约束** | $f(\vec{r}_1,\vec{r}_2,...,t)=0$（显含时间 $t$） | 约束随时间变化 | 虚位移 ≠ 实位移方向！ |

**定常约束——约束"静止不动"**

固定半径的单摆（绳子长 $l$ 不变）：

$$x^2+y^2-l^2=0$$

方程中不出现 $t$——无论何时，摆锤都只能在同一个圆弧上运动。虚位移沿圆弧切向，可能的实位移也沿圆弧切向。

**非定常约束——约束在"动"**

摆长随时间变化的单摆（有人在拉绳子）：

$$x^2+y^2-[l(t)]^2=0$$

方程中出现 $t$——摆锤的"允许活动范围"随时间在变。此时**虚位移和实位移不再是同一批方向**！因为虚位移冻结了时间（$\delta t=0$），只看到瞬时约束——那一瞬间绳子还是 $l(t_0)$ 长，但实位移经历了 $dt$ 时间，这期间绳子变了。

> **考试极其爱考**：非定常约束下 $\delta\vec{r} \neq d\vec{r}$ 的子集！虚位移的集合是在"时间冻结"前提下，满足 $t=t_0$ 时约束的所有无穷小位移。而实位移只有一条路径。如果有 $k$ 个完整约束，$N$ 个质点→ $3N-k$ 个自由度。非定常约束不改变自由度计数，但改变了虚位移和实位移的关系。

---

#### 第三层分类：双侧约束 vs 单侧约束

| 类型 | 数学形式 | 物理例子 |
|------|----------|----------|
| **双侧约束** | $f(\vec{r})=0$（等号） | 穿在铁丝上的珠子（不能离开铁丝） |
| **单侧约束** | $f(\vec{r})\geq 0$（不等号） | 球在地面上方（$z\geq 0$，可以跳起来但不可以钻下去） |

> 理论力学课本99%讨论的是**双侧约束**，因为约束方程是等号时，分析力学才有简洁的数学处理。单侧约束需要不等式，用到更复杂的接触力学。

---

#### 约束如何决定自由度和广义坐标

**自由度 = 坐标总数 - 独立完整约束的个数**

| 系统 | 坐标数 | 完整约束 | 自由度 | 可选广义坐标 |
|------|--------|----------|--------|------------|
| 自由质点 | 3 | 0 | 3 | $x,y,z$ 或 $r,\theta,\varphi$ |
| 球面上质点 | 3 | $1$（$x^2+y^2+z^2=R^2$） | 2 | 纬度+经度 $(\theta,\varphi)$ |
| 双摆（两杆连接） | 4（每质点 $x,y$） | 2（两杆长度固定）+ 1（连接点固定） | 2 | 两杆与竖直线的夹角 $\theta_1,\theta_2$ |
| 双原子分子（自由空间） | 6 | 1（两原子间距固定 $l$） | 5 | 质心坐标 $(X,Y,Z)$ + 方位角 $(\theta,\varphi)$ |

**广义坐标的核心思想**：

对于 $N$ 个质点、$k$ 个完整约束的系统，你可以放弃 $3N$ 个笛卡尔坐标，改用 $(3N-k)$ 个**广义坐标** $q_1,q_2,...,q_{3N-k}$ 来完全描述系统。这些 $q$ 可以不是长度单位——可以是角度、面积、电荷……可以是任何方便描述约束的量。

> **拉格朗日方程只适用于完整约束系统！** 因为拉格朗日方程需要独立广义坐标，而非完整约束无法用来消去不独立坐标。

---

#### 约束力——为什么我们拼命想消去它

约束力（绳子张力、接触面法向力、铰链力……）是系统**被动产生**的力——它们自己不会"主动"改变运动状态，只是确保约束条件始终满足。

约束力有三个让物理学家头疼的特点：

1. **事先未知**——你不知道它多大，直到算出来
2. **不做功**（理想约束情况下）——约束力始终与虚位移垂直
3. **方程数量爆炸**——复杂系统可能有几十个约束力，每个都要列方程再消去

这正是达朗贝尔原理和虚功原理存在的意义：**通过虚位移的数学技巧，把约束力从方程中"消掉"**，只留下我们关心的主动力。

> **一句话总结**：约束 → 缩小自由度 → 引入广义坐标 → 在广义坐标中列方程 → 约束力自动消失（对理想约束）

## 🔥 约束精讲——用生活类比彻底搞懂三层分类

> 如果你觉得约束分类还是云里雾里，这一节就是为你准备的。我们用最土的话、最直观的例子，把三层分类串起来。

### 第一层：完整约束 vs 非完整约束（"限制你在哪" vs "限制你怎么走"）

#### 🎯 一句话区分

| | 完整约束 | 非完整约束 |
|---|---|---|
| **本质** | 限制你的**位置** | 限制你的**运动方式** |
| **类比** | 把你关在一个房间里 | 允许你在操场任何地方，但只能走直线 |
| **自由度** | 直接减少独立坐标数 | 不减少坐标数，但减少**速度的独立分量** |

#### 🏠 生活类比：宿舍管理规定

**完整约束 = "晚上11点必须在宿舍楼内"**
- 这是一个**位置限制**——你的坐标被限制在宿舍楼这个区域内
- 不管你走哪条路回来、跑多快、是不是骑自行车——只要11点你的身体在楼里就满足
- 约束方程只涉及你的位置坐标，不涉及你的速度

**非完整约束 = "在走廊里只能走，不能跑"**
- 这限制的是你的**运动方式**（速度不能超过某个值），而不是你在哪里
- 你可以站在走廊任何位置，但移动速度受限制
- 约束方程涉及速度（$\dot{x}, \dot{y}$）

#### 🔬 物理例子对比

**完整约束举例**

① **单摆（绳子约束）**
```
小球被绳子拴着 → 小球只能在以悬挂点为圆心、绳长为半径的圆弧上运动
约束方程：x² + y² = l²
含义：无论小球怎么动，它到圆心的距离永远是 l
只含坐标 x, y，不含速度 → 完整约束
```

② **火车在铁轨上**
```
铁轨限制了火车的位置——火车只能沿着铁轨的几何形状走
约束方程：y = f(x)（铁轨的形状曲线）
只含坐标 → 完整约束
```

③ **活塞在气缸里**
```
活塞只能在气缸的直线方向上滑动
约束方程：y = 0, z = 0（假设沿 x 轴滑动）
只含坐标 → 完整约束
```

**非完整约束举例**

① **冰刀问题（经典必考！）**
```
冰刀在冰面上自由滑动，但有一个限制：
冰刀只能沿着刀刃的方向滑，不能侧向打滑

约束方程：ẋ sinθ − ẏ cosθ = 0
含义：速度方向必须和刀刃朝向一致

为什么是非完整的？
→ 因为 θ（刀刃朝向）是独立变量，由动力学决定
→ 你无法把这个速度关系积分成纯坐标关系
→ 如果你强行积分，你会发现结果依赖于你走了哪条路径
```

> 💡 **直观理解**：想象你开车。车可以在停车场任何位置，但有一个约束——**车轮不能侧滑**（即垂直于车轮平面的速度分量为零）。你能写出坐标必须满足的方程吗？不能。因为车轮的朝向随时在变，而这个朝向取决于你之前怎么开。所以"不能侧滑"是非完整约束。

② **滚动的硬币**
```
一个硬币在桌面上滚动，不滑动。
硬币的中心位置 (x, y) 可以在地面任意位置，
但硬币的滚动方向必须和硬币的朝向一致。
涉及速度 (ẋ, ẏ) 和朝向角 φ 的变化率 φ̇ 之间的关系。
两个非完整约束。
```

#### 🎓 考试必备：完整 vs 非完整 判断三步法

```
步骤1：看约束方程里有没有速度（ẋ, ẏ, ż 等）
  ├─ 没有速度 → 完整约束 ✓
  └─ 有速度 → 进入步骤2

步骤2：尝试积分
  ├─ 能积分成只含坐标的方程 → 完整约束（伪装的微分约束）
  │   例子：ẋ dt + ẏ dt = 0 → 积分为 x + y = const
  └─ 不能积分 → 进入步骤3

步骤3：判断是否可以写成 f(q₁, q₂, ..., t) = 0 的形式
  ├─ 可以 → 完整约束
  └─ 不可以 → 非完整约束 ✗（拉格朗日方程不能直接用！）
```

> ⚠️ **关键陷阱**：约束方程写成微分形式不一定就是非完整约束！比如 `ẋ + ẏ = 0` 可以积分为 `x + y = const`，它本质上是完整约束。判断标准是**可积性**，不是写了ẋ就是非完整。

### 第二层：定常约束 vs 非定常约束（"牢笼不动的" vs "牢笼在变形的"）

#### 🎯 一句话区分

| | 定常约束 | 非定常约束 |
|---|---|---|
| **本质** | 约束条件不随时间变化 | 约束条件随时间变化 |
| **类比** | 固定大小的笼子 | 笼子的大小/形状在随时间改变 |
| **方程** | 不含 $t$ | 显含 $t$ |
| **虚位移 vs 实位移** | 虚位移 = 实位移的可能方向 | 虚位移 ≠ 实位移的方向！ |

#### 🏠 生活类比：教室规则

**定常约束 = 固定座位表**
```
教室的座位排列从开学到期末不变。
你坐在第3排第4列——今天是这样，明天也是这样。
你的"允许位置"永不改变。
```

**非定常约束 = 老师每节课随机调座位**
```
每节课开始，老师重新排座位——你的"允许位置"随时间变化。
第5周你坐3排4列，第6周可能坐5排2列。
你的"允许位置"是时间的函数。
```

#### 🔬 物理例子深挖

**定常约束——固定单摆**

```
悬挂点固定，绳子长度 l 不变：
x² + y² = l²

问：小球虚位移有哪些方向？
答：沿圆弧切向（上下两个方向）

问：小球的实际位移沿哪个方向？
答：也是沿圆弧切向（由运动决定是往上还是往下）

→ 虚位移的集合 = 实位移可能的集合
```

**非定常约束——绳长变化的单摆（有人缓慢拉绳子）**

```
绳长随时间变化：x² + y² = [l(t)]²

在时刻 t₀：
  - 虚位移（冻结时间）：沿着以 l(t₀) 为半径的圆弧切向
  - 实位移（经历 dt 时间）：既包括沿圆弧的运动，也包括绳子伸长/缩短
    带来的径向分量

→ 虚位移和实位移不是同一个集合！
→ 虚位移只有切向，实位移还有径向分量
```

> 🔥 **这就是为什么"虚位移"要冻结时间（δt = 0）**——你要分析某一瞬间系统的"可能性空间"，而不是系统随时间演化的真实路径。

#### 🎓 考试陷阱：非定常约束下约束力可能做功

```
定常约束：小球在固定圆弧上滑 → 约束力（法向力）始终垂直于速度 → 做功为零
非定常约束：绳子在拉小球 → 约束力（绳子张力）在小球的径向运动上做功！

这就是为什么在非定常约束下，"约束力不做功"不一定成立。
但在达朗贝尔-拉格朗日方程中，我们讨论的是虚功——约束力在虚位移
上做的虚功（冻结时间），所以对理想约束仍为零。
```

#### 📊 定常 vs 非定常 速查表

| 判断 | 定常 | 非定常 |
|------|------|--------|
| 方程含 t？ | 否 | 是 |
| 约束面/线是否移动？ | 否 | 是 |
| δr 和 dr 的关系 | δr 的集合 ⊇ dr 的方向 | δr 的集合 ≠ dr 的方向 |
| 约束力做实功？ | 不（理想约束） | 可能做 |
| 约束力做虚功？ | 不（理想约束） | 不（理想约束） |
| 例子 | 固定单摆、珠子在固定铁丝上 | 绳长变化的单摆、移动的斜面 |

### 第三层：双侧约束 vs 单侧约束（"两边都关死" vs "只关一边"）

#### 🎯 一句话区分

| | 双侧约束 | 单侧约束 |
|---|---|---|
| **本质** | 质点被"夹"在约束面上 | 质点被"挡"在约束面一侧 |
| **方程** | $f = 0$（等号） | $f \geq 0$（不等号） |
| **能否脱离？** | 不能——两边都限制 | 能——可以从不受限制的一侧离开 |

#### 🏠 生活类比

**双侧约束 = 穿在晾衣绳上的衣架**
```
衣架套在绳子上 → 衣架不能离开绳子
既不能往上离开，也不能往下离开 → 双边都"锁死"
约束方程：衣架位置 = 绳子上的某个点
```

**单侧约束 = 地面上弹跳的球**
```
球可以在地面上（z = 0），也可以在空中（z > 0）
但不能钻到地下（z < 0 不允许）
约束方程：z ≥ 0（不等式！）
一侧（下方）有约束，一侧（上方）自由
```

#### 🔬 更多物理例子

**双侧约束**（课本99%讨论的）
- 珠子穿在铁丝上——不能离开铁丝
- 滑块嵌在滑轨里——只能沿轨道方向移动
- 双原子分子（刚性连接）——原子间距固定为 l，不能大也不能小

**单侧约束**
- 书放在桌面上——书可以放在桌上，也可以被拿起来，但不能穿透桌面
- 摆锤用绳子挂——绳子可以拉但不能推（绳子松弛时摆锤自由下落）
- 球在碗里——球可以在碗底，也可以滚出碗

#### 🤔 为什么课本只讨论双侧约束？

因为双侧约束用**等号** `=` 处理，拉格朗日乘子法等数学工具直接可用。
单侧约束用**不等号** `≥` 处理，需要判断"约束是否激活"——更复杂，超出本科范围。

> 💡 但你考试时可能遇到"绳子拉小球"——绳子只能拉不能推。这时要额外判断：如果算出的绳子张力为负（即绳子在推小球），说明小球已经脱离约束，用自由落体重新算。

### 🧩 三层分类的综合练习

> 读下面的例子，在脑子里给每个系统打三个标签

**例子1：固定长度的单摆**
- 完整约束 ✓（方程 x²+y²=l² 只含坐标）
- 定常约束 ✓（方程不含 t）
- 双侧约束 ✓（小球既不能脱离绳子往外飞，也不能缩短绳子往里缩）

**例子2：绳长随时间增长的摆（缓慢拉绳）**
- 完整约束 ✓（虽然绳长变化，但 x²+y²=l²(t) 仍只含坐标）
- 非定常约束 ✗（方程含 t）
- 双侧约束 ✓

**例子3：冰刀在冰面上滑动**
- 非完整约束 ✗（约束方程含速度且不可积）
- 定常约束 ✓（冰面条件不随时间变）
- 双侧约束 ✓（冰刀不能离开冰面）

**例子4：球在地面上弹跳（完全弹性）**
- 完整约束 ✓（在地面接触时 z=0，离开后无约束）
- 定常约束 ✓（地面位置固定）
- 单侧约束 ✗（z ≥ 0）

**例子5：滚动中的硬币**
- 非完整约束 ✗（纯滚动条件含速度且不可积）
- 定常约束 ✓（桌面固定）
- 双侧约束 ✓（硬币始终接触桌面）

---

### 📋 为什么要分这么多层？——拉格朗日力学的适用范围

```
拉格朗日方程的标准形式要求：
  1. 完整约束（否则不能定义独立广义坐标）
  2. 理想约束（约束力在虚位移上不做虚功）

定常/非定常的区分：影响的是虚位移和实位移的关系，以及约束力是否做实功。
双侧/单侧的区分：影响的是约束方程的形式（等号 vs 不等号）。
```

> 🎯 **一句话**：只要是完整约束 + 理想约束 → 拉格朗日方程就能用。其他分类帮你理解细枝末节。

## 🔢 自由度计算——傻瓜式入门教程

> **这一节的目标**：用最笨的方法，让你看到任何系统都能一步一步数出自由度。
> 
> **预备知识**：你只需要知道 $x, y, z$ 是三个数，描述一个点的位置。

---

### 🐣 第0步：先搞懂三个词

**什么是"坐标"？**
> 描述一个东西在哪儿，需要的数。

```
一个蚂蚁在桌面上爬 → 需要 x, y 两个数描述它的位置 → 2个坐标
一只苍蝇在空中飞 → 需要 x, y, z 三个数 → 3个坐标
一支笔直直地立在桌上 → 需要：笔中心x, 笔中心y, 笔的倾角θ → 3个坐标
```

**什么是"约束"？**
> 限制物体不能随便动的东西。每个约束就是一个"必须满足的等式"。

```
小球被绳子拴住 → 绳子长度不能变 → 约束等式：x²+y²+z² = l²
小球在桌面上 → 不能陷进桌子 → 约束等式：z = 0
两根杆用铰链连着 → 连接点位置必须一样 → 两个等式：x₁=x₂, y₁=y₂
```

**什么是"自由度"？**
> 回答一个问题：**你到底需要几个数，才能完全确定这个系统的状态？**

> 自由度的数目 = 你想让一个人从远处复现你这个系统，你最少要打电话告诉他几个数。

---

### 🐣 第1步：质点系统——死记这个方法

**质点是什么？**  
> 把物体看成一个小点。只有位置，没有朝向。如果你在题目里看到的物体能简化成一个"小球"或者一个"滑块"——那就用质点法。

---

### 📐 质点法三步走

```
┌─────────────────────────────────────────────────────────┐
│  第一步：写出所有质点的所有坐标                           │
│          → 三维：每个质点 3 个坐标 (xᵢ, yᵢ, zᵢ)           │
│          → 二维：每个质点 2 个坐标 (xᵢ, yᵢ)               │
│          → 一维：每个质点 1 个坐标 (xᵢ)                    │
│                                                         │
│  第二步：列出所有约束等式                                 │
│          → 每根绷紧的绳子 → 1 个等式（距离=l²）            │
│          → 每个卡住的轨道/桌面 → 1 个等式                  │
│          → 每根不能弯的杆（两端连着质点）→ 1 个等式         │
│                                                         │
│  第三步：自由度 = 总坐标数 - 独立约束等式数                 │
│          → 数"有几个等号"                                 │
└─────────────────────────────────────────────────────────┘
```

> 🔴 **重要**：非完整约束（带速度的，比如"纯滚动"）**不能**在这里减！后面单独讲。

---

### 🖼️ 例1：单摆（三维空间里荡的小球）

```
     ┌────── 天花板 (固定点)
     │
     │  ← 绳子，长度 l
     │
     ●  ← 小球
```

一步一步数：

```
第一步：写出所有坐标
  小球1个质点 → N=1
  三维空间 → 3个坐标：(x, y, z)
  总坐标数 = 3

第二步：列出约束等式
  绳子绷紧，小球到固定点(0,0,0)距离始终为 l
  → 等式：x² + y² + z² = l²  ← 这是 1 个等号
  所以约束数 k = 1

第三步：相减
  自由度 s = 3 − 1 = 2

理解：要描述这个摆球在哪儿，只需要两个数（比如仰角θ和方位角φ）。
```

---

### 🖼️ 例2：双摆（三维空间，两个球，两根绳）

```
     ┌── 天花板
     │
     │ 绳1，长 l₁
     │
     ● 球1
     │
     │ 绳2，长 l₂
     │
     ● 球2
```

一步一步数：

```
第一步：坐标数
  球1 → (x₁, y₁, z₁) → 3个
  球2 → (x₂, y₂, z₂) → 3个
  总坐标数 = 6

第二步：约束等式
  绳1：x₁² + y₁² + z₁² = l₁² → 1个等号
  绳2：(x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)² = l₂² → 1个等号
  约束数 k = 2

第三步：自由度 = 6 − 2 = 4
```

> 想想：描述双摆需要 4 个数——比如球1的两个角度 + 球2的两个角度。

---

### 🖼️ 例3：滑块在桌面上，用弹簧连着墙（二维）

```
┌────墙壁─────────────────────────┐
│                                  │
│  ┌┐  弹簧  ■滑块                  │
│  └┘~~~~~~~ ■                    │
│                                  │
└────────────────── 光滑桌面 ──────┘
```

一步一步数：

```
第一步：坐标数
  滑块 1 个质点 → 二维 → (x, y) → 2个坐标
  总坐标数 = 2

第二步：约束等式
  滑块在桌面上不能陷进去 → y = 0 → 1个等号
  弹簧：没有"长度必须等于某个值"的约束，弹簧可以伸缩！
       → 弹簧不是约束！它提供力，但不限制运动。
  约束数 k = 1（只有桌面）

第三步：自由度 = 2 − 1 = 1

理解：滑块只能沿 x 方向来回动。1 个坐标 x 就够描述了。
```

> 🔑 **关键**：弹簧 ≠ 约束！约束是限制不能动，弹簧是让你能动但有阻力。两者完全不同。

---

### 🖼️ 例4：两个滑块用连杆连着（二维，都在地面上）

```
┌────────────────── 地面（滑块在地面上）──────────────────┐
│                                                        │
│   ■滑块1 ───── 连杆(长l) ───── ■滑块2                  │
│                                                        │
└─────────────────────────────────────────────────────────┘
```

一步一步数：

```
第一步：坐标数
  滑块1 → (x₁, y₁)
  滑块2 → (x₂, y₂)
  总坐标数 = 4

第二步：约束等式
  地面约束1：y₁ = 0 → 1个等号
  地面约束2：y₂ = 0 → 1个等号
  连杆约束：(x₁−x₂)² + (y₁−y₂)² = l² → 但 y₁=y₂=0（地面上）
            → 化简：(x₁−x₂)² = l² → |x₁−x₂| = l → 1个等号
  约束数 k = 3

第三步：自由度 = 4 − 3 = 1

理解：你只需要告诉我滑块1的 x₁，滑块2的位置就被连杆长度确定了。
```

---

### 🤖 第2步：刚体系统——用另一个方法

**什么时候用刚体法而不是质点法？**

> 如果一个物体有**大小和朝向**，不能简化成一个点，那就用刚体法。
> 
> 标志：题目说"一根杆"、"一个圆盘"、"一个木块"——这些东西能**转动**。

---

### 📐 刚体法——死记公式

> **一个刚体在空间里的自由度 = 它能在几个方向上移动 + 它能在几个方向上转动。**

| 空间 | 移动方向数 | 转动方向数 | 一个刚体的自由度 |
|------|------------|------------|------------------|
| 三维（空中） | 3（x,y,z方向） | 3（绕x,y,z轴） | **6** |
| 二维（桌面） | 2（x,y方向） | 1（绕垂直轴转） | **3** |
| 一维（轨道） | 1 | 0（不能转） | **1** |

> 别想为什么，**先背下来**：
> - 三维刚体 → 6
> - 二维刚体 → 3
> - 一维刚体 → 1

---

### 🖼️ 例5：平面上的一根杆（二维，自由运动）

```
       杆
   ●==========●
   (可以在桌面上随便平移 + 转圈)
```

```
第一步：刚体数 = 1
  二维刚体 → 初始自由度 = 3

第二步：约束等式
  没有。杆是自由的。
  k = 0

第三步：自由度 = 3

理解：需要 3 个数 → 质心 x, 质心 y, 杆的倾角 θ。
```

---

### 🖼️ 例6：单摆（用刚体法看——一根杆挂在天花板上）

```
     ┌── 天花板 (铰链点)
     │
     │
     │  杆（刚体，长 l，挂在铰链上可自由摆动）
     │
     │
```

> 这里杆不是绳子——杆有粗细、能转动。但在二维单摆里，杆只摆不转。

```
方法一（刚体法）：
  杆是刚体 → 二维 → 初始自由度 = 3
  约束：杆的一端固定在铰链上
        固定点的坐标不动 → x固定端=0, y固定端=0 → 2个等号（二维）
        杆的另一端自由
  约束数 k = 2
  自由度 = 3 − 2 = 1 ✓

方法二（质点法——把杆简化成末端小球）：
  小球 1 个质点 → 二维 → 2 个坐标
  杆长约束：小球到固定点距离 = l → 1 个等号
  自由度 = 2 − 1 = 1 ✓

→ 两种方法都能得到 1。
```

---

### 🖼️ 例7：双摆（二维，用刚体法）

```
     ┌── 天花板
     │  铰链0（固定）
     │
     │  杆1，长 l₁
     │
     ○  铰链1（杆1下端 = 杆2上端）
     │
     │  杆2，长 l₂
     │
     ●  自由端
```

```
刚体法：
  杆1是刚体 → 二维 → 初始 3 自由度
  杆1上端固定（铰链0）→ 减 2 个约束（该点 x=0, y=0）
  杆1现有 3−2=1 自由度

  杆2是刚体 → 二维 → 初始 3 自由度
  杆2上端和杆1下端铰接（铰链1）→ 减 2 个约束（该点两杆坐标一致）
  杆2现有 3−2=1 自由度

  总自由度 = 1 + 1 = 2 ✓（常用 θ₁, θ₂ 描述）
```

---

### 🪢 第3步：铰链到底减几个约束？——死记这张表

> 铰链 = 两根杆连在一起的那个点，两根杆在连接点处必须坐标一致。

```
一个铰链减几个约束？

二维（桌面）：
  铰链让两个物体的连接点 x 和 y 分别相等
  → 2 个等式：x₁=x₂, y₁=y₂
  → 减 2

三维（空中）：
  铰链让两个物体的连接点 x, y, z 分别相等
  → 3 个等式：x₁=x₂, y₁=y₂, z₁=z₂
  → 减 3
```

> 死记：**二维铰链减2，三维铰链减3。**

---

### 🖼️ 例8：四连杆机构（二维，一个闭环，4根杆用4个铰链连接）

```
   铰链0（固定在地面）  铰链1
         ○──────────────────○
        /                    \
       /                      \
      /                        \
     ○──────────────────────────○
   铰链3（固定在地面）        铰链2

   杆1：铰链0--铰链1
   杆2：铰链1--铰链2
   杆3：铰链2--铰链3
   杆4（地面）：铰链0--铰链3（固定）
```

```
刚体法一步一步来：

杆1 是刚体 → 二维 → 初始 3
  约束：铰链0固定在地面 → 减 2
        （杆1上端连接铰链0，铰链0位置固定 → 杆1上端的x,y被定住）
  杆1现有：3−2=1

杆2 是刚体 → 二维 → 初始 3
  约束：上端和杆1下端铰接（铰链1）→ 减 2
  杆2现有：3−2=1

杆3 是刚体 → 二维 → 初始 3
  约束：上端和杆2下端铰接（铰链2）→ 减 2
        下端和地面铰接（铰链3，固定）→ 减 2
  杆3现有：3−2−2= −1？？
```

> 💥 这里出现了**约束冗余**——4根杆形成闭环后，不是每个约束都是独立的。  
> 👉 **本科阶段不考闭环判断，只要记结论：一个n连杆平面机构（两端固定），自由度 = n−2**。比如4连杆 = 2自由度。

> ✅ 简化版结论：四连杆机构 = 1 自由度（梁昆淼教材标准答案）。你不需要自己算闭环，**直接记结论**。

### ⚡ 第4步：非完整约束——考试最容易扣分的地方

**先搞清楚完整约束和非完整约束的区别：**

```
完整约束（Holonomic）：
  → 只涉及坐标，不涉及速度
  → 等式里没有 ẋ, θ̇ 这种带点的东西
  → 例子：x²+y²=1, z=0, |x₁−x₂|=l

非完整约束（Nonholonomic）：
  → 涉及速度
  → 等式里有 ẋ, θ̇ 这种带点的东西
  → 例子：ẋ = Rθ̇, ẋ·sinθ = ẏ·cosθ
```

> **铁律：非完整约束不能用来减自由度！**  
> 看到带速度的等式 → 无视它，别用它做减法。

---

### 🖼️ 例9：冰刀在冰面上（经典非完整约束）

```
         冰刀  →
     ═══════●═══════
          ↑
    只能沿着刀尖方向滑，不能横着滑
```

```
分析：
  冰刀在二维冰面上 → 需要 (x, y, θ) 三个坐标
    x：刀尖的x坐标
    y：刀尖的y坐标
    θ：冰刀的朝向

  约束：冰刀不能横着滑
    → 横滑速度 = 0 → ẋ·sinθ − ẏ·cosθ = 0
    → 等号里带 ẋ, ẏ ← 这是速度！→ 非完整约束！

  完整约束数 k = 0（没有只含坐标的等式）
  自由度数 s = 3 − 0 = 3（不是 2！）

常见错误：s = 3 − 1 = 2 → ❌ 大错特错！
```

---

### 🖼️ 例10：硬币在桌面上纯滚动（非完整约束）

```
    ┌── 硬币（从侧面看是个圆盘竖着滚）
    │    ○→ x
    │    θ
  桌面 ──────────────
```

# 第二章 达朗贝尔原理——分析力学的第一块基石

> **本章目标**：透彻理解虚位移、虚功、达朗贝尔原理，为第三章拉格朗日方程铺路。这一章在梁昆淼教材中是**独立的一章**，是分析力学的地基。

---

## 2.1 从牛顿到达朗贝尔——把动力学"装进"静力学

### 牛顿力学的困境

对于 $N$ 个质点的系统，牛顿力学需要：

1. 对每个质点列 $\vec{F}=m\vec{a}$（共 $3N$ 个方程）
2. 列出所有约束力的表达式
3. 联立求解

对于复杂系统（如由多根杆连接的机构），约束力的数目可能比自由度多很多倍——大量方程是冗余的。

### 达朗贝尔的天才想法

把 $m\vec{a}$ 移到方程左边：

$$\vec{F} - m\vec{a} = 0$$

**把 $-m\vec{a}$ 当成一个"假想的力"（惯性力）**，那么这个方程就变成了"合力为零"——**形式上与静力学平衡方程完全一样**！

> **物理直觉**：你在加速上升的电梯里感觉"变重"——那就是惯性力在起作用。电梯向上加速 $a$，你感觉多了一个向下的力 $ma$。把惯性力加上去，你就可以用"静止"的思维处理加速运动。

### 达朗贝尔原理的数学形式

对系统中每个质点 $i$：

$$\vec{F}_i + \vec{R}_i - m_i\ddot{\vec{r}}_i = 0$$

其中：

- $\vec{F}_i$：主动力（已知的力，如重力、弹簧力）
- $\vec{R}_i$：约束力（未知的，如绳子张力、接触面法向力）
- $-m_i\ddot{\vec{r}}_i$：达朗贝尔惯性力（假想的）

> **核心突破**：动力学方程变成了"平衡方程"——你可以用静力学的全部工具来处理动力学问题！

---

## 2.2 虚位移——时间冻结下的"可能位移"

### 为什么需要虚位移？

达朗贝尔原理还是含约束力 $\vec{R}_i$ 的。我们的目标是**消去约束力**——因为大多数情况下我们不需要知道绳子张力是多少，只关心物体的运动。

消去约束力的关键工具就是**虚位移**。

### 实位移 vs 虚位移——一张表讲清楚

| | 实位移 $d\vec{r}$ | 虚位移 $\delta\vec{r}$ |
|---|---|---|
| **发生条件** | 真实运动，经历时间 $dt$ | 假想的，时间"冻结"（$dt=0$） |
| **与约束的关系** | 满足约束且满足运动方程 | 只满足瞬时约束 |
| **方向** | 唯一（由初始条件决定） | 无穷多种可能（约束曲面上的任意切方向） |
| **记号** | $d$ | $\delta$ |

> **形象比喻**：一个珠子穿在弯曲的铁丝上。实位移是珠子在 $dt$ 时间内实际移动的方向和距离（由力和初速度决定）。虚位移是**你用手在铁丝上拨一下珠子**——珠子只能沿铁丝方向移动，但你可以往左拨也可以往右拨，拨多拨少也随意。

### 虚位移的数学定义

满足以下条件的无穷小位移 $\delta\vec{r}$ 叫虚位移：

1. **瞬时性**：时间冻结，$\delta t = 0$
2. **约束相容**：满足瞬时约束条件

对于完整约束 $f(\vec{r},t)=0$：

$$\nabla f \cdot \delta\vec{r} = 0$$

即 $\delta\vec{r}$ 在约束曲面的切平面内。

### 定常约束 vs 非定常约束下的虚位移（重要！）

**定常约束**（如固定的铁丝）：虚位移 = 可能的实位移方向——铁丝不动，珠子只能沿铁丝走。

**非定常约束**（如振动的铁丝）：虚位移方向 ≠ 实位移方向！因为约束本身在动——铁丝在振动，珠子的实位移包含了"跟着铁丝一起动"的部分，但虚位移冻结了时间，铁丝在那一瞬间是"静止"的。

> **考试常考**：非定常约束下，虚位移的集合 ≠ 可能实位移的集合！

---

## 2.3 理想约束——消去约束力的关键

### 定义

**理想约束**：约束力在任何虚位移上不做功（或做功之和为零）：

$$\sum_i \vec{R}_i \cdot \delta\vec{r}_i = 0$$

### 哪些约束是理想的？

| 约束类型 | 为什么是理想约束 |
|----------|------------------|
| **光滑曲面** | 约束力沿法向，虚位移沿切向——点乘为零 |
| **刚性杆连接** | 两端约束力等大反向，沿杆方向——在刚性杆约束下两端虚位移沿杆分量相等，总功为零 |
| **不可伸长绳** | 类似刚性杆，张力沿绳方向，总虚功为零 |
| **纯滚动（无滑动）** | 接触点瞬时速度为零，约束力不做功 |

### 摩擦力怎么办？

摩擦力在虚位移上**做功不为零**——所以摩擦力不是理想约束力。处理方法：把摩擦力当**主动力**处理，移到 $\vec{F}_i$ 那边去。

---

## 2.4 虚功原理——静力学的优雅解法

### 从静力学平衡到虚功原理

系统静止平衡时，每个质点合力为零：$\vec{F}_i + \vec{R}_i = 0$。

乘上虚位移并求和：

$$\sum_i (\vec{F}_i + \vec{R}_i) \cdot \delta\vec{r}_i = 0$$

利用理想约束条件 $\sum_i \vec{R}_i \cdot \delta\vec{r}_i = 0$，得到：

> $$\boxed{\sum_i \vec{F}_i \cdot \delta\vec{r}_i = 0}$$

**虚功原理**：理想约束下，系统静平衡的充要条件是所有主动力在任意虚位移上做的虚功之和为零。

### 虚功原理的强大之处

**不需要求约束力！** 方程中只有主动力 $\vec{F}_i$。对于复杂的静力学问题（滑轮组、杠杆系统、桁架），用牛顿法需要列出大量含约束力的方程，虚功原理一步到位。

### 经典例题：滑轮组的平衡

> 两个质量 $m_1$, $m_2$ 通过无摩擦滑轮组连接（阿特伍德机）。求平衡条件。

**牛顿法**：对每个质量列方程 → $T-m_1g=0$, $T-m_2g=0$ → $m_1=m_2$。需要引入张力 $T$ 再消去。

**虚功法**：

- 选绳子位移 $x$ 为广义坐标（向上为正）
- 虚位移：$\delta x_1 = \delta x$，$\delta x_2 = -\delta x$（绳子总长不变）
- 主动力虚功：$m_1g \cdot \delta x_1 + m_2g \cdot \delta x_2 = m_1g\,\delta x - m_2g\,\delta x = (m_1-m_2)g\,\delta x = 0$
- $\delta x$ 任意 → **$m_1 = m_2$**。没有张力出现！

> 对于更复杂的滑轮组（如 3 个动滑轮 + 4 段绳），虚功法比牛顿法省 5 个以上方程。

---

## 2.5 达朗贝尔-拉格朗日方程——分析力学的"母方程"

### 推导

从达朗贝尔原理出发：$\vec{F}_i + \vec{R}_i - m_i\ddot{\vec{r}}_i = 0$。

乘以虚位移 $\delta\vec{r}_i$ 并求和：

$$\sum_i (\vec{F}_i - m_i\ddot{\vec{r}}_i) \cdot \delta\vec{r}_i + \sum_i \vec{R}_i \cdot \delta\vec{r}_i = 0$$

利用理想约束条件（第二项为零）：

> $$\boxed{\sum_i (\vec{F}_i - m_i\ddot{\vec{r}}_i) \cdot \delta\vec{r}_i = 0}$$

这就是**达朗贝尔-拉格朗日方程**（也叫动力学虚功原理）。

### 为什么叫"母方程"？

- 从这个方程出发，经过广义坐标变换 → **拉格朗日方程**（第三章）
- 从拉格朗日方程出发，经勒让德变换 → **哈密顿正则方程**（第七章）
- 从哈密顿原理（第八章）也可以反推回来

整个分析力学的大厦，就建在这一条方程上。

### 每一项的物理含义

| 项 | 含义 |
|----|------|
| $\vec{F}_i \cdot \delta\vec{r}_i$ | 主动力在虚位移上做的虚功 |
| $-m_i\ddot{\vec{r}}_i \cdot \delta\vec{r}_i$ | 惯性力在虚位移上做的虚功 |
| 总和 $= 0$ | "主动力虚功 + 惯性力虚功 = 0"——动力学的"平衡"条件 |

---

## 2.6 从直角坐标到广义坐标——为拉格朗日方程铺路

达朗贝尔-拉格朗日方程中还用的是直角坐标 $\vec{r}_i$。对于受约束系统，$\vec{r}_i$ 不独立——需要切换到广义坐标 $q_j$。

### 坐标变换

$$\vec{r}_i = \vec{r}_i(q_1, q_2, ..., q_s, t)$$

其中 $s =$ 自由度数目。对时间求导：

$$\dot{\vec{r}}_i = \sum_j \frac{\partial \vec{r}_i}{\partial q_j}\dot{q}_j + \frac{\partial \vec{r}_i}{\partial t}$$

### 两个重要恒等式（考试推导可能用到）

> **恒等式①**：
> $$\frac{\partial \dot{\vec{r}}_i}{\partial \dot{q}_j} = \frac{\partial \vec{r}_i}{\partial q_j}$$

> **恒等式②**：
> $$\frac{d}{dt}\left(\frac{\partial \vec{r}_i}{\partial q_j}\right) = \frac{\partial \dot{\vec{r}}_i}{\partial q_j}$$

### 虚位移的广义坐标表示

时间冻结（$\delta t = 0$）：

$$\delta\vec{r}_i = \sum_j \frac{\partial \vec{r}_i}{\partial q_j}\delta q_j$$

---

### 详细推导：从达朗贝尔-拉格朗日方程到基本拉格朗日方程

> 这是全书最关键的推导之一。如果考试出"请推导拉格朗日方程"的证明题，以下就是标准答案。

---

#### 出发点：达朗贝尔-拉格朗日方程

从第二章第 2.5 节，已有（**理想约束条件下**）：

$$\sum_i (\vec{F}_i - m_i\ddot{\vec{r}}_i) \cdot \delta\vec{r}_i = 0$$

拆成两项：

$$\underbrace{\sum_i \vec{F}_i \cdot \delta\vec{r}_i}_{\text{主动力虚功}} - \underbrace{\sum_i m_i\ddot{\vec{r}}_i \cdot \delta\vec{r}_i}_{\text{惯性力虚功}} = 0$$

---

#### 第一步：把 $\delta\vec{r}_i$ 换成用 $\delta q_j$ 表示

用虚位移的广义坐标表达式：

$$\delta\vec{r}_i = \sum_j \frac{\partial \vec{r}_i}{\partial q_j}\delta q_j$$

---

#### 第二步：第一项（主动力虚功）——定义广义力

把 $\delta\vec{r}_i$ 换入第一项：

$$\sum_i \vec{F}_i \cdot \delta\vec{r}_i = \sum_i \vec{F}_i \cdot \left(\sum_j \frac{\partial \vec{r}_i}{\partial q_j}\delta q_j\right)$$

交换求和顺序（$\sum_i$ 和 $\sum_j$ 互换）：

$$= \sum_j \left(\sum_i \vec{F}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}\right) \delta q_j$$

定义括号里的东西为**广义力**：

> $$\boxed{Q_j \equiv \sum_i \vec{F}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}}$$

于是第一项 $= \sum_j Q_j \delta q_j$。

**物理直觉**：

- 如果 $q_j$ 是普通坐标 $x$，则 $\partial\vec{r}/\partial x =$ 单位向量 $\hat{e}_x$ → $Q_x = F_x$（普通力的 $x$ 分量）
- 如果 $q_j$ 是角度 $\theta$，则 $\partial\vec{r}/\partial\theta$ 是旋转方向 → $Q_\theta$ 是力矩
- $Q_j\delta q_j$ 可以不是"焦耳"的量纲——取决于 $q_j$ 的量纲

---

#### 第三步：第二项（惯性力虚功）——建立与动能 $T$ 的联系

这是推导的核心难点。

动能：

$$T = \frac{1}{2}\sum_i m_i |\dot{\vec{r}}_i|^2 = \frac{1}{2}\sum_i m_i (\dot{\vec{r}}_i \cdot \dot{\vec{r}}_i)$$

我们要处理的是：

$$\sum_i m_i \ddot{\vec{r}}_i \cdot \delta\vec{r}_i = \sum_i m_i \ddot{\vec{r}}_i \cdot \left(\sum_j \frac{\partial \vec{r}_i}{\partial q_j}\delta q_j\right) = \sum_j \left(\sum_i m_i \ddot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}\right) \delta q_j$$

现在需要把括号里这个含 $\ddot{\vec{r}}_i$ 的复杂表达式，变成只含 $T$ 的某种导数的简洁形式。

---

**第三步(a)：计算 $\dfrac{\partial T}{\partial \dot{q}_j}$**

对 $T = \frac{1}{2}\sum_i m_i (\dot{\vec{r}}_i \cdot \dot{\vec{r}}_i)$ 求偏导（用链式法则）：

$$\frac{\partial T}{\partial \dot{q}_j} = \sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \dot{\vec{r}}_i}{\partial \dot{q}_j}$$

现在用**恒等式①**：

$$\frac{\partial \dot{\vec{r}}_i}{\partial \dot{q}_j} = \frac{\partial \vec{r}_i}{\partial q_j}$$

代回：

> $$\frac{\partial T}{\partial \dot{q}_j} = \sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}$$

---

**第三步(b)：计算 $\dfrac{d}{dt}\left(\dfrac{\partial T}{\partial \dot{q}_j}\right)$**

对上面结果两边求全时间导数（用乘法法则）：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) = \sum_i m_i \ddot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j} + \sum_i m_i \dot{\vec{r}}_i \cdot \frac{d}{dt}\left(\frac{\partial \vec{r}_i}{\partial q_j}\right)$$

现在用**恒等式②**：

$$\frac{d}{dt}\left(\frac{\partial \vec{r}_i}{\partial q_j}\right) = \frac{\partial \dot{\vec{r}}_i}{\partial q_j}$$

代回：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) = \sum_i m_i \ddot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j} + \sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \dot{\vec{r}}_i}{\partial q_j}$$

---

**第三步(c)：计算 $\dfrac{\partial T}{\partial q_j}$**

对 $T$ 直接求对 $q_j$ 的偏导：

$$\frac{\partial T}{\partial q_j} = \sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \dot{\vec{r}}_i}{\partial q_j}$$

---

**第三步(d)：组合——关键一步！**

用第三步(b)减去第三步(c)：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j}$$

$$= \left[\sum_i m_i \ddot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j} + \sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \dot{\vec{r}}_i}{\partial q_j}\right] - \left[\sum_i m_i \dot{\vec{r}}_i \cdot \frac{\partial \dot{\vec{r}}_i}{\partial q_j}\right]$$

两个带 $\partial\dot{\vec{r}}_i/\partial q_j$ 的求和项**恰好抵消**！

> $$\boxed{\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} = \sum_i m_i \ddot{\vec{r}}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}}$$

---

#### 第四步：回到达朗贝尔-拉格朗日方程

原方程现在变成：

$$\sum_j Q_j \delta q_j - \sum_j \left[\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j}\right] \delta q_j = 0$$

即：

> $$\boxed{\sum_j \left[\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} - Q_j\right]\delta q_j = 0}$$

其中 $T$ 是系统总动能，$Q_j = \sum_i \vec{F}_i \cdot \dfrac{\partial \vec{r}_i}{\partial q_j}$ 是**广义力**。

---

#### 第五步：$\delta q_j$ 独立 → 系数为零

由于广义坐标互相**独立**，各 $\delta q_j$ 可以取任意值，上述求和为零要求**每一项的系数都必须为零**：

> $$\boxed{\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} = Q_j \qquad (j=1,2,...,s)}$$

这就是**基本拉格朗日方程**——第三章的起点。

---

#### 推导路线图总结

> 起点：达朗贝尔-拉格朗日方程
>
> $$\sum_i (\vec{F}_i - m_i\ddot{\vec{r}}_i) \cdot \delta\vec{r}_i = 0$$
>
> ↓ 第一项：$\vec{F}_i \cdot \delta\vec{r}_i$ → 用 $\delta\vec{r} = \sum \frac{\partial\vec{r}}{\partial q}\delta q$ → 得 $\sum_j Q_j \delta q_j$（广义力）
>
> ↓ 第二项：$m_i\ddot{\vec{r}}_i \cdot \delta\vec{r}_i$ → 需要与动能 $T$ 建立联系
>
> ↓ 计算 $\dfrac{\partial T}{\partial \dot{q}_j}$ → 用恒等式① $\dfrac{\partial \dot{\vec{r}}}{\partial \dot{q}} = \dfrac{\partial \vec{r}}{\partial q}$
>
> ↓ 计算 $\dfrac{d}{dt}\!\left(\dfrac{\partial T}{\partial \dot{q}_j}\right)$ → 用恒等式② $\dfrac{d}{dt}\!\left(\dfrac{\partial \vec{r}}{\partial q}\right) = \dfrac{\partial \dot{\vec{r}}}{\partial q}$
>
> ↓ 计算 $\dfrac{\partial T}{\partial q_j}$
>
> ↓ $\dfrac{d}{dt}\!\left(\dfrac{\partial T}{\partial \dot{q}}\right) - \dfrac{\partial T}{\partial q} = \sum m_i \ddot{\vec{r}}_i \cdot \dfrac{\partial \vec{r}_i}{\partial q}$（两个 $\partial\dot{\vec{r}}/\partial q$ 项恰好抵消）
>
> ↓ 代回原式：$\sum_j \left[ \dfrac{d}{dt}\!\left(\dfrac{\partial T}{\partial \dot{q}_j}\right) - \dfrac{\partial T}{\partial q_j} - Q_j \right] \delta q_j = 0$
>
> ↓ $\delta q_j$ 独立
>
> ↓ **基本拉格朗日方程**：$\dfrac{d}{dt}\!\left(\dfrac{\partial T}{\partial \dot{q}_j}\right) - \dfrac{\partial T}{\partial q_j} = Q_j$

---

# 第三章 拉格朗日动力学——理论力学的核心工具

> **本章目标**：完全掌握拉格朗日方程的使用。这是全书最重要的工具——后面哈密顿、变分原理、H-J 方程全都建立在它之上。

---

## 3.1 拉格朗日方程的最终形态

对于保守力系统（$\vec{F} = -\nabla V$），定义拉格朗日函数：

$$L = T - V$$

> $$\boxed{\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j} = 0 \qquad (j = 1,2,...,s)}$$

**这就是理论力学最重要的方程。全书其余部分几乎都是它的应用和推广。**

---

## 3.2 从基本拉格朗日方程到保守力拉格朗日方程——完整推导

> **很多同学卡在这里**："为什么右边突然从 $Q_j$ 变成 0 了？为什么要引入 $L = T - V$？"
>
> 别急，下面把每一步拆开讲，保证讲明白。

---

### 3.2.1 回顾：基本拉格朗日方程的起点

第二章推导出的**非保守形式**：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} = Q_j \qquad (j = 1,2,...,s)$$

左边是"动能 $T$ 的各种导数组合"，右边是**广义力 $Q_j$**。

广义力的定义：

$$Q_j = \sum_i \vec{F}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j}$$

> 上式是一个"通用形式"——无论什么力（摩擦力、磁力、外加力...）都适用，只要把对应的 $Q_j$ 放到右边就行。

---

### 3.2.2 什么是"保守力"？

**保守力**的定义（三个等价说法）：

1. **做功与路径无关**：从 $A$ 到 $B$ 的功只取决于 $A$ 和 $B$ 的位置，与走什么路无关
2. **可用势能函数 $V(\vec{r})$ 描述**：
   $$\vec{F} = -\nabla V = -\left(\frac{\partial V}{\partial x}\hat{x} + \frac{\partial V}{\partial y}\hat{y} + \frac{\partial V}{\partial z}\hat{z}\right)$$
3. **沿任意闭合回路做功为零**：
   $$\oint \vec{F}\cdot d\vec{r} = 0$$

**常见的保守力**：重力、弹簧弹性力、万有引力、静电力  
**非保守力**：摩擦力、空气阻力、磁力（磁力不做功但也不是保守力）

> **关键**：保守力中那个**负号**是物理的，不是随便加的。$\vec{F} = -\nabla V$ 的意思是：力指向势能**降低**的方向。
>
> 例子——重力：取向上为正，$V = mgy$，则 $F_y = -\partial(mgy)/\partial y = -mg$（力向下 ✓）

---

### 3.2.3 第一步：保守力下的广义力 $Q_j$ 是什么？

把 $\vec{F}_i = -\nabla_i V$（梯度只对第 $i$ 个质点的坐标求）代入广义力定义：

$$Q_j = \sum_i \vec{F}_i \cdot \frac{\partial \vec{r}_i}{\partial q_j} = \sum_i (-\nabla_i V) \cdot \frac{\partial \vec{r}_i}{\partial q_j}$$

$$= -\sum_i \left(\frac{\partial V}{\partial x_i}\frac{\partial x_i}{\partial q_j} + \frac{\partial V}{\partial y_i}\frac{\partial y_i}{\partial q_j} + \frac{\partial V}{\partial z_i}\frac{\partial z_i}{\partial q_j}\right)$$

括号里的东西——根据**链式法则**——正是 $\partial V/\partial q_j$！

> **数学证明**（链式法则）：
>
> $V$ 是各质点坐标的函数：$V(x_1,y_1,z_1, x_2,y_2,z_2, ..., t)$
>
> $q_j$ 变化 → 各质点的 $x_i, y_i, z_i$ 都跟着变：
>
> $$\frac{\partial V}{\partial q_j} = \sum_i \left(\frac{\partial V}{\partial x_i}\frac{\partial x_i}{\partial q_j} + \frac{\partial V}{\partial y_i}\frac{\partial y_i}{\partial q_j} + \frac{\partial V}{\partial z_i}\frac{\partial z_i}{\partial q_j}\right)$$
>
> 就是上面括号里的东西！所以：

> $$\boxed{Q_j = -\frac{\partial V}{\partial q_j}}$$

**物理学含义**：

- $\partial V/\partial q_j > 0$ → $q_j$ 增大时势能升高 → 力在 $q_j$ 负方向（阻止 $q_j$ 增大）
- $\partial V/\partial q_j < 0$ → $q_j$ 增大时势能降低 → 力在 $q_j$ 正方向（推动 $q_j$ 增大）

> 广义力 $Q_j$ 和广义坐标 $q_j$ 的乘积 $Q_j\delta q_j$ 是**虚功**——在这个意义下，$Q_j = -\partial V/\partial q_j$ 表示"力总是推动系统朝势能降低的方向运动"。

---

### 3.2.4 第二步：把 $Q_j = -\partial V/\partial q_j$ 代入基本拉格朗日方程

基本拉格朗日方程现在变成：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} = -\frac{\partial V}{\partial q_j}$$

移项（把右边拿到左边）：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} + \frac{\partial V}{\partial q_j} = 0$$

---

### 3.2.5 第三步：为什么 $V$ 可以合并进对 $\dot{q}$ 的导数里？

关键观察：**势能 $V$ 只依赖于坐标 $q$，不依赖于速度 $\dot{q}$**。

因为 $V$ 是位置的函数（$V = V(x_1,y_1,z_1,...,t) = V(q_1,q_2,...,q_s,t)$），所以：

$$\frac{\partial V}{\partial \dot{q}_j} = 0 \quad \text{（$V$ 不含速度→对速度求偏导为零）}$$

这意味着：对任何函数，在 $\partial/\partial\dot{q}$ 前面加一个 $V$ 不会改变结果：

$$\frac{\partial T}{\partial \dot{q}_j} = \frac{\partial (T - V)}{\partial \dot{q}_j}$$

因为 $\partial V/\partial\dot{q} = 0$，加不加 $V$ 都对 $\partial/\partial\dot{q}$ 没影响。

同样：

$$-\frac{\partial T}{\partial q_j} + \frac{\partial V}{\partial q_j} = -\frac{\partial (T - V)}{\partial q_j}$$

---

### 3.2.6 第四步：定义 $L = T - V$，完成合并

把原式用 $L = T - V$ 重写：

$$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j}\right) - \frac{\partial T}{\partial q_j} + \frac{\partial V}{\partial q_j}$$

$$= \frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}_j} - \underbrace{\frac{\partial V}{\partial \dot{q}_j}}_{=0}\right) - \left(\frac{\partial T}{\partial q_j} - \frac{\partial V}{\partial q_j}\right)$$

$$= \frac{d}{dt}\left(\frac{\partial (T-V)}{\partial \dot{q}_j}\right) - \frac{\partial (T-V)}{\partial q_j}$$

$$= \frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j}$$

令这个等于零（来自移项后的方程），得到：

> $$\boxed{\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j} = 0 \qquad (j = 1,2,...,s)}$$

这就是**保守系统的拉格朗日方程**——右边为零，所有"力"的信息都包含在 $L = T - V$ 里了！

---

### 3.2.7 为什么是减号（$T - V$）而不是加号（$T + V$）？

**一句话：减号来自 $\vec{F} = -\nabla V$。**

- 基本方程的右边是 $+Q_j$（力在坐标增大方向的虚功）
- 保守力 $Q_j = -\partial V/\partial q_j$ → 右边变成 $-\partial V/\partial q_j$
- $-\partial V/\partial q_j$ 移项到左边，变成 $+\partial V/\partial q_j$
- 和 $-\partial T/\partial q_j$ 合并 → $-(\partial T/\partial q_j - \partial V/\partial q_j) = -\partial(T-V)/\partial q_j$

**物理本质**：

- 动能 $T$：系统运动的"活力"——$\frac{d}{dt}(\partial T/\partial \dot{q})$ 表示 $q$ 方向的"惯性力"
- $-\partial T/\partial q$：坐标变化时动能如何变化——在曲线坐标中即使没有势能也有这个"几何力"
- $-\partial V/\partial q$：保守力——总是把系统推向势能低谷
- 两项合并：$-(\partial(T-V)/\partial q) = -\partial L/\partial q$ → 系统自然地朝 $L$ 增大的方向"被推"

> $L = T - V$ 衡量的是"动能相对于势能的盈余"。
> 当 $L$ 大 → 系统运动空间大（动能多、势能束缚少）。
> 自然界选择的路径是让"动能 $-$ 势能"这个量取极值（最小作用量原理——后面讲）。
> 这就是 $T-V$ 而非 $T+V$ 的深层原因。

> 如果还不直观——先记住结论：**保守力 $Q_j = -\partial V/\partial q_j$ 代入方程右边，然后 $\partial V/\partial\dot{q} = 0$ 让 $V$ 可以和 $T$ 合并成 $T - V$。**

---

### 3.2.8 总结：从 $Q_j$ 到 $L = T - V$ 的完整思维链

> 基本拉格朗日方程：
>
> $$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}}\right) - \frac{\partial T}{\partial q} = Q_j$$
>
> ↓ 假设：所有力都是保守力 $\vec{F}_i = -\nabla_i V$
>
> ↓ $Q_j = \sum \vec{F}_i \cdot \frac{\partial\vec{r}_i}{\partial q_j} = \sum (-\nabla_i V) \cdot \frac{\partial\vec{r}_i}{\partial q_j} = -\frac{\partial V}{\partial q_j}$（链式法则）
>
> ↓ 代入基本方程右侧
>
> $$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}}\right) - \frac{\partial T}{\partial q} = -\frac{\partial V}{\partial q}$$
>
> ↓ $-\partial V/\partial q$ 移项到左边
>
> $$\frac{d}{dt}\left(\frac{\partial T}{\partial \dot{q}}\right) - \frac{\partial T}{\partial q} + \frac{\partial V}{\partial q} = 0$$
>
> ↓ 关键：$\partial V/\partial\dot{q} = 0$（$V$ 不含速度）
>
> ↓ 所以：$\partial T/\partial\dot{q} = \partial(T-V)/\partial\dot{q}$，同时 $-\partial T/\partial q + \partial V/\partial q = -\partial(T-V)/\partial q$
>
> ↓ 令 $L = T - V$
>
> $$\boxed{\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\right) - \frac{\partial L}{\partial q} = 0}$$

---

## 3.3 拉格朗日方程逐项拆解

把 $L = T - V$ 拆开：

| 项 | 物理含义 |
|----|----------|
| $\partial L/\partial \dot{q}_j$ | **广义动量** $p_j$。对普通坐标 $x$，$\partial L/\partial \dot{x} = m\dot{x} =$ 普通动量 |
| $\frac{d}{dt}(\partial L/\partial \dot{q}_j)$ | 广义动量的时间变化率 = "广义力" |
| $\partial L/\partial q_j$ | 势能负梯度（$-\partial V/\partial q_j$）+ 动能的"惯性力"部分 |

整条方程就是：**广义动量变化率 = 广义力**——和 $\vec{F}=m\vec{a}$ 完全等价，只是换了一套坐标系。

---

## 3.4 能量积分——什么时候能量守恒

### 3.4.1 问题引入

拉格朗日方程是二阶微分方程，解起来有时候很麻烦。但如果能找到**守恒量**（运动积分），就能降阶，大大简化。

最常见的守恒量就是**能量**——但拉格朗日力学里，"能量守恒"和 $\partial L/\partial t = 0$ 是什么关系？为什么有时候 $L$ 不显含 $t$ 但 $H$ 却不等于 $T+V$？

> **考试陷阱**：$H$ 守恒和能量守恒不是同一回事！本节给你讲清楚。

---

### 3.4.2 雅可比积分（广义能量积分）的推导

考虑拉格朗日函数对时间的全导数：

$$\frac{dL}{dt} = \sum_j \left( \frac{\partial L}{\partial q_j}\dot{q}_j + \frac{\partial L}{\partial \dot{q}_j}\ddot{q}_j \right) + \frac{\partial L}{\partial t}$$

利用拉格朗日方程 $\partial L/\partial q_j = \frac{d}{dt}(\partial L/\partial\dot{q}_j)$ 替换第一项：

$$\frac{dL}{dt} = \sum_j \left[ \frac{d}{dt}\left(\frac{\partial L}{\partial\dot{q}_j}\right)\dot{q}_j + \frac{\partial L}{\partial\dot{q}_j}\ddot{q}_j \right] + \frac{\partial L}{\partial t}$$

注意到方括号里是乘积的导数：

$$\frac{d}{dt}\left(\dot{q}_j\frac{\partial L}{\partial\dot{q}_j}\right) = \ddot{q}_j\frac{\partial L}{\partial\dot{q}_j} + \dot{q}_j\frac{d}{dt}\left(\frac{\partial L}{\partial\dot{q}_j}\right)$$

所以：

$$\frac{dL}{dt} = \frac{d}{dt}\sum_j \left(\dot{q}_j\frac{\partial L}{\partial\dot{q}_j}\right) + \frac{\partial L}{\partial t}$$

移项：

$$\frac{d}{dt}\left[\sum_j \dot{q}_j\frac{\partial L}{\partial\dot{q}_j} - L\right] = -\frac{\partial L}{\partial t}$$

这就是**能量积分定理**。现在定义：

> $$\boxed{H = \sum_j \dot{q}_j\frac{\partial L}{\partial\dot{q}_j} - L}$$

$H$ 叫做**雅可比积分**（Jacobi integral），也叫**广义能量函数**。

**关键结论**：

> 如果 $L$ **不显含时间** $t$（即 $\boxed{\partial L/\partial t = 0}$），则：
>
> $$\boxed{\frac{dH}{dt} = 0 \quad \Rightarrow \quad H = \text{常数}}$$

这叫**广义能量积分**。注意条件：是 $\partial L/\partial t = 0$，不是 $dL/dt = 0$。

---

### 3.4.3 $H = T + V$ 还是 $H \neq T + V$？——关键区别

$H$ 什么时候等于总能量 $E = T + V$？这里有一个**考试极容易错**的坑。

#### 情况一：定常约束（$\partial\vec{r}_i/\partial t = 0$）→ $H = E$

如果约束不随时间变化（定常约束 = 完整、稳定约束），坐标变换 $\vec{r}_i = \vec{r}_i(q_1,...,q_s)$ **不显含 $t$**。

此时系统动能 $T$ 可以写为广义速度的**二次齐次函数**：

$$T = \frac{1}{2}\sum_{jk} a_{jk}(q)\dot{q}_j\dot{q}_k \quad \text{（纯二次型，无一次项和零次项）}$$

根据**欧拉齐次函数定理**（二次齐次函数有性质 $\sum_j \dot{q}_j \partial f/\partial\dot{q}_j = 2f$）：

$$\sum_j \dot{q}_j\frac{\partial T}{\partial\dot{q}_j} = 2T$$

由于 $V = V(q)$ 不依赖速度，$\partial V/\partial\dot{q}_j = 0$，所以：

$$\sum_j \dot{q}_j\frac{\partial L}{\partial\dot{q}_j} = \sum_j \dot{q}_j\frac{\partial(T-V)}{\partial\dot{q}_j} = \sum_j \dot{q}_j\frac{\partial T}{\partial\dot{q}_j} = 2T$$

因此：

$$H = \sum_j \dot{q}_j\frac{\partial L}{\partial\dot{q}_j} - L = 2T - (T - V) = T + V = E$$

> **结论**：对于定常约束的保守系统，$H = T + V = E$，$H$ 守恒就是总能量守恒。

---

#### 情况二：非定常约束（$\partial\vec{r}_i/\partial t \neq 0$）→ $H \neq E$

如果约束随时间变化（例如：悬点振动的摆、加速的参考系），坐标变换含 $t$：

$$\vec{r}_i = \vec{r}_i(q_1,...,q_s, t)$$

速度变成：

$$\dot{\vec{r}}_i = \sum_j \frac{\partial\vec{r}_i}{\partial q_j}\dot{q}_j + \frac{\partial\vec{r}_i}{\partial t}$$

动能就会出现一般形式：

$$T = T_2 + T_1 + T_0$$

其中：

- $T_2 = \frac{1}{2}\sum m_{ij}\dot{q}_i\dot{q}_j$——广义速度的二次项（"正常的"动能）
- $T_1 = \sum b_j\dot{q}_j$——广义速度的一次项
- $T_0$——与广义速度无关的项（来自 $\partial\vec{r}_i/\partial t$ 的贡献）

此时欧拉定理给出：

$$\sum_j \dot{q}_j\frac{\partial T_2}{\partial\dot{q}_j} = 2T_2, \quad \sum_j \dot{q}_j\frac{\partial T_1}{\partial\dot{q}_j} = T_1, \quad \sum_j \dot{q}_j\frac{\partial T_0}{\partial\dot{q}_j} = 0$$

所以：

$$\sum_j \dot{q}_j\frac{\partial L}{\partial\dot{q}_j} = 2T_2 + T_1$$

$$H = (2T_2 + T_1) - (T_2 + T_1 + T_0 - V) = T_2 - T_0 + V$$

> **重点**：对于非定常约束，$H = T_2 - T_0 + V \neq T + V$！**$H$ 仍然守恒**（只要 $\partial L/\partial t = 0$），但它**不是总能量 $E$**！

---

### 3.4.4 直观对比表

| 条件 | $T$ 的形式 | $H =$ ? | $H$ 守恒吗？ | $H = E$ 吗？ |
|------|----------|---------|-----------|-----------|
| 定常约束 + $\partial L/\partial t = 0$ | $T = T_2$（纯二次型） | $H = T + V = E$ | 守恒 | 等于 |
| 非定常约束 + $\partial L/\partial t = 0$ | $T = T_2 + T_1 + T_0$ | $H = T_2 - T_0 + V$ | 守恒 | 不等于 |
| 任何情况 + $\partial L/\partial t \neq 0$ | 任意 | $H = \sum \dot{q}\cdot\partial L/\partial\dot{q} - L$ | 不守恒 | — |

> **记忆口诀**：「定常约束 $H$ 就是 $E$，非定常 $H$ 不是 $E$；$H$ 守恒看 $\partial L/\partial t$，$E$ 守恒看约束。」

---

### 3.4.5 经典例题

#### 例题 1：定常约束——单摆

取 $\theta$ 为广义坐标，摆长 $l$，质量 $m$。

**动能**（纯二次型）：

$$T = \frac{1}{2}ml^2\dot{\theta}^2$$

**势能**（取悬点为零势能面）：

$$V = -mgl\cos\theta$$

**拉格朗日函数**：

$$L = T - V = \frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta$$

因为 $L$ 不显含 $t$，所以 $\partial L/\partial t = 0$，$H$ 守恒。

**广义动量**：

$$p_\theta = \frac{\partial L}{\partial\dot{\theta}} = ml^2\dot{\theta}$$

**雅可比积分 $H$**：

$$
\begin{aligned}
H &= p_\theta\dot{\theta} - L \\
  &= ml^2\dot{\theta}^2 - \left(\frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta\right) \\
  &= \frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta \\
  &= T + V = E
\end{aligned}
$$

> $T$ 是 $\dot{\theta}$ 的纯二次型，约束定常 → $H = E$（总能量守恒）。

---

#### 例题 2：非定常约束——滑块在加速斜面上

质量为 $m$ 的滑块放在倾角 $\alpha$ 的光滑斜面上，斜面以速度 $u(t)$ 水平运动。取滑块沿斜面向上的位移 $s$ 为广义坐标。

**位置矢量**（含 $t$ → 非定常约束）：

$$\vec{r} = \bigl( u(t) + s\cos\alpha,\; -s\sin\alpha \bigr)$$

**速度矢量**：

$$\dot{\vec{r}} = \bigl( \dot{u} + \dot{s}\cos\alpha,\; -\dot{s}\sin\alpha \bigr)$$

**动能**：

$$
\begin{aligned}
T &= \frac{1}{2}m\left[(\dot{u} + \dot{s}\cos\alpha)^2 + (-\dot{s}\sin\alpha)^2\right] \\
  &= \frac{1}{2}m\left[\dot{u}^2 + 2\dot{u}\dot{s}\cos\alpha + \dot{s}^2\cos^2\alpha + \dot{s}^2\sin^2\alpha\right] \\
  &= \frac{1}{2}m\dot{s}^2 + m\dot{u}\cos\alpha\,\dot{s} + \frac{1}{2}m\dot{u}^2
\end{aligned}
$$

**分解 $T = T_2 + T_1 + T_0$**：

$$T_2 = \frac{1}{2}m\dot{s}^2,\qquad
T_1 = m\dot{u}\cos\alpha\,\dot{s},\qquad
T_0 = \frac{1}{2}m\dot{u}^2$$

**势能**：

$$V = -mgs\sin\alpha$$

**拉格朗日函数**：

$$L = T - V = \frac{1}{2}m\dot{s}^2 + m\dot{u}\cos\alpha\,\dot{s} + \frac{1}{2}m\dot{u}^2 + mgs\sin\alpha$$

如果斜面匀速运动 $u = v_0 t$（$v_0$ 为常数），则 $\dot{u} = v_0$。此时 $L$ 中 $u$ 显含 $t$，但经检查：

$$\frac{\partial L}{\partial t} = 0$$

因为 $L$ 中只有 $\dot{u}$（= 常数 $v_0$），不含 $u$ 本身。所以 $H$ 守恒。

**广义动量**：

$$p_s = \frac{\partial L}{\partial\dot{s}} = m\dot{s} + mv_0\cos\alpha$$

**雅可比积分 $H$**：

$$
\begin{aligned}
H &= p_s\dot{s} - L \\
  &= (m\dot{s} + mv_0\cos\alpha)\dot{s} - \left(\frac{1}{2}m\dot{s}^2 + mv_0\cos\alpha\,\dot{s} + \frac{1}{2}mv_0^2 + mgs\sin\alpha\right) \\
  &= m\dot{s}^2 + mv_0\cos\alpha\,\dot{s} - \frac{1}{2}m\dot{s}^2 - mv_0\cos\alpha\,\dot{s} - \frac{1}{2}mv_0^2 - mgs\sin\alpha \\
  &= \frac{1}{2}m\dot{s}^2 - \frac{1}{2}mv_0^2 - mgs\sin\alpha
\end{aligned}
$$

**对比总能量**：

$$E = T + V = \frac{1}{2}m\dot{s}^2 + mv_0\cos\alpha\,\dot{s} + \frac{1}{2}mv_0^2 - mgs\sin\alpha$$

显然：

$$H = \frac{1}{2}m\dot{s}^2 - \frac{1}{2}mv_0^2 - mgs\sin\alpha \neq E$$

> **结论**：$H$ 守恒（$\partial L/\partial t = 0$），但 $H \neq E$！原因——约束非定常（斜面在运动），外部约束通过 $T_1$ 项向系统注入/抽取能量，而 $H = T_2 - T_0 + V$ 恰好抵消了这部分贡献。**$H$ 守恒但总能量 $E$ 不守恒**，这是考试中最容易出错的判断题。

---

### 3.4.6 欧拉齐次函数定理（补数学）

如果 $f(x_1, ..., x_n)$ 是**$k$ 次齐次函数**，即满足：

$$f(\lambda x_1, ..., \lambda x_n) = \lambda^k f(x_1, ..., x_n)$$

则：

$$\boxed{\sum_i x_i\frac{\partial f}{\partial x_i} = k f}$$

> 这是上面 $T_2 \to 2T_2$、$T_1 \to T_1$ 的数学依据。考试时可以当已知结论用，但要知道它叫什么。

---

### 3.4.7 总结：三步判断能量守恒

**第一步** — 写出 $L = T - V$

**第二步** — 判断 $\dfrac{\partial L}{\partial t} = 0$ 是否成立

- **否** → $H$ 不守恒，能量一般不守恒
- **是** → $H$ 守恒，进入第三步

**第三步** — 判断约束类型

- **定常** $\left(\dfrac{\partial \vec{r}_i}{\partial t} = 0\right)$ → $H = T + V = E$（总能量守恒）
- **非定常** $\left(\dfrac{\partial \vec{r}_i}{\partial t} \neq 0\right)$ → $H \neq E$（$H$ 守恒但总能量不守恒）

> **考试重点**：$\dfrac{\partial L}{\partial t} = 0 \Rightarrow H$ 守恒——这是**必要充分条件**。但「$H =$ 总能量」还需要**定常约束**这个额外条件。老师最喜欢在这里出判断题坑人。

---

## 3.5 循环坐标与守恒量——拉格朗日最美的性质

如果拉格朗日函数 $L$ 不显含某个坐标 $q_j$：

$$\frac{\partial L}{\partial q_j} = 0$$

则称 $q_j$ 为**循环坐标**。代入拉格朗日方程：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) = 0 \quad \Rightarrow \quad p_j = \frac{\partial L}{\partial \dot{q}_j} = \text{常数}$$

**广义动量守恒！**

| $L$ 不显含... | 对应的守恒量 | 背后的对称性 |
|--------------|-------------|-------------|
| 某个角度 $\theta$ | 角动量守恒 | 旋转对称 |
| 某个位置 $x$ | 线动量守恒 | 平移对称 |
| 时间 $t$ | 雅可比积分守恒 | 时间平移对称 |

> **这就是诺特定理的前身**：每一种连续对称性对应一个守恒量。拉格朗日力学让对称性与守恒律的关联一目了然——找出循环坐标就是找出守恒量。

---

## 3.6 含时约束与非保守力

如果 $L$ 显含时间 $t$，拉格朗日方程的形式不变——直接代入使用。

如果有非保守力（如摩擦力），方程右边加上广义力：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}_j}\right) - \frac{\partial L}{\partial q_j} = Q_j^{\text{非保守}}$$

---

## 3.7 拉格朗日方程解题标准流程

**六步法**：

1. **确定自由度 $s$**，选广义坐标 $q_1,...,q_s$
2. **写直角坐标用广坐表示**：$x_i = x_i(q,t)$
3. **写动能 $T$** $= \frac{1}{2}\sum m_i(\dot{x}_i^2 + \dot{y}_i^2 + \dot{z}_i^2)$，用 $q$ 和 $\dot{q}$ 表达
4. **写势能 $V$** $= V(q)$
5. **构造 $L = T - V$**
6. **代入拉格朗日方程**，逐个 $q_j$ 列方程

---

## 3.8 经典例题大全

### 例题 1：单摆

选摆角 $\theta$ 为广义坐标（1 个自由度）。

直角坐标：$x = l\sin\theta$, $y = -l\cos\theta$（最低点为原点）

速度：$\dot{x} = l\dot{\theta}\cos\theta$, $\dot{y} = l\dot{\theta}\sin\theta$

动能：$T = \frac{1}{2}m(\dot{x}^2 + \dot{y}^2) = \frac{1}{2}ml^2\dot{\theta}^2$

势能：$V = -mgl\cos\theta$（最低点 $V = -mgl$）

拉格朗日函数：$L = T - V = \frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta$

计算：

- $\dfrac{\partial L}{\partial \dot{\theta}} = ml^2\dot{\theta}$ → $\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot{\theta}}\right) = ml^2\ddot{\theta}$
- $\dfrac{\partial L}{\partial \theta} = -mgl\sin\theta$

运动方程：$ml^2\ddot{\theta} + mgl\sin\theta = 0$ → **$\ddot{\theta} + (g/l)\sin\theta = 0$**

小角度（$\sin\theta \approx \theta$）：$\ddot{\theta} + (g/l)\theta = 0$ → $\omega = \sqrt{g/l}$

> 用牛顿法：需要分解重力和张力，列出向心加速度方程，然后小角度近似。拉格朗日法只要写 $L$ 然后求导——没有矢量分解，没有几何推理。

---

### 例题 2：阿特伍德机（滑轮组）

两个质量 $m_1$, $m_2$ 用不可伸长绳通过无摩擦滑轮连接。求加速度。

**选广义坐标**：绳子一端下降的距离 $x$（1 个自由度）。

约束：$x_1 = x$, $x_2 = -x$（绳长不变）

$$T = \frac{1}{2}m_1\dot{x}^2 + \frac{1}{2}m_2(-\dot{x})^2 = \frac{1}{2}(m_1+m_2)\dot{x}^2$$

$$V = -m_1gx - m_2g(-x) = (m_2 - m_1)gx$$（选滑轮位置为势能零点）

$$L = T - V = \frac{1}{2}(m_1+m_2)\dot{x}^2 - (m_2-m_1)gx$$

- $\dfrac{\partial L}{\partial\dot{x}} = (m_1+m_2)\dot{x}$ → $\dfrac{d}{dt} = (m_1+m_2)\ddot{x}$
- $\dfrac{\partial L}{\partial x} = -(m_2-m_1)g = (m_1-m_2)g$

方程：$(m_1+m_2)\ddot{x} = (m_1-m_2)g$ → **$\ddot{x} = (m_1-m_2)g/(m_1+m_2)$**

> 牛二法需要：$T-m_1g=m_1a$, $m_2g-T=m_2a$ → 两个方程联立消 $T$。拉格朗日法：$T$ 根本没出现。

---

### 例题 3：弹簧摆（两自由度）

质量为 $m$ 的质点通过劲度系数 $k$ 的弹簧连接到固定点。弹簧原长 $l_0$。设弹簧始终在竖直平面内运动。

**选广义坐标**：弹簧长度 $r$，摆角 $\theta$（2 个自由度）。

$$x = r\sin\theta,\quad y = -r\cos\theta$$
$$\dot{x} = \dot{r}\sin\theta + r\dot{\theta}\cos\theta,\quad \dot{y} = -\dot{r}\cos\theta + r\dot{\theta}\sin\theta$$

$$T = \frac{1}{2}m(\dot{x}^2+\dot{y}^2) = \frac{1}{2}m(\dot{r}^2 + r^2\dot{\theta}^2)$$
$$V = \frac{1}{2}k(r - l_0)^2 - mgr\cos\theta$$

$$L = \frac{1}{2}m(\dot{r}^2 + r^2\dot{\theta}^2) - \frac{1}{2}k(r-l_0)^2 + mgr\cos\theta$$

**对 $r$ 列方程**：

- $\dfrac{\partial L}{\partial \dot{r}} = m\dot{r}$ → $\dfrac{d}{dt} = m\ddot{r}$
- $\dfrac{\partial L}{\partial r} = mr\dot{\theta}^2 - k(r-l_0) + mg\cos\theta$

$r$-方程：$m\ddot{r} - mr\dot{\theta}^2 + k(r-l_0) - mg\cos\theta = 0$

**对 $\theta$ 列方程**：

- $\dfrac{\partial L}{\partial \dot{\theta}} = mr^2\dot{\theta}$ → $\dfrac{d}{dt} = 2mr\dot{r}\dot{\theta} + mr^2\ddot{\theta}$
- $\dfrac{\partial L}{\partial \theta} = -mgr\sin\theta$

$\theta$-方程：$mr^2\ddot{\theta} + 2mr\dot{r}\dot{\theta} + mgr\sin\theta = 0$

> 有了 $L$，列方程就是机械化操作——求导、代公式、整理。不需要画受力图，不需要分析向心力、科里奥利力——它们自动出现在方程里（注意 $\theta$-方程中的 $2mr\dot{r}\dot{\theta}$ 项，那就是科里奥利力的等效项）。

---

### 例题 4：在旋转圆环上的珠子

光滑圆环以角速度 $\omega$ 绕竖直直径匀速旋转。质量为 $m$ 的珠子穿在环上。求平衡位置和振动频率。

**广义坐标**：珠子在环上的角度 $\theta$（从底部量起，1 个自由度）。

圆环半径 $R$。珠子在环上——这是一个**非定常约束**（环在转）。但约束是完整的（珠子始终在环上），拉格朗日方程处理起来很自然。

珠子位置：$x = R\sin\theta\cos(\omega t)$, $y = R\sin\theta\sin(\omega t)$, $z = -R\cos\theta$（原点在环心）

$$T = \frac{1}{2}m(\dot{x}^2+\dot{y}^2+\dot{z}^2) = \frac{1}{2}mR^2(\dot{\theta}^2 + \omega^2\sin^2\theta)$$
$$V = -mgR\cos\theta$$

$$L = \frac{1}{2}mR^2\dot{\theta}^2 + \frac{1}{2}mR^2\omega^2\sin^2\theta + mgR\cos\theta$$

- $\dfrac{\partial L}{\partial \dot{\theta}} = mR^2\dot{\theta}$ → $\dfrac{d}{dt} = mR^2\ddot{\theta}$
- $\dfrac{\partial L}{\partial \theta} = mR^2\omega^2\sin\theta\cos\theta - mgR\sin\theta$

运动方程：$mR^2\ddot{\theta} - mR^2\omega^2\sin\theta\cos\theta + mgR\sin\theta = 0$

**平衡位置**（$\ddot{\theta}=0$）：$\sin\theta\,(g/R - \omega^2\cos\theta) = 0$

解：

- $\sin\theta = 0$ → $\theta=0$（底部）或 $\theta=\pi$（顶部）
- $\cos\theta = g/(R\omega^2)$ → 当 $\omega^2R > g$ 时存在偏离底部的平衡位置

> **科里奥利力和离心力都藏在 $L$ 里自动冒出来了！** 不需要"在旋转参考系中分析虚拟力"——写出 $L$，求导，完事。

### 例题 5：双摆（两自由度）——完整推导版

> 🎯 这是**考试最常考**的多自由度拉格朗日系统。两个完全相同摆长的双摆在竖直平面内运动，看似简单，但动能里的交叉项 $\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)$ 是很多学生翻车的地方。**下面每一步都完整展示，不留任何跳步。**

---

#### 📐 系统描述与坐标系

**物理场景**：固定点 $O$，长度为 $l$ 的轻绳（或轻杆）连接质点 1（质量 $m$）；质点 1 上又用同样长度为 $l$ 的轻绳连接质点 2（质量 $m$）。两个质点只能在竖直平面（$xOy$ 平面）内运动。

**广义坐标**：
- $\theta_1$：上绳与竖直方向的夹角（向右偏为正）
- $\theta_2$：下绳与竖直方向的夹角（向右偏为正）

自由度 = 2（两个角度完全确定系统状态——约束是两绳长固定和悬挂点固定，全部为完整定常约束）。

**坐标系**：悬挂点 $O$ 为原点，$x$ 轴向右，$y$ 轴**向下**为正（方便写重力势能）。

> ⚠️ 这里的 $y$ 轴向下是为了与前面的双耦合摆例子一致。但原版例题用了 $y$ 轴向上（$y = -l\cos\theta$）。两种都可以——只要重力势能的符号处理正确即可。**考试时保持符号自洽就不会扣分。**

---

#### 🧱 第一步：用广义坐标表达直角坐标

**质点 1**（从 $O$ 开始，绳长 $l$，角度 $\theta_1$）：
$$x_1 = l\sin\theta_1$$
$$y_1 = l(1-\cos\theta_1)$$

**质点 2**（从质点 1 开始，再延伸绳长 $l$，角度 $\theta_2$）：
$$x_2 = x_1 + l\sin\theta_2 = l\sin\theta_1 + l\sin\theta_2$$
$$y_2 = y_1 + l(1-\cos\theta_2) = l(1-\cos\theta_1) + l(1-\cos\theta_2) = l(2-\cos\theta_1-\cos\theta_2)$$

> 📌 注意 $y$ 坐标的写法：$l(1-\cos\theta)$ 表示该质点在悬挂点下方 $l(1-\cos\theta)$ 处。当 $\theta=0$ 时（竖直下垂），$y=0$，所有质点都在同一水平线上等待——因为我们取悬挂点为原点并让 $y$ 向下递增。

---

#### 🧱 第二步：写出动能 $T$

**质点 1 的速度**：
$$\dot{x}_1 = l\cos\theta_1 \cdot \dot{\theta}_1, \quad \dot{y}_1 = l\sin\theta_1 \cdot \dot{\theta}_1$$
$$\dot{x}_1^2 + \dot{y}_1^2 = l^2(\cos^2\theta_1 + \sin^2\theta_1)\dot{\theta}_1^2 = l^2\dot{\theta}_1^2$$

**质点 2 的速度**（关键——这里有交叉项）：
$$\dot{x}_2 = l\cos\theta_1 \dot{\theta}_1 + l\cos\theta_2 \dot{\theta}_2$$
$$\dot{y}_2 = l\sin\theta_1 \dot{\theta}_1 + l\sin\theta_2 \dot{\theta}_2$$

$$\dot{x}_2^2 + \dot{y}_2^2 = (l\cos\theta_1\dot{\theta}_1 + l\cos\theta_2\dot{\theta}_2)^2 + (l\sin\theta_1\dot{\theta}_1 + l\sin\theta_2\dot{\theta}_2)^2$$

展开第一项：
$$l^2\cos^2\theta_1\dot{\theta}_1^2 + 2l^2\cos\theta_1\cos\theta_2\dot{\theta}_1\dot{\theta}_2 + l^2\cos^2\theta_2\dot{\theta}_2^2$$

展开第二项：
$$l^2\sin^2\theta_1\dot{\theta}_1^2 + 2l^2\sin\theta_1\sin\theta_2\dot{\theta}_1\dot{\theta}_2 + l^2\sin^2\theta_2\dot{\theta}_2^2$$

相加：
$$\dot{x}_2^2 + \dot{y}_2^2 = l^2(\cos^2\theta_1+\sin^2\theta_1)\dot{\theta}_1^2 + l^2(\cos^2\theta_2+\sin^2\theta_2)\dot{\theta}_2^2 + 2l^2(\cos\theta_1\cos\theta_2+\sin\theta_1\sin\theta_2)\dot{\theta}_1\dot{\theta}_2$$

利用 $\cos^2\alpha + \sin^2\alpha = 1$ 和 $\cos\alpha\cos\beta+\sin\alpha\sin\beta = \cos(\alpha-\beta)$：
$$\boxed{\dot{x}_2^2 + \dot{y}_2^2 = l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)}$$

**总动能**：
$$T = \frac{1}{2}m(\dot{x}_1^2+\dot{y}_1^2) + \frac{1}{2}m(\dot{x}_2^2+\dot{y}_2^2)$$
$$= \frac{1}{2}m l^2\dot{\theta}_1^2 + \frac{1}{2}m\left[l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]$$

$$\boxed{T = \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]}$$

> 🔑 **关键观察**：交叉项 $\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)$ 来自质点 2 的速度计算中 $\dot{\theta}_1$ 和 $\dot{\theta}_2$ 的"叠乘效应"——质点 2 既有上摆转动的牵连速度，又有自身摆动的相对速度。**这就是为什么双摆的动能矩阵不是对角的——两个广义坐标通过惯性产生了耦合。**

---

#### 🧱 第三步：写出势能 $V$

$y$ 轴向下，取 $y=0$ 为势能零点。重力势能 $V_g = -mgy$（$y$ 增大，势能减小）：
$$V = -mg y_1 - mg y_2$$
$$= -mgl(1-\cos\theta_1) - mgl(2-\cos\theta_1-\cos\theta_2)$$
$$= -mgl(1-\cos\theta_1 + 2 - \cos\theta_1 - \cos\theta_2)$$
$$= -mgl(3 - 2\cos\theta_1 - \cos\theta_2)$$

去掉常数项（$-3mgl$ 不影响运动方程），保留与 $\theta$ 相关的部分：
$$\boxed{V(\theta_1,\theta_2) = -mgl(2\cos\theta_1 + \cos\theta_2)}$$

或者写成去掉常数后更方便的形式：
$$V = mgl[2(1-\cos\theta_1) + (1-\cos\theta_2)]$$

> 📌 两种写法等价。如果要与前面的双耦合摆例题保持一致，可以写为 $V = mgl(2-2\cos\theta_1 + 1-\cos\theta_2)$ 再去掉常数 $3mgl$。

---

#### 🧱 第四步：拉格朗日函数

$$L = T - V = \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right] + mgl(2\cos\theta_1 + \cos\theta_2)$$

> ⚠️ 注意 $-V = +mgl(2\cos\theta_1 + \cos\theta_2)$（因为 $V$ 前面自带负号）

---

#### 🧱 第五步：代入拉格朗日方程得到运动方程

拉格朗日方程：$\displaystyle\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} - \frac{\partial L}{\partial \theta_1} = 0$

**对 $\theta_1$**：

(1) $\displaystyle\frac{\partial L}{\partial \dot{\theta}_1} = ml^2[2\dot{\theta}_1 + \dot{\theta}_2\cos(\theta_1-\theta_2)]$

(2) $\displaystyle\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} = ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2(\dot{\theta}_1-\dot{\theta}_2)\sin(\theta_1-\theta_2)]$

(3) $\displaystyle\frac{\partial L}{\partial \theta_1} = ml^2\dot{\theta}_1\dot{\theta}_2[-\sin(\theta_1-\theta_2)] - 2mgl\sin\theta_1$
$$= -ml^2\dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) - 2mgl\sin\theta_1$$

(4) 方程：
$$ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2(\dot{\theta}_1-\dot{\theta}_2)\sin(\theta_1-\theta_2)] - [-ml^2\dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) - 2mgl\sin\theta_1] = 0$$

化简：
$$ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2\dot{\theta}_1\sin(\theta_1-\theta_2) + \dot{\theta}_2^2\sin(\theta_1-\theta_2) + \dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2)] + 2mgl\sin\theta_1 = 0$$

注意到 $-\dot{\theta}_2\dot{\theta}_1\sin(\theta_1-\theta_2) + \dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) = 0$，两项消去：
$$\boxed{2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) + \dot{\theta}_2^2\sin(\theta_1-\theta_2) + \frac{2g}{l}\sin\theta_1 = 0}$$

**对 $\theta_2$**（类似推导，这里展示结果）：

$$\boxed{\ddot{\theta}_1\cos(\theta_1-\theta_2) + \ddot{\theta}_2 - \dot{\theta}_1^2\sin(\theta_1-\theta_2) + \frac{g}{l}\sin\theta_2 = 0}$$

> 🏋️ 这是两个**强非线性**耦合二阶方程——包含了 $\sin\theta$, $\cos(\theta_1-\theta_2)$, $\dot{\theta}^2\sin(\theta_1-\theta_2)$ 等非线性项。这就是为什么双摆在较大振幅下会出现**混沌**现象！

---

#### 🧱 第六步：小角度线性化

小角度近似：
$$\sin\theta \approx \theta, \quad \cos\theta \approx 1 - \frac{\theta^2}{2} \approx 1$$
$$\cos(\theta_1-\theta_2) \approx 1, \quad \sin(\theta_1-\theta_2) \approx \theta_1 - \theta_2$$

但还需要保留到**二阶**才能得到正确的线性化方程。在动能中，$\cos(\theta_1-\theta_2)$ 那项乘的是 $\dot{\theta}_1\dot{\theta}_2$（已经是二阶小量），所以保留 $\cos \approx 1$ 就够；在势能中，$\cos\theta$ 需要展开到 $\approx 1 - \theta^2/2$。

**线性化的 $L$**（略去 $O(\theta^3)$ 及以上）：
$$T \approx \frac{1}{2}ml^2(2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2)$$
$$V \approx mgl\left[2\left(1 - (1-\frac{\theta_1^2}{2})\right) + \left(1 - (1-\frac{\theta_2^2}{2})\right)\right] = mgl(\theta_1^2 + \frac{1}{2}\theta_2^2)$$

去掉 $V$ 中无关的常数项后：
$$\boxed{L_{\text{线性化}} = \frac{1}{2}ml^2(2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2) - mgl\left(\theta_1^2 + \frac{1}{2}\theta_2^2\right)}$$

写成矩阵二次型形式：
$$L = \frac{1}{2}\dot{\boldsymbol{\theta}}^T \mathbf{M} \dot{\boldsymbol{\theta}} - \frac{1}{2}\boldsymbol{\theta}^T \mathbf{K} \boldsymbol{\theta}$$

其中：
$$\mathbf{M} = ml^2\begin{pmatrix}2 & 1 \\ 1 & 1\end{pmatrix}, \quad \mathbf{K} = mgl\begin{pmatrix}2 & 0 \\ 0 & 1\end{pmatrix}$$

> 🔥 **对比前面例题4双耦合摆**：耦合摆的 $\mathbf{M}$ 是对角阵、$\mathbf{K}$ 有非对角元（势能耦合）；双摆恰好相反——$\mathbf{M}$ 有非对角元（惯性耦合）、$\mathbf{K}$ 是对角阵（势能无耦合）。**这两种耦合是完全等价的——都是耦合小振动，只是耦合的"通道"不同。**

---

#### 🧱 第七步：假设谐振解，求简正频率

设 $\boldsymbol{\theta} = \mathbf{A} e^{i\omega t}$，代入 $\mathbf{M}\ddot{\boldsymbol{\theta}} + \mathbf{K}\boldsymbol{\theta} = 0$：

$$(-\omega^2\mathbf{M} + \mathbf{K})\mathbf{A} = 0$$

有非零解 ⇒ $\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$：

$$\det\begin{pmatrix}2mgl - 2ml^2\omega^2 & -ml^2\omega^2 \\ -ml^2\omega^2 & mgl - ml^2\omega^2\end{pmatrix} = 0$$

令 $\lambda = \omega^2 l / g$（无量纲化）：
$$\det\begin{pmatrix}2(1-\lambda) & -\lambda \\ -\lambda & 1-\lambda\end{pmatrix} = 0$$

展开行列式：
$$2(1-\lambda)^2 - \lambda^2 = 0$$
$$2(1 - 2\lambda + \lambda^2) - \lambda^2 = 0$$
$$2 - 4\lambda + 2\lambda^2 - \lambda^2 = 0$$
$$\lambda^2 - 4\lambda + 2 = 0$$

解得：
$$\lambda = \frac{4 \pm \sqrt{16 - 8}}{2} = 2 \pm \sqrt{2}$$

$$\lambda_1 = 2 - \sqrt{2} \approx 0.586, \quad \lambda_2 = 2 + \sqrt{2} \approx 3.414$$

还原频率：
$$\boxed{\omega_1 = \sqrt{(2-\sqrt{2})\frac{g}{l}} \approx 0.765\sqrt{\frac{g}{l}}}$$
$$\boxed{\omega_2 = \sqrt{(2+\sqrt{2})\frac{g}{l}} \approx 1.848\sqrt{\frac{g}{l}}}$$

---

#### 🧱 第八步：求简正模式（特征向量）

对 $\lambda_1 = 2-\sqrt{2}$，代回本征方程：

第一行：$2(1-\lambda_1)A_1 - \lambda_1 A_2 = 0$
$$A_2 = \frac{2(1-\lambda_1)}{\lambda_1}A_1$$

代入 $\lambda_1 = 2-\sqrt{2}$：
$$1-\lambda_1 = 1-(2-\sqrt{2}) = \sqrt{2}-1$$
$$\frac{2(\sqrt{2}-1)}{2-\sqrt{2}} = \frac{2(\sqrt{2}-1)(2+\sqrt{2})}{(2-\sqrt{2})(2+\sqrt{2})} = \frac{2(2\sqrt{2}+2-2-\sqrt{2})}{2} = \sqrt{2}$$

所以 $A_2 = \sqrt{2}A_1$。**模式1**：两个摆**同向**摆动，下摆振幅是上摆的 $\sqrt{2}$ 倍。频率较低（慢模）。

---

对 $\lambda_2 = 2+\sqrt{2}$：

$A_2 = \dfrac{2(1-\lambda_2)}{\lambda_2}A_1 = \dfrac{2(1-2-\sqrt{2})}{2+\sqrt{2}}A_1 = \dfrac{-2(1+\sqrt{2})}{2+\sqrt{2}}A_1$

有理化：
$$\frac{-2(1+\sqrt{2})(2-\sqrt{2})}{(2+\sqrt{2})(2-\sqrt{2})} = \frac{-2(2-\sqrt{2}+2\sqrt{2}-2)}{2} = -\sqrt{2}$$

所以 $A_2 = -\sqrt{2}A_1$。**模式2**：两个摆**反向**摆动，下摆振幅是上摆的 $\sqrt{2}$ 倍但方向相反。频率较高（快模）。

---

#### 🎨 物理图像总结

| | 模式1（同相模 / 慢模） | 模式2（反相模 / 快模） |
|---|---|---|
| **频率** | $\omega_1 = \sqrt{(2-\sqrt{2})g/l}$ | $\omega_2 = \sqrt{(2+\sqrt{2})g/l}$ |
| **大体形态** | 两摆同向摆动 | 两摆反向摆动 |
| **振幅比** | $A_2 : A_1 = \sqrt{2} : 1$ | $A_2 : A_1 = -\sqrt{2} : 1$ |
| **下摆的相对运动** | 下摆在上摆基础上同向叠加 | 下摆在上摆基础上反向拉回 |
| **恢复力** | 等效恢复力较小（慢） | 等效恢复力较大（快） |

> 📖 **一句话总结**：双摆在小角度下的线性化运动由两种"自然节奏"叠加而成——慢模（同相，$\omega_1 \approx 0.765\sqrt{g/l}$）和快模（反相，$\omega_2 \approx 1.848\sqrt{g/l}$）。任意初始条件下的运动 = 两种模式按初始条件分配的线性叠加。这种"模式分解"的思想就是第五章小振动的核心：**系统的自然运动不是各个质点"各动各的"，而是以一些特定的空间模式集体振动。**

> 🔗 **第5.5节将从这个系统的线性化拉格朗日出发，用矩阵方法重新求解简正频率和简正坐标——展示小振动的标准操作流程。现在在这里先把完整的非线性运动方程和拉格朗日形式掌握好。**

#### 🧱 第一步：用广义坐标表达直角坐标

**质点 1**（从 $O$ 开始，绳长 $l$，角度 $\theta_1$）：

> 📌 注意 $y$ 坐标的写法：$l(1-\cos\theta)$ 表示该质点在悬挂点下方 $l(1-\cos\theta)$ 处。当 $\theta=0$ 时（竖直下垂），$y=0$，所有质点都在同一水平线上等待——因为我们取悬挂点为原点并让 $y$ 向下递增。

**质点 1 的速度**：
$$\dot{x}_1 = l\cos\theta_1 \cdot \dot{\theta}_1, \quad \dot{y}_1 = l\sin\theta_1 \cdot \dot{\theta}_1$$
$$\dot{x}_1^2 + \dot{y}_1^2 = l^2(\cos^2\theta_1 + \sin^2\theta_1)\dot{\theta}_1^2 = l^2\dot{\theta}_1^2$$

**质点 2 的速度**（关键——这里有交叉项）：
$$\dot{x}_2 = l\cos\theta_1 \dot{\theta}_1 + l\cos\theta_2 \dot{\theta}_2$$
$$\dot{y}_2 = l\sin\theta_1 \dot{\theta}_1 + l\sin\theta_2 \dot{\theta}_2$$

$$\dot{x}_2^2 + \dot{y}_2^2 = (l\cos\theta_1\dot{\theta}_1 + l\cos\theta_2\dot{\theta}_2)^2 + (l\sin\theta_1\dot{\theta}_1 + l\sin\theta_2\dot{\theta}_2)^2$$

展开第一项：
$$l^2\cos^2\theta_1\dot{\theta}_1^2 + 2l^2\cos\theta_1\cos\theta_2\dot{\theta}_1\dot{\theta}_2 + l^2\cos^2\theta_2\dot{\theta}_2^2$$

展开第二项：
$$l^2\sin^2\theta_1\dot{\theta}_1^2 + 2l^2\sin\theta_1\sin\theta_2\dot{\theta}_1\dot{\theta}_2 + l^2\sin^2\theta_2\dot{\theta}_2^2$$

相加：
$$\dot{x}_2^2 + \dot{y}_2^2 = l^2(\cos^2\theta_1+\sin^2\theta_1)\dot{\theta}_1^2 + l^2(\cos^2\theta_2+\sin^2\theta_2)\dot{\theta}_2^2 + 2l^2(\cos\theta_1\cos\theta_2+\sin\theta_1\sin\theta_2)\dot{\theta}_1\dot{\theta}_2$$

利用 $\cos^2\alpha + \sin^2\alpha = 1$ 和 $\cos\alpha\cos\beta+\sin\alpha\sin\beta = \cos(\alpha-\beta)$：
$$\boxed{\dot{x}_2^2 + \dot{y}_2^2 = l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)}$$

**总动能**：
$$T = \frac{1}{2}m(\dot{x}_1^2+\dot{y}_1^2) + \frac{1}{2}m(\dot{x}_2^2+\dot{y}_2^2)$$
$$= \frac{1}{2}m l^2\dot{\theta}_1^2 + \frac{1}{2}m\left[l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]$$

$$\boxed{T = \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]}$$

> 🔑 **关键观察**：交叉项 $\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)$ 来自质点 2 的速度计算中 $\dot{\theta}_1$ 和 $\dot{\theta}_2$ 的"叠乘效应"——质点 2 既有上摆转动的牵连速度，又有自身摆动的相对速度。**这就是为什么双摆的动能矩阵不是对角的——两个广义坐标通过惯性产生了耦合。**

---

#### 🧱 第三步：写出势能 $V$

$y$ 轴向下，取 $y=0$ 为势能零点。重力势能 $V_g = -mgy$（$y$ 增大，势能减小）：
$$V = -mg y_1 - mg y_2$$
$$= -mgl(1-\cos\theta_1) - mgl(2-\cos\theta_1-\cos\theta_2)$$
$$= -mgl(1-\cos\theta_1 + 2 - \cos\theta_1 - \cos\theta_2)$$
$$= -mgl(3 - 2\cos\theta_1 - \cos\theta_2)$$

去掉常数项（$-3mgl$ 不影响运动方程），保留与 $\theta$ 相关的部分：
$$\boxed{V(\theta_1,\theta_2) = -mgl(2\cos\theta_1 + \cos\theta_2)}$$

或者写成去掉常数后更方便的形式：
$$V = mgl[2(1-\cos\theta_1) + (1-\cos\theta_2)]$$

> 📌 两种写法等价。如果要与前面的双耦合摆例题保持一致，可以写为 $V = mgl(2-2\cos\theta_1 + 1-\cos\theta_2)$ 再去掉常数 $3mgl$。

---

#### 🧱 第四步：拉格朗日函数

$$L = T - V = \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right] + mgl(2\cos\theta_1 + \cos\theta_2)$$

> ⚠️ 注意 $-V = +mgl(2\cos\theta_1 + \cos\theta_2)$（因为 $V$ 前面自带负号）

---

#### 🧱 第五步：代入拉格朗日方程得到运动方程

拉格朗日方程：$\displaystyle\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} - \frac{\partial L}{\partial \theta_1} = 0$

**对 $\theta_1$**：

(1) $\displaystyle\frac{\partial L}{\partial \dot{\theta}_1} = ml^2[2\dot{\theta}_1 + \dot{\theta}_2\cos(\theta_1-\theta_2)]$

(2) $\displaystyle\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} = ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2(\dot{\theta}_1-\dot{\theta}_2)\sin(\theta_1-\theta_2)]$

(3) $\displaystyle\frac{\partial L}{\partial \theta_1} = ml^2\dot{\theta}_1\dot{\theta}_2[-\sin(\theta_1-\theta_2)] - 2mgl\sin\theta_1$
$$= -ml^2\dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) - 2mgl\sin\theta_1$$

(4) 方程：
$$ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2(\dot{\theta}_1-\dot{\theta}_2)\sin(\theta_1-\theta_2)] - [-ml^2\dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) - 2mgl\sin\theta_1] = 0$$

化简：
$$ml^2[2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) - \dot{\theta}_2\dot{\theta}_1\sin(\theta_1-\theta_2) + \dot{\theta}_2^2\sin(\theta_1-\theta_2) + \dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2)] + 2mgl\sin\theta_1 = 0$$

注意到 $-\dot{\theta}_2\dot{\theta}_1\sin(\theta_1-\theta_2) + \dot{\theta}_1\dot{\theta}_2\sin(\theta_1-\theta_2) = 0$，两项消去：
$$\boxed{2\ddot{\theta}_1 + \ddot{\theta}_2\cos(\theta_1-\theta_2) + \dot{\theta}_2^2\sin(\theta_1-\theta_2) + \frac{2g}{l}\sin\theta_1 = 0}$$

**对 $\theta_2$**（类似推导，这里展示结果）：

$$\boxed{\ddot{\theta}_1\cos(\theta_1-\theta_2) + \ddot{\theta}_2 - \dot{\theta}_1^2\sin(\theta_1-\theta_2) + \frac{g}{l}\sin\theta_2 = 0}$$

> 🏋️ 这是两个**强非线性**耦合二阶方程——包含了 $\sin\theta$, $\cos(\theta_1-\theta_2)$, $\dot{\theta}^2\sin(\theta_1-\theta_2)$ 等非线性项。这就是为什么双摆在较大振幅下会出现**混沌**现象！

---

#### 🧱 第六步：小角度线性化

小角度近似：
$$\sin\theta \approx \theta, \quad \cos\theta \approx 1 - \frac{\theta^2}{2} \approx 1$$
$$\cos(\theta_1-\theta_2) \approx 1, \quad \sin(\theta_1-\theta_2) \approx \theta_1 - \theta_2$$

但还需要保留到**二阶**才能得到正确的线性化方程。在动能中，$\cos(\theta_1-\theta_2)$ 那项乘的是 $\dot{\theta}_1\dot{\theta}_2$（已经是二阶小量），所以保留 $\cos \approx 1$ 就够；在势能中，$\cos\theta$ 需要展开到 $\approx 1 - \theta^2/2$。

**线性化的 $L$**（略去 $O(\theta^3)$ 及以上）：
$$T \approx \frac{1}{2}ml^2(2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2)$$
$$V \approx mgl\left[2\left(1 - (1-\frac{\theta_1^2}{2})\right) + \left(1 - (1-\frac{\theta_2^2}{2})\right)\right] = mgl(\theta_1^2 + \frac{1}{2}\theta_2^2)$$

去掉 $V$ 中无关的常数项后：
$$\boxed{L_{\text{线性化}} = \frac{1}{2}ml^2(2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2) - mgl\left(\theta_1^2 + \frac{1}{2}\theta_2^2\right)}$$

写成矩阵二次型形式：
$$L = \frac{1}{2}\dot{\boldsymbol{\theta}}^T \mathbf{M} \dot{\boldsymbol{\theta}} - \frac{1}{2}\boldsymbol{\theta}^T \mathbf{K} \boldsymbol{\theta}$$

其中：
$$\mathbf{M} = ml^2\begin{pmatrix}2 & 1 \\ 1 & 1\end{pmatrix}, \quad \mathbf{K} = mgl\begin{pmatrix}2 & 0 \\ 0 & 1\end{pmatrix}$$

> 🔥 **对比前面例题4双耦合摆**：耦合摆的 $\mathbf{M}$ 是对角阵、$\mathbf{K}$ 有非对角元（势能耦合）；双摆恰好相反——$\mathbf{M}$ 有非对角元（惯性耦合）、$\mathbf{K}$ 是对角阵（势能无耦合）。**这两种耦合是完全等价的——都是耦合小振动，只是耦合的"通道"不同。**

---

#### 🧱 第七步：假设谐振解，求简正频率

设 $\boldsymbol{\theta} = \mathbf{A} e^{i\omega t}$，代入 $\mathbf{M}\ddot{\boldsymbol{\theta}} + \mathbf{K}\boldsymbol{\theta} = 0$：

$$(-\omega^2\mathbf{M} + \mathbf{K})\mathbf{A} = 0$$

有非零解 ⇒ $\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$：

$$\det\begin{pmatrix}2mgl - 2ml^2\omega^2 & -ml^2\omega^2 \\ -ml^2\omega^2 & mgl - ml^2\omega^2\end{pmatrix} = 0$$

令 $\lambda = \omega^2 l / g$（无量纲化）：
$$\det\begin{pmatrix}2(1-\lambda) & -\lambda \\ -\lambda & 1-\lambda\end{pmatrix} = 0$$

展开行列式：
$$2(1-\lambda)^2 - \lambda^2 = 0$$
$$2(1 - 2\lambda + \lambda^2) - \lambda^2 = 0$$
$$2 - 4\lambda + 2\lambda^2 - \lambda^2 = 0$$
$$\lambda^2 - 4\lambda + 2 = 0$$

解得：
$$\lambda = \frac{4 \pm \sqrt{16 - 8}}{2} = 2 \pm \sqrt{2}$$

$$\lambda_1 = 2 - \sqrt{2} \approx 0.586, \quad \lambda_2 = 2 + \sqrt{2} \approx 3.414$$

还原频率：
$$\boxed{\omega_1 = \sqrt{(2-\sqrt{2})\frac{g}{l}} \approx 0.765\sqrt{\frac{g}{l}}}$$
$$\boxed{\omega_2 = \sqrt{(2+\sqrt{2})\frac{g}{l}} \approx 1.848\sqrt{\frac{g}{l}}}$$

---

#### 🧱 第八步：求简正模式（特征向量）

对 $\lambda_1 = 2-\sqrt{2}$，代回本征方程：

第一行：$2(1-\lambda_1)A_1 - \lambda_1 A_2 = 0$
$$A_2 = \frac{2(1-\lambda_1)}{\lambda_1}A_1$$

代入 $\lambda_1 = 2-\sqrt{2}$：
$$1-\lambda_1 = 1-(2-\sqrt{2}) = \sqrt{2}-1$$
$$\frac{2(\sqrt{2}-1)}{2-\sqrt{2}} = \frac{2(\sqrt{2}-1)(2+\sqrt{2})}{(2-\sqrt{2})(2+\sqrt{2})} = \frac{2(2\sqrt{2}+2-2-\sqrt{2})}{2} = \sqrt{2}$$

所以 $A_2 = \sqrt{2}A_1$。**模式1**：两个摆**同向**摆动，下摆振幅是上摆的 $\sqrt{2}$ 倍。频率较低（慢模）。

---

对 $\lambda_2 = 2+\sqrt{2}$：

$A_2 = \dfrac{2(1-\lambda_2)}{\lambda_2}A_1 = \dfrac{2(1-2-\sqrt{2})}{2+\sqrt{2}}A_1 = \dfrac{-2(1+\sqrt{2})}{2+\sqrt{2}}A_1$

有理化：
$$\frac{-2(1+\sqrt{2})(2-\sqrt{2})}{(2+\sqrt{2})(2-\sqrt{2})} = \frac{-2(2-\sqrt{2}+2\sqrt{2}-2)}{2} = -\sqrt{2}$$

所以 $A_2 = -\sqrt{2}A_1$。**模式2**：两个摆**反向**摆动，下摆振幅是上摆的 $\sqrt{2}$ 倍但方向相反。频率较高（快模）。

---

#### 🎨 物理图像总结

| | 模式1（同相模 / 慢模） | 模式2（反相模 / 快模） |
|---|---|---|
| **频率** | $\omega_1 = \sqrt{(2-\sqrt{2})g/l}$ | $\omega_2 = \sqrt{(2+\sqrt{2})g/l}$ |
| **大体形态** | 两摆同向摆动 | 两摆反向摆动 |
| **振幅比** | $A_2 : A_1 = \sqrt{2} : 1$ | $A_2 : A_1 = -\sqrt{2} : 1$ |
| **下摆的相对运动** | 下摆在上摆基础上同向叠加 | 下摆在上摆基础上反向拉回 |
| **恢复力** | 等效恢复力较小（慢） | 等效恢复力较大（快） |

> 📖 **一句话总结**：双摆在小角度下的线性化运动由两种"自然节奏"叠加而成——慢模（同相，$\omega_1 \approx 0.765\sqrt{g/l}$）和快模（反相，$\omega_2 \approx 1.848\sqrt{g/l}$）。任意初始条件下的运动 = 两种模式按初始条件分配的线性叠加。这种"模式分解"的思想就是第五章小振动的核心：**系统的自然运动不是各个质点"各动各的"，而是以一些特定的空间模式集体振动。**

> 🔗 **第5.5节将从这个系统的线性化拉格朗日出发，用矩阵方法重新求解简正频率和简正坐标——展示小振动的标准操作流程。现在在这里先把完整的非线性运动方程和拉格朗日形式掌握好。**

## 3.9 拉格朗日方程 vs 牛顿定律——为什么拉格朗日更强大

| 方面 | 牛顿力学 | 拉格朗日力学 |
|------|----------|-------------|
| 基本量 | 力（矢量） | 能量（标量） |
| 约束处理 | 逐个分析约束力，再消去 | 选广义坐标，约束自动满足 |
| 坐标系 | 固定在直角坐标 | 任意广义坐标 |
| 对称性 | 不明显 | 循环坐标直接给出守恒量 |
| 复杂系统 | 方程数量爆炸 | 方程数 = 自由度数 |
| 推广到其他领域 | 困难 | 可直接推广到量子力学、场论 |

## 3.10 常见误区

| 误区 | 正确做法 |
|------|----------|
| $L = T+V$ | $L = T-V$！减号来自保守力 $\vec{F} = -\nabla V$ |
| $\dfrac{\partial L}{\partial q}$ 和 $\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot{q}}\right)$ 混淆 | 前者是**偏**导数（$\dot{q}$ 当常数），后者是**全**时间导数 |
| 选了不独立的广义坐标 | 广义坐标必须互相独立——一个自由度一个坐标 |
| 忘写势能零点 | $V$ 的零点任意，不影响运动方程 |
| $T$ 中漏了交叉项 | 如双摆中 $\cos(\theta_1-\theta_2)$ 项——那是耦合的关键 |

# 第四章 有心力场与两体问题

## 4.1 约化质量——两体变单体

两个质点 $m_1$, $m_2$ 在万有引力下互相作用。引入**约化质量** $\mu$：

$$\mu = \frac{m_1 m_2}{m_1 + m_2}$$

相对运动等价于质量为 $\mu$ 的假想质点在固定力心下的运动。

## 4.2 有效势——理解轨道的关键

在有心力场中角动量守恒 $l = \mu r^2\dot{\varphi} =$ 常数。径向运动：

$$L = \frac{1}{2}\mu\dot{r}^2 - \left[V(r) + \frac{l^2}{2\mu r^2}\right]$$

**有效势**：

$$V_{\text{eff}}(r) = V(r) + \frac{l^2}{2\mu r^2}$$

> 离心势垒 $l^2/(2\mu r^2)$ 阻止质点掉入力心。有效势的"坑"决定了轨道类型：能量在坑底部=圆轨道，坑中=椭圆，坑上方=双曲线。

## 4.3 轨道方程

对于万有引力 $V = -k/r$，轨道为圆锥曲线：

$$r = \frac{p}{1 + e\cos\varphi}$$

- $e=0$：圆 | $0<e<1$：椭圆 | $e=1$：抛物线 | $e>1$：双曲线

# 第五章 小振动——简正坐标的美妙世界

> **本章目标**：掌握如何将耦合的多自由度振动解耦为独立简谐振子。这是从"看得见的运动"到"看不见的模式"的思维跃升。

---

## 5.1 为什么要研究小振动？

很多物理系统在稳定平衡点附近做**微小偏离**时，运动退化为简谐振动——因为任何光滑势能在极小值附近展开，第一非零项就是二次项：

$$V(q) \approx V(q_0) + \frac{1}{2}\left.\frac{\partial^2 V}{\partial q^2}\right|_{q_0} (q-q_0)^2$$

这就是"谐振近似"——自然界最常见的近似，从分子振动到晶体晶格到 LC 电路都适用。

---

## 5.2 平衡点的稳定性

**平衡条件**：$\partial V/\partial q_j = 0$（所有广义力的势能部分为零）

**稳定平衡**：势能矩阵 $k_{ij} = \partial^2 V/\partial q_i\partial q_j$ 是**正定**的（所有特征值 $> 0$）

> **直觉**：势能曲面像一个碗。碗底 = 稳定平衡。碗沿（鞍点）= 不稳定平衡。小振动只在碗底附近成立。

---

## 5.3 多自由度小振动的一般理论

### 动能和势能的二次型

在平衡点附近展开，取广义坐标偏离量为 $q_i$（原点=平衡位置）：

**动能**（恒为正定的二次型）：

$$T = \frac{1}{2}\sum_{i,j} m_{ij}\dot{q}_i\dot{q}_j$$

其中 $m_{ij} = m_{ji}$ 是**质量矩阵**的元素，常数（在平衡点取值）。

**势能**（展开到二阶）：

$$V = \frac{1}{2}\sum_{i,j} k_{ij} q_i q_j$$

其中 $k_{ij} = k_{ji} = \partial^2 V/\partial q_i\partial q_j|_{\text{平衡点}}$ 是**劲度矩阵**（刚度矩阵）。

### 运动方程

代入拉格朗日方程：

$$\sum_j (m_{ij}\ddot{q}_j + k_{ij}q_j) = 0$$

矩阵形式：

$$\mathbf{M}\ddot{\mathbf{q}} + \mathbf{K}\mathbf{q} = 0$$

---

## 5.4 简正坐标——解耦的魔法

### 核心思想

$\mathbf{M}\ddot{\mathbf{q}} + \mathbf{K}\mathbf{q} = 0$ 中的自由度是**耦合**的（因为 $\mathbf{M}$ 和 $\mathbf{K}$ 一般不是对角矩阵）。我们要找一组线性变换 $\mathbf{q} = \mathbf{A}\cdot\mathbf{Q}$，使得新坐标 $\mathbf{Q}$ 的方程各自独立——每个 $Q_i$ 满足独立的简谐振动方程。

### 数学方法

假设解为 $\mathbf{q} = \mathbf{A}e^{i\omega t}$（$\mathbf{A}$ 是常向量），代入运动方程：

$$(-\omega^2\mathbf{M} + \mathbf{K})\mathbf{A} = 0$$

有非零解的条件——**特征值方程**：

> $$\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$$

解出 $s$ 个特征值 $\omega_\alpha^2$（$\alpha=1,2,...,s$）——**简正频率**。每个 $\omega_\alpha$ 对应一个特征向量 $\mathbf{A}^{(\alpha)}$——**简正模式**。

### 简正坐标的性质

1. **独立振动**：每个简正坐标做独立的简谐振动 $\ddot{Q}_\alpha + \omega_\alpha^2 Q_\alpha = 0$
2. **能量可加**：总能量 = $\sum$ 各模式能量
3. **同时对角化**：通过适当的线性变换，$\mathbf{M}$ 和 $\mathbf{K}$ 被**同时对角化**——$\mathbf{M}\to$ 单位矩阵，$\mathbf{K}\to$ 对角矩阵（对角元 = $\omega_\alpha^2$）

---

## 5.5 经典例题

### 例题 1：双耦合摆（完整推导版）

> 🎯 **考试极高频**：双耦合摆是小振动一章的"母题"——后面的三原子分子、耦合电路、晶格振动都是它的变形。**把这一道题彻底吃透，本章就掌握了80%。**

---

#### 📐 系统描述与建系

**物理场景**：两个完全相同的单摆（质量均为 $m$，摆长均为 $l$），悬挂在同一水平高度、间距为 $d$ 的两个固定点。两个摆锤之间用一根劲度系数为 $k$ 的轻弹簧水平连接。当两个摆都处于竖直下垂位置时，弹簧恰好处于**原长**（既不拉伸也不压缩）。

**取广义坐标**：两个摆各自与竖直方向的夹角——$\theta_1$ 和 $\theta_2$。

- $\theta_1$：摆1偏离竖直方向的角度（向右偏为正）
- $\theta_2$：摆2偏离竖直方向的角度（向右偏为正）

系统自由度 = 2（两个角度确定了，整个系统的几何状态就完全确定了——约束是两绳长固定和悬点固定，全部是完整定常约束）。

---

#### 🧱 第一步：用广义坐标表达直角坐标（为写动能做准备）

以摆1的悬点为原点，建立坐标系（$x$ 向右为正，$y$ 向下为正）。

**摆1的锤位置**：
$$x_1 = l\sin\theta_1, \quad y_1 = l(1-\cos\theta_1)$$

**摆2的锤位置**（注意：悬点2在悬点1右边距离 $d$ 处）：
$$x_2 = d + l\sin\theta_2, \quad y_2 = l(1-\cos\theta_2)$$

---

#### 🧱 第二步：写出动能 $T$

两个质点各自动能相加：

$$T = \frac{1}{2}m(\dot{x}_1^2 + \dot{y}_1^2) + \frac{1}{2}m(\dot{x}_2^2 + \dot{y}_2^2)$$

对 $x_1 = l\sin\theta_1$ 求导：$\dot{x}_1 = l\cos\theta_1 \cdot \dot{\theta}_1$

对 $y_1 = l(1-\cos\theta_1)$ 求导：$\dot{y}_1 = l\sin\theta_1 \cdot \dot{\theta}_1$

$$\dot{x}_1^2 + \dot{y}_1^2 = l^2\cos^2\theta_1 \dot{\theta}_1^2 + l^2\sin^2\theta_1 \dot{\theta}_1^2 = l^2\dot{\theta}_1^2$$

同理 $\dot{x}_2^2 + \dot{y}_2^2 = l^2\dot{\theta}_2^2$。

> 💡 **关键简化**：这里用了三角恒等式 $\cos^2\theta + \sin^2\theta = 1$，摆的速率恰好就是 $v = l|\dot{\theta}|$——这不奇怪，单摆本来就是沿圆弧运动，速率 = 半径 × 角速率。

因此：
$$\boxed{T = \frac{1}{2}ml^2(\dot{\theta}_1^2 + \dot{\theta}_2^2)}$$

质量矩阵 $\mathbf{M} = ml^2\begin{pmatrix}1 & 0 \\ 0 & 1\end{pmatrix}$——对角阵，因为两个摆的动能没有交叉项（广义坐标的选取使得惯性耦合为零，这是设计好的）。

---

#### 🧱 第三步：写出势能 $V$（分为重力势能 + 弹性势能）

**3.1 重力势能**

以悬点为势能零点（向下为正的 $y$ 坐标，所以重力势能 = $-mgy$）：

$$V_g = -mg y_1 - mg y_2 = -mgl(1-\cos\theta_1) - mgl(1-\cos\theta_2)$$

**小角度近似**（$\cos\theta \approx 1 - \theta^2/2$，常数项无关紧要——势能零点可以任意选）：

$$V_g = -mgl\left(1 - (1 - \frac{\theta_1^2}{2})\right) - mgl\left(1 - (1 - \frac{\theta_2^2}{2})\right)$$

$$V_g = -mgl\frac{\theta_1^2}{2} - mgl\frac{\theta_2^2}{2} + \text{常数}$$

去掉无关的常数（势能零点的平移不影响运动方程），得到：
$$V_g = \frac{1}{2}mgl(\theta_1^2 + \theta_2^2)$$

> ⚠️ **符号检查**：因为 $y$ 向下为正，重力势能在 $y$ 增大时减小，所以表达式里是负号。但经过小角度展开后，去掉常数项后变成正号——这符合"摆偏离平衡位置 → 重力试图拉回 → 等效于弹簧"的物理直觉。

**3.2 弹性势能**

弹簧连接两个摆锤。弹簧的**实际伸长量** = 两个摆锤之间的水平距离 − 原长。

两个摆锤的水平坐标分别为 $x_1 = l\sin\theta_1$ 和 $x_2 = d + l\sin\theta_2$。

两个摆锤水平方向的距离：
$$x_2 - x_1 = (d + l\sin\theta_2) - (l\sin\theta_1) = d + l(\sin\theta_2 - \sin\theta_1)$$

当 $\theta_1 = \theta_2 = 0$ 时，$x_2 - x_1 = d$，这就是弹簧的原长。

实际伸长量：
$$\Delta x = (x_2 - x_1) - d = l(\sin\theta_2 - \sin\theta_1)$$

> 💡 这里我们忽略了弹簧的**垂直**偏移。严格的弹簧长度是 $\sqrt{(\Delta x_{\text{水平}})^2 + (\Delta y)^2}$，但在小角度近似下，$|\Delta y| \sim l \cdot O(\theta^2)$，而水平位移是 $l\theta$ 量级。$\Delta y$ 的贡献是 $O(\theta^4)$ 对势能——在二阶近似中忽略不计。**考试写到"小角度近似下只考虑水平位移"就得分了**。

**小角度近似** $\sin\theta \approx \theta$：
$$\Delta x \approx l(\theta_2 - \theta_1)$$

弹性势能：
$$V_e = \frac{1}{2}k(\Delta x)^2 = \frac{1}{2}kl^2(\theta_2 - \theta_1)^2$$

**3.3 总势能**

$$V = V_g + V_e = \frac{1}{2}mgl(\theta_1^2 + \theta_2^2) + \frac{1}{2}kl^2(\theta_1 - \theta_2)^2$$

展开弹性项（为了方便后续提取刚度矩阵）：
$$V = \frac{1}{2}mgl\theta_1^2 + \frac{1}{2}mgl\theta_2^2 + \frac{1}{2}kl^2(\theta_1^2 - 2\theta_1\theta_2 + \theta_2^2)$$
$$V = \frac{1}{2}(mgl + kl^2)\theta_1^2 + \frac{1}{2}(mgl + kl^2)\theta_2^2 - kl^2\theta_1\theta_2$$

写成矩阵二次型 $V = \frac{1}{2}\boldsymbol{\theta}^T \mathbf{K} \boldsymbol{\theta}$ 的形式：
$$\boxed{V = \frac{1}{2}\begin{pmatrix}\theta_1 & \theta_2\end{pmatrix}\begin{pmatrix}mgl+kl^2 & -kl^2 \\ -kl^2 & mgl+kl^2\end{pmatrix}\begin{pmatrix}\theta_1 \\ \theta_2\end{pmatrix}}$$

刚度矩阵：
$$\mathbf{K} = \begin{pmatrix}mgl+kl^2 & -kl^2 \\ -kl^2 & mgl+kl^2\end{pmatrix}$$

---

#### 🧱 第四步：拉格朗日函数

$$L = T - V = \frac{1}{2}ml^2(\dot{\theta}_1^2 + \dot{\theta}_2^2) - \frac{1}{2}(mgl+kl^2)(\theta_1^2 + \theta_2^2) + kl^2\theta_1\theta_2$$

---

#### 🧱 第五步：代入拉格朗日方程得到运动方程

拉格朗日方程：$\displaystyle\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_i} - \frac{\partial L}{\partial \theta_i} = 0$（$i = 1, 2$）

**对 $\theta_1$**：

$$\frac{\partial L}{\partial \dot{\theta}_1} = ml^2\dot{\theta}_1 \quad \Rightarrow \quad \frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} = ml^2\ddot{\theta}_1$$

$$\frac{\partial L}{\partial \theta_1} = -(mgl+kl^2)\theta_1 + kl^2\theta_2$$

方程：
$$ml^2\ddot{\theta}_1 - [-(mgl+kl^2)\theta_1 + kl^2\theta_2] = 0$$
$$ml^2\ddot{\theta}_1 + (mgl+kl^2)\theta_1 - kl^2\theta_2 = 0$$

**对 $\theta_2$**（对称性）：

$$ml^2\ddot{\theta}_2 + (mgl+kl^2)\theta_2 - kl^2\theta_1 = 0$$

**矩阵形式**：
$$ml^2\begin{pmatrix}1 & 0 \\ 0 & 1\end{pmatrix}\begin{pmatrix}\ddot{\theta}_1 \\ \ddot{\theta}_2\end{pmatrix} + \begin{pmatrix}mgl+kl^2 & -kl^2 \\ -kl^2 & mgl+kl^2\end{pmatrix}\begin{pmatrix}\theta_1 \\ \theta_2\end{pmatrix} = 0$$

即 $\mathbf{M}\ddot{\boldsymbol{\theta}} + \mathbf{K}\boldsymbol{\theta} = 0$。

> 📌 **观察方程结构**：第一项的系数是 $ml^2\ddot{\theta}_1$，第二项的交叉项 $-kl^2\theta_2$ 把 $\theta_2$ 和 $\theta_1$ 的方程勾连在一起——这就是"耦合"的数学体现。如果没有弹簧（$k=0$），两个方程就退化为各自独立：$ml^2\ddot{\theta}_i + mgl\theta_i = 0$，即 $\ddot{\theta}_i + \frac{g}{l}\theta_i = 0$——两个独立的单摆。

---

#### 🧱 第六步：设谐振解，建立本征值方程

假设系统以单一频率 $\omega$ 做简谐振动：
$$\begin{pmatrix}\theta_1(t) \\ \theta_2(t)\end{pmatrix} = \begin{pmatrix}A_1 \\ A_2\end{pmatrix} e^{i\omega t}$$

代入运动方程（$\ddot{\boldsymbol{\theta}} = -\omega^2\boldsymbol{\theta}$）：

$$-\omega^2 ml^2\begin{pmatrix}1 & 0 \\ 0 & 1\end{pmatrix}\begin{pmatrix}A_1 \\ A_2\end{pmatrix} + \begin{pmatrix}mgl+kl^2 & -kl^2 \\ -kl^2 & mgl+kl^2\end{pmatrix}\begin{pmatrix}A_1 \\ A_2\end{pmatrix} = \begin{pmatrix}0 \\ 0\end{pmatrix}$$

整理为：
$$\begin{pmatrix}mgl+kl^2 - ml^2\omega^2 & -kl^2 \\ -kl^2 & mgl+kl^2 - ml^2\omega^2\end{pmatrix}\begin{pmatrix}A_1 \\ A_2\end{pmatrix} = \begin{pmatrix}0 \\ 0\end{pmatrix}$$

这是 $\mathbf{A} \cdot \vec{v} = \vec{0}$ 的形式。有**非零解**（即 $A_1, A_2$ 不全为零）的充要条件是系数矩阵的行列式为零：

$$\det\begin{pmatrix}mgl+kl^2 - ml^2\omega^2 & -kl^2 \\ -kl^2 & mgl+kl^2 - ml^2\omega^2\end{pmatrix} = 0$$

---

#### 🧱 第七步：求解本征值方程（完整代数推导）

令 $\lambda = ml^2\omega^2$ 以简化书写（或者直接展开上面那个行列式）：

行列式 = 主对角线乘积 − 副对角线乘积：
$$(mgl+kl^2 - ml^2\omega^2)^2 - (-kl^2)^2 = 0$$
$$(mgl+kl^2 - ml^2\omega^2)^2 - (kl^2)^2 = 0$$

利用平方差公式 $a^2 - b^2 = (a-b)(a+b)$：
$$(mgl+kl^2 - ml^2\omega^2 - kl^2)(mgl+kl^2 - ml^2\omega^2 + kl^2) = 0$$
$$(mgl - ml^2\omega^2)(mgl + 2kl^2 - ml^2\omega^2) = 0$$

由第一个因子 $=0$：
$$mgl - ml^2\omega^2 = 0 \quad \Rightarrow \quad \omega^2 = \frac{g}{l} \quad \Rightarrow \quad \boxed{\omega_1 = \sqrt{\frac{g}{l}}}$$

由第二个因子 $=0$：
$$mgl + 2kl^2 - ml^2\omega^2 = 0 \quad \Rightarrow \quad \omega^2 = \frac{g}{l} + \frac{2k}{m} \quad \Rightarrow \quad \boxed{\omega_2 = \sqrt{\frac{g}{l} + \frac{2k}{m}}}$$

> 🎉 这就是两个**简正频率**（特征频率 / 本征频率 / normal mode frequency）。

---

#### 🧱 第八步：求特征向量（简正模式 / normal modes）

对每个频率，代回本征方程求振幅比 $A_1 : A_2$。

**模式1：$\omega_1 = \sqrt{g/l}$**（同相模 / in-phase mode）

代入本征方程的第一行：
$$(mgl+kl^2 - ml^2\omega_1^2)A_1 - kl^2 A_2 = 0$$

$\omega_1^2 = g/l$，所以 $ml^2\omega_1^2 = mgl$：
$$(mgl + kl^2 - mgl)A_1 - kl^2 A_2 = 0$$
$$kl^2 A_1 - kl^2 A_2 = 0$$

即 $kl^2(A_1 - A_2) = 0$。由于 $k \neq 0, l \neq 0$：
$$\boxed{A_1 = A_2}$$

**物理解释**：两个摆以**完全相同**的振幅和相位运动——它们同步摆动，就像一个整体。此时弹簧的相对伸长量恒为零（$\theta_1 - \theta_2 = 0$），弹簧完全不参与运动。这就是为什么频率恰好就是自由单摆的频率 $\sqrt{g/l}$——弹簧形同虚设。

**简正模式向量**（归一化表示）：
$$\mathbf{A}^{(1)} = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ 1\end{pmatrix}$$

---

**模式2：$\omega_2 = \sqrt{g/l + 2k/m}$**（反相模 / out-of-phase mode）

代入 $\omega_2^2 = g/l + 2k/m$，则 $ml^2\omega_2^2 = mgl + 2kl^2$：
$$(mgl + kl^2 - (mgl + 2kl^2))A_1 - kl^2 A_2 = 0$$
$$-kl^2 A_1 - kl^2 A_2 = 0$$
$$-kl^2(A_1 + A_2) = 0$$
$$\boxed{A_2 = -A_1}$$

**物理解释**：两个摆以**幅度相同、方向相反**的方式运动——你向左时我向右，你向右时我向左。弹簧不断地被拉伸和压缩，弹簧的恢复力**叠加**在重力的恢复力上，使总恢复力更大 → 频率更高。

**简正模式向量**：
$$\mathbf{A}^{(2)} = \frac{1}{\sqrt{2}}\begin{pmatrix}1 \\ -1\end{pmatrix}$$

---

> 🔑 **核心洞见**：这两个模式是相互**独立**的——系统可以纯以模式1振动，也可以纯以模式2振动，它们之间不会互相转换能量。这就是"简正模式"的本质：系统的"自然运动方式"不是两个摆各自独立摆动，而是两种"集体舞步"。

---

#### 🧱 第九步：通解——任意初始条件的运动

两个简正模式的线性叠加构成**通解**：

$$\begin{pmatrix}\theta_1(t) \\ \theta_2(t)\end{pmatrix} = C_1\begin{pmatrix}1 \\ 1\end{pmatrix}\cos(\omega_1 t + \delta_1) \;+\; C_2\begin{pmatrix}1 \\ -1\end{pmatrix}\cos(\omega_2 t + \delta_2)$$

其中 $C_1, C_2, \delta_1, \delta_2$ 由初始条件（$t=0$ 时 $\theta_1, \theta_2, \dot{\theta}_1, \dot{\theta}_2$ 的值）确定。

展开分量：
$$\theta_1(t) = C_1\cos(\omega_1 t + \delta_1) + C_2\cos(\omega_2 t + \delta_2)$$
$$\theta_2(t) = C_1\cos(\omega_1 t + \delta_1) - C_2\cos(\omega_2 t + \delta_2)$$

---

#### 🧱 第十步：简正坐标——"解耦的魔法"

定义新的坐标：
$$Q_1 = \frac{\theta_1 + \theta_2}{\sqrt{2}}, \quad Q_2 = \frac{\theta_1 - \theta_2}{\sqrt{2}}$$

（你很快会发现，$Q_1$ 代表同相运动——"质心"模式；$Q_2$ 代表反相运动——"相对"模式）

用 $Q_1, Q_2$ 重新表达 $T$ 和 $V$：

$$\theta_1 = \frac{Q_1 + Q_2}{\sqrt{2}}, \quad \theta_2 = \frac{Q_1 - Q_2}{\sqrt{2}}$$

**动能**：
$$T = \frac{1}{2}ml^2(\dot{\theta}_1^2 + \dot{\theta}_2^2) = \frac{1}{2}ml^2\left[\frac{(\dot{Q}_1 + \dot{Q}_2)^2}{2} + \frac{(\dot{Q}_1 - \dot{Q}_2)^2}{2}\right]$$
$$= \frac{1}{2}ml^2\left[\frac{2\dot{Q}_1^2 + 2\dot{Q}_2^2}{2}\right] = \frac{1}{2}ml^2(\dot{Q}_1^2 + \dot{Q}_2^2)$$

**势能**（利用前面 $V$ 的表达式，代入即可——这里展示结果）：
$$V = \frac{1}{2}mgl(\theta_1^2 + \theta_2^2) + \frac{1}{2}kl^2(\theta_1 - \theta_2)^2 = \frac{1}{2}mgl(Q_1^2 + Q_2^2) + \frac{1}{2}kl^2(2Q_2^2)$$
$$= \frac{1}{2}mgl Q_1^2 + \frac{1}{2}(mgl + 2kl^2)Q_2^2$$

于是在新坐标下：
$$L = \frac{1}{2}ml^2\dot{Q}_1^2 - \frac{1}{2}mgl Q_1^2 \;+\; \frac{1}{2}ml^2\dot{Q}_2^2 - \frac{1}{2}(mgl + 2kl^2)Q_2^2$$

拉格朗日方程给出的运动方程**完全解耦**：
$$\ddot{Q}_1 + \frac{g}{l}Q_1 = 0 \quad \Rightarrow \quad Q_1 = A\cos(\omega_1 t + \delta_1)$$
$$\ddot{Q}_2 + \left(\frac{g}{l} + \frac{2k}{m}\right)Q_2 = 0 \quad \Rightarrow \quad Q_2 = B\cos(\omega_2 t + \delta_2)$$

> 🏆 **这是小振动理论最核心的成就**：通过线性变换找到简正坐标，把 $s$ 个耦合的二阶方程变成 $s$ 个独立的简谐振动方程。无论初始条件多么复杂，系统的运动永远是这 $s$ 种独立"节奏"的叠加。

---

#### 🎨 物理图像总结

| | 模式1（同相/对称模） | 模式2（反相/反对称模） |
|---|---|---|
| **频率** | $\omega_1 = \sqrt{g/l}$ | $\omega_2 = \sqrt{g/l + 2k/m}$ |
| **运动形态** | $\theta_1 = \theta_2$，同步摆动 | $\theta_1 = -\theta_2$，相反摆动 |
| **弹簧状态** | 弹簧始终保持原长，不受力 | 弹簧不断地拉伸和压缩 |
| **恢复力来源** | 只有重力 | 重力 + 弹簧力（所以频率更高） |
| **简正坐标** | $Q_1 = (\theta_1+\theta_2)/\sqrt{2}$ | $Q_2 = (\theta_1-\theta_2)/\sqrt{2}$ |
| **极限情况 $k \to 0$** | 退化为单摆 $\sqrt{g/l}$ | 退化为单摆 $\sqrt{g/l}$（两个摆不再相互作用） |
| **极限情况 $k \to \infty$** | 仍是 $\sqrt{g/l}$（同相时弹簧不影响） | $\omega_2 \to \infty$（弹簧变成刚性杆，反相被锁死） |

> 📖 **一句话总结**：耦合摆不是两个独立的摆！系统有两种"集体节奏"——同相（慢，$\sqrt{g/l}$）和反相（快，$\sqrt{g/l + 2k/m}$）。任意初始条件下的运动 = 两种节奏的线性叠加。这就是"模式"的概念——系统的自然运动不是"每个质点各跳各的"，而是"整个系统以一些特定的空间模式集体振动"。把这道题吃透，后面三原子分子、耦合电路、晶格振动都是同理——不过就是矩阵变大了一点而已。

### 例题 2：三原子线性分子（$\text{CO}_2$ 模型）

三个等质量 $m$ 的质点排列在一条直线上，相邻用劲度系数 $k$ 的弹簧连接。两端质点只和中间质点有弹簧连接。

坐标：$x_1$, $x_2$, $x_3$（偏离平衡位置的位移）

$$T = \frac{1}{2}m(\dot{x}_1^2 + \dot{x}_2^2 + \dot{x}_3^2)$$
$$V = \frac{1}{2}k\left[(x_2-x_1)^2 + (x_3-x_2)^2\right]$$

$$\mathbf{M} = m\mathbf{I}_3,\quad
\mathbf{K} = k\begin{pmatrix}1 & -1 & 0\\ -1 & 2 & -1\\ 0 & -1 & 1\end{pmatrix}$$

$\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$ → 三个简正频率：

$$\omega_1 = 0 \quad \text{（平移模——整体平动，没有恢复力）}$$
$$\omega_2 = \sqrt{\frac{k}{m}} \quad \text{（呼吸模——中间不动，两端反向）}$$
$$\omega_3 = \sqrt{\frac{3k}{m}} \quad \text{（反对称伸缩模）}$$

> $\omega_1=0$ 对应整体的平动——系统有平移对称性，没有恢复力。这个"零模"是诺特定理的体现。

## 5.6 简并情况——等频率怎么办？

如果两个不同的模式具有相同的频率，称为**简并**。简并来自系统的对称性。

处理方式：在简并子空间内任意取正交的基向量即可。例如二维各向同性谐振子——$x$ 和 $y$ 方向频率相同，任何方向的直线振动都是简正模式。

# 第六章 刚体运动学与动力学

## 6.1 刚体的自由度

不受约束的刚体：**6 个自由度**——3 平动（质心位置）+ 3 转动（取向）。

## 6.2 转动惯量张量

$$I_{ij} = \int \rho(\vec{r})(r^2\delta_{ij} - x_i x_j) dV$$

转动惯量是**二阶张量**——角动量 $\vec{L} = \mathbf{I}\cdot\vec{\omega}$，$\vec{L}$ 不一定平行于 $\vec{\omega}$！

**惯量主轴**：使 $\mathbf{I}$ 对角化的方向。沿主轴转动时 $\vec{L} \parallel \vec{\omega}$。

## 6.3 欧拉动力学方程（本体主轴系）

$$
\begin{aligned}
I_1\dot{\omega}_1 - (I_2-I_3)\omega_2\omega_3 &= N_1 \\
I_2\dot{\omega}_2 - (I_3-I_1)\omega_3\omega_1 &= N_2 \\
I_3\dot{\omega}_3 - (I_1-I_2)\omega_1\omega_2 &= N_3
\end{aligned}
$$

**每一项的含义**：第一项 $I\dot{\omega}$ = 角加速度效应；第二项 $(I\text{差})\omega\omega$ = 旋转坐标系带来的交叉耦合。

## 6.4 网球拍定理（中间轴定理）

绕**中间大小**的惯量主轴的转动是**不稳定**的。拿起网球拍绕三根轴旋转——绕中间轴的那次会自动翻转。

## 6.5 陀螺的进动

重力力矩使角动量方向旋转（而非大小改变）→ 进动。进动角速度 $\omega_p = mgl/(I_s\omega_s)$。

# 第七章 哈密顿力学——从 $L$ 到 $H$ 的华丽升级

> **本章目标**：理解为什么要从拉格朗日升级到哈密顿——二阶方程降为一阶、坐标与动量平起平坐、相空间、泊松括号。**本章是整本书里最容易被老师出难题的章节，必须逐字逐句吃透。**

---

## 🐣 第0步：先搞懂——"勒让德变换"到底是什么？

### 你已经在生活中用过无数次了

想象你记账：
- **方式A**：记录「买了几个苹果 × 苹果单价 + 买了几斤米 × 米单价」→ 总花费 = $f$(数量, 单价)
- **方式B**：记录「总花费 − 苹果花费」和「苹果单价」→ 换个角度描述同一件事

这两种方式记录的是**同一笔账**，只是**自变量不同**。方式A的自变量是（数量, 单价），方式B的自变量是（单价, 总花费的一部分）。

**这就是勒让德变换**：换个角度描述同一个东西，把一部分旧自变量换成新的自变量。

### 热力学里的勒让德变换（你已经见过了）

| 热力学势 | 自然变量 | 勒让德变换来源 |
|----------|---------|---------------|
| 内能 $U$ | $S, V$ | 原始函数 |
| 亥姆霍兹自由能 $F = U - TS$ | $T, V$ | 把 $S$ 换成 $T$ |
| 焓 $H = U + PV$ | $S, P$ | 把 $V$ 换成 $P$ |
| 吉布斯自由能 $G = U - TS + PV$ | $T, P$ | 把 $S,V$ 换成 $T,P$ |

每次变换，都是用「旧变量的共轭量」去替代「旧变量自己」。
- $U(S,V)$ → $F(T,V)$：因为 $T = \partial U/\partial S$，用 $T$ 替换 $S$
- $U(S,V)$ → $H(S,P)$：因为 $P = -\partial U/\partial V$，用 $P$ 替换 $V$

### 力学里的勒让德变换

拉格朗日函数的自然变量是 $(q, \dot{q})$。
哈密顿函数的自然变量是 $(q, p)$。

怎么从 $\dot{q}$ 换成 $p$？定义 $p = \partial L/\partial \dot{q}$（广义动量），然后：

> $$\boxed{H(q,p,t) = p\cdot\dot{q} - L(q,\dot{q},t)}$$

**关键**：右边的 $\dot{q}$ 必须用 $p$ 和 $q$ 表示——即反解 $p = \partial L/\partial \dot{q}$ 得到 $\dot{q} = \dot{q}(q,p)$，然后**全部代入**把 $H$ 写成只含 $(q,p)$ 的函数。

> 🍎 **生活类比**：$L$ 是靠「位置 + 速度」来描述系统，$H$ 是靠「位置 + 动量」来描述系统。就像你可以用「吃了多少 + 运动量」描述体重变化，也可以用「摄入热量 + 消耗热量」——同一回事，换了一组变量。

## 7.1 为什么要从拉格朗日升级到哈密顿？

拉格朗日方程已经很优雅了——$s$ 个二阶方程。但它有几个"遗憾"：

### 遗憾1：二阶方程不如一阶好解

$s$ 个二阶方程等价于 $2s$ 个一阶方程，但拉格朗日没有把它写成标准的一阶形式。

哈密顿直接给你 $2s$ 个一阶方程——**格式统一，适合计算机数值求解**。

### 遗憾2：坐标和动量不对称

拉格朗日里，$q$ 是主角，$\dot{q}$ 是配角（只是 $q$ 的导数）。但在物理里，位置和动量应该是**平起平坐**的——量子力学的不确定关系 $\Delta x \Delta p \geq \hbar/2$ 就体现了这种平等。

哈密顿力学里，$q$ 和 $p$ 是完全对称的独立变量——这在拉格朗日框架里做不到（因为 $\dot{q}$ 天然是 $q$ 的导数）。

### 遗憾3：对称性不够直观

拉格朗日里，循环坐标 $\partial L/\partial q = 0$ 给出守恒量 $p$。但有些对称性（如时间平移）藏在 $\partial L/\partial t = 0$ 里，不够显眼。

哈密顿力学里，守恒量 = 和 $H$ 的泊松括号为零——一个统一的代数判据。

### 遗憾4：不易推广到量子力学

量子力学里，物理量是算符，运动方程是海森堡方程 $\frac{d\hat{A}}{dt} = \frac{1}{i\hbar}[\hat{A}, \hat{H}]$——这直接对应哈密顿力学里的 $\frac{dF}{dt} = \{F, H\}$。

拉格朗日力学没有这么自然的量子对应。

## 7.2 勒让德变换的数学操作——一步一步走

### 第一步：从 $L(q,\dot{q},t)$ 出发

你已经有了拉格朗日函数 $L$。

### 第二步：定义广义动量

$$p_j = \frac{\partial L}{\partial \dot{q}_j}$$

这是**定义**，没得商量。$p_j$ 叫「和 $q_j$ 共轭的广义动量」。

> ⚠️ **注意**：$p_j$ 不一定是「质量 × 速度」！对于带电粒子在磁场中，$p_j$ 会多出一项。它只是 $\partial L/\partial \dot{q}_j$ 的缩写。

### 第三步：反解出 $\dot{q}_j = \dot{q}_j(q,p,t)$

从 $p = \partial L/\partial \dot{q}$ 这个方程里，把 $\dot{q}$ 解出来，用 $q$ 和 $p$ 表达。

**这步是唯一的难点**——有时候 $\dot{q}$ 不能显式解出（这时需要用到「约束哈密顿系统」的理论，本科一般不考）。

### 第四步：代入构造 $H$

$$H(q,p,t) = \sum_j p_j \dot{q}_j(q,p,t) - L\!\left(q,\;\dot{q}(q,p,t),\;t\right)$$

**关键**：等号右边所有的 $\dot{q}$ 必须用第三步的结果消掉，最终 $H$ 是 $(q,p,t)$ 的函数，不能留任何 $\dot{q}$。

### 第五步：验证

检查 $\partial H/\partial p_j$ 是否等于 $\dot{q}_j$。如果等于——恭喜，做对了。

---

## 7.3 哈密顿正则方程——两条方程统领一切

> $$\boxed{\dot{q}_j = \frac{\partial H}{\partial p_j}, \qquad \dot{p}_j = -\frac{\partial H}{\partial q_j}}$$

### 方程的对称性

| 方面 | $q$ 的方程 | $p$ 的方程 |
|------|-----------|-----------|
| 形式 | $\dot{q} = +\partial H/\partial p$ | $\dot{p} = -\partial H/\partial q$ |
| 物理含义 | 「广义速度 = 能量对动量的变化率」 | 「广义力的负值 = 能量对位置的变化率」 |
| 简单系统 | $\dot{q} = p/m$（恢复 $p=mv$） | $\dot{p} = -\partial V/\partial q = F$（恢复 $F=ma$） |

**合在一起就是 $F=ma$**：
第一条告诉你 $p=mv$，第二条告诉你 $\dot{p}=F$ → $m\ddot{q}=F$。

### 这两条方程是从哪里来的？

拉格朗日方程：
$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\right) - \frac{\partial L}{\partial q} = 0$$

用 $p = \partial L/\partial \dot{q}$ 替换第一项：
$$\dot{p} - \frac{\partial L}{\partial q} = 0 \quad \Rightarrow \quad \dot{p} = \frac{\partial L}{\partial q}$$

现在计算 $\partial H/\partial q$（注意 $H = p\dot{q} - L$）：
$$\frac{\partial H}{\partial q} = p\frac{\partial \dot{q}}{\partial q} - \frac{\partial L}{\partial q} - \frac{\partial L}{\partial \dot{q}}\frac{\partial \dot{q}}{\partial q} = p\frac{\partial \dot{q}}{\partial q} - \frac{\partial L}{\partial q} - p\frac{\partial \dot{q}}{\partial q} = -\frac{\partial L}{\partial q}$$

所以 $\partial L/\partial q = -\partial H/\partial q$。代入拉格朗日方程：
$$\dot{p} = -\frac{\partial H}{\partial q}$$

这就是第二条正则方程！第一条 $\dot{q} = \partial H/\partial p$ 直接来自 $p$ 的定义在勒让德变换下的反函数性质。

## 7.4 哈密顿量的物理含义——什么时候 $H$ 等于能量？

### 一般情况

$$H = \sum p_j \dot{q}_j - L$$

### 如果 $T$ 是 $\dot{q}$ 的纯二次型（大多数普通力学系统）

利用欧拉齐次函数定理：
$$\sum \dot{q}_j\frac{\partial T}{\partial \dot{q}_j} = 2T$$

如果 $L = T - V$，且 $T$ 是 $\dot{q}$ 的纯二次型（没有一次项和常数项）：
$$p_j = \frac{\partial L}{\partial \dot{q}_j} = \frac{\partial T}{\partial \dot{q}_j}$$
$$\sum p_j\dot{q}_j = 2T$$
$$H = 2T - (T - V) = T + V = E$$

> ✅ **结论**：定常约束 + $T$ 是 $\dot{q}$ 的纯二次型 → $H = T+V =$ 总能量，守恒。

### 如果 $T$ 含一次项（$\dot{q}$ 的线性项）——非定常约束系统

$T = T_2 + T_1 + T_0$，则 $\sum \dot{q} \partial T/\partial \dot{q} = 2T_2 + T_1$，$H = T_2 - T_0 + V \neq T + V$。

> ⚠️ **死亡陷阱**：$H$ 守恒不代表总能量守恒！如果斜面在加速、约束在运动，外部在给系统注入/抽取能量，总能量 $E = T+V$ 会变，但 $H$ 仍然守恒（只要 $\partial L/\partial t = 0$）。考试判断题最爱考这个。

---

## 7.5 构造哈密顿量的标准流程（土法炼钢五步走）

> 🛠️ **万能模板**：以下五步适用于任何常规力学系统。

**第一步** — 写出 $L(q, \dot{q}, t) = T - V$

**第二步** — 计算广义动量：$p_j = \partial L/\partial \dot{q}_j$

**第三步** — 反解：从 $p_j$ 的表达式反解出 $\dot{q}_j = \dot{q}_j(q,p)$（如果可能的话）

**第四步** — 代入：$H = \sum p_j \dot{q}_j(q,p) - L(q, \dot{q}(q,p), t)$，化简使 $H$ 只含 $q, p, t$

**第五步** — 代正则方程：$\dot{q}_j = \partial H/\partial p_j$，$\dot{p}_j = -\partial H/\partial q_j$

## 7.6 例题暴击——从新手到高手

### 🟢 例题1：一维谐振子（零级难度）

$$L = \frac{1}{2}m\dot{x}^2 - \frac{1}{2}kx^2$$

**第二步**：$p = \partial L/\partial \dot{x} = m\dot{x}$

**第三步**：$\dot{x} = p/m$

**第四步**：
$$H = p\cdot\frac{p}{m} - \left[\frac{1}{2}m\left(\frac{p}{m}\right)^2 - \frac{1}{2}kx^2\right] = \frac{p^2}{2m} + \frac{1}{2}kx^2$$

**第五步**：
$$\dot{x} = \frac{\partial H}{\partial p} = \frac{p}{m}$$
$$\dot{p} = -\frac{\partial H}{\partial x} = -kx$$

结合得：$m\ddot{x} = -kx$ → $\ddot{x} + \omega^2 x = 0$（$\omega = \sqrt{k/m}$）。✅

---

### 🟢 例题2：单摆

$$L = \frac{1}{2}ml^2\dot{\theta}^2 + mgl\cos\theta$$

**第二步**：$p_\theta = \partial L/\partial\dot{\theta} = ml^2\dot{\theta}$

**第三步**：$\dot{\theta} = p_\theta/(ml^2)$

**第四步**：
$$H = p_\theta\cdot\frac{p_\theta}{ml^2} - \left[\frac{1}{2}ml^2\left(\frac{p_\theta}{ml^2}\right)^2 + mgl\cos\theta\right] = \frac{p_\theta^2}{2ml^2} - mgl\cos\theta$$

**第五步**：
$$\dot{\theta} = \frac{\partial H}{\partial p_\theta} = \frac{p_\theta}{ml^2}$$
$$\dot{p}_\theta = -\frac{\partial H}{\partial\theta} = -mgl\sin\theta$$

结合：$ml^2\ddot{\theta} + mgl\sin\theta = 0$ → $\ddot{\theta} + (g/l)\sin\theta = 0$。✅

> 📌 **注意**：$H$ 里势能项是 $-mgl\cos\theta$，而 $V = -mgl\cos\theta$（取最低点为零势能时 $V = -mgl\cos\theta$）。所以 $H = T + V$，总能量守恒。

---

### 🟡 例题3：中心力场中的质点（极坐标）

质点 $m$ 在势场 $V(r)$ 中做平面运动（如行星绕太阳）。

$$L = \frac{1}{2}m(\dot{r}^2 + r^2\dot{\phi}^2) - V(r)$$

**两个自由度：$r$ 和 $\phi$**

**第二步**：
$$p_r = \frac{\partial L}{\partial \dot{r}} = m\dot{r}$$
$$p_\phi = \frac{\partial L}{\partial \dot{\phi}} = mr^2\dot{\phi}$$

**$p_\phi$ 的物理意义**：角动量 $L_z = mr^2\dot{\phi}$。所以在中心力场里，$p_\phi$ 就是角动量！

**第三步**：
$$\dot{r} = p_r/m$$
$$\dot{\phi} = p_\phi/(mr^2)$$

**第四步**：
$$H = p_r\dot{r} + p_\phi\dot{\phi} - L$$
$$= p_r\cdot\frac{p_r}{m} + p_\phi\cdot\frac{p_\phi}{mr^2} - \left[\frac{1}{2}m\left(\frac{p_r^2}{m^2} + r^2\frac{p_\phi^2}{m^2r^4}\right) - V(r)\right]$$
$$= \frac{p_r^2}{2m} + \frac{p_\phi^2}{2mr^2} + V(r)$$

**第五步**——正则方程：
$$\dot{r} = \frac{\partial H}{\partial p_r} = \frac{p_r}{m}$$
$$\dot{\phi} = \frac{\partial H}{\partial p_\phi} = \frac{p_\phi}{mr^2}$$
$$\dot{p}_r = -\frac{\partial H}{\partial r} = \frac{p_\phi^2}{mr^3} - \frac{dV}{dr}$$
$$\dot{p}_\phi = -\frac{\partial H}{\partial \phi} = 0$$

> 🔑 **重要发现**：$\dot{p}_\phi = 0$ → $p_\phi =$ 常数 → 角动量守恒！而且因为 $\phi$ 不显含在 $H$ 里（$\partial H/\partial \phi = 0$），$\phi$ 是**循环坐标**。
>
> 在哈密顿力学里，循环坐标的判据更加简洁：**只要 $\partial H/\partial q_j = 0$，则 $\dot{p}_j = 0$，$p_j$ 守恒**。

---

### 🟡 例题4：带电粒子在电磁场中（挑战题）

质量为 $m$、电荷为 $q$ 的粒子在电磁场中运动。拉格朗日函数为：
$$L = \frac{1}{2}m\dot{\vec{r}}^2 + q\dot{\vec{r}}\cdot\vec{A}(\vec{r},t) - q\varphi(\vec{r},t)$$

其中 $\vec{A}$ 是磁矢势，$\varphi$ 是电势。

**广义动量（注意不是 $m\dot{\vec{r}}$！）**：
$$\vec{p} = \frac{\partial L}{\partial \dot{\vec{r}}} = m\dot{\vec{r}} + q\vec{A}$$

> ⚠️ **死亡陷阱**：这里的「广义动量」$\vec{p}$ 不等于「机械动量」$m\dot{\vec{r}}$！多了一项 $q\vec{A}$。这在量子力学里极其重要——规范场的最小耦合原理就来源于此。

**机械动量**：$m\dot{\vec{r}} = \vec{p} - q\vec{A}$

**哈密顿量**：
$$H = \vec{p}\cdot\dot{\vec{r}} - L$$
$$= \vec{p}\cdot\frac{\vec{p}-q\vec{A}}{m} - \left[\frac{1}{2}m\left(\frac{\vec{p}-q\vec{A}}{m}\right)^2 + q\frac{\vec{p}-q\vec{A}}{m}\cdot\vec{A} - q\varphi\right]$$
$$= \frac{1}{2m}(\vec{p} - q\vec{A})^2 + q\varphi$$

> 📌 **这就是量子力学里的哈密顿算符的经典原型**！$\hat{H} = \frac{1}{2m}(-i\hbar\nabla - q\vec{A})^2 + q\varphi$。

---

### 🔴 例题5：双摆的哈密顿量（高手题）

回顾双摆（第五章例题5）：
$$L = \frac{1}{2}ml^2[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)] + mgl(2\cos\theta_1 + \cos\theta_2)$$

**第二步**：
$$p_1 = \frac{\partial L}{\partial \dot{\theta}_1} = ml^2[2\dot{\theta}_1 + \dot{\theta}_2\cos(\theta_1-\theta_2)]$$
$$p_2 = \frac{\partial L}{\partial \dot{\theta}_2} = ml^2[\dot{\theta}_2 + \dot{\theta}_1\cos(\theta_1-\theta_2)]$$

**第三步（反解，这是最恶心的部分）**：
写成矩阵形式：
$$\begin{pmatrix}p_1\\ p_2\end{pmatrix} = ml^2 \begin{pmatrix}2 & \cos\Delta\\ \cos\Delta & 1\end{pmatrix} \begin{pmatrix}\dot{\theta}_1\\ \dot{\theta}_2\end{pmatrix}$$
其中 $\Delta = \theta_1 - \theta_2$。

求逆矩阵：
$$\begin{pmatrix}\dot{\theta}_1\\ \dot{\theta}_2\end{pmatrix} = \frac{1}{ml^2(2-\cos^2\Delta)} \begin{pmatrix}1 & -\cos\Delta\\ -\cos\Delta & 2\end{pmatrix} \begin{pmatrix}p_1\\ p_2\end{pmatrix}$$

**第四步**：
$$H = p_1\dot{\theta}_1 + p_2\dot{\theta}_2 - L$$
经过化简（利用动能是 $\dot{\theta}$ 的纯二次型）：
$$H = T + V = \frac{p_1^2 + 2p_2^2 - 2p_1p_2\cos(\theta_1-\theta_2)}{2ml^2[2-\cos^2(\theta_1-\theta_2)]} - mgl(2\cos\theta_1 + \cos\theta_2)$$

> 🔑 **重点**：$H = T+V$ 是因为 $T$ 是 $\dot{\theta}$ 的纯二次型，约束定常。但 $H$ 的表达式极其复杂——这说明为什么有些系统在哈密顿形式下也不一定简单。但好处是：现在方程是一阶的，可以数值求解。

## 7.7 相空间——经典力学的"地图"

### 什么是相空间？

> 对于 $s$ 个自由度的系统，**相空间**是由 $s$ 个广义坐标 $q_j$ 和 $s$ 个广义动量 $p_j$ 共同张成的 $2s$ 维空间。

- 系统在某一时刻的**状态** = 相空间中的一个**点** $(q_1, ..., q_s, p_1, ..., p_s)$
- 系统随时间的**演化** = 点在相空间中画出的一条**曲线**（相轨迹）

### 为什么相空间如此重要？

在牛顿力学里，说「系统的状态」你需要知道所有质点的位置和速度。
在哈密顿力学里，说「系统的状态」就是一个点 $(q, p)$——简洁统一。

> 🍎 **类比**：普通空间像一张地图告诉你位置，相空间像一张"状态地图"——同时告诉你位置和动量。给定一个初始点，整个未来就由哈密顿方程唯一确定。

### 相轨迹的核心性质

**性质1：相轨迹永不相交**（对于不显含 $t$ 的 $H$）
- 如果两条相轨迹相交于某点，从那点出发系统有两个不同的未来——矛盾！
- 所以相轨迹要么是闭合曲线（周期运动），要么永不相交。

**性质2：经过相空间每个点的轨迹只有一条**
- 哈密顿方程是一阶的，给定初始条件解唯一（只要 $H$ 足够光滑）。

## 7.8 相图详解——用眼睛看懂运动

### 🟢 一维谐振子的相图

$$H = \frac{p^2}{2m} + \frac{1}{2}m\omega^2 x^2 = E$$

改写为：
$$\frac{p^2}{2mE} + \frac{x^2}{2E/(m\omega^2)} = 1$$

**这是椭圆！** 每个 $E$ 对应一个椭圆。

| 能量 $E$ | 相轨迹 |
|---------|--------|
| 小 $E$ | 小椭圆（振幅小） |
| 大 $E$ | 大椭圆（振幅大） |
| $E=0$ | 退化为原点（停在平衡点） |

- 相轨迹是闭合曲线 → 运动是**周期**的
- 椭圆的长轴方向取决于 $m$ 和 $\omega$
- 系统沿椭圆**顺时针**运动（第一象限 $x>0, p>0$ → $x$ 增大，$p$ 减小）

> 📌 **关键**：谐振子的相轨迹是同心椭圆簇——这就是为什么谐振子如此特殊：所有轨道都是周期的，且周期与振幅无关。

---

### 🟡 单摆的相图（重点！）

$$H = \frac{p_\theta^2}{2ml^2} - mgl\cos\theta$$

能量守恒：$\frac{p_\theta^2}{2ml^2} - mgl\cos\theta = E$

**解出**：$p_\theta = \pm \sqrt{2ml^2(E + mgl\cos\theta)}$

#### 不同类型的轨迹

| 能量范围 | 相轨迹形状 | 物理运动 |
|---------|-----------|---------|
| $E = -mgl$ | 一个点 $(0,0)$ 和 $(\pm\pi,0)$ | 最低点（稳定平衡）和最高点（不稳定平衡） |
| $-mgl < E < mgl$ | 绕 $(0,0)$ 的闭合曲线 | 来回摆动（振动） |
| $E = mgl$ | 连接 $(-\pi,0)$ 和 $(\pi,0)$ 的曲线 | **分离线（separatrix）** |
| $E > mgl$ | 不闭合的波浪形曲线 | 单向旋转（转圈） |

#### 🔥 分离线详解（考试概念题最爱）

当 $E = mgl$ 时，摆刚好有足够的能量到达最高点 $\theta = \pi$（但到达不了更高）。

- 分离线连接了两个**不稳定平衡点**（$\theta=-\pi$ 和 $\theta=\pi$，其实是同一个物理点）
- 沿分离线运动的时间是**无穷大**——摆无限接近最高点但永远到不了
- 分离线是一道"分水岭"：线内是振动，线外是旋转
- **分离线的存在是单摆这种非线性系统的特征**——谐振子没有分离线

> 🎨 **视觉描述**：想象二维相图（横轴 $\theta$ 从 $-\pi$ 到 $\pi$，纵轴 $p$）。
> - 中心 $(\theta=0, p=0)$ 有一族闭合的"蛋形"曲线——那是振动区
> - 蛋形外面，在 $p$ 较大/较小处，曲线变成不闭合的波浪线——那是旋转区
> - 蛋形最外层的"蛋壳"恰好碰到 $\theta=\pm\pi, p=0$ 这两个点——那就是分离线

## 7.9 刘维尔定理——相体积永恒

### 定理陈述

> 在哈密顿演化下，相空间的体积元 $d\Gamma = dq_1...dq_s dp_1...dp_s$ 保持不变。

$$\frac{d}{dt}(d\Gamma) = 0$$

或者等价地说：相空间中一群代表点的"密度"在随体运动中不变。

### 直观理解

把一群初始条件接近的系统在相空间中用一个小区域包围起来。随着时间演化，这个区域会**变形**（拉长、扭曲），但它的**体积**不变——就像揉面团：形状变，但面团总量不变。

### 为什么成立？

哈密顿方程的"相流"是**无散度**的：
$$\nabla\cdot\vec{v}_{\text{相}} = \sum_j\left(\frac{\partial \dot{q}_j}{\partial q_j} + \frac{\partial \dot{p}_j}{\partial p_j}\right) = \sum_j\left(\frac{\partial^2 H}{\partial q_j\partial p_j} - \frac{\partial^2 H}{\partial p_j\partial q_j}\right) = 0$$

相空间流体的"速度"散度为零 → 流体不可压缩 → 相体积守恒。

### 深远影响

- **统计力学的基础**：如果相体积不守恒，等概率假设就没有意义——概率密度会自发聚集或稀释。刘维尔定理保证了微正则系综的合理性。
- **没有"绝对吸引子"**：哈密顿系统不可能有所有轨迹都汇集到一点的"吸引子"——因为相体积必须守恒。所有"吸引"现象都来自耗散。
- **时间箭头**：微观物理规律是时间可逆的（哈密顿方程里把 $t$ 换成 $-t$，同时 $p \to -p$，方程不变）。宏观不可逆性来自初始条件+大量自由度。

## 7.10 泊松括号——哈密顿力学的"代数学"

### 定义

> $$\boxed{\{F, G\} = \sum_{j=1}^s \left(\frac{\partial F}{\partial q_j}\frac{\partial G}{\partial p_j} - \frac{\partial F}{\partial p_j}\frac{\partial G}{\partial q_j}\right)}$$

其中 $F(q,p,t)$ 和 $G(q,p,t)$ 是相空间上的任意两个函数。

### 人话版解释

泊松括号 $\{F, G\}$ 衡量「$F$ 在 $G$ 生成的变换下的变化率」，或者反过来理解也行——它是经典力学版的"对易子"。

### 简单例题：计算 $\{x, p_x\}$

一维情况，$F = x$，$G = p_x$：
$$\{x, p_x\} = \frac{\partial x}{\partial x}\frac{\partial p_x}{\partial p_x} - \frac{\partial x}{\partial p_x}\frac{\partial p_x}{\partial x} = 1\cdot 1 - 0\cdot 0 = 1$$

> 🔑 **这就是海森堡对易关系 $[\hat{x}, \hat{p}] = i\hbar$ 的经典对应！** 把 $\{,\} \to [,]/(i\hbar)$，经典 $\{x,p\}=1$ → 量子 $[\hat{x},\hat{p}] = i\hbar$。

### 基本泊松括号

$$\{q_i, q_j\} = 0$$
$$\{p_i, p_j\} = 0$$
$$\{q_i, p_j\} = \delta_{ij}$$

**人话**：坐标之间可交换，动量之间可交换，坐标和「自己对应的」动量不交换（=1），和「不对应的」动量可交换（=0）。

---

### 物理量 $F$ 的时间演化

> $$\boxed{\frac{dF}{dt} = \{F, H\} + \frac{\partial F}{\partial t}}$$

**推导**：
$$\frac{dF}{dt} = \sum_j\left(\frac{\partial F}{\partial q_j}\dot{q}_j + \frac{\partial F}{\partial p_j}\dot{p}_j\right) + \frac{\partial F}{\partial t}$$
代入正则方程 $\dot{q}_j = \partial H/\partial p_j$，$\dot{p}_j = -\partial H/\partial q_j$：
$$= \sum_j\left(\frac{\partial F}{\partial q_j}\frac{\partial H}{\partial p_j} - \frac{\partial F}{\partial p_j}\frac{\partial H}{\partial q_j}\right) + \frac{\partial F}{\partial t} = \{F, H\} + \frac{\partial F}{\partial t}$$

### 守恒量的泊松括号判据

> ✅ **$F$ 是守恒量** $\iff \{F, H\} = 0$ 且 $\partial F/\partial t = 0$

**检查能量守恒**：$F = H$，
$$\frac{dH}{dt} = \{H, H\} + \frac{\partial H}{\partial t} = 0 + \frac{\partial H}{\partial t}$$
所以 $\partial H/\partial t = 0$ → 能量守恒。与第三章的结论一致！

**检查动量守恒**：$F = p_x$，
$$\frac{dp_x}{dt} = \{p_x, H\} = -\frac{\partial H}{\partial x}$$
所以 $\partial H/\partial x = 0$（$H$ 不显含 $x$）→ $x$ 方向动量守恒。$x$ 是循环坐标！

**检查角动量守恒**：$F = L_z = x p_y - y p_x$，
$$\frac{dL_z}{dt} = \{L_z, H\}$$
如果 $\{L_z, H\} = 0$ → 角动量守恒。

---

### 角动量的泊松括号——通向量子力学的大门

角动量分量（定义 $L_x = y p_z - z p_y$，$L_y = z p_x - x p_z$，$L_z = x p_y - y p_x$）：

> $$\boxed{\{L_x, L_y\} = L_z}$$
> $$\boxed{\{L_y, L_z\} = L_x}$$
> $$\boxed{\{L_z, L_x\} = L_y}$$

**推导 $\{L_x, L_y\}$（考试可能考推导过程）**：

$L_x = y p_z - z p_y$，$L_y = z p_x - x p_z$

逐个计算泊松括号中非零的贡献：
$$\{y p_z, z p_x\} = y\{p_z, z\}p_x = y(-\delta_{zz})p_x = -y p_x$$
$$\{y p_z, -x p_z\} = -\{y, x\}p_z p_z = 0$$
$$\{-z p_y, z p_x\} = -\{z, z\}p_y p_x = 0$$
$$\{-z p_y, -x p_z\} = z p_y\{z, p_z\}x = z p_y(\delta_{zz})x = x p_y$$

$$\{L_x, L_y\} = -y p_x + x p_y = x p_y - y p_x = L_z$$ ✅

> 🔑 **量子力学对应**：海森堡直接把泊松括号替换为对易子除以 $i\hbar$：
> $$[\hat{L}_x, \hat{L}_y] = i\hbar \hat{L}_z$$
> $$[\hat{L}_y, \hat{L}_z] = i\hbar \hat{L}_x$$
> $$[\hat{L}_z, \hat{L}_x] = i\hbar \hat{L}_y$$
>
> 经典的泊松括号代数是量子力学的对易关系代数的「经典极限」。你在经典力学里学的泊松括号，到量子力学里直接变成海森堡方程和算符对易关系——一字不改地平移。

---

### 泊松括号的代数性质（背下来！）

| 性质 | 公式 |
|------|------|
| 反对称性 | $\{F, G\} = -\{G, F\}$ |
| 线性 | $\{aF+bG, K\} = a\{F,K\} + b\{G,K\}$ |
| 莱布尼茨法则 | $\{FG, K\} = F\{G,K\} + \{F,K\}G$ |
| 雅可比恒等式 | $\{F,\{G,K\}\} + \{G,\{K,F\}\} + \{K,\{F,G\}\} = 0$ |

> 📌 雅可比恒等式保证了泊松括号定义的代数结构是**李代数**。这三个性质（反对称+双线性+雅可比）恰好是李代数的定义——角动量算符满足的正是 $\mathfrak{so}(3)$ 李代数。

## 7.11 哈密顿力学里的对称性与守恒量

> 🎯 **诺特定理的哈密顿版**：每一个连续对称性对应一个守恒量，守恒量就是生成该对称变换的"母函数"的泊松括号判据。

### 时间平移对称 → 能量守恒

如果 $H$ 不显含 $t$（$\partial H/\partial t = 0$），则 $\{H, H\} = 0$ → $dH/dt = 0$ → $H$ 守恒。

### 空间平移对称 → 动量守恒

如果 $H$ 不显含 $x$（$\partial H/\partial x = 0$），则 $\{p_x, H\} = -\partial H/\partial x = 0$ → $p_x$ 守恒。

### 旋转对称 → 角动量守恒

如果 $H$ 在绕某轴旋转下不变，则 $\{L_{\text{轴}}, H\} = 0$ → 该方向的角动量守恒。

> 🏆 **统一表述**：守恒量 $G$ ⟺ $\{G, H\} = 0$ 且 $\partial G/\partial t = 0$。这是哈密顿力学里判断守恒量的万能判据——不需要分析力的性质，只要算泊松括号。

> **本章目标**：理解哈密顿原理和最小作用量原理——分析力学不仅是一套计算方法，更是一种**世界观**。大自然选择"作用量极值"的路径。

## 8.1 变分法的数学基础

### 泛函——函数的函数

普通函数：数 → 数（如 $f(3) = 9$）

**泛函**：函数 → 数（如曲线 $y(x)$ 的长度 $= \int\sqrt{1+y'^2}\,dx$——输入一条曲线，输出一个数）

### 变分 $\delta$ vs 微分 $d$

| | 微分 $d$ | 变分 $\delta$ |
|---|---|---|
| 对象 | 函数 $f(x)$ 在某点的变化 | 泛函 $S[y]$ 随函数 $y(x)$ 的变化 |
| 操作 | $\Delta x\to 0$ | 比较两条不同曲线 $y(x)$ 和 $y(x)+\delta\eta(x)$ |
| 记号 | $df = f'(x)dx$ | $\delta S = S[y+\delta\eta] - S[y]$ |

### 欧拉方程的一般推导

考虑泛函 $S[y] = \int_a^b F(x, y, y')\, dx$。对 $y(x)$ 做微小变化 $y(x)\to y(x)+\delta\eta(x)$（端点不变 $\eta(a)=\eta(b)=0$）：

$$\delta S = \int_a^b \left(\frac{\partial F}{\partial y}\eta + \frac{\partial F}{\partial y'}\eta'\right)dx$$

第二项分部积分，利用 $\eta(a)=\eta(b)=0$：

$$= \int_a^b \left[\frac{\partial F}{\partial y} - \frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right)\right]\eta(x)\,dx$$

要求 $\delta S = 0$ 对任意 $\eta(x)$ 成立 → 括号内必须恒为零：

> $$\frac{d}{dx}\left(\frac{\partial F}{\partial y'}\right) - \frac{\partial F}{\partial y} = 0$$

这就是**欧拉-拉格朗日方程**——变分法的核心结果。

### 推广

- **多元函数**（$F$ 含 $y_1, y_2, ...$）：每个 $y_i$ 各有一个欧拉方程
- **高阶导数**（$F$ 含 $y, y', y'', ...$）：欧拉方程含高阶导数项
- **多变量函数**（$F$ 含 $y, \partial y/\partial x_1, \partial y/\partial x_2, ...$）：导出偏微分方程——这就是场论的基础

---

## 8.2 哈密顿原理——分析力学的最高原理

### 内容

**在所有满足约束、连接固定初态和末态的可能路径中，真实运动的路径使作用量 $S$ 取极值（通常为极小值）。**

$$\delta S = \delta \int_{t_1}^{t_2} L(q,\dot{q},t)\,dt = 0$$

边界条件：$\delta q(t_1) = \delta q(t_2) = 0$（初始和末了位置固定）。

### 从哈密顿原理到拉格朗日方程

把 $F = L(q,\dot{q},t)$，$x = t$，$y = q$ 代入欧拉方程：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\right) - \frac{\partial L}{\partial q} = 0$$

**直接得到拉格朗日方程！**

> 这就是拉格朗日方程的"终极版本"——它来自一个更基本的原理（哈密顿原理），而不是从牛顿定律推导出来的。从逻辑上讲，哈密顿原理是第一性的，拉格朗日方程是它的推论。

---

## 8.3 哈密顿原理的物理深意

### 大自然"选择"什么？

在所有可能的运动路径中，大自然选择了使**作用量** $S = \int L\,dt$ 取极值的那条。

不是"力决定了运动"——而是"作用量极值条件决定了运动"。

### 类比：光学的费马原理

费马原理：光总是沿**时间最短**的路径传播。

哈密顿原理：质点总是沿**作用量最小**的路径运动。

光的折射定律 ← 费马原理
质点的运动方程 ← 哈密顿原理

> 德布罗意就是从这里获得灵感，提出"物质波"的概念——光有波粒二象性，如果物质粒子也满足类似的"变分原理"，那物质粒子也应该有波动性！$\lambda = h/p$ 就是这么来的。

## 8.4 最小作用量原理（莫培督原理）——哈密顿原理的特例

对于**能量守恒**的系统，可以把时间积掉：

$$\Delta \int_{q_1}^{q_2} p\,dq = 0$$

或者更一般地：

$$\Delta \int \sum p_j\,dq_j = 0$$

与哈密顿原理的区别：哈密顿原理变分时固定**时间端点**（$t_1$ 和 $t_2$ 固定），莫培督原理固定**能量**（能量为常数，但时间可以变化）。

- **量子力学的薛定谔方程**（费曼路径积分）
- **粒子物理的标准模型**（规范场的拉格朗日量）

> 变分原理是理论物理的"通用语言"。一旦你为一个物理系统写出了拉格朗日函数（或拉格朗日密度），剩下的"推导运动方程"就是机械化的变分操作。这就是为什么理解变分原理如此重要——它是从经典力学通往现代物理的大门。

## 8.6 例题：用哈密顿原理导最速降线

问题：质点从 $A$ 沿光滑曲线下滑到 $B$，什么曲线用时最短？

时间：$T = \int ds/v = \int \sqrt{1+y'^2}/\sqrt{2gy}\,dx$

被积函数 $F = \sqrt{1+y'^2}/\sqrt{2gy}$ 不显含 $x$。利用欧拉方程的**第一积分**（$F$ 不显含 $x$ 时，$F - y'\partial F/\partial y' = C$）：

经过计算得到：$y(1+y'^2) =$ 常数 → **摆线（旋轮线）**。

> 最速降线不是直线（虽然直线最短），不是圆弧（虽然圆弧最光滑），而是摆线——在重力作用下，摆线恰好让质点以最短时间到达。这就是变分法的威力。

# 第九章 正则变换 + 哈密顿-雅可比方程

> **本章目标**：掌握分析力学最抽象也最强大的部分——通过"换坐标系"简化问题，以及 H-J 方程——经典力学的最终形式。

## 9.1 正则变换——换一套"好"的变量

### 动机

有些系统在 $(q,p)$ 坐标下很复杂，但换个变量就可能变得极其简单。例如：有心力场的拉格朗日函数用极坐标比直角坐标清爽得多。

正则变换是**保持哈密顿方程形式不变**的最一般的变量变换。注意——$q$ 和 $p$ 可以混着变！新坐标 $Q$ 可以同时依赖旧的 $q$ 和 $p$。

### 定义

变换 $(q,p) \to (Q,P)$ 是**正则变换**，当且仅当存在新哈密顿 $K(Q,P,t)$ 使得：

$$\dot{Q} = \frac{\partial K}{\partial P}, \quad \dot{P} = -\frac{\partial K}{\partial Q}$$

### 正则变换的等价条件

**泊松括号不变**：

$$\{Q_i, Q_j\}_{q,p} = 0, \quad \{P_i, P_j\}_{q,p} = 0, \quad \{Q_i, P_j\}_{q,p} = \delta_{ij}$$

**辛条件**：雅可比矩阵 $\mathbf{M} = \partial(Q,P)/\partial(q,p)$ 满足 $\mathbf{M}\mathbf{J}\mathbf{M}^T = \mathbf{J}$（$\mathbf{J}$ 是辛矩阵）。

---

## 9.2 母函数——正则变换的生成器

任意正则变换可以由**母函数**生成。有四类（通过勒让德变换互相转化）：

### 第一类母函数 $F_1(q, Q, t)$

$$p_j = \frac{\partial F_1}{\partial q_j}, \quad P_j = -\frac{\partial F_1}{\partial Q_j}, \quad K = H + \frac{\partial F_1}{\partial t}$$

### 第二类母函数 $F_2(q, P, t)$——最常用！

$$p_j = \frac{\partial F_2}{\partial q_j}, \quad Q_j = \frac{\partial F_2}{\partial P_j}, \quad K = H + \frac{\partial F_2}{\partial t}$$

### 第三类母函数 $F_3(p, Q, t)$

$$q_j = -\frac{\partial F_3}{\partial p_j}, \quad P_j = -\frac{\partial F_3}{\partial Q_j}$$

### 第四类母函数 $F_4(p, P, t)$

$$q_j = -\frac{\partial F_4}{\partial p_j}, \quad Q_j = \frac{\partial F_4}{\partial P_j}$$

### 四类母函数的关系

> $F_1(q,Q) \longleftrightarrow F_2(q,P)$（$Q \leftrightarrow P$ 勒让德变换）
>
> $F_3(p,Q) \longleftrightarrow F_4(p,P)$（$q \leftrightarrow p$ 勒让德变换）

> **记忆诀窍**：母函数名的下标告诉你它依赖哪两个变量——$F_1(q,Q)$、$F_2(q,P)$、$F_3(p,Q)$、$F_4(p,P)$。$F_2$ 最常用——$q$ 和 $P$。

---

## 9.3 母函数例题

### 例题 1：恒等变换 $F_2 = q\cdot P$

$$p = \frac{\partial F_2}{\partial q} = P, \quad Q = \frac{\partial F_2}{\partial P} = q$$

新旧变量完全一样——这就是为什么母函数是 $F_2(q,P)$。

### 例题 2：坐标-动量交换 $F_1 = q\cdot Q$

$$p = \frac{\partial F_1}{\partial q} = Q, \quad P = -\frac{\partial F_1}{\partial Q} = -q$$

新"坐标" $Q = p$，新"动量" $P = -q$。**坐标和动量互换了！** 这在哈密顿力学中是完全合法的——说明 $q$ 和 $p$ 的地位确实是对称的。

### 例题 3：点变换 $F_2(q,P) = f(q)\cdot P$

$$p = \frac{\partial F_2}{\partial q} = f'(q)P, \quad Q = \frac{\partial F_2}{\partial P} = f(q)$$

这就是普通的坐标变换 $q \to Q = f(q)$ 的正则化版本。

---

## 9.4 无穷小正则变换与对称性——诺特定理

### 无穷小变换

考虑由函数 $G(q,p)$ 生成的无穷小正则变换（$\varepsilon$ 为无穷小参数）：

$$\delta q_j = \varepsilon\frac{\partial G}{\partial p_j}, \quad \delta p_j = -\varepsilon\frac{\partial G}{\partial q_j}$$

### 时间演化就是正则变换！

取 $\varepsilon = dt$，$G = H$，得：

$$\delta q_j = dt\frac{\partial H}{\partial p_j} = \dot{q}_j\,dt$$
$$\delta p_j = -dt\frac{\partial H}{\partial q_j} = \dot{p}_j\,dt$$

**时间演化就是哈密顿函数生成的无穷小正则变换！**

### 对称性 → 守恒量

如果 $G$ 与 $H$ 的泊松括号为零：$\{G, H\} = 0$ → $G$ 是守恒量 → $G$ 生成的变换是系统的对称性。

| 生成元 $G$ | 生成的变换 | 若 $\{G,H\}=0$，守恒量 |
|-----------|-----------|---------------------|
| $H$ | 时间平移 | 能量守恒 |
| $p_x$ | $x$ 方向平移 | $x$ 方向动量守恒 |
| $L_z$ | 绕 $z$ 轴旋转 | $z$ 方向角动量守恒 |

> **诺特定理的哈密顿版**：守恒量 = 对称变换的生成元。每一个连续对称性对应一个守恒量，守恒量就是该对称性变换的"母函数"。

---

## 9.5 哈密顿-雅可比方程——经典力学的最终形式

### 基本思想

找一个正则变换，使得**新哈密顿函数 $K$ 恒为零**。如果成功——所有新变量 $P, Q$ 都是常数，问题瞬间解决！

### H-J 方程

寻找母函数 $S(q, P, t)$（第二类）使得 $K=0$：

> $$\boxed{H\left(q_1,...,q_s; \frac{\partial S}{\partial q_1},...,\frac{\partial S}{\partial q_s}; t\right) + \frac{\partial S}{\partial t} = 0}$$

$S$ 叫**哈密顿主函数**。方程中所有 $p_j$ 被 $\partial S/\partial q_j$ 替代。

### 如果不显含时间

设 $S(q, t) = W(q) - Et$（分离变量），$W(q)$ 叫**哈密顿特征函数**，满足：

$$H\left(q, \frac{\partial W}{\partial q}\right) = E$$

这是一个不含时间的偏微分方程——相当于能量守恒条件。

---

## 9.6 分离变量法

如果 $H$ 可以写成每个自由度独立部分之和，H-J 方程可以**分离变量**为 $s$ 个常微分方程。

假设 $S = \sum S_j(q_j)$（完全可分离），代入 H-J 方程后，每个方程只含一个变量——逐个积分即可。

**步骤**：

1. 写 H-J 方程
2. 假设 $S = \sum S_j(q_j)$（或 $S = W(q) - Et$）
3. 分离变量，每项只含一个 $q_j$
4. 令每项 $=$ 分离常数 $\alpha_j$
5. 积分得 $S(q, \alpha, t)$（含 $s$ 个积分常数 $\alpha_j$）
6. $\partial S/\partial\alpha_j = \beta_j$（常数）→ 这给出轨道方程

---

## 9.7 H-J 方程例题

### 例题 1：一维谐振子

$$H = \frac{p^2}{2m} + \frac{1}{2}m\omega^2 q^2$$

H-J 方程（不显含 $t$，设 $S = W(q) - Et$）：

$$\frac{1}{2m}\left(\frac{dW}{dq}\right)^2 + \frac{1}{2}m\omega^2 q^2 = E$$

解：$\dfrac{dW}{dq} = \sqrt{2mE - m^2\omega^2 q^2}$

$$W(q) = \int \sqrt{2mE - m^2\omega^2 q^2}\,dq\text{（含积分常数）}$$

$$S(q, E, t) = W(q) - Et$$

$\partial S/\partial E =$ 常数 → $t + \text{常数} = (1/\omega)\arcsin(q\sqrt{m\omega^2/(2E)})$

→ $q = \sqrt{2E/(m\omega^2)}\sin(\omega t + \varphi)$——标准的谐振子解。

---

### 例题 2：均匀重力场中的抛体

$$H = \frac{p_x^2 + p_y^2 + p_z^2}{2m} + mgz$$

设 $S = W_x(x) + W_y(y) + W_z(z) - Et$，分离变量：

- $(dW_x/dx)^2 =$ 常数 $= p_{x0}^2$ → $W_x = p_{x0}x$
- $(dW_y/dy)^2 =$ 常数 $= p_{y0}^2$ → $W_y = p_{y0}y$
- $(dW_z/dz)^2 = 2m(E - mgz) - p_{x0}^2 - p_{y0}^2$

$\partial S/\partial p_{x0} =$ 常数 → $x = x_0 + p_{x0}t/m$（匀速）

$\partial S/\partial E =$ 常数 → 解出 $z(t) = z_0 + v_{z0}t - \frac{1}{2}gt^2$

> H-J 方程统一地处理了所有自由度——守恒的动量直接变成分离常数。

---

## 9.8 作用-角变量——连接旧量子论

对于**周期运动**，定义**作用变量**：

$$J = \oint p\,dq = \oint \frac{\partial W}{\partial q}\,dq$$

$J$ 对周期积分。角变量 $w$：

$$w = \frac{\partial S}{\partial J}, \quad \dot{w} = \frac{\partial H}{\partial J} = \nu = \text{常数}$$

**物理含义**：$J$ 决定轨道的"大小"（能量），$w$ 是轨道上的"相位"（线性增长）。

$J$ 是**绝热不变量**——系统参数缓慢变化时，$J$ 保持不变。这解释了为什么荡秋千时慢慢蹲下/站起可以增加振幅——你在改变"有效摆长"。

> **历史注记**：玻尔-索末菲量子化条件 $\oint p\,dq = nh$。旧量子论在量子力学诞生之前就"猜"出了正确的量子化规则——正是基于作用-角变量理论。

## 9.9 全图景：从牛顿到量子力学

> 牛顿 $\vec{F} = m\vec{a}$（力，矢量）
>
> ↓ 达朗贝尔原理（第二章）
>
> 拉格朗日 $\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot{q}}\right) - \dfrac{\partial L}{\partial q} = 0$（能量，标量）
>
> ↓ 勒让德变换（第七章）
>
> 哈密顿 $\dot{q} = \dfrac{\partial H}{\partial p}$, $\dot{p} = -\dfrac{\partial H}{\partial q}$（相空间，对称）
>
> ↓ 变分原理（第八章）
>
> $\delta S = \delta\int L\,dt = 0$（大自然的最优选择）
>
> ↓ 找 $K=0$ 的变换（第九章）
>
> H-J 方程 $H\!\left(q,\dfrac{\partial S}{\partial q},t\right) + \dfrac{\partial S}{\partial t} = 0$
>
> ↓ $\hbar\to 0$ 极限
>
> 薛定谔方程 $i\hbar\dfrac{\partial\psi}{\partial t} = \hat{H}\psi$

> 经典力学的最终形式 H-J 方程，其数学结构几乎就是"量子力学在 $\hbar\to 0$ 时的几何光学近似"。路径积分量子化（费曼）更是直接把 $\delta S = 0$ 推广为"所有路径以 $e^{iS/\hbar}$ 的权重叠加"。理论力学的终点，恰好是量子力学的起点。

# 五大考试题型解题模板

## 题型一：拉格朗日方程求运动方程

1. 确定自由度 $s$，选广义坐标 $q_j$
2. 写动能 $T$（用 $q, \dot{q}$）和势能 $V$
3. $L = T - V$
4. 代入 $\dfrac{d}{dt}\!\left(\dfrac{\partial L}{\partial \dot{q}_j}\right) - \dfrac{\partial L}{\partial q_j} = 0$
5. 检查循环坐标 → 守恒量

## 题型二：小振动求简正频率

1. 找稳定平衡点（$\partial V/\partial q = 0$）
2. 展开 $T$ 和 $V$ 到二阶 → $\mathbf{M}$ 和 $\mathbf{K}$ 矩阵
3. 解特征值方程 $\det(\mathbf{K} - \omega^2\mathbf{M}) = 0$
4. $\omega$ 即简正频率，特征向量即简正模式

## 题型三：有心力轨道分析

1. 写有效势 $V_{\text{eff}} = V(r) + l^2/(2\mu r^2)$
2. 根据 $E$ 判断轨道类型（负能椭圆、正能双曲）
3. 用比内公式求轨道方程

## 题型四：哈密顿形式化

1. 从 $L$ 求 $p = \partial L/\partial\dot{q}$
2. 构造 $H = \sum p\dot{q} - L$（消去 $\dot{q}$）
3. 写正则方程 $\dot{q} = \partial H/\partial p$, $\dot{p} = -\partial H/\partial q$

## 题型五：正则变换

1. 验证泊松括号是否满足（$\{Q,P\}=1$ 等）
2. 求给定母函数对应的变换
3. 利用母函数求新的哈密顿函数 $K$
