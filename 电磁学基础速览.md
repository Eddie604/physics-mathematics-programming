## 1. 静电场

### 一句话核心
**电荷是电场的源，电场对电荷施加力。**

---

### 1.1 库仑定律

两个点电荷之间的力：

$$F = \frac{1}{4\pi\varepsilon_0}\frac{q_1 q_2}{r^2}$$

- $\varepsilon_0 \approx 8.85 \times 10^{-12}$ ——真空介电常数
- 同号相斥，异号相吸
- 力沿连线方向

---

### 1.2 电场强度

$$\vec{E} = \frac{\vec{F}}{q}$$

**点电荷的电场**：

$$E = \frac{1}{4\pi\varepsilon_0}\frac{Q}{r^2}$$

方向：正电荷向外辐射，负电荷向内汇聚。

**电场叠加原理**：多个电荷的总电场 = 每个电荷单独产生的电场矢量和。

---

### 1.3 电势（电压）

**定义**：单位正电荷从参考点（无穷远）移到某点，电场力做的功的负值。

$$\varphi = \frac{1}{4\pi\varepsilon_0}\frac{Q}{r}$$

**电场与电势的关系**：

$$\vec{E} = -\nabla\varphi$$

> 白话：电场指向电势下降最快的方向。等势面越密，电场越强。

**电势差（电压）**：$U_{AB} = \varphi_A - \varphi_B$

---

### 1.4 高斯定理 🔑

$$\oint_S \vec{E}\cdot d\vec{S} = \frac{Q_{\text{内}}}{\varepsilon_0}$$

**白话**：穿过一个闭合曲面的电场"总通量"，只与曲面内部包含的电荷有关。

**典型应用**（考试常考）：

| 情况 | 电场结果 |
|------|----------|
| 点电荷 Q | $E = \frac{Q}{4\pi\varepsilon_0 r^2}$ |
| 均匀带电球体（外） | 同上（相当于所有电荷在球心） |
| 无限长均匀带电直线 | $E = \frac{\lambda}{2\pi\varepsilon_0 r}$ |
| 无限大均匀带电平面 | $E = \frac{\sigma}{2\varepsilon_0}$（与距离无关！） |
| 平行板电容器 | $E = \frac{\sigma}{\varepsilon_0}$（板间均匀） |

---

### 1.5 电容

$$C = \frac{Q}{U}$$

**平行板电容器**：$C = \frac{\varepsilon_0 S}{d}$

**电容器储能**：$W = \frac{1}{2}CU^2 = \frac{Q^2}{2C}$

---

### 1.6 静电场的能量

$$W = \frac{1}{2}\int \varepsilon_0 E^2\,dV$$

> 能量储存在电场中，不是储存在电荷上。

---

## 2. 静磁场

### 一句话核心
**电流是磁场的源，磁场对运动电荷施加力。**

---

### 2.1 磁感应强度 B

磁场的基本物理量。单位：特斯拉（T）。

地球磁场 ≈ 5×10⁻⁵ T，医用 MRI ≈ 1.5~3 T。

---

### 2.2 洛伦兹力 🔑

$$\vec{F} = q\vec{v} \times \vec{B}$$

**关键性质**：
- 力始终垂直于速度 → **磁场力不做功**，只改变运动方向
- 带电粒子在均匀磁场中做**匀速圆周运动**

**回旋半径**：$r = \frac{mv}{qB}$，**回旋频率**：$\omega = \frac{qB}{m}$

---

### 2.3 毕奥-萨伐尔定律

电流元 Idl 产生的磁场微元：

$$dB = \frac{\mu_0}{4\pi}\frac{I\,dl \times \hat{r}}{r^2}$$

$\mu_0 = 4\pi \times 10^{-7}$ ——真空磁导率。

**典型结果**：

| 情况 | 磁场 |
|------|------|
| 无限长直导线 | $B = \frac{\mu_0 I}{2\pi r}$（绕导线的同心圆） |
| 圆环中心 | $B = \frac{\mu_0 I}{2R}$ |
| 长直螺线管内部 | $B = \mu_0 nI$（n 为单位长度匝数，管内均匀） |
| 亥姆霍兹线圈中心 | 均匀磁场区域 |

---

### 2.4 安培力

一段通电导线在磁场中受的力：
$$\vec{F} = I\vec{L} \times \vec{B}$$

**两根平行载流导线之间的力**：
- 同向电流相吸，反向电流相斥
- 单位长度受力：$F/L = \frac{\mu_0 I_1 I_2}{2\pi d}$

---

### 2.5 安培环路定理 🔑

$$\oint_L \vec{B}\cdot d\vec{l} = \mu_0 I_{\text{内}}$$

**典型应用**：长直螺线管内部 $B = \mu_0 nI$，环形螺线管（螺绕环）$B = \mu_0 NI/(2\pi r)$。

---

### 2.6 磁通量

$$\Phi_B = \int_S \vec{B}\cdot d\vec{S}$$

**磁场的高斯定理**（说明没有磁单极子）：
$$\oint_S \vec{B}\cdot d\vec{S} = 0$$

---

### 2.7 电感

**自感**：$L = \frac{\Phi}{I}$（长直螺线管：$L = \mu_0 n^2 V$）

**互感**：$\mathcal{E}_2 = -M\frac{dI_1}{dt}$

**电感储能**：$W = \frac{1}{2}LI^2$

---

## 3. 电磁感应

### 一句话核心
**变化的磁场产生电场（电动势）。**

---

### 3.1 法拉第电磁感应定律 🔑

$$\mathcal{E} = -\frac{d\Phi_B}{dt}$$

**两类情况**：

| 类型 | 原因 | 例子 |
|------|------|------|
| 感生电动势 | 磁场本身在变，导体不动 | 变压器 |
| 动生电动势 | 磁场不变，导体在动 | 发电机 |

**动生电动势公式**：$\mathcal{E} = \int (\vec{v} \times \vec{B})\cdot d\vec{l}$

---

### 3.2 楞次定律 🔑

> **感应电流的方向，总是"反对"引起它的磁通量变化。**

- 磁铁靠近线圈 → 线圈产生排斥磁铁靠近的磁场
- 磁铁远离线圈 → 线圈产生吸引磁铁回来的磁场

这是**能量守恒**的体现——如果反过来，就成了永动机。

---

### 3.3 涡旋电场

变化的磁场产生的电场是**涡旋电场**——电场线是闭合环，不是从正电荷出发。这意味着：
- 涡旋电场**不是保守场**
- 不能定义标量电势
- 但能驱动电荷绕圈流动——这就是涡流和变压器的原理

---

### 3.4 涡流（傅科电流）

导体块在变化磁场中，内部产生闭合感应电流——涡流。

**利**：电磁炉加热、金属感应熔炼  
**弊**：变压器铁芯损耗 → 铁芯用叠片或铁氧体抑制涡流

---

## 4. 麦克斯韦方程组

### 一句话核心
**四条方程完整描述所有电磁现象，是经典电磁理论的终极框架。**

---

### 4.1 积分形式（物理含义最清晰）

| # | 方程 | 说的是什么 |
|---|------|-----------|
| 1 | $\oint_S \vec{E}\cdot d\vec{S} = \frac{Q}{\varepsilon_0}$ | 电荷产生电场（高斯定理） |
| 2 | $\oint_S \vec{B}\cdot d\vec{S} = 0$ | 没有磁单极子 |
| 3 | $\oint_L \vec{E}\cdot d\vec{l} = -\frac{d\Phi_B}{dt}$ | 变化的磁场产生电场（法拉第） |
| 4 | $\oint_L \vec{B}\cdot d\vec{l} = \mu_0 I + \mu_0\varepsilon_0\frac{d\Phi_E}{dt}$ | 电流 + 变化的电场产生磁场 |

### 4.2 微分形式（计算常用）

| # | 方程 |
|---|------|
| 1 | $\nabla\cdot\vec{E} = \frac{\rho}{\varepsilon_0}$ |
| 2 | $\nabla\cdot\vec{B} = 0$ |
| 3 | $\nabla\times\vec{E} = -\frac{\partial\vec{B}}{\partial t}$ |
| 4 | $\nabla\times\vec{B} = \mu_0\vec{J} + \mu_0\varepsilon_0\frac{\partial\vec{E}}{\partial t}$ |

---

### 4.3 位移电流——麦克斯韦的天才修正 🔑

**问题**：正在充电的电容——导线中有电流，极板间是真空没有电流——安培环路定理出现了矛盾。

**麦克斯韦的解决方案**：变化的电场也产生磁场！把 $\varepsilon_0 \frac{\partial E}{\partial t}$ 当作"位移电流"。

这导致方程④多出一项——而正是这一项，数学上必然推出电磁波的存在。

---

## 5. 电磁波基础

### 一句话核心
**麦克斯韦方程组 → 波动方程 → 电磁波以光速传播 → 光就是电磁波。**

---

### 5.1 电磁波的产生

振荡的电偶极子（如天线中的交变电流）发出电磁波。变化的电场产生变化的磁场，变化的磁场又产生变化的电场——互相激发，向外传播。

---

### 5.2 电磁波的基本性质

- **横波**：E 和 B 都垂直于传播方向
- **E ⊥ B**：电场和磁场互相垂直
- **E, B, k 右手关系**：$\vec{B} = \frac{1}{c}\hat{k} \times \vec{E}$
- **真空中 E = cB**
- **传播速度** $c = 1/\sqrt{\mu_0\varepsilon_0} \approx 3\times 10^8$ m/s

---

### 5.3 电磁波谱

| 波段 | 波长范围 | 典型应用 |
|------|----------|----------|
| 无线电波 | > 0.1 m | 广播、通信 |
| 微波 | 1 mm ~ 0.1 m | 雷达、微波炉、WiFi |
| 红外线 | 0.7 μm ~ 1 mm | 热成像、遥控 |
| 可见光 | 400~700 nm | 人眼可见 |
| 紫外线 | 10~400 nm | 杀菌、验钞 |
| X 射线 | 0.01~10 nm | 医学影像 |
| γ 射线 | < 0.01 nm | 核医学、放疗 |

---

### 5.4 电磁波的能量——坡印廷矢量

$$\vec{S} = \vec{E} \times \vec{H}$$

S 代表**单位时间穿过单位面积的电磁能量**（能流密度）。

**对于平面波**：$S = \frac{1}{\mu_0}EB = \frac{E^2}{\mu_0 c}$

---

## 6. 电路中的电磁学

### 6.1 欧姆定律

微观：$\vec{J} = \sigma\vec{E}$（电流密度 = 电导率 × 电场）

宏观：$I = \frac{U}{R}$

**电阻**：$R = \rho\frac{L}{S}$（电阻率 × 长度 / 截面积）

---

### 6.2 基尔霍夫定律

| 定律 | 内容 | 依据 |
|------|------|------|
| KCL（电流定律） | 节点流入流出电流和为零 | 电荷守恒 |
| KVL（电压定律） | 闭合回路电压升等于电压降 | 静电场是保守场 |

---

### 6.3 RC 电路（充放电）

**充电**：$q(t) = C\mathcal{E}(1 - e^{-t/RC})$

**放电**：$q(t) = Q_0 e^{-t/RC}$

**时间常数**：$\tau = RC$

---

### 6.4 RL 电路

**通电**：$I(t) = \frac{\mathcal{E}}{R}(1 - e^{-Rt/L})$

**断电**：$I(t) = I_0 e^{-Rt/L}$

**时间常数**：$\tau = L/R$

---

### 6.5 LC 振荡

$$\omega = \frac{1}{\sqrt{LC}}$$

电荷和电流作简谐振荡，电场能量和磁场能量互相转化——总能量守恒。

---

### 6.6 RLC 串联电路

**阻抗**：$Z = \sqrt{R^2 + (\omega L - \frac{1}{\omega C})^2}$

**谐振频率**：$\omega_0 = \frac{1}{\sqrt{LC}}$

**品质因数**：$Q = \frac{\omega_0 L}{R}$

---

## 7. 常用公式速查

### 力与场

| 公式 | 含义 |
|------|------|
| $\vec{F} = q\vec{E}$ | 电场力 |
| $\vec{F} = q\vec{v}\times\vec{B}$ | 磁场力（洛伦兹力） |
| $\vec{F} = I\vec{L}\times\vec{B}$ | 安培力 |

### 电场

| 公式 | 含义 |
|------|------|
| $\oint \vec{E}\cdot d\vec{S} = Q/\varepsilon_0$ | 高斯定理 |
| $\vec{E} = -\nabla\varphi$ | 电场与电势 |
| $C = Q/U$ | 电容 |
| $C = \varepsilon_0 S/d$ | 平行板电容 |
| $W = \frac{1}{2}CU^2$ | 电容储能 |

### 磁场

| 公式 | 含义 |
|------|------|
| $B = \frac{\mu_0 I}{2\pi r}$ | 长直导线磁场 |
| $B = \mu_0 nI$ | 长直螺线管内部 |
| $\oint \vec{B}\cdot d\vec{l} = \mu_0 I$ | 安培环路定理 |
| $\Phi_B = \int \vec{B}\cdot d\vec{S}$ | 磁通量 |

### 感应

| 公式 | 含义 |
|------|------|
| $\mathcal{E} = -d\Phi_B/dt$ | 法拉第定律 |
| $\mathcal{E} = BLv$ | 动生电动势（垂直切割） |
| $L = \Phi/I$ | 自感 |
| $W = \frac{1}{2}LI^2$ | 电感储能 |

### 电路

| 公式 | 含义 |
|------|------|
| $\tau = RC$ | RC 时间常数 |
| $\tau = L/R$ | RL 时间常数 |
| $\omega_0 = 1/\sqrt{LC}$ | LC 谐振频率 |
