## 1. 仿真是什么，为什么物理系必须会

### 一句话定义
**仿真（Simulation）= 用计算机数值求解物理方程，模拟真实物理过程。**

### 三种认知层次

| 层次 | 能力 | 典型场景 |
|------|------|----------|
| 使用者 | 调用现成软件（COMSOL, ANSYS, LAMMPS） | 课设、简单课题 |
| 定制者 | 自己写代码求解特定问题 | 科研、毕设 |
| 开创者 | 提出新算法、新模型 | 发论文、做方法 |

**你的目标：从"使用者"跃升到"定制者"。**

### 仿真 vs 理论 vs 实验

```
理论 ──→ 解析解（只有极少数问题有）
  ↓
仿真 ──→ 数值解（几乎所有问题都能算）
  ↓
实验 ──→ 测量值（验证理论和仿真）
```

**仿真 = 在"可算但不可解"的鸿沟上架桥。**

### 3.2 数值微积分核心公式

**数值导数（三点中心差分，O(h²)精度）：**

```
f'(x) ≈ [f(x+h) - f(x-h)] / (2h)
f''(x) ≈ [f(x+h) - 2f(x) + f(x-h)] / h²
```

**数值积分（Simpson 法则，O(h⁴)精度）：**

```
∫[a,b] f(x)dx ≈ (h/3) × [f(a) + 4 Σ f(奇数点) + 2 Σ f(偶数点) + f(b)]
```

**实战：用 5 行代码验证精度**

```python
import numpy as np

def derivative(f, x, h=1e-5):
    """三点中心差分求导"""
    return (f(x + h) - f(x - h)) / (2 * h)

# 验证：sin(x) 在 x=π/3 处导数应为 cos(π/3)=0.5
f = np.sin
result = derivative(f, np.pi/3)
print(f"数值: {result:.6f}, 解析: {np.cos(np.pi/3):.6f}")
```

## 5. 第三阶段：核心算法精讲（5天）

### 5.1 ODE 求解器：从欧拉到 RK45

这是仿真中最常用的工具。物理定律大多是微分方程。

#### Euler 法（入门，不可用于生产）

```
y_{n+1} = y_n + h · f(t_n, y_n)
```

误差 ∝ h，全局误差 ∝ h。**精度太差，仅用于理解概念。**

#### RK4（经典，性价比最高）

```
k1 = h · f(t_n, y_n)
k2 = h · f(t_n + h/2, y_n + k1/2)
k3 = h · f(t_n + h/2, y_n + k2/2)
k4 = h · f(t_n + h,   y_n + k3)

y_{n+1} = y_n + (k1 + 2k2 + 2k3 + k4)/6
```

误差 ∝ h⁴，全局误差 ∝ h³。**这是你的默认选择。**

#### 实战：阻尼谐振子（完整代码）

```python
import numpy as np
import matplotlib.pyplot as plt
from scipy.integrate import solve_ivp

def damped_oscillator(t, y, b, omega0):
    """
    阻尼谐振子: mx'' + bx' + kx = 0
    化为一阶组:
        y[0] = x
        y[1] = x'
        y[0]' = y[1]
        y[1]' = -2β y[1] - ω₀² y[0]
    """
    beta = b / 2
    return [y[1], -2 * beta * y[1] - omega0**2 * y[0]]

# 参数
omega0 = 2.0   # 固有频率
b = 0.5        # 阻尼系数
y0 = [1.0, 0.0]  # 初始位移=1, 初速=0

# 求解
sol = solve_ivp(
    damped_oscillator, [0, 20], y0,
    args=(b, omega0),
    method='RK45',          # 自适应步长 RK45
    rtol=1e-8, atol=1e-10   # 精度控制
)

# 可视化
plt.figure(figsize=(10, 4))
plt.plot(sol.t, sol.y[0], 'b-', label='位移 x(t)')
plt.plot(sol.t, sol.y[1], 'r--', label='速度 v(t)')
plt.axhline(0, color='gray', lw=0.5)
plt.xlabel('时间 t'); plt.ylabel('响应')
plt.legend(); plt.grid(alpha=0.3)
plt.title('阻尼谐振子数值解 (RK45)')
plt.show()
```

### 5.2 PDE 有限差分法（FDM）

**核心思想：用差分代替微分。**

#### 一维热传导方程

```
∂u/∂t = α · ∂²u/∂x²

离散化（FTCS 格式）：
u_j^{n+1} = u_j^n + αΔt/Δx² · (u_{j+1}^n - 2u_j^n + u_{j-1}^n)
```

**稳定性条件（CFL 条件）：αΔt/Δx² ≤ 1/2**

这是仿真中第一个让你崩溃的概念——**时间步不是随便取的，取大了直接爆炸！**

#### 实战：热传导完整代码

```python
import numpy as np
import matplotlib.pyplot as plt

# 参数
L = 1.0           # 杆长
Nx = 50           # 空间网格数
dx = L / (Nx - 1)
alpha = 0.01      # 热扩散系数

# CFL 稳定性条件
dt_max = dx**2 / (2 * alpha)
dt = dt_max * 0.9  # 取 90% 保证安全
Nt = 2000

print(f"dx={dx:.4f}, dt_max={dt_max:.6f}, dt={dt:.6f}")

# 初始条件：中间高温
x = np.linspace(0, L, Nx)
u = np.exp(-100 * (x - 0.5)**2)  # 高斯热脉冲

# 时间推进
r = alpha * dt / dx**2
for n in range(Nt):
    u[1:-1] = u[1:-1] + r * (u[2:] - 2*u[1:-1] + u[:-2])
    # 边界条件：两端恒温 0
    u[0] = u[-1] = 0

# 可视化
plt.plot(x, u, 'r-', lw=2)
plt.xlabel('位置 x'); plt.ylabel('温度 u')
plt.title(f'一维热传导 (t={Nt*dt:.2f}s)')
plt.grid(alpha=0.3); plt.show()
```

### 5.3 蒙特卡洛方法

**核心公式（定积分估计）：**

```
∫ f(x)dx ≈ (b-a)/N · Σ f(x_i),  x_i ~ U(a,b)

误差 ∝ 1/√N  （与维度无关！这是高维积分的杀手锏）
```

#### 实战：用蒙特卡洛算 π

```python
import numpy as np

N = 1_000_000
x, y = np.random.uniform(-1, 1, (2, N))
inside = x**2 + y**2 <= 1
pi_est = 4 * np.sum(inside) / N
print(f"估计值: {pi_est:.6f}, 真值: {np.pi:.6f}, 误差: {abs(pi_est-np.pi):.2e}")
```

#### Metropolis 算法（采样任意分布）

这是蒙特卡洛方法最厉害的应用——**你不需要知道分布的归一化常数就能采样。**

```python
import numpy as np

def metropolis(target_pdf, proposal_std, N):
    """Metropolis-Hastings 采样"""
    samples = np.zeros(N)
    x = 0.0  # 初始点
    for i in range(N):
        x_new = x + np.random.normal(0, proposal_std)  # 提议步
        alpha = target_pdf(x_new) / target_pdf(x)      # 接受率
        if np.random.uniform() < alpha:
            x = x_new   # 接受
        samples[i] = x  # 记录（拒绝也记录旧值）
    return samples

# 采样目标：双峰分布 p(x) ∝ exp(-(x²-4)²/8)
target = lambda x: np.exp(-(x**2 - 4)**2 / 8)
samples = metropolis(target, proposal_std=1.0, N=50000)
```

### 5.4 分子动力学（MD）基础

**Verlet 算法是 MD 的灵魂：**

```
速度 Verlet:
  v(t + Δt/2) = v(t) + a(t)·Δt/2
  r(t + Δt)   = r(t) + v(t + Δt/2)·Δt
  a(t + Δt)   = F(r(t+Δt)) / m    ← 最耗时的一步
  v(t + Δt)   = v(t + Δt/2) + a(t + Δt)·Δt/2
```

**优点：时间可逆、辛结构保持、能量长期稳定。**

#### 实战：Lennard-Jones 粒子

```python
import numpy as np

def lj_force(r, epsilon=1.0, sigma=1.0):
    """Lennard-Jones 力（已向量化）"""
    r2 = np.sum(r**2, axis=1, keepdims=True)
    r6 = r2**3
    r12 = r6**2
    # F = 24ε/σ² · [2(σ/r)^14 - (σ/r)^8] · r
    force_mag = 24 * epsilon * (2 * sigma**12 / r12 - sigma**6 / r6) / r2
    return force_mag * r

def velocity_verlet_step(pos, vel, forces, mass, dt):
    """单步速度 Verlet（含周期边界）"""
    vel_half = vel + 0.5 * forces / mass * dt
    pos_new = pos + vel_half * dt
    # 周期边界条件（盒子 L=10）
    pos_new = pos_new % 10.0
    forces_new = compute_all_forces(pos_new)
    vel_new = vel_half + 0.5 * forces_new / mass * dt
    return pos_new, vel_new, forces_new
```

### 6.1 经典力学：三体问题

```python
from scipy.integrate import solve_ivp
import numpy as np

def three_body(t, y, G=1.0):
    """三体问题运动方程，y 格式: [r1, r2, r3, v1, v2, v3] (各3分量，共18维)"""
    r = y[:9].reshape(3, 3)  # 3体 × 3维坐标
    v = y[9:].reshape(3, 3)
    
    a = np.zeros((3, 3))
    m = np.array([1.0, 1.0, 1.0])  # 三体质量
    
    for i in range(3):
        for j in range(3):
            if i != j:
                dr = r[j] - r[i]
                dist = np.linalg.norm(dr)
                a[i] += G * m[j] * dr / (dist**3 + 0.01**3)  # 加 softening 防发散
    
    return np.concatenate([v.flatten(), a.flatten()])

# 初始条件（八字形轨道）
y0 = np.array([
    -0.970, 0.243, 0,  0.970, -0.243, 0,  0, 0, 0,  # 坐标
    0.466, 0.432, 0,  0.466, 0.432, 0,  -0.932, -0.864, 0  # 速度
])

sol = solve_ivp(three_body, [0, 10], y0, max_step=0.01, rtol=1e-9, atol=1e-12)
print(f"积分步数: {len(sol.t)}")
```

### 6.4 统计物理：Ising 模型

```python
import numpy as np

def ising_mc_step(lattice, beta, L):
    """单步 Metropolis 更新（2D Ising）"""
    i, j = np.random.randint(0, L, 2)
    s = lattice[i, j]
    # 最近邻能量变化
    nb = lattice[(i+1)%L, j] + lattice[(i-1)%L, j] + lattice[i, (j+1)%L] + lattice[i, (j-1)%L]
    dE = 2 * s * nb  # ΔE = 2J·s·Σs_nb (J=1)
    
    if dE <= 0 or np.random.uniform() < np.exp(-beta * dE):
        lattice[i, j] = -s  # 翻转
    
    return lattice
```

## 7. 重难点深度讲解

### 7.1 线性代数在仿真中的核心用法

#### 为什么矩阵是仿真的语言

物理仿真从数学上讲就是：
1. **离散化** 物理场 → 向量
2. **离散化** 微分算子 → 矩阵
3. **演化/求解** → 矩阵-向量乘法 / 解线性方程组

**一维 Laplace 算子的离散矩阵（三对角）：**

```
        [ -2   1   0   0  ]
  1/h²  [  1  -2   1   0  ]  ← 这就是仿真引擎的心脏
        [  0   1  -2   1  ]
        [  0   0   1  -2  ]
```

**稀疏性是关键：** 1000×1000 的矩阵只有 3000 个非零元素（密度 0.3%），必须用 `scipy.sparse`。

#### SVD 的物理直觉

```
A = U Σ V^T

U 的列 = 输出空间的正交基（系统响应模式）
Σ 的对角 = 各模式的放大倍数（奇异值）
V 的列 = 输入空间的正交基（系统激发模式）
```

**仿真中的应用：**
- 分析多体系统的集体运动模式
- 从噪声数据中提取物理信号（PCA 降噪）
- 求解病态线性方程组（截断 SVD）

### 7.2 稳定性与刚性方程

#### 数值不稳定性的本质

**热传导方程 FTCS 格式的稳定性：**

```
r = α·Δt / Δx²

当 r > 1/2 时，误差指数增长 → 解爆炸
当 r ≤ 1/2 时，误差衰减 → 解稳定
```

**直觉理解：** 你的时间步不能"跳过"物理信息的传播速度。Δt 太大意味着信息在一个时间步内传播了超过一个网格间距——这在物理上是不可能的，数值上也必然发散。

#### 刚性方程（Stiff ODE）

**定义：** 系统中同时存在快尺度和慢尺度过程。

**例子：** 化学反应网络中，有的反应在纳秒级完成，有的需要几秒。

```
如果用来求解刚性：用显式方法（RK4）→ Δt 被最快过程限制 → 算到天荒地老
如果用隐式方法（BDF, Radau）→ Δt 由精度要求决定 → 实际可行
```

**Scipy 中求解刚性方程：**

```python
sol = solve_ivp(f, t_span, y0, method='BDF', rtol=1e-6, atol=1e-9)
# 或者
sol = solve_ivp(f, t_span, y0, method='Radau', rtol=1e-6, atol=1e-9)
```

**判断是否刚性：** 如果 RK45 在某个时间区间步长骤降到极小，大概率是刚性问题。

### 7.3 重要性采样与马尔可夫链蒙特卡洛 (MCMC)

#### 为什么要重要性采样

直接用均匀分布采样做积分 `∫f(x)dx`：
- 如果 f(x) 只在很小的区域非零 → 绝大多数采样点浪费
- 误差 ∝ 1/√N，要增加一位精度需要 100 倍采样点

**重要性采样的思想：**

```
∫ f(x) dx = ∫ [f(x)/p(x)] · p(x) dx ≈ (1/N) Σ f(x_i)/p(x_i)

其中 x_i ~ p(x)，且 p(x) 在 f(x) 大的地方也大
```

**最优 p(x) ∝ |f(x)|**，此时方差最小。

#### Metropolis 的物理直觉

Metropolis 算法本质上模拟了**热平衡态的 Boltzmann 分布**：

```
P(接受 新状态) = min(1, exp(-ΔE / kT))

ΔE > 0 → 接受概率 < 1（可能拒绝）
ΔE ≤ 0 → 总是接受
```

这正是热力学系统达到平衡时的细致平衡条件。

#### 常见陷阱

1. **Burn-in 期：** 前 N 个样本不是目标分布，必须丢弃
2. **自相关：** 相邻样本高度相关，有效样本数远小于 N
3. **收敛诊断：** 需要从多个不同初始点运行，比较链内方差 vs 链间方差（Gelman-Rubin 统计量）

### 7.4 有限元法 (FEM)  vs  有限差分法 (FDM)

| 维度 | 有限差分 FDM | 有限元 FEM |
|------|-------------|-----------|
| 核心思想 | 微分 → 差分 | 弱形式 + 基函数展开 |
| 网格 | 规则矩形 | 任意三角形/四面体 |
| 精度 | 固定精度 | 可增阶 (p-refinement) |
| 复杂几何 | 差 | 强 |
| 理论基础 | 泰勒展开 | 泛函分析 |
| 上手难度 | 低 | 高 |
| 代表软件 | 自己写 | FEniCS, COMSOL |

**选择指南：**
- 矩形/立方体区域 + 简单方程 → FDM（自己写）
- 复杂几何 + 复杂边界 → FEM（用 FEniCS）
- 需要自适应加密 → FEM

### 7.5 并行计算入门要点

仿真中最耗时的部分：**力的计算（N² 粒子相互作用）** 和 **线性方程组求解**。

```python
# Numba 加速示例：一行装饰器实现 GPU 级别加速
from numba import jit

@jit(nopython=True)
def compute_forces(positions, N):
    """计算 N 个粒子的两两作用力"""
    forces = np.zeros((N, 3))
    for i in range(N):
        for j in range(i+1, N):
            dr = positions[j] - positions[i]
            dist = np.sqrt(np.sum(dr**2))
            f = 24 * (2/dist**14 - 1/dist**8) * dr / dist**2
            forces[i] += f
            forces[j] -= f
    return forces
# 这个 double for 循环，加了 @jit 之后跑得跟 C 一样快
```

## 结语：从"学仿真"到"用仿真思考"

仿真的最高境界不是会调库，而是形成一种**数值直觉**：

- 看到偏微分方程，脑中自动浮现离散网格
- 看到一个积分，立刻判断 Monte Carlo 是否比高斯求积更合适
- 调试结果不对，第一反应是检查 CFL 条件和网格收敛性
- 写代码时，默认写成向量化形式

**最后的建议：** 不要试图读完所有教材再动手。选一个你最熟悉的物理问题（比如谐振子），今天就写出它的仿真程序。哪怕只用了 Euler 法，哪怕只画了一条曲线——**动手的那一刻，你已经是仿真者了。**
