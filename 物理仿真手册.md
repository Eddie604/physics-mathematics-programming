## 第一章 线性代数选讲

### 1.1 为什么线性代数是仿真的"语言"

物理仿真从数学上讲只有三件事：
1. 将连续的物理场离散化为向量 $\mathbf{u} \in \mathbb{R}^N$
2. 将微分算子离散化为矩阵 $\mathbf{A} \in \mathbb{R}^{N \times N}$
3. 求解 $\mathbf{A}\mathbf{u} = \mathbf{b}$ 或演化 $\mathbf{u}^{n+1} = \mathbf{A}\mathbf{u}^n$

> 📘 **核心观点**：在计算机眼中，一切物理方程都是大规模线性系统。

### 1.2 向量与矩阵：物理视角

#### 内积的物理意义

对于两个离散化的函数 $f(x)$ 和 $g(x)$，它们在网格点上的值构成向量 $\mathbf{f}, \mathbf{g}$，则：

$$
\langle f, g \rangle \approx \sum_{i=1}^N f_i g_i \Delta x = \Delta x \cdot \mathbf{f}^T \mathbf{g}
$$

这是函数内积的矩形积分近似。

#### 特征值分解的物理直觉

对于一个振动系统的矩阵表示，特征值分解 $\mathbf{A}\mathbf{v}_k = \lambda_k \mathbf{v}_k$ 给出：

| 量 | 物理意义 |
|:---|:---|
| $\lambda_k$ | 第 $k$ 个简正模的固有频率平方（乘上适当的系数） |
| $\mathbf{v}_k$ | 第 $k$ 个简正模的振型 |

📐 **推导**：一维耦合谐振子链

考虑 $N$ 个质量为 $m$ 的小球，相邻小球间用劲度系数为 $k$ 的弹簧连接。第 $i$ 个小球的位移为 $x_i$。

运动方程：
$$m\ddot{x}_i = k(x_{i+1} - 2x_i + x_{i-1})$$

写为矩阵形式：
$$m\ddot{\mathbf{x}} = -k \mathbf{D} \mathbf{x}$$

其中 $\mathbf{D}$ 是离散 Laplace 算符（三对角矩阵）：

$$\mathbf{D} = \begin{bmatrix}
2 & -1 & 0 & \cdots & 0 \\
-1 & 2 & -1 & \cdots & 0 \\
0 & -1 & 2 & \cdots & 0 \\
\vdots & \vdots & \vdots & \ddots & \vdots \\
0 & 0 & 0 & \cdots & 2
\end{bmatrix}_{N \times N}$$

设 $\mathbf{x} = \mathbf{v} e^{i\omega t}$，则 $-\omega^2 m \mathbf{v} = -k \mathbf{D} \mathbf{v}$，即：

$$\mathbf{D}\mathbf{v} = \frac{m\omega^2}{k} \mathbf{v}$$

特征向量 $\mathbf{v}_k$ 是离散正弦波：$v_k(j) = \sin\left(\frac{k j\pi}{N+1}\right)$。

🖥️ **代码实现**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

# 构建一维离散 Laplace 矩阵
N = 50
D = np.diag(2 * np.ones(N)) - np.diag(np.ones(N-1), 1) - np.diag(np.ones(N-1), -1)

# 求特征值和特征向量
eigenvalues, eigenvectors = np.linalg.eigh(D)

# 显示前 4 个简正模
fig, axes = plt.subplots(2, 2, figsize=(10, 8))
for k, ax in enumerate(axes.flat):
    ax.plot(eigenvectors[:, k], 'b-', lw=1.5)
    omega_sq = eigenvalues[k]  # 正比于 ω²
    ax.set_title(f"模 {k+1}: λ={omega_sq:.3f}, ω*√(m/k)={np.sqrt(omega_sq):.3f}")
    ax.set_xlabel("粒子编号"); ax.set_ylabel("位移")
    ax.grid(alpha=0.3)

fig.suptitle("一维耦合谐振子的简正模", fontsize=14)
plt.tight_layout()
plt.show()
```

### 1.3 稀疏矩阵与迭代求解

⚠️ **关键认知**：物理仿真中出现的矩阵几乎都是**稀疏的**。一个 $N \times N$ 的三维 Poisson 矩阵只有 $7N$ 个非零元素，而非 $N^2$ 个。永远不要用稠密格式存储或求解这些矩阵。

#### 共轭梯度法 (Conjugate Gradient)

用于求解 $\mathbf{A}\mathbf{x} = \mathbf{b}$，其中 $\mathbf{A}$ 是对称正定矩阵。

📐 **算法步骤**：

```
1: r₀ = b - A·x₀,  p₀ = r₀
2: for k = 0, 1, 2, ...:
3:     α_k = (r_k · r_k) / (p_k · A·p_k)
4:     x_{k+1} = x_k + α_k · p_k
5:     r_{k+1} = r_k - α_k · A·p_k
6:     if ||r_{k+1}|| < tolerance: break
7:     β_k = (r_{k+1} · r_{k+1}) / (r_k · r_k)
8:     p_{k+1} = r_{k+1} + β_k · p_k
```

💡 **收敛速度**：迭代次数 $\propto \sqrt{\kappa}$，其中 $\kappa = \lambda_{\max}/\lambda_{\min}$ 是条件数。对于 Poisson 方程，$\kappa \propto N^{2/d}$，因此 CG 在 $N$ 个迭代内收敛（$d$ 为空间维度）。

### 1.4 SVD 分解及其在仿真中的应用

📘 **奇异值分解 (SVD)**：

$$\mathbf{A} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^T$$

其中 $\mathbf{U}, \mathbf{V}$ 是正交矩阵，$\boldsymbol{\Sigma} = \operatorname{diag}(\sigma_1, \sigma_2, \dots)$ 包含奇异值 $\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$。

**在仿真中的三大应用**：

| 应用 | 原理 | 例子 |
|:---|:---|:---|
| 数据降噪 | 丢弃小奇异值对应的分量 | 从含噪实验数据提取物理信号 |
| 主成分分析 (PCA) | 最大方差方向 = 最大奇异值方向 | 多体系统集体运动模式分解 |
| 病态矩阵正则化 | 截断 SVD 去除接近 0 的奇异值 | 求解反问题时稳定数值结果 |

🖥️ **PCA 示例：多粒子轨迹的运动模式分解**：

```python
# 依赖: numpy
import numpy as np

# 模拟 N 个粒子在 2D 平面上的轨迹（加噪声）
np.random.seed(42)
n_particles, n_frames = 20, 100
# 真实信号：整体平动 + 椭圆振动
t = np.linspace(0, 4*np.pi, n_frames)
true_traj = np.zeros((n_frames, 2*n_particles))
for i in range(n_particles):
    true_traj[:, 2*i]   = np.cos(t + i*0.3)      # x 坐标
    true_traj[:, 2*i+1] = 0.5 * np.sin(t + i*0.3)  # y 坐标
# 加噪声
noisy_traj = true_traj + 0.1 * np.random.randn(n_frames, 2*n_particles)

# SVD 分解
U, S, Vt = np.linalg.svd(noisy_traj - noisy_traj.mean(axis=0), full_matrices=False)

print("奇异值分布:")
for k, sigma in enumerate(S[:8]):
    energy = sigma**2 / np.sum(S**2) * 100
    print(f"  σ{k+1} = {sigma:.2f}  (能量占比 {energy:.1f}%)")
print("  → 只有前 2 个模式显著，对应整体的 x-y 平面椭圆运动")
```

## 第二章 数值微积分

### 2.1 数值微分

#### 2.1.1 从泰勒展开推导差分公式

📐 **推导三点中心差分**：

设在 $x$ 处展开 $f(x+h)$ 和 $f(x-h)$：

$$
\begin{aligned}
f(x+h) &= f(x) + hf'(x) + \frac{h^2}{2}f''(x) + \frac{h^3}{6}f'''(x) + \mathcal{O}(h^4) \\
f(x-h) &= f(x) - hf'(x) + \frac{h^2}{2}f''(x) - \frac{h^3}{6}f'''(x) + \mathcal{O}(h^4)
\end{aligned}
$$

两式相减：
$$f(x+h) - f(x-h) = 2hf'(x) + \frac{h^3}{3}f'''(x) + \mathcal{O}(h^5)$$

因此：
$$f'(x) = \frac{f(x+h) - f(x-h)}{2h} - \frac{h^2}{6}f'''(x) + \mathcal{O}(h^4)$$

**结论：三点中心差分的截断误差为 $\mathcal{O}(h^2)$**——比前向/后向差分的 $\mathcal{O}(h)$ 好一个量级。

📘 **常用差分格式汇总**：

| 名称 | 公式 | 精度 | 用途 |
|:---|:---|:---|:---|
| 前向差分 | $f'(x) \approx \frac{f(x+h)-f(x)}{h}$ | $\mathcal{O}(h)$ | 时间推进（显式） |
| 后向差分 | $f'(x) \approx \frac{f(x)-f(x-h)}{h}$ | $\mathcal{O}(h)$ | 时间推进（隐式） |
| 中心差分 | $f'(x) \approx \frac{f(x+h)-f(x-h)}{2h}$ | $\mathcal{O}(h^2)$ | 空间导数 |
| 二阶中心差分 | $f''(x) \approx \frac{f(x+h)-2f(x)+f(x-h)}{h^2}$ | $\mathcal{O}(h^2)$ | Laplace 算子 |

⚠️ **舍入误差 vs 截断误差**：当 $h$ 太小时，差分公式中的减法会放大浮点舍入误差。最佳 $h$ 大约在 $10^{-5} \sim 10^{-7}$（双精度）。

🖥️ **验证差分精度**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

def central_diff(f, x, h):
    """三点中心差分"""
    return (f(x + h) - f(x - h)) / (2 * h)

def second_diff(f, x, h):
    """二阶中心差分"""
    return (f(x + h) - 2*f(x) + f(x - h)) / h**2

# 以 sin(x) 为例
x0 = np.pi / 3
exact_deriv = np.cos(x0)       # 一阶导精确值 = 0.5
exact_second = -np.sin(x0)     # 二阶导精确值 = -0.8660...

# 不同 h 下的误差
h_values = np.logspace(-1, -15, 50, base=10)
errors_deriv = []
errors_second = []

for h in h_values:
    err1 = abs(central_diff(np.sin, x0, h) - exact_deriv)
    err2 = abs(second_diff(np.sin, x0, h) - exact_second)
    errors_deriv.append(err1)
    errors_second.append(err2)

plt.figure(figsize=(8, 5))
plt.loglog(h_values, errors_deriv, 'b.-', label="一阶导误差")
plt.loglog(h_values, errors_second, 'r.-', label="二阶导误差")
plt.loglog(h_values, h_values**2 * 0.1, 'k--', label=r"$\propto h^2$ (理论)")
plt.xlabel("步长 h"); plt.ylabel("绝对误差")
plt.title("数值微分的精度检验：截断误差 vs 舍入误差")
plt.legend(); plt.grid(True, alpha=0.3)
plt.show()
# 观察：h 过小时误差反而增大（舍入误差主导）
# 最佳 h 大约在 1e-5 附近
```

### 2.2 数值积分

#### 2.2.1 Newton-Cotes 公式族

📐 **从插值到求积**：

数值积分的基本思路是用插值多项式代替被积函数，然后对多项式精确积分。

**梯形法则**（线性插值）：
$$\int_a^b f(x)dx \approx \frac{h}{2}[f(a) + f(b)], \quad 误差 \propto h^3 f''(\xi)$$

**Simpson 法则**（二次插值）：
$$\int_a^b f(x)dx \approx \frac{h}{3}[f(a) + 4f(\frac{a+b}{2}) + f(b)], \quad 误差 \propto h^5 f^{(4)}(\xi)$$

复合 Simpson 法则（$N$ 为偶数个区间）：
$$\int_a^b f(x)dx \approx \frac{h}{3}\left[f(x_0) + 4\sum_{i\ \text{odd}} f(x_i) + 2\sum_{i\ \text{even}} f(x_i) + f(x_N)\right]$$

#### 2.2.2 Gauss 求积：最优的数值积分

📘 **核心思想**：放弃等距节点，选择正交多项式的根作为求积节点，使 $N$ 个节点能达到 $2N-1$ 次代数精度。

**Gauss-Legendre 求积**（区间 $[-1,1]$）：
$$\int_{-1}^1 f(x)dx \approx \sum_{i=1}^n w_i f(x_i)$$

其中 $x_i$ 是 Legendre 多项式 $P_n(x)$ 的根，$w_i = \frac{2}{(1-x_i^2)[P_n'(x_i)]^2}$。

💡 **选择指南**：
- 光滑周期函数 → 直接用复合梯形法则（误差指数衰减！）
- 光滑非周期函数 → 复合 Simpson
- 包含奇异点（如 $1/\sqrt{x}$）→ Gauss 求积
- 高维积分 → [[#5.1 蒙特卡洛积分]]

🖥️ **各方法对比**：

```python
# 依赖: numpy, scipy
import numpy as np
from scipy import integrate

# 被积函数：一个有尖锐峰值的函数
def f(x):
    return np.exp(-50 * (x - 0.3)**2) + 0.1 * np.sin(20 * x)

# 精确值（用高精度 scipy 计算）
exact, _ = integrate.quad(f, 0, 1)

# Gauss-Legendre (5 阶)
from numpy.polynomial.legendre import leggauss
nodes, weights = leggauss(5)
# 映射 [-1,1] → [0,1]
nodes_mapped = (nodes + 1) / 2
weights_mapped = weights / 2
gauss_result = np.sum(weights_mapped * f(nodes_mapped))

# 复合 Simpson (5 点 = 相同函数调用次数)
x_simpson = np.linspace(0, 1, 5)
h = x_simpson[1] - x_simpson[0]
simpson_result = h/3 * (f(x_simpson[0]) + 
                        4*np.sum(f(x_simpson[1:-1:2])) + 
                        2*np.sum(f(x_simpson[2:-1:2])) + 
                        f(x_simpson[-1]))

print(f"精确值:       {exact:.8f}")
print(f"Gauss (5点):  {gauss_result:.8f}  误差: {abs(gauss_result-exact):.2e}")
print(f"Simpson (5点):{simpson_result:.8f}  误差: {abs(simpson_result-exact):.2e}")
# Gauss 求积在相同节点数下通常优于 Newton-Cotes
```

## 第三章 常微分方程的数值求解

### 3.1 初值问题的一般形式

一阶 ODE 系统：
$$\frac{d\mathbf{y}}{dt} = \mathbf{f}(t, \mathbf{y}), \quad \mathbf{y}(t_0) = \mathbf{y}_0$$

> 任何高阶 ODE 都可以化为一阶系统。例如 $\ddot{x} + \omega^2 x = 0$，设 $\mathbf{y} = [x, \dot{x}]^T$，则 $\dot{\mathbf{y}} = [y_2, -\omega^2 y_1]^T$。

### 3.2 Euler 方法

#### 3.2.1 显式 Euler（前向 Euler）

📐 **推导**：将导数用前向差分近似

$$\frac{\mathbf{y}_{n+1} - \mathbf{y}_n}{\Delta t} \approx \mathbf{f}(t_n, \mathbf{y}_n)$$

整理得：
$$\mathbf{y}_{n+1} = \mathbf{y}_n + \Delta t \cdot \mathbf{f}(t_n, \mathbf{y}_n)$$

**局部截断误差**：$\mathcal{O}(\Delta t^2)$  
**全局误差**：$\mathcal{O}(\Delta t)$  
**稳定性**：条件稳定，稳定域为复平面上以 $(-1, 0)$ 为圆心、半径为 $1$ 的圆内部。

⚠️ **Euler 法的适用场景**：仅用于教学目的或极小步长的简单系统。**永远不要用它求解实际物理问题。**

#### 3.2.2 隐式 Euler（后向 Euler）

$$\mathbf{y}_{n+1} = \mathbf{y}_n + \Delta t \cdot \mathbf{f}(t_{n+1}, \mathbf{y}_{n+1})$$

- 每步需要解非线性方程（通常用 Newton 迭代）
- **无条件稳定**（A-稳定），适合[[#3.6 刚性方程与隐式方法|刚性方程]]
- 代价：每步计算量大

📐 **从显式到隐式的动机**：考虑测试方程 $\dot{y} = -\lambda y$（$\lambda > 0$，真实解衰减）。

- 显式 Euler：$y_{n+1} = (1 - \lambda\Delta t) y_n$，要求 $|1-\lambda\Delta t| < 1$ → $\Delta t < 2/\lambda$
- 隐式 Euler：$y_{n+1} = \frac{1}{1+\lambda\Delta t} y_n$，恒有 $< 1$ → 无条件稳定

### 3.3 Runge-Kutta 方法

#### 3.3.1 RK4：经典的四阶方法

📐 **Butcher 表（经典 RK4）**：

$$\begin{array}{c|cccc}
0 & & & & \\
\frac{1}{2} & \frac{1}{2} & & & \\
\frac{1}{2} & 0 & \frac{1}{2} & & \\
1 & 0 & 0 & 1 & \\
\hline
& \frac{1}{6} & \frac{1}{3} & \frac{1}{3} & \frac{1}{6}
\end{array}$$

对应的算法：
$$\begin{aligned}
\mathbf{k}_1 &= \Delta t \cdot \mathbf{f}(t_n, \mathbf{y}_n) \\
\mathbf{k}_2 &= \Delta t \cdot \mathbf{f}(t_n + \tfrac{\Delta t}{2}, \mathbf{y}_n + \tfrac{\mathbf{k}_1}{2}) \\
\mathbf{k}_3 &= \Delta t \cdot \mathbf{f}(t_n + \tfrac{\Delta t}{2}, \mathbf{y}_n + \tfrac{\mathbf{k}_2}{2}) \\
\mathbf{k}_4 &= \Delta t \cdot \mathbf{f}(t_n + \Delta t, \mathbf{y}_n + \mathbf{k}_3) \\
\mathbf{y}_{n+1} &= \mathbf{y}_n + \frac{1}{6}(\mathbf{k}_1 + 2\mathbf{k}_2 + 2\mathbf{k}_3 + \mathbf{k}_4)
\end{aligned}$$

**局部截断误差**：$\mathcal{O}(\Delta t^5)$  
**全局误差**：$\mathcal{O}(\Delta t^4)$  

💡 **RK4 是物理仿真中最常用的单步显式方法**，精度和计算量之间达到了极好的平衡。

#### 3.3.2 自适应步长 Runge-Kutta-Fehlberg (RK45)

📘 **核心思想**：用 4 阶和 5 阶两个 RK 方法的差值估计局部误差，自动调节步长。

误差估计：
$$\mathbf{e}_{n+1} = \mathbf{y}_{n+1}^{(5)} - \mathbf{y}_{n+1}^{(4)}$$

步长调整：
$$\Delta t_{\text{new}} = \Delta t_{\text{old}} \cdot \min\left(\alpha_{\max}, \max\left(\alpha_{\min}, \beta \left(\frac{\text{tol}}{\|\mathbf{e}\|}\right)^{1/5}\right)\right)$$

> $\text{tol} = \text{atol} + \text{rtol} \cdot \max(\|\mathbf{y}_n\|, \|\mathbf{y}_{n+1}\|)$

### 3.4 经典例题：阻尼驱动谐振子

#### 3.4.1 物理模型

运动方程：
$$\ddot{x} + 2\beta\dot{x} + \omega_0^2 x = \frac{F_0}{m}\cos(\omega_d t)$$

其中：
- $\omega_0 = \sqrt{k/m}$：固有频率
- $\beta = b/(2m)$：阻尼系数
- $\omega_d$：驱动频率
- $F_0$：驱动力幅值

化为一阶系统：
$$\begin{cases}
\dot{x} = v \\
\dot{v} = -2\beta v - \omega_0^2 x + \frac{F_0}{m}\cos(\omega_d t)
\end{cases}$$

#### 3.4.2 解析解（欠阻尼 $\beta < \omega_0$）

齐次解（瞬态）：
$$x_h(t) = e^{-\beta t}\left[A\cos(\omega_1 t) + B\sin(\omega_1 t)\right], \quad \omega_1 = \sqrt{\omega_0^2 - \beta^2}$$

稳态解（驱动频率）：
$$x_p(t) = \frac{F_0/m}{\sqrt{(\omega_0^2 - \omega_d^2)^2 + (2\beta\omega_d)^2}} \cos(\omega_d t - \phi)$$

$$\phi = \arctan\left(\frac{2\beta\omega_d}{\omega_0^2 - \omega_d^2}\right)$$

📐 **共振条件**：振幅最大时 $\omega_d = \sqrt{\omega_0^2 - 2\beta^2}$（振幅共振），能量吸收最大时 $\omega_d = \omega_0$（速度共振）。

#### 3.4.3 数值求解与可视化

🖥️ **完整代码**：

```python
# 依赖: numpy, scipy, matplotlib
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# ========== 参数设置 ==========
omega0 = 2.0       # 固有频率
beta = 0.2         # 阻尼系数
F0_over_m = 0.5    # F0/m，驱动力/质量
omega_d = 2.5      # 驱动频率

# ========== 运动方程 ==========
def driven_oscillator(t, y):
    """y = [x, v]"""
    x, v = y
    dxdt = v
    dvdt = -2 * beta * v - omega0**2 * x + F0_over_m * np.cos(omega_d * t)
    return [dxdt, dvdt]

# ========== 数值求解 ==========
t_span = (0, 50)
y0 = [1.0, 0.0]  # 初始位移=1, 初速=0

sol = solve_ivp(
    driven_oscillator, t_span, y0,
    method='RK45',
    rtol=1e-8, atol=1e-10,
    max_step=0.05,
    dense_output=True
)

# 均匀时间网格用于绘图
t_fine = np.linspace(0, 50, 2000)
y_fine = sol.sol(t_fine)
x, v = y_fine[0], y_fine[1]

# ========== 解析稳态解 ==========
denom = np.sqrt((omega0**2 - omega_d**2)**2 + (2*beta*omega_d)**2)
A = F0_over_m / denom                  # 振幅
phi = np.arctan2(2*beta*omega_d, omega0**2 - omega_d**2)  # 相位
x_steady = A * np.cos(omega_d * t_fine - phi)

# ========== 可视化 ==========
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 8))

# 位置时程
ax1.plot(t_fine, x, 'b-', lw=1, alpha=0.8, label='数值解')
ax1.plot(t_fine, x_steady, 'r--', lw=1.5, label='解析稳态解')
ax1.set_ylabel('位移 x(t)'); ax1.set_title('阻尼驱动谐振子')
ax1.legend(); ax1.grid(alpha=0.3)

# 相空间
ax2.plot(x, v, 'b-', lw=0.5, alpha=0.7)
ax2.set_xlabel('位移 x'); ax2.set_ylabel('速度 v')
ax2.set_title('相空间轨迹 (吸引到极限环)')
ax2.grid(alpha=0.3)

plt.tight_layout(); plt.show()

print(f"稳态振幅 (数值): {np.max(x[t_fine > 40]):.4f}")
print(f"稳态振幅 (解析): {A:.4f}")
print(f"相对误差: {abs(np.max(x[t_fine > 40]) - A)/A*100:.3f}%")
```

### 3.5 经典例题：行星运动（Kepler 问题）

#### 3.5.1 物理模型与守恒量

万有引力下的二体运动，在质心系中简化为等效单体问题：

$$\ddot{\mathbf{r}} = -\frac{GM}{r^3}\mathbf{r}$$

其中 $M = m_1 + m_2$，$r = |\mathbf{r}|$。

此系统有两个守恒量（用于检验数值精度）：

**能量守恒**：
$$E = \frac{1}{2}m\dot{\mathbf{r}}^2 - \frac{GMm}{r} = \text{const}$$

**角动量守恒**：
$$\mathbf{L} = m\,\mathbf{r} \times \dot{\mathbf{r}} = \text{const}$$

🔥 **关键问题**：显式 RK 方法会系统性地漂移能量。长时间积分必须用 **辛积分器**。

📐 **辛积分器（Symplectic Integrator）**：保持相空间体积守恒，能量在精确值附近振荡而不漂移。

二阶 Störmer-Verlet（速度形式）：
$$\begin{aligned}
\mathbf{v}_{n+1/2} &= \mathbf{v}_n + \frac{\Delta t}{2}\,\mathbf{a}(\mathbf{r}_n) \\
\mathbf{r}_{n+1} &= \mathbf{r}_n + \Delta t \cdot \mathbf{v}_{n+1/2} \\
\mathbf{v}_{n+1} &= \mathbf{v}_{n+1/2} + \frac{\Delta t}{2}\,\mathbf{a}(\mathbf{r}_{n+1})
\end{aligned}$$

#### 3.5.2 对比：RK45 vs 辛积分器

🖥️ **完整代码**：

```python
# 依赖: numpy, scipy, matplotlib
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

# ========== 参数 ==========
GM = 4 * np.pi**2  # 使周期 = 1 年 (a=1 AU)
T_orbit = 1.0
dt = T_orbit / 200  # 每轨道 200 步
n_orbits = 8        # 积分 8 个轨道

# 初始条件：近日点
r0 = np.array([0.5, 0.0])          # 偏心率 e=0.5
v0 = np.array([0.0, np.sqrt(GM * (1+0.5)/(0.5*(1-0.5)))])  # Kepler 速度

# ========== 方法 1: RK45 (scipy) ==========
def kepler(t, y):
    r = y[:2]; v = y[2:]
    r_norm = np.linalg.norm(r)
    return np.concatenate([v, -GM * r / r_norm**3])

sol = solve_ivp(kepler, [0, n_orbits], np.concatenate([r0, v0]),
                method='RK45', max_step=0.01, rtol=1e-8, atol=1e-10)
r_rk = sol.y[:2]

# ========== 方法 2: 二阶辛积分器 (Stormer-Verlet) ==========
n_steps = int(n_orbits / dt)
r_sym = np.zeros((2, n_steps+1))
v_sym = np.zeros((2, n_steps+1))
r_sym[:, 0] = r0
v_sym[:, 0] = v0
energy_sym = np.zeros(n_steps+1)

def acceleration(r):
    return -GM * r / np.linalg.norm(r)**3

for n in range(n_steps):
    a_n = acceleration(r_sym[:, n])
    v_half = v_sym[:, n] + 0.5 * dt * a_n
    r_sym[:, n+1] = r_sym[:, n] + dt * v_half
    a_next = acceleration(r_sym[:, n+1])
    v_sym[:, n+1] = v_half + 0.5 * dt * a_next
    
    # 记录能量
    ke = 0.5 * np.sum(v_sym[:, n+1]**2)
    pe = -GM / np.linalg.norm(r_sym[:, n+1])
    energy_sym[n+1] = ke + pe

# RK45 能量
r_rk_mag = np.sqrt(r_rk[0]**2 + r_rk[1]**2)
v_rk_mag = np.sqrt(sol.y[2]**2 + sol.y[3]**2)
energy_rk = 0.5 * v_rk_mag**2 - GM / r_rk_mag

# ========== 可视化 ==========
fig, axes = plt.subplots(1, 2, figsize=(14, 6))

# 轨道
axes[0].plot(r_sym[0], r_sym[1], 'b-', lw=0.8, label=f'辛积分器 ({n_orbits} 圈)')
axes[0].plot(r_rk[0], r_rk[1], 'r--', lw=0.8, label='RK45')
axes[0].plot(0, 0, 'ko', ms=8, label='太阳')
axes[0].set_xlabel('x (AU)'); axes[0].set_ylabel('y (AU)')
axes[0].set_title('轨道对比')
axes[0].axis('equal'); axes[0].legend(); axes[0].grid(alpha=0.3)

# 能量漂移
E0 = energy_sym[0]
axes[1].semilogy(np.arange(n_steps+1)*dt, np.abs(energy_sym - E0)/np.abs(E0),
                 'b-', lw=1.5, label='辛积分器')
axes[1].semilogy(sol.t, np.abs(energy_rk - E0)/np.abs(E0),
                 'r-', lw=1.5, label='RK45')
axes[1].set_xlabel('时间 (年)'); axes[1].set_ylabel('|ΔE/E₀|')
axes[1].set_title('能量相对误差 (对数坐标)')
axes[1].legend(); axes[1].grid(True, alpha=0.3)

plt.tight_layout(); plt.show()
print("注意：辛积分器的能量误差在零附近振荡，而 RK45 的能量误差单调增长。")
print("这就是辛方法的优势——长时间守恒性质。")
```

🔗 辛积分器思想在分子动力学中有决定性应用，见 [[#5.3 分子动力学基础|分子动力学章节]]。

### 3.6 刚性方程与隐式方法

#### 3.6.1 什么是"刚性"

📘 **定义**：一个 ODE 系统被称为**刚性的**（stiff），如果显式方法必须取极小的步长才能保持稳定，甚至当解本身变化很慢时也是如此。

数学判据：系统的 Jacobian 矩阵 $\partial\mathbf{f}/\partial\mathbf{y}$ 的特征值实部跨度极大，即：

$$\frac{\max|\operatorname{Re}(\lambda_i)|}{\min|\operatorname{Re}(\lambda_i)|} \gg 1$$

**物理来源**：系统中同时存在快过程和慢过程。
- 化学动力学：快反应（ns）和慢反应（s）
- 电路模拟：寄生电容产生的极快衰减模式
- 扩散方程空间离散后：高频模式极快衰减

⚠️ **判断刚性的实用方法**：如果用 RK45 时步长自动降到极小（如 $10^{-8}$），而解本身变化平稳，那就是刚性问题。应该换用 `method='BDF'` 或 `method='Radau'`。

🖥️ **刚性问题的求解**：

```python
# 依赖: numpy, scipy
import numpy as np
from scipy.integrate import solve_ivp
import time

# Robertson 化学动力学问题（经典刚性测试）
def robertson(t, y):
    y1, y2, y3 = y
    return [
        -0.04*y1 + 1e4*y2*y3,
         0.04*y1 - 1e4*y2*y3 - 3e7*y2**2,
                             3e7*y2**2
    ]

y0 = [1.0, 0.0, 0.0]
t_span = (0, 1e5)

# 显式 RK45 → 会极慢
t0 = time.time()
sol_explicit = solve_ivp(robertson, t_span, y0, method='RK45', rtol=1e-4, atol=1e-8)
print(f"RK45: 步数={len(sol_explicit.t)}, 耗时={time.time()-t0:.1f}s")
print(f"  可能失败或步数极多" if len(sol_explicit.t) > 10000 else f"  步数={len(sol_explicit.t)}")

# 隐式 BDF（专为刚性问题设计）
t0 = time.time()
sol_stiff = solve_ivp(robertson, t_span, y0, method='BDF', rtol=1e-4, atol=1e-8)
print(f"BDF:  步数={len(sol_stiff.t)}, 耗时={time.time()-t0:.1f}s")
print("→ BDF 用极少步数完成，RK45 几乎不可能完成。这就是隐式方法的威力。")
```

## 第四章 偏微分方程的数值方法

### 4.1 偏微分方程的分类

📘 **二阶线性 PDE 的标准形式**：

$$a\frac{\partial^2 u}{\partial x^2} + b\frac{\partial^2 u}{\partial x\partial y} + c\frac{\partial^2 u}{\partial y^2} + \text{低阶项} = 0$$

| 判别式 | 类型 | 物理方程 | 定解条件 |
|:---|:---|:---|:---|
| $b^2 - 4ac < 0$ | **椭圆型** | Laplace, Poisson | 边值问题（边界上全部给定） |
| $b^2 - 4ac = 0$ | **抛物型** | 热传导, 扩散 | 初边值问题 |
| $b^2 - 4ac > 0$ | **双曲型** | 波动方程 | 初值问题 |

### 4.2 有限差分法 (FDM)

#### 4.2.1 离散化的标准流程

以 **Poisson 方程**为例：
$$-\nabla^2 u = f \quad \text{在 } \Omega \subset \mathbb{R}^d \text{ 内}, \quad u|_{\partial\Omega} = g$$

**步骤 1**：网格剖分。在 1D 中：$x_j = a + j\Delta x$，$j = 0,1,\dots,N$，其中 $\Delta x = (b-a)/N$。

**步骤 2**：差分替换。用二阶中心差分替代 $\partial^2/\partial x^2$：
$$\frac{u_{j+1} - 2u_j + u_{j-1}}{\Delta x^2} = -f(x_j)$$

**步骤 3**：组装线性系统。结合边界条件：
$$\frac{1}{\Delta x^2}\underbrace{\begin{bmatrix}
2 & -1 &  &  & \\
-1 & 2 & -1 &  & \\
 & \ddots & \ddots & \ddots & \\
 &  & -1 & 2
\end{bmatrix}}_{= \mathbf{A}}
\underbrace{\begin{bmatrix} u_1 \\ u_2 \\ \vdots \\ u_{N-1} \end{bmatrix}}_{= \mathbf{u}} =
\underbrace{\begin{bmatrix} f_1 + u_0/\Delta x^2 \\ f_2 \\ \vdots \\ f_{N-1} + u_N/\Delta x^2 \end{bmatrix}}_{= \mathbf{b}}$$

**步骤 4**：求解 $\mathbf{A}\mathbf{u} = \mathbf{b}$。

#### 4.2.2 稳定性和 CFL 条件

📘 **von Neumann 稳定性分析**：

对于时间依赖的 PDE，将数值格式作用于单个 Fourier 模 $u_j^n = \xi^n e^{ikj\Delta x}$，分析放大因子 $\xi(k)$。稳定性要求：

$$|\xi(k)| \leq 1 \quad \text{对所有 } k$$

📐 **以热传导方程的 FTCS 格式为例**：

$$\frac{u_j^{n+1} - u_j^n}{\Delta t} = \alpha \frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{\Delta x^2}$$

代入 $u_j^n = \xi^n e^{ikj\Delta x}$：
$$\xi = 1 - \frac{4\alpha\Delta t}{\Delta x^2}\sin^2\left(\frac{k\Delta x}{2}\right)$$

稳定性要求 $|\xi| \leq 1$，得：

$$\boxed{\frac{\alpha\Delta t}{\Delta x^2} \leq \frac{1}{2}}$$

💡 **这是数值仿真中最著名的稳定性条件**——CFL 条件在扩散问题中的具体形式。

⚠️ **为什么 CFL 条件不是"建议"而是"必须"**：违反 CFL 条件意味着你的数值格式在物理上是**不因果的**——信息在一个时间步内传播了超过一个网格间距。解会指数发散，不会"差一点就对了"，而是直接变成 NaN。

#### 4.2.3 经典例题：一维热传导（三种格式对比）

🖥️ **完整代码**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

# ========== 参数 ==========
L = 1.0; alpha = 1.0
Nx = 50; dx = L / (Nx - 1)
x = np.linspace(0, L, Nx)

# CFL 条件决定的 dt
dt_cfl = dx**2 / (2 * alpha)
dt = dt_cfl * 0.4  # 取 40% 保证 FTCS 稳定

# 初始条件：方波
u0 = np.where((x > 0.3) & (x < 0.7), 1.0, 0.0)

# ========== FTCS (显式) ==========
def ftcs(u, dt, dx, alpha, Nt):
    u_hist = [u.copy()]
    r = alpha * dt / dx**2
    for _ in range(Nt):
        u_new = u.copy()
        u_new[1:-1] = u[1:-1] + r * (u[2:] - 2*u[1:-1] + u[:-2])
        u_new[0] = u_new[-1] = 0.0
        u = u_new
        u_hist.append(u.copy())
    return u, u_hist

# ========== BTCS (隐式) ==========
def btcs(u, dt, dx, alpha, Nt):
    N = len(u) - 2
    r = alpha * dt / dx**2
    # 三对角矩阵
    main_diag = (1 + 2*r) * np.ones(N)
    off_diag = -r * np.ones(N-1)
    u_hist = [u.copy()]
    for _ in range(Nt):
        rhs = u[1:-1].copy()
        rhs[0] += r * u[0]
        rhs[-1] += r * u[-1]
        # Thomas 算法（三对角求解）
        u_new = np.zeros_like(u)
        u_new[0] = u[0]; u_new[-1] = u[-1]
        u_new[1:-1] = thomas_solver(main_diag.copy(), off_diag.copy(), rhs)
        u = u_new
        u_hist.append(u.copy())
    return u, u_hist

def thomas_solver(a, b, d):
    """三对角矩阵求解器 [b, a, b]（Thomas 算法）"""
    n = len(d)
    c_prime = np.zeros(n-1)
    d_prime = np.zeros(n)
    c_prime[0] = b[0] / a[0]
    d_prime[0] = d[0] / a[0]
    for i in range(1, n-1):
        denom = a[i] - b[i-1] * c_prime[i-1]
        c_prime[i] = b[i] / denom
    for i in range(1, n):
        denom = a[i] - b[i-1] * c_prime[i-1]
        d_prime[i] = (d[i] - b[i-1] * d_prime[i-1]) / denom
    x = np.zeros(n)
    x[-1] = d_prime[-1]
    for i in range(n-2, -1, -1):
        x[i] = d_prime[i] - c_prime[i] * x[i+1]
    return x

# ========== Crank-Nicolson ==========
def crank_nicolson(u, dt, dx, alpha, Nt):
    N = len(u) - 2
    r = alpha * dt / (2 * dx**2)
    main_diag = (1 + 2*r) * np.ones(N)
    off_diag = -r * np.ones(N-1)
    u_hist = [u.copy()]
    for _ in range(Nt):
        rhs = (1 - 2*r) * u[1:-1] + r * (u[2:] + u[:-2])
        rhs[0] += r * u[0]; rhs[-1] += r * u[-1]
        u_new = np.zeros_like(u)
        u_new[0] = u[0]; u_new[-1] = u[-1]
        u_new[1:-1] = thomas_solver(main_diag.copy(), off_diag.copy(), rhs)
        u = u_new
        u_hist.append(u.copy())
    return u, u_hist

# ========== 运行 & 画图 ==========
T_final = 0.05
Nt = int(T_final / dt)

u_ftcs, hist_ftcs = ftcs(u0.copy(), dt, dx, alpha, Nt)
u_btcs, hist_btcs = btcs(u0.copy(), dt, dx, alpha, Nt)
u_cn, hist_cn = crank_nicolson(u0.copy(), dt, dx, alpha, Nt)

fig, axes = plt.subplots(1, 3, figsize=(15, 4))
snapshots = [0, Nt//4, Nt//2, Nt]
colors = plt.cm.viridis(np.linspace(0, 1, len(snapshots)))

for ax, method, hist in zip(axes, ['FTCS (显式)', 'BTCS (隐式)', 'Crank-Nicolson'],
                              [hist_ftcs, hist_btcs, hist_cn]):
    for k, (s, c) in enumerate(zip(snapshots, colors)):
        ax.plot(x, hist[s], color=c, lw=1.5, label=f't={s*dt:.3f}')
    ax.set_xlabel('x'); ax.set_ylabel('u(x,t)')
    ax.set_title(method); ax.legend(fontsize=7); ax.grid(alpha=0.3)

fig.suptitle(f'热传导方程：三种格式对比  (CFL={alpha*dt/dx**2:.3f})', fontsize=14)
plt.tight_layout(); plt.show()
```

### 4.3 有限元法 (FEM) 入门

#### 4.3.1 变分原理

📐 **Poisson 方程的弱形式**：

强形式：$-\nabla^2 u = f$ 在 $\Omega$，$u = 0$ 在 $\partial\Omega$

两边乘以测试函数 $v$（满足 $v|_{\partial\Omega} = 0$）并在 $\Omega$ 上积分：

$$-\int_\Omega v\nabla^2 u \,d\Omega = \int_\Omega vf \,d\Omega$$

分部积分（Green 恒等式）：

$$\int_\Omega \nabla v \cdot \nabla u \,d\Omega - \cancel{\int_{\partial\Omega} v\frac{\partial u}{\partial n}\,d\Gamma} = \int_\Omega vf \,d\Omega$$

> 边界项消失是因为 $v|_{\partial\Omega}=0$。

**弱形式**：求 $u \in H_0^1(\Omega)$，使得对所有 $v \in H_0^1(\Omega)$ 有：

$$\int_\Omega \nabla v \cdot \nabla u \,d\Omega = \int_\Omega vf \,d\Omega$$

写成内积符号：
$$a(u, v) = L(v)$$
其中 $a(u,v) = (\nabla u, \nabla v)$ 是双线性形式，$L(v) = (f, v)$ 是线性泛函。

#### 4.3.2 Galerkin 离散

将 $u$ 用有限维基函数逼近：$u_h = \sum_{j=1}^N U_j \phi_j(x)$

取测试函数为基函数 $v = \phi_i$，得：

$$\sum_{j=1}^N a(\phi_j, \phi_i) U_j = L(\phi_i), \quad i=1,\dots,N$$

写成矩阵形式：
$$\mathbf{A}\mathbf{U} = \mathbf{b}$$
其中 $A_{ij} = a(\phi_j, \phi_i) = \int_\Omega \nabla\phi_j \cdot \nabla\phi_i \,d\Omega$。

#### 4.3.3 FEniCS 示例

💡 **FEniCS** 是 Python 中最流行的 FEM 库，直接使用近似的数学语言编写。

🖥️ **FEniCS 求解 Poisson 方程**：

### 4.4 经典例题：波动方程的数值解

📐 **一维波动方程**：
$$\frac{\partial^2 u}{\partial t^2} = c^2\frac{\partial^2 u}{\partial x^2}$$

二阶中心差分离散：
$$\frac{u_j^{n+1} - 2u_j^n + u_j^{n-1}}{\Delta t^2} = c^2\frac{u_{j+1}^n - 2u_j^n + u_{j-1}^n}{\Delta x^2}$$

整理得显式公式：
$$u_j^{n+1} = 2u_j^n - u_j^{n-1} + \left(\frac{c\Delta t}{\Delta x}\right)^2(u_{j+1}^n - 2u_j^n + u_{j-1}^n)$$

**CFL 条件**：$c\Delta t / \Delta x \leq 1$

🖥️ **完整代码**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt
from matplotlib.animation import FuncAnimation

# ========== 参数 ==========
L = 1.0; c = 1.0
Nx = 200; dx = L / (Nx - 1)
x = np.linspace(0, L, Nx)

# CFL = 0.5
dt = 0.5 * dx / c
T = 2.0
Nt = int(T / dt)

# 初始条件：高斯波包
u = np.exp(-200 * (x - 0.3)**2)
u_old = np.exp(-200 * (x - 0.3 + c*dt)**2)  # 用平移模拟初始速度

# 时间推进（固定边界 u=0）
u_history = [u_old.copy()]

for n in range(Nt):
    u_new = np.zeros_like(u)
    u_new[0] = u_new[-1] = 0
    r = (c * dt / dx)**2
    u_new[1:-1] = (2*u[1:-1] - u_old[1:-1] + 
                    r*(u[2:] - 2*u[1:-1] + u[:-2]))
    u_old = u.copy()
    u = u_new.copy()
    
    if n % 5 == 0:
        u_history.append(u.copy())

# 可视化
fig, ax = plt.subplots(figsize=(10, 4))
for k, u_snap in enumerate(u_history):
    alpha = k / len(u_history)
    ax.plot(x, u_snap, color=plt.cm.plasma(alpha), lw=0.8,
            label=f't={k*5*dt:.2f}' if k % 6 == 0 else '')

ax.set_xlabel('x'); ax.set_ylabel('u(x,t)')
ax.set_title('波动方程：高斯波包的传播和反射')
ax.legend(fontsize=7, loc='upper right'); ax.grid(alpha=0.3)
ax.set_ylim(-1.1, 1.1)
plt.tight_layout(); plt.show()
```

### 5.1 蒙特卡洛积分

📘 **核心定理**（大数定律在求积中的应用）：

$$\int_\Omega f(\mathbf{x})\,d\mathbf{x} \approx \frac{|\Omega|}{N} \sum_{i=1}^N f(\mathbf{x}_i), \quad \mathbf{x}_i \sim \mathcal{U}(\Omega)$$

**误差估计**（中心极限定理）：
$$\sigma_{\text{MC}} = \frac{\sigma_f}{\sqrt{N}}, \quad \sigma_f^2 = \frac{1}{N-1}\sum_{i=1}^N (f_i - \bar{f})^2$$

🔥 **蒙特卡洛的优势**：误差 $\propto 1/\sqrt{N}$，**与积分空间的维度无关**。在 $d$ 维空间中，确定性求积的误差 $\propto N^{-k/d}$（$k$ 为方法阶数），在 $d$ 很大时远差于 Monte Carlo。

📐 **重要性采样**：

若 $f(\mathbf{x})$ 只在很小的区域显著非零，均匀采样效率极低。引入重要性分布 $p(\mathbf{x})$：

$$\int f(\mathbf{x})\,d\mathbf{x} = \int \frac{f(\mathbf{x})}{p(\mathbf{x})} p(\mathbf{x})\,d\mathbf{x} \approx \frac{1}{N}\sum_{i=1}^N \frac{f(\mathbf{x}_i)}{p(\mathbf{x}_i)}, \quad \mathbf{x}_i \sim p$$

**最优重要性分布**：$p^*(\mathbf{x}) \propto |f(\mathbf{x})|$（方差最小化）。

🖥️ **π 的 Monte Carlo 估计（附收敛性分析）**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

# ========== 多次实验观察收敛 ==========
N_max = 10_000_000
N_samples = np.logspace(2, 7, 30, base=10, dtype=int)

pi_estimates = []
pi_errors = []

for N in N_samples:
    x = np.random.uniform(-1, 1, N)

    pi_estimates.append(pi_est)

# ========== 理论误差线：1/√N ==========

ax.set_xlabel('采样点数 N'); ax.set_ylabel('|π_est - π|')
ax.set_title('Monte Carlo 积分收敛性验证\nπ 的估计')

### 5.2 Markov Chain Monte Carlo (MCMC)

#### 5.2.1 Metropolis-Hastings 算法

📘 **目标**：从已知 $\tilde{p}(\mathbf{x}) = Z p(\mathbf{x})$（$Z$ 未知）的分布中采样。

**算法**：
```
1: 初始化 x₀
2: for t = 0, 1, 2, ...:
3:     提议: x' ~ q(x' | x_t)    (通常为高斯扰动)
4:     接受率: α = min(1, p̃(x')/p̃(x_t) * q(x_t|x')/q(x'|x_t))
5:     以概率 α 接受: x_{t+1} = x'
6:     否则: x_{t+1} = x_t
```

📐 **Metropolis 形式的物理直觉**：

当提议分布对称时（$q(x'|x) = q(x|x')$），接受率简化为：
$$\alpha = \min\left(1, \frac{\tilde{p}(x')}{\tilde{p}(x)}\right) = \min\left(1, e^{-\beta[E(x') - E(x)]}\right)$$

这正是 Boltzmann 分布的细致平衡条件！Metropolis 算法本质上在模拟热平衡态的随机热涨落。

### 5.3 分子动力学基础

#### 5.3.1 物理模型

📐 **Newton 运动方程**（$N$ 个粒子的系统）：

$$m_i \ddot{\mathbf{r}}_i = \mathbf{F}_i = -\nabla_{\mathbf{r}_i} U(\mathbf{r}_1, \dots, \mathbf{r}_N)$$

**Lennard-Jones 势**（最常用的原子间势）：
$$U_{\text{LJ}}(r) = 4\varepsilon \left[\left(\frac{\sigma}{r}\right)^{12} - \left(\frac{\sigma}{r}\right)^6\right]$$

> $r^{-12}$ 项 = Pauli 排斥（电子云重叠），$r^{-6}$ 项 = van der Waals 吸引（诱导偶极）

对应力：
$$\mathbf{F}_{ij} = 24\varepsilon \left[2\left(\frac{\sigma}{r_{ij}}\right)^{12} - \left(\frac{\sigma}{r_{ij}}\right)^6\right] \frac{\mathbf{r}_j - \mathbf{r}_i}{r_{ij}^2}$$

#### 5.3.2 速度 Verlet 算法

📐 **Verlet 的辛结构**：Verlet 算法保持相空间的辛结构，这意味着：

1. **时间可逆**：反向积分回到起点
2. **体积守恒**（Liouville 定理）：相空间体积元不变
3. **能量长期稳定**：能量在精确值附近振荡而不漂移（与 RK45 的根本区别）

速度 Verlet 形式（最实用）：
$$\begin{aligned}
\mathbf{v}(t+\frac{\Delta t}{2}) &= \mathbf{v}(t) + \frac{\Delta t}{2m}\mathbf{F}(t) \\
\mathbf{r}(t+\Delta t) &= \mathbf{r}(t) + \Delta t \cdot \mathbf{v}(t+\frac{\Delta t}{2}) \\
\mathbf{F}(t+\Delta t) &= -\nabla U(\mathbf{r}(t+\Delta t)) \\
\mathbf{v}(t+\Delta t) &= \mathbf{v}(t+\frac{\Delta t}{2}) + \frac{\Delta t}{2m}\mathbf{F}(t+\Delta t)
\end{aligned}$$

💡 **为什么只用一次力计算就可以推进半步**：因为第一步只用旧力（已知），最后一步只用新力（刚算出），恰好两次力计算完成完整的一步。

🖥️ **Lennard-Jones 粒子系统**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

# ========== 参数 ==========
N = 64                        # 粒子数
L = 10.0                      # 盒子长度
dt = 0.002                    # 时间步长
n_steps = 5000                # 总步数
epsilon = 1.0; sigma = 1.0
m = 1.0

# ========== 初始化：面心立方晶格排列 ==========
n_per_side = int(np.ceil(N**(1/3)))
positions = []
spacing = L / n_per_side
count = 0
for i in range(n_per_side):
    for j in range(n_per_side):
        for k in range(n_per_side):
            if count >= N: break
            positions.append([(i+0.5)*spacing, (j+0.5)*spacing, (k+0.5)*spacing])
            count += 1
positions = np.array(positions[:N])
velocities = np.random.randn(N, 3) * 0.1

# ========== 力计算（最小镜像约定 + 截断） ==========
rcut = 2.5 * sigma
rcut2 = rcut**2

def compute_forces(positions):
    forces = np.zeros_like(positions)
    energy = 0.0
    for i in range(N):
        for j in range(i+1, N):
            dr = positions[j] - positions[i]
            # 最小镜像
            dr -= L * np.round(dr / L)
            r2 = np.sum(dr**2)
            if r2 < rcut2:
                r2i = 1.0 / r2
                r6i = r2i**3
                r12i = r6i**2
                f_mag = 24 * epsilon * (2*r12i - r6i) * r2i
                f = f_mag * dr
                forces[i] += f
                forces[j] -= f
                energy += 4 * epsilon * (r12i - r6i)
    return forces, energy

# ========== 速度 Verlet ==========
forces, pe = compute_forces(positions)
ke = 0.5 * m * np.sum(velocities**2)

energy_history = []

for step in range(n_steps):
    # 半步速度更新
    velocities += 0.5 * dt * forces / m
    # 位置更新（周期边界）
    positions = (positions + dt * velocities) % L
    # 新力
    forces, pe = compute_forces(positions)
    # 剩下半步速度
    velocities += 0.5 * dt * forces / m
    # 记录
    ke = 0.5 * m * np.sum(velocities**2)
    energy_history.append(ke + pe)

# ========== 能量漂移分析 ==========
E = np.array(energy_history)
E0 = E[0]
drift = np.abs(E - E0) / np.abs(E0)

plt.figure(figsize=(10, 5))
plt.plot(np.arange(n_steps)*dt, E, 'b-', lw=0.8, alpha=0.7)
plt.axhline(E0, color='r', ls='--', label=f'E₀ = {E0:.2f}')
plt.xlabel('时间'); plt.ylabel('总能量')
plt.title('速度 Verlet：长时间能量守恒')
plt.legend(); plt.grid(alpha=0.3)
plt.tight_layout(); plt.show()

print(f"能量相对漂移: {drift[-1]:.2e}")
print("速度 Verlet 保持能量在精确值附近振荡，无长期漂移。")
```

#### 5.3.3 恒温器：从 NVE 到 NVT 系综

⚠️ 上面的模拟是**微正则系综 (NVE)**——总能量守恒。但大多数物理场景需要在**正则系综 (NVT)** 下模拟，即温度恒定。

##### Berendsen 恒温器（最简单）

通过速度标度将系统"推向"目标温度：

$$v_i^{\text{new}} = v_i \cdot \lambda, \quad \lambda = \sqrt{1 + \frac{\Delta t}{\tau_T}\left(\frac{T_0}{T} - 1\right)}$$

其中 $T_0$ 是目标温度，$\tau_T$ 是耦合时间常数（越大越温和），$T$ 是瞬时温度（由能量均分 $k_B T = 2 \langle E_k \rangle / 3N$ 求得）。

💡 **适用场景**：快速平衡阶段，简单有效。但不能严格给出正则系综分布。

##### Nosé-Hoover 恒温器（严格正则系综）

引入一个额外的"热浴"自由度 $s$ 及其共轭动量 $p_s$，通过扩展 Lagrangian 严格实现 NVT：

$$\ddot{\mathbf{r}}_i = \frac{\mathbf{F}_i}{m_i} - \xi \dot{\mathbf{r}}_i, \quad \dot{\xi} = \frac{1}{Q}\left(\sum_i m_i \dot{\mathbf{r}}_i^2 - N_f k_B T_0\right)$$

其中 $\xi = p_s/Q$ 是热浴摩擦系数，$Q$ 是热浴"质量"（控制温度波动频率），$N_f$ 是自由度数。

🖥️ **Nosé-Hoover 实现**（接续上面的 LJ 系统）：

```python
# ========== Nosé-Hoover 恒温器参数 ==========
T0 = 1.0                     # 目标温度 (ε/kB 单位)
Q = 100.0                    # 热浴质量
xi = 0.0                     # 初始摩擦系数
Nf = 3 * N                   # 自由度数

T_history = []

# 重新初始化
# positions, velocities = 原始位置, 原始速度
# forces, pe = compute_forces(positions)

for step in range(n_steps):
    # 速度 Verlet 半步
    velocities += 0.5 * dt * forces / m
    
    # Nosé-Hoover: 半步摩擦更新
    ke = 0.5 * m * np.sum(velocities**2)
    T_inst = 2 * ke / (Nf * 1.0)  # kB=1
    xi += 0.5 * dt * (ke - 0.5 * Nf * T0) / Q
    scale = np.exp(-xi * dt)
    velocities *= scale
    
    # 位置更新
    positions = (positions + dt * velocities) % L
    
    # 新力
    forces, pe = compute_forces(positions)
    
    # 另一半速度更新
    velocities += 0.5 * dt * forces / m
    
    # Nosé-Hoover: 另一半摩擦更新
    ke = 0.5 * m * np.sum(velocities**2)
    scale = np.exp(-xi * 0.5 * dt)
    velocities *= scale
    xi += 0.5 * dt * (ke - 0.5 * Nf * T0) / Q
    
    T_history.append(2 * ke / (Nf * 1.0))

# 验证温度趋向 T0
plt.figure(figsize=(8, 4))
plt.axhline(T0, color='r', ls='--', label=f'目标温度 T₀={T0}')
plt.plot(np.arange(n_steps)*dt, T_history, 'b-', lw=0.8)
plt.xlabel('时间'); plt.ylabel('温度 T')
plt.title('Nosé-Hoover 恒温器：温度随时间收敛到 T₀')
plt.legend(); plt.grid(alpha=0.3); plt.show()
```

| 恒温器 | 系综正确性 | 实现难度 | 适用场景 |
|:---|:---|:---|:---|
| **Berendsen** | 近似 | ★☆☆ | 快速平衡 |
| **Nosé-Hoover** | 严格 NVT | ★★☆ | 平衡态采样（科研标准） |
| **Langevin** | 严格 NVT | ★☆☆ | 粗粒化模型，溶剂隐式处理 |

## 第六章 物理领域经典问题

### 6.1 三体问题与混沌

📘 **混沌的定义**：对初始条件极其敏感的确定性动力学。初值相差无穷小，轨道以指数速度分离。

**Lyapunov 指数**：
$$\lambda = \lim_{t\to\infty} \frac{1}{t} \ln \frac{|\delta\mathbf{r}(t)|}{|\delta\mathbf{r}(0)|}$$

若 $\lambda > 0$ → 混沌。

🖥️ **三体混沌演示**：

```python
# 依赖: numpy, scipy, matplotlib
import numpy as np
from scipy.integrate import solve_ivp
import matplotlib.pyplot as plt

def three_body(t, y, G=1.0):
    """三体问题：18 维 ODE (3×3 坐标 + 3×3 速度)"""
    r = y[:9].reshape(3, 3)
    v = y[9:].reshape(3, 3)
    m = np.array([1.0, 1.0, 1.0])
    a = np.zeros((3, 3))
    for i in range(3):
        for j in range(3):
            if i != j:
                dr = r[j] - r[i]
                dist = np.linalg.norm(dr)
                softening = 0.01  # Plummer 软化因子，避免奇点
                a[i] += G * m[j] * dr / (dist**3 + softening**3)
    return np.concatenate([v.flatten(), a.flatten()])

# 初始条件（八字形轨道）
y0 = np.array([
    -0.97000436,  0.24308753, 0,  0.97000436, -0.24308753, 0,  0, 0, 0,
     0.466203685, 0.43236573, 0,  0.466203685, 0.43236573, 0,
    -0.93240737, -0.86473146, 0
])

# 两个极微不同的初值
delta = 1e-8
y0_perturbed = y0.copy()
y0_perturbed[0] += delta  # 第 1 体的 x 坐标偏移 1e-8

sol1 = solve_ivp(three_body, [0, 15], y0, max_step=0.005, rtol=1e-10, atol=1e-12)
sol2 = solve_ivp(three_body, [0, 15], y0_perturbed, max_step=0.005, rtol=1e-10, atol=1e-12)

# 差异随时间增长
diff = np.sqrt(np.sum((sol1.y[:, -min(len(sol1.t), len(sol2.t)):] - 
                        sol2.y[:, -min(len(sol1.t), len(sol2.t)):])**2, axis=0))

plt.figure(figsize=(10, 5))
plt.semilogy(sol1.t[-len(diff):], diff, 'r-', lw=1)
plt.xlabel('时间'); plt.ylabel('$||\\delta\\mathbf{y}(t)||$')
plt.title(f'三体混沌：初始偏差 {delta:.0e} 的指数放大')
plt.grid(True, alpha=0.3)
plt.tight_layout(); plt.show()
print("观察到：差异在短时间内指数增长 → Lyapunov 指数 > 0 → 混沌")
```

📐 **定律的积分形式**：

### 6.3 量子力学：含时薛定谔方程

#### 6.3.1 Crank-Nicolson 方法

📐 **一维含时薛定谔方程**（无量纲化，$\hbar = 1$）：
$$i\frac{\partial\psi}{\partial t} = -\frac{1}{2}\frac{\partial^2\psi}{\partial x^2} + V(x)\psi$$

空间离散化后写成矩阵形式：
$$i\frac{d\boldsymbol{\psi}}{dt} = \mathbf{H}\boldsymbol{\psi}, \quad \mathbf{H} = -\frac{1}{2}\mathbf{D}_{xx} + \operatorname{diag}(V_j)$$

Crank-Nicolson 格式（**无条件稳定，保持幺正性**）：
$$\left(\mathbf{I} + \frac{i\Delta t}{2}\mathbf{H}\right)\boldsymbol{\psi}^{n+1} = \left(\mathbf{I} - \frac{i\Delta t}{2}\mathbf{H}\right)\boldsymbol{\psi}^n$$

> Crank-Nicolson 是量子力学仿真的标准选择，因为它保持波函数的归一化（幺正演化）。

🖥️ **量子隧穿示例**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt
from scipy.sparse import diags, eye
from scipy.sparse.linalg import spsolve

# ========== 参数 ==========
L = 20.0; Nx = 500
dx = L / Nx
x = np.linspace(-L/2, L/2, Nx)
dt = 0.005
T_final = 2.0
Nt = int(T_final / dt)

# ========== 势垒 ==========
V0 = 50.0; w = 0.3
V = np.where(np.abs(x) < w, V0, 0.0)

# ========== 初始波包 (高斯, 向右运动) ==========
x0 = -5.0; k0 = 15.0; sigma = 0.5
psi = np.exp(-(x-x0)**2/(2*sigma**2)) * np.exp(1j*k0*x)
psi = psi / np.sqrt(np.trapz(np.abs(psi)**2, x))  # 归一化

# ========== Crank-Nicolson 矩阵 ==========
r = dt / (4 * dx**2)
main_diag = 1.0 + 2*r*1j + 1j*dt/2 * V
off_diag = -r*1j * np.ones(Nx-1)

A = diags([off_diag, main_diag, off_diag], [-1, 0, 1], format='csc')
# B = I - i (dt/2) H：对角元素为 1 - (main_diag-1) = 2 - main_diag
# 非对角元素取反：-(i(dt/2)H_off) = -(-ir) = ir = -off_diag
B = diags([-off_diag, 2-main_diag, -off_diag], [-1, 0, 1], format='csc')

# ========== 时间推进 ==========
prob_history = []

for n in range(Nt):
    psi = spsolve(A, B.dot(psi))
    if n % 20 == 0:
        prob_history.append(np.abs(psi)**2)

# ========== 可视化 ==========
fig, (ax1, ax2) = plt.subplots(2, 1, figsize=(12, 8))

# 势垒 + 概率密度快照
ax1.fill_between(x, 0, V/V0*np.max(prob_history[0]), 
                  color='gray', alpha=0.3, label='势垒 V(x)')
for k, prob in enumerate(prob_history):
    alpha = k / len(prob_history)
    ax1.plot(x, prob, color=plt.cm.plasma(alpha), lw=1,
             label=f't={k*20*dt:.2f}' if k%4==0 else '')
ax1.set_xlabel('x'); ax1.set_ylabel('|ψ(x,t)|²')
ax1.set_title('量子隧穿：波包穿过势垒')
ax1.legend(fontsize=7); ax1.grid(alpha=0.3)

# 时空图
prob_array = np.array(prob_history)
ax2.contourf(x, np.arange(len(prob_history))*20*dt, prob_array, 
             levels=50, cmap='hot')
ax2.set_xlabel('x'); ax2.set_ylabel('t')
ax2.set_title('|ψ(x,t)|² 时空演化图')

plt.tight_layout(); plt.show()
```

### 6.4 Ising 模型与相变

📘 **二维 Ising 模型的 Hamilton 量**：
$$H = -J\sum_{\langle i,j\rangle} s_i s_j, \quad s_i = \pm 1$$

Onsager (1944) 精确解：
$$T_c = \frac{2J}{k_B \ln(1+\sqrt{2})} \approx 2.269\frac{J}{k_B}$$

$$M(T) = \begin{cases}
[1 - \sinh^{-4}(2J/k_BT)]^{1/8}, & T < T_c \\
0, & T \geq T_c
\end{cases}$$

#### Metropolis 算法采样

每个 Monte Carlo 步（MCS）尝试翻转 $L^2$ 个自旋。单次翻转的接受概率为：
$$\alpha(s_i \rightarrow -s_i) = \min\left(1, e^{-\beta \Delta E}\right), \quad \Delta E = 2 J s_i \sum_{\langle j\rangle} s_j$$

🖥️ **完整实现**：

```python
# 依赖: numpy, matplotlib
import numpy as np
import matplotlib.pyplot as plt

def ising_metropolis(beta, L, n_equil, n_sample):
    """二维 Ising 模型 Metropolis 模拟"""
    # 随机初始化
    lattice = np.random.choice([-1, 1], (L, L))
    n_sites = L * L
    
    mag = []
    energy = []
    
    for step in range(n_equil + n_sample):
        # 一个 MCS = L² 次尝试翻转
        for _ in range(n_sites):
            i, j = np.random.randint(0, L, 2)
            s = lattice[i, j]
            # 最近邻和（周期边界）
            nb_sum = (lattice[(i+1)%L, j] + lattice[(i-1)%L, j] +
                      lattice[i, (j+1)%L] + lattice[i, (j-1)%L])
            dE = 2 * s * nb_sum  # J=1
            
            if dE <= 0 or np.random.uniform() < np.exp(-beta * dE):
                lattice[i, j] = -s
        
        # 热化后开始记录
        if step >= n_equil:
            M = np.abs(np.sum(lattice)) / n_sites
            E = -np.sum(lattice * (
                np.roll(lattice, 1, axis=0) + np.roll(lattice, 1, axis=1)
            )) / n_sites
            mag.append(M)
            energy.append(E)
    
    return np.array(mag), np.array(energy)

# ========== 参数扫描 ==========
L = 32  # 系统尺寸
betas = np.linspace(0.2, 0.8, 30)  # β = 1/T
T_vals = 1.0 / betas

M_mean, M2, M4 = [], [], []
for beta in betas:
    mag, _ = ising_metropolis(beta, L, n_equil=5000, n_sample=20000)
    M_mean.append(np.mean(mag))
    M2.append(np.mean(mag**2))
    M4.append(np.mean(mag**4))

M_mean, M2, M4 = map(np.array, [M_mean, M2, M4])

# ========== Binder 累积量 ==========
📘 **Binder 累积量**（四阶累积量比）：
$$U_L = 1 - \frac{\langle M^4 \rangle}{3\langle M^2 \rangle^2}$$

**物理含义**：
- $T < T_c$（有序相）：$U_L \to 2/3$
- $T = T_c$：$U_L$ 与 $L$ 无关 → **不同 $L$ 的 $U_L(T)$ 曲线交于 $T_c$**
- $T > T_c$（无序相）：$U_L \to 0$

```python
# 计算 Binder 累积量
U = 1 - M4 / (3 * M2**2)

# 可视化
fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(12, 5))

ax1.plot(T_vals, M_mean, 'bo-', lw=1.5, markersize=4)
ax1.axvline(2/np.log(1+np.sqrt(2)), color='r', ls='--',
            label=f'$T_c$ = {2/np.log(1+np.sqrt(2)):.3f}')
ax1.set_xlabel('T'); ax1.set_ylabel('|M|')
ax1.set_title(f'磁化强度 (L={L})')
ax1.legend(); ax1.grid(alpha=0.3)

ax2.plot(T_vals, U, 'ro-', lw=1.5, markersize=4)
ax2.axhline(2/3, color='gray', ls=':', label='U → 2/3 (T<T_c)')
ax2.axhline(0, color='gray', ls=':', label='U → 0 (T>T_c)')
ax2.axvline(2/np.log(1+np.sqrt(2)), color='r', ls='--',
            label=f'$T_c (精确解)$')
ax2.set_xlabel('T'); ax2.set_ylabel('U (Binder)')
ax2.set_title(f'Binder 累积量 (L={L})')
ax2.legend(); ax2.grid(alpha=0.3)

plt.tight_layout(); plt.show()
```

## 第七章 误差分析与收敛性

### 7.1 误差来源的总分类

| 误差类型 | 来源 | 控制方法 |
|:---|:---|:---|
| **建模误差** | 物理模型与真实世界的差异 | 更精确的模型 |
| **离散误差** | 用差分/有限元近似微分算子 | 加密网格 → 收敛性检验 |
| **截断误差** | 泰勒展开截断高阶项 | 高阶方法 |
| **舍入误差** | 浮点运算精度有限 | 双精度，避免灾难性抵消 |
| **统计误差** | Monte Carlo 的有限采样 | $\propto 1/\sqrt{N}$，增加采样 |

### 7.2 网格收敛性验证

📐 **Richardson 外推**：

若数值方法具有形式 $u_h = u_{\text{exact}} + C h^p + \text{高阶项}$，则在两个不同网格间距 $h$ 和 $h/2$ 的结果可以外推：

$$u_{\text{exact}} \approx \frac{2^p u_{h/2} - u_h}{2^p - 1}$$

经验收敛阶：
$$p_{\text{obs}} = \log_2\left(\frac{u_{2h} - u_h}{u_h - u_{h/2}}\right)$$

🖥️ **Richardson 外推验证**：

```python
# 依赖: numpy
import numpy as np

def poisson_fdm_1d(N):
    """一维 Poisson: -u'' = sin(x), u(0)=u(π)=0, 精确解 u=sin(x)"""
    x = np.linspace(0, np.pi, N)
    h = x[1] - x[0]
    f = np.sin(x[1:-1])
    A = (np.diag(2*np.ones(N-2)) - np.diag(np.ones(N-3), 1) - np.diag(np.ones(N-3), -1)) / h**2
    u_interior = np.linalg.solve(A, f)
    u = np.zeros(N)
    u[1:-1] = u_interior
    return x, u

Ns = [8, 16, 32, 64, 128]
errors = []

for N in Ns:
    x, u_num = poisson_fdm_1d(N)
    u_exact = np.sin(x)
    error = np.max(np.abs(u_num - u_exact))
    errors.append(error)

# 计算收敛阶
print(f"  {'N':>5s}  {'h':>10s}  {'Error':>12s}  {'Order':>8s}")
for k in range(1, len(Ns)):
    h = np.pi / (Ns[k] - 1)
    p = np.log2(errors[k-1] / errors[k])
    print(f"  {Ns[k]:5d}  {h:10.4f}  {errors[k]:12.2e}  {p:8.3f}")

print("\n→ 二阶精度已确认（中心差分 → O(h²)）")
```

### 7.3 Monte Carlo 误差的 Blocking 分析

📘 **问题**：MCMC 采样的连续样本高度自相关，$\sqrt{\text{Var}(\bar{O})} \neq \sigma/\sqrt{N}$。

**解法**：将 $N$ 个样本分成 $B$ 个块，每块长度 $L = N/B$，用块均值估计真实方差。

📐 **块分析的原理**：
$$\text{Var}(\bar{O})_{\text{block}} = \frac{1}{B(B-1)}\sum_{b=1}^B (\bar{O}_b - \bar{O})^2$$

当块长大于自相关时间 $\tau$ 时，块间近似独立，块方差趋于真实方差。
