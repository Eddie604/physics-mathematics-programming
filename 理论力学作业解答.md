## 一、释义题（每题4分，共20分）

### 1. 简正坐标

**简正坐标**是描述多自由度耦合振动系统的一种特殊广义坐标。在线性小振动问题中，通过适当的线性变换，将原始坐标变换为简正坐标后，系统的拉格朗日量（或哈密顿量）变为一系列独立谐振子的拉格朗日量之和——即各简正坐标之间不存在耦合。

**数学表述**：设系统的原始广义坐标为 $q_i (i=1,2,\ldots,n)$，存在正交变换 $Q_\alpha = \sum_i a_{\alpha i} q_i$，使得系统的拉格朗日量化为：
$$L = \frac{1}{2}\sum_{\alpha}(\dot{Q}_\alpha^2 - \omega_\alpha^2 Q_\alpha^2)$$
其中 $\omega_\alpha$ 是系统的**简正频率**（本征频率），每个 $Q_\alpha$ 作独立的简谐振动。

**物理意义**：每个简正坐标对应系统一种独立的集体振动模式（简正模式），系统的任何振动都可以分解为这些简正模式的叠加。

---

### 2. 哈密顿原理

**哈密顿原理**（Hamilton's Principle）是分析力学中最基本的变分原理，也是最普遍的力学原理之一。

**表述**：保守的、完整的力学系统，在从时刻 $t_1$ 到 $t_2$ 的所有可能的运动中，真实运动使**作用量** $S$ 取极值（通常为极小值）：

$$\delta S = \delta\int_{t_1}^{t_2} L(q_i, \dot{q}_i, t)\,dt = 0$$

其中 $L = T - V$ 是系统的拉格朗日量，$S = \int L\,dt$ 称为**作用量**。

**要点**：
- 这是**积分形式的变分原理**（不同于虚功原理等微分形式原理）
- 变分时端点固定：$\delta q_i(t_1) = \delta q_i(t_2) = 0$
- 从哈密顿原理可导出拉格朗日方程，因此它是整个分析力学的出发点
- 适用范围极广，可推广至连续介质力学、电磁场理论等

---

### 3. 正则变换

**正则变换**（Canonical Transformation）是在哈密顿力学框架下，从一组正则变量 $(q_i, p_i)$ 变换到另一组正则变量 $(Q_i, P_i)$，使得哈密顿方程的形式保持不变。

**定义**：如果变换 $(q_i, p_i) \to (Q_i, P_i)$ 满足
$$\sum_i p_i dq_i - \sum_i P_i dQ_i + (K - H)dt = dF$$
其中 $F$ 是某个任意函数（称为**母函数**或生成函数），$K$ 是新哈密顿量，则此变换为正则变换。

**四种母函数**：
| 类型 | 母函数 | 变换关系 |
|------|--------|---------|
| 第一类 | $F_1(q, Q, t)$ | $p_i = \partial F_1/\partial q_i$，$P_i = -\partial F_1/\partial Q_i$ |
| 第二类 | $F_2(q, P, t)$ | $p_i = \partial F_2/\partial q_i$，$Q_i = \partial F_2/\partial P_i$ |
| 第三类 | $F_3(p, Q, t)$ | $q_i = -\partial F_3/\partial p_i$，$P_i = -\partial F_3/\partial Q_i$ |
| 第四类 | $F_4(p, P, t)$ | $q_i = -\partial F_4/\partial p_i$，$Q_i = \partial F_4/\partial P_i$ |

**意义**：正则变换是求解复杂力学问题的重要工具。通过选择合适的正则变换，可以使新哈密顿量 $K$ 大大简化（如变为零、变为常数等），从而简化运动方程甚至直接求解。**哈密顿-雅可比理论**就是寻找"使 $K=0$"的正则变换。

---

### 4. 拉格朗日方程

**拉格朗日方程**（Lagrange's Equations）是分析力学中描述完整系统运动的基本方程，用广义坐标表达。

**形式**：对于具有 $s$ 个自由度的完整系统，选定 $s$ 个广义坐标 $q_\alpha (\alpha=1,2,\ldots,s)$，拉格朗日量为 $L = T - V$，则运动方程为：

$$\boxed{\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha} = 0 \quad (\alpha = 1,2,\ldots,s)}$$

**对于有非保守力的情况**（引入广义力 $Q_\alpha$）：
$$\frac{d}{dt}\frac{\partial L}{\partial \dot{q}_\alpha} - \frac{\partial L}{\partial q_\alpha} = Q_\alpha$$

**优势**：
- 不需要考虑约束力（理想约束自动消除）
- 不依赖于坐标系（可使用任何方便的广义坐标）
- 形式简洁、统一
- 与牛顿力学等价但适用范围更广

**来源**：可从达朗贝尔原理推导，也可从哈密顿原理 $\delta\int L dt = 0$ 直接导出。

---

### 5. 达朗贝尔原理

**达朗贝尔原理**（d'Alembert's Principle）是分析力学中的基本原理之一，它将动力学问题转化为静力学（平衡）问题来处理。

**表述**：在理想约束条件下，系统在任意虚位移上满足：

$$\boxed{\sum_i (\vec{F}_i - m_i\ddot{\vec{r}}_i)\cdot\delta\vec{r}_i = 0}$$

其中 $\vec{F}_i$ 是作用在第 $i$ 个质点上的主动力（不包括约束力），$m_i\ddot{\vec{r}}_i$ 是惯性力（达朗贝尔力），$\delta\vec{r}_i$ 是满足约束条件的虚位移。

**物理意义**：将 $-m\ddot{r}$ 视为一种"惯性力"后，作用于系统的主动力和惯性力在任意虚位移上所做的虚功之和为零——这就是**动力学问题的"静力学平衡方程"**。

**重要性**：
- 是拉格朗日方程的出发点（对达朗贝尔原理中的虚功项用广义坐标表示即得拉格朗日方程）
- 不需要求约束力（理想约束条件保证了约束反力虚功为零）
- 为"动力学平衡"提供了统一的表述

### 第1题：双摆的小振动（15分）

**题目**：质量为 $m$ 的质点用长 $l$ 的轻绳系在固定点 $O$，该质点上又用同样长 $l$ 的轻绳系另一质量也为 $m$ 的质点。求此"双摆"系统在竖直平面内作小振动的情况。

**解**：

#### (1) 建立坐标系和广义坐标

取固定点 $O$ 为原点，竖直向下为 $y$ 轴正方向。设上摆角为 $\theta_1$（与竖直方向的夹角），下摆角为 $\theta_2$（与竖直方向的夹角），两角均以逆时针为正。取 $\theta_1, \theta_2$ 为广义坐标。

上质点坐标：
$$x_1 = l\sin\theta_1, \quad y_1 = l\cos\theta_1$$

下质点坐标：
$$x_2 = l\sin\theta_1 + l\sin\theta_2, \quad y_2 = l\cos\theta_1 + l\cos\theta_2$$

#### (2) 求动能 $T$

上质点速度：
$$\dot{x}_1 = l\dot{\theta}_1\cos\theta_1, \quad \dot{y}_1 = -l\dot{\theta}_1\sin\theta_1$$
$$v_1^2 = \dot{x}_1^2 + \dot{y}_1^2 = l^2\dot{\theta}_1^2$$

下质点速度：
$$\dot{x}_2 = l\dot{\theta}_1\cos\theta_1 + l\dot{\theta}_2\cos\theta_2$$
$$\dot{y}_2 = -l\dot{\theta}_1\sin\theta_1 - l\dot{\theta}_2\sin\theta_2$$

$$v_2^2 = l^2[\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2(\cos\theta_1\cos\theta_2 + \sin\theta_1\sin\theta_2)]$$
$$= l^2[\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1 - \theta_2)]$$

动能：
$$T = \frac{1}{2}mv_1^2 + \frac{1}{2}mv_2^2$$
$$T = \frac{1}{2}m\left[l^2\dot{\theta}_1^2 + l^2\dot{\theta}_1^2 + l^2\dot{\theta}_2^2 + 2l^2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]$$
$$T = \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\cos(\theta_1-\theta_2)\right]$$

#### (3) 求势能 $V$（以 $O$ 点为零势能面）

$$V = -mgy_1 - mgy_2 = -mgl(\cos\theta_1 + \cos\theta_1 + \cos\theta_2)$$
$$V = -mgl(2\cos\theta_1 + \cos\theta_2)$$

为方便小振动分析，定义势能以平衡位置为零：
$$V_{\text{平}} = -3mgl$$
$$V_{\text{相对}} = -mgl(2\cos\theta_1 + \cos\theta_2) + 3mgl$$

对于小振动，$\cos\theta \approx 1 - \theta^2/2$：
$$V \approx -mgl\left[2\left(1 - \frac{\theta_1^2}{2}\right) + \left(1 - \frac{\theta_2^2}{2}\right)\right] + 3mgl$$
$$= \frac{1}{2}mgl(2\theta_1^2 + \theta_2^2)$$

#### (4) 小振动近似下的拉格朗日量

小振动时 $\cos(\theta_1 - \theta_2) \approx 1$，动能近似为：
$$T \approx \frac{1}{2}ml^2\left[2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2\right]$$

$$L = T - V$$
$$\boxed{L = \frac{1}{2}ml^2(2\dot{\theta}_1^2 + \dot{\theta}_2^2 + 2\dot{\theta}_1\dot{\theta}_2) - \frac{1}{2}mgl(2\theta_1^2 + \theta_2^2)}$$

#### (5) 运动方程

对 $\theta_1$：
$$\frac{\partial L}{\partial \dot{\theta}_1} = ml^2(2\dot{\theta}_1 + \dot{\theta}_2)$$
$$\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_1} = ml^2(2\ddot{\theta}_1 + \ddot{\theta}_2)$$
$$\frac{\partial L}{\partial \theta_1} = -2mgl\,\theta_1$$

$$\Rightarrow ml^2(2\ddot{\theta}_1 + \ddot{\theta}_2) + 2mgl\,\theta_1 = 0$$
$$2l\ddot{\theta}_1 + l\ddot{\theta}_2 + 2g\theta_1 = 0 \quad\cdots (1)$$

对 $\theta_2$：
$$\frac{\partial L}{\partial \dot{\theta}_2} = ml^2(\dot{\theta}_2 + \dot{\theta}_1)$$
$$\frac{d}{dt}\frac{\partial L}{\partial \dot{\theta}_2} = ml^2(\ddot{\theta}_2 + \ddot{\theta}_1)$$
$$\frac{\partial L}{\partial \theta_2} = -mgl\,\theta_2$$

$$\Rightarrow ml^2(\ddot{\theta}_1 + \ddot{\theta}_2) + mgl\,\theta_2 = 0$$
$$l\ddot{\theta}_1 + l\ddot{\theta}_2 + g\theta_2 = 0 \quad\cdots (2)$$

#### (6) 求简正频率

设 $\theta_1 = A_1 e^{i\omega t}, \theta_2 = A_2 e^{i\omega t}$，代入运动方程：

由(1)：$(-2l\omega^2 + 2g)A_1 - l\omega^2 A_2 = 0$
由(2)：$-l\omega^2 A_1 + (-l\omega^2 + g)A_2 = 0$

即矩阵形式式：
$$\begin{pmatrix} 2g - 2l\omega^2 & -l\omega^2 \\ -l\omega^2 & g - l\omega^2 \end{pmatrix} \begin{pmatrix} A_1 \\ A_2 \end{pmatrix} = 0$$

令行列式为零：
$$(2g - 2l\omega^2)(g - l\omega^2) - l^2\omega^4 = 0$$
$$2g^2 - 2gl\omega^2 - 2gl\omega^2 + 2l^2\omega^4 - l^2\omega^4 = 0$$
$$l^2\omega^4 - 4gl\omega^2 + 2g^2 = 0$$

解此关于 $\omega^2$ 的一元二次方程：
$$\omega^2 = \frac{4gl \pm \sqrt{16g^2l^2 - 8g^2l^2}}{2l^2} = \frac{4gl \pm 2\sqrt{2}gl}{2l^2}$$

$$\boxed{\omega^2 = \frac{g}{l}(2 \pm \sqrt{2})}$$

$$\boxed{\omega_1 = \sqrt{\frac{g}{l}(2 - \sqrt{2})} \approx 0.765\sqrt{\frac{g}{l}}}$$

$$\boxed{\omega_2 = \sqrt{\frac{g}{l}(2 + \sqrt{2})} \approx 1.848\sqrt{\frac{g}{l}}}$$

#### (7) 简正模式

**第一模式**（同相摆动，$\omega_1$ 低频）：
$$\frac{A_2}{A_1} = \frac{2g - 2l\omega_1^2}{l\omega_1^2} = \frac{2g - 2g(2-\sqrt{2})}{g(2-\sqrt{2})} = \frac{2g(\sqrt{2}-1)}{g(2-\sqrt{2})} = \sqrt{2}$$

两摆**同向摆动**，下摆振幅约为上摆的 $\sqrt{2}$ 倍。

**第二模式**（反相摆动，$\omega_2$ 高频）：
$$\frac{A_2}{A_1} = \frac{2g - 2g(2+\sqrt{2})}{g(2+\sqrt{2})} = \frac{-2g\sqrt{2}}{g(2+\sqrt{2})} = -\sqrt{2}$$

两摆**反向摆动**，下摆振幅也约为上摆的 $\sqrt{2}$ 倍，但方向相反。

#### 答案总结

双摆小振动的两个简正频率为：
$$\omega_1 = \sqrt{(2-\sqrt{2})\frac{g}{l}}, \quad \omega_2 = \sqrt{(2+\sqrt{2})\frac{g}{l}}$$

对应简正模式分别为同相摆动和反相摆动。系统的任意小振动都可表示为这两个简正模式的线性叠加。

### 第2题：长杆平衡条件（共15分）

**题目**：质量均匀的长杆靠在光滑墙面且保持静止，墙面距右侧顶点水平距离为 $d$，杆长 $2L$，质量 $m$。分别用牛顿力学方法和虚功原理求解平衡条件。

**解**：

首先分析物理场景：一根长度为 $2L$ 的均匀长杆，靠在光滑的竖直墙面上。杆的左端靠在墙上（接触点为墙面），右端放在地面上。墙面是光滑的（无摩擦），地面情况题中未明确说明，假设也是光滑的（或求一般平衡条件）。

设杆与水平面的夹角为 $\theta$，墙面距右侧顶点的水平距离为 $d$。

由几何关系（设右端 $B$ 在地面上，左端 $A$ 在墙面上）：
- $A$ 点（墙上）：$x_A = 0, y_A$
- $B$ 点（地面）：$x_B = d, y_B = 0$

由于杆长 $2L$：
$$(x_B - x_A)^2 + (y_B - y_A)^2 = (2L)^2$$
$$d^2 + y_A^2 = 4L^2$$
$$y_A = \sqrt{4L^2 - d^2}$$

杆与水平面的夹角：
$$\tan\theta = \frac{y_A}{d} = \frac{\sqrt{4L^2 - d^2}}{d}$$

#### 方法一：牛顿力学方法（7分）

杆所受的力：
1. **重力** $mg$：作用在杆的中心（质心），方向竖直向下
2. **墙面支持力** $N_A$：水平向右（墙面光滑，无摩擦力）
3. **地面支持力** $N_B$：竖直向上（若地面也光滑）
4. **地面摩擦力** $f_B$：水平方向（若地面有摩擦，维持平衡的必需条件）

质心位置：
$$x_C = \frac{d}{2}, \quad y_C = \frac{y_A}{2} = \frac{\sqrt{4L^2 - d^2}}{2}$$

**力平衡方程**：

水平方向：$N_A - f_B = 0 \quad\cdots (1)$

竖直方向：$N_B - mg = 0 \quad\cdots (2)$

**力矩平衡**（对 $B$ 点取矩，逆时针为正）：

重力力矩：$-mg \cdot \frac{d}{2}$（力 × 水平力臂）

墙面力力矩：$+N_A \cdot y_A$（力 × 竖直力臂）

(注：$N_B$ 和 $f_B$ 对 $B$ 点力矩为零)

$$N_A \cdot y_A - mg\cdot\frac{d}{2} = 0 \quad\cdots (3)$$

由(2)：$N_B = mg$

由(3)：$N_A = \frac{mgd}{2y_A} = \frac{mgd}{2\sqrt{4L^2 - d^2}}$

由(1)：$f_B = N_A = \frac{mgd}{2\sqrt{4L^2 - d^2}}$

**平衡条件**（若地面有摩擦）：
$$f_B \leq \mu N_B = \mu mg$$
$$\frac{mgd}{2\sqrt{4L^2 - d^2}} \leq \mu mg$$
$$\frac{d}{2\sqrt{4L^2 - d^2}} \leq \mu$$

若地面**光滑**（$f_B = 0$），则由(1) $N_A = 0$，再由(3)得 $d = 0$ 或 $mg = 0$——这是平平凡解，意味着光滑地面无法维持倾斜杆的平衡。必须有**摩擦力**才能平衡。

**平衡条件**：
$$\boxed{f = \frac{mgd}{2\sqrt{4L^2 - d^2}} \quad \text{（维持平衡所需的静摩擦力）}}$$

$$\boxed{\mu \geq \frac{d}{2\sqrt{4L^2 - d^2}} \quad \text{（摩擦系数条件）}}$$

写成 $\theta$ 的形式（$\tan\theta = \sqrt{4L^2 - d^2}/d$）：
$$\boxed{\mu \geq \frac{1}{2\tan\theta}}$$

---

#### 方法二：虚功原理（8分）

**虚功原理**：理想约束条件下，处于静平衡的系统，所有主动力在任意虚位移上所做的虚功之和为零。

$$\delta W = \sum_i \vec{F}_i \cdot \delta\vec{r}_i = 0$$

系统中主动力只有重力 $mg$（作用在质心）。若地面有摩擦力，摩擦力也是主动力。

**情况1：假设地面也是理想光滑的（只有墙面光滑）**

这时摩擦力的虚功需要纳入。地面摩擦力和墙面支持力都是约束力（静力），在光滑墙面上 $N_A$ 不做功（墙面垂直于水平位移），但地面的摩擦力需要考察。

**情况2：直接利用平衡的广义坐标表述**

取杆与水平面夹角 $\theta$ 为广义坐标。

质心坐标：
$$x_C = \frac{d}{2} = \frac{2L\cos\theta}{2} = L\cos\theta$$
$$y_C = \frac{\sqrt{4L^2 - d^2}}{2} = L\sin\theta$$

虚位移（对 $\theta$ 的变分）：
$$\delta x_C = -L\sin\theta\,\delta\theta$$
$$\delta y_C = L\cos\theta\,\delta\theta$$

**解法A**：若地面光滑（$f=0$），只有重力做虚功：
$$\delta W = -mg\,\delta y_C = -mgL\cos\theta\,\delta\theta = 0$$

$(mgL\cos\theta = 0)$ 对任意 $\delta\theta$ 成立，但 $mgL \neq 0$，$\cos\theta = 0$ 才成立即 $\theta = \pi/2$——即杆竖直贴墙。这不是一般平衡位置。

因此**地面必须存在摩擦力**才能有非平凡平衡。

**解法B**：含摩擦力的虚功原理。

将摩擦力 $f$ 视为主动力（或广义力）。$B$ 点的虚位移方向是水平方向：
$$x_B = 2L\cos\theta, \quad \delta x_B = -2L\sin\theta\,\delta\theta$$

摩擦力 $f$ 的方向与可能的水平滑动方向相反：
$$\delta W_{\text{摩擦}} = f \cdot \delta x_B = -2fL\sin\theta\,\delta\theta$$

重力虚功：
$$\delta W_{\text{重量}} = -mg\,\delta y_C = -mgL\cos\theta\,\delta\theta$$

平衡条件 $\delta W_{\text{重量}} + \delta W_{\text{摩擦}} = 0$：
$$-mgL\cos\theta\,\delta\theta - 2fL\sin\theta\,\delta\theta = 0$$
$$\Rightarrow f = -\frac{mg\cos\theta}{2\sin\theta} = -\frac{mg}{2\tan\theta}$$

负号表示摩擦力的实际方向与我假设的 $\delta x_B$ 正方向相反。取绝对值：
$$\boxed{f = \frac{mg}{2\tan\theta} = \frac{mgd}{2\sqrt{4L^2 - d^2}}}$$

与牛顿法结果一致。平衡条件为：
$$\boxed{\mu \geq \frac{1}{2\tan\theta}}$$

### 第3题：作用量的计算与对比（15分）

**题目**：重力场中质量 $m$ 的质点从 $A$ 点无初速度自由下落到 $B$ 点，经历时间 $t_0$。求：
(1) 自由下落过程的作用量；
(2) 若匀速从 $A$ 到 $B$（时间仍为 $t_0$），作用量为多少？
(3) 对比两种情景，可得到什么结论？

**解**：

设 $A$ 点位于高处，$B$ 点位于低处，竖直向下为 $y$ 轴正方向，$A$ 点 $y=0$，$B$ 点 $y=h$。

#### (1) 自由下落的作用量

**运动方程**：自由下落加速度为 $g$，初速度 $v_0 = 0$：
$$y(t) = \frac{1}{2}gt^2$$

到达 $B$ 点（$y=h$）时 $t = t_0$：
$$h = \frac{1}{2}gt_0^2 \quad\Rightarrow\quad t_0 = \sqrt{\frac{2h}{g}}$$

**速度**：
$$v(t) = \dot{y}(t) = gt$$

**动能**：
$$T = \frac{1}{2}mv^2 = \frac{1}{2}mg^2t^2$$

**势能**（以 $A$ 点为势能零点）：
$$V = -mgy(t) = -\frac{1}{2}mg^2t^2$$

**拉格朗日量**：
$$L = T - V = \frac{1}{2}mg^2t^2 - \left(-\frac{1}{2}mg^2t^2\right) = mg^2t^2$$

**作用量**：
$$S_{\text{自由}} = \int_0^{t_0} L\,dt = \int_0^{t_0} mg^2t^2\,dt = mg^2\cdot\left[\frac{t^3}{3}\right]_0^{t_0} = \frac{1}{3}mg^2t_0^3$$

代入 $t_0 = \sqrt{2h/g}$：
$$S_{\text{自由}} = \frac{1}{3}mg^2\left(\frac{2h}{g}\right)^{3/2} = \frac{1}{3}mg^2\frac{(2h)^{3/2}}{g^{3/2}} = \frac{m\sqrt{8gh^3}}{3}$$

若保留 $t_0$ 形式：
$$\boxed{S_{\text{自由}} = \frac{1}{3}mg^2t_0^3}$$

---

#### (2) 匀速下降的作用量

匀速下降：$y(t) = \frac{h}{t_0}t$（$v = h/t_0$ 为常数）

**速度**：
$$v = \dot{y} = \frac{h}{t_0} = \text{常数}$$

**动能**：
$$T = \frac{1}{2}mv^2 = \frac{1}{2}m\left(\frac{h}{t_0}\right)^2 = \frac{1}{2}m\frac{h^2}{t_0^2}$$

**势能**：
$$V = -mgy = -mg\frac{h}{t_0}t$$

**拉格朗日量**：
$$L = T - V = \frac{1}{2}m\frac{h^2}{t_0^2} + mg\frac{h}{t_0}t$$

**作用量**：
$$S_{\text{匀速}} = \int_0^{t_0} L\,dt = \int_0^{t_0} \left[\frac{1}{2}m\frac{h^2}{t_0^2} + mg\frac{h}{t_0}t\right]dt$$
$$= \frac{1}{2}m\frac{h^2}{t_0^2}\cdot t_0 + mg\frac{h}{t_0}\cdot\left[\frac{t^2}{2}\right]_0^{t_0}$$
$$= \frac{1}{2}m\frac{h^2}{t_0} + \frac{1}{2}mgh\,t_0$$

由自由下落条件 $h = \frac{1}{2}gt_0^2$ 代入：
$$S_{\text{匀速}} = \frac{1}{2}m\frac{(\frac{1}{2}gt_0^2)^2}{t_0} + \frac{1}{2}mg\cdot\frac{1}{2}gt_0^2\cdot t_0$$
$$= \frac{1}{2}m\frac{g^2t_0^4}{4t_0} + \frac{1}{4}mg^2t_0^3$$
$$= \frac{1}{8}mg^2t_0^3 + \frac{1}{4}mg^2t_0^3$$
$$= \frac{3}{8}mg^2t_0^3$$

$$\boxed{S_{\text{匀速}} = \frac{3}{8}mg^2t_0^3}$$

---

#### (3) 对比与结论

$$S_{\text{自由}} = \frac{1}{3}mg^2t_0^3 \approx 0.333\,mg^2t_0^3$$
$$S_{\text{匀速}} = \frac{3}{8}mg^2t_0^3 = 0.375\,mg^2t_0^3$$

**结论**：

> 🔑 **$S_{\text{自由}} < S_{\text{匀速}}$**，即真实的自由下落运动对应的作用量**小于**非真实的匀速运动的作用量。

这**验证了哈密顿原理**：在所有连接相同起点和终点、经历相同时间的可能运动中，真实的物理运动使作用量取**极小值**。

具体地说：
- 自由下落是保守力场中只受重力的真实运动，满足拉格朗日方程（即 $d/dt(\partial L/\partial \dot{y}) - \partial L/\partial y = 0$）
- 匀速下降是人为强制的运动（需要额外的非保守力才能维持），不满足拉格朗日方程
- 真实运动的作用量确实小于这个非真实运动的作用量，体现了**最小作用量原理**

数值上：$S_{\text{匀速}}/S_{\text{自由}} = 9/8 = 1.125$，匀速运动的作用量比真实运动大 $12.5\%$。

### 第4题：最速降线方程（15分）

**题目**：小球从原点 $A$ 无初速度向下滑动到固定点 $B$，分别用**折射定律类比法**和**欧拉-拉格朗日方程**求解最速降线方程。

**解**：

最速降线问题（Brachistochrone Problem）：在重力场中，质点从点 $A(0,0)$ 无初速度地沿着某条光滑曲线下滑到点 $B(x_1, y_1)$（$y$ 轴向下），求使得时间最短的曲线形状。

---

#### 方法一：折射定律类比法（光-力类比）

**核心思想**：利用费马原理（光走最短时间路径）与最速降线问题的类比——"光在连续变化的折射率介质中的传播路径"对应"质点在重力场中的最速下降路径"。

**物理类比**：

质点下降过程中，由能量守恒：
$$\frac{1}{2}mv^2 = mgy \quad\Rightarrow\quad v = \sqrt{2gy}$$

光的斯涅尔折射定律：在分层介质中，$\sin\alpha/v = \text{常数}$（其中 $\alpha$ 是光线与法线的夹角）。

将最速降线问题视为光在折射率 $n \propto 1/v \propto 1/\sqrt{y}$ 的介质中传播。

设曲线切线方向与竖直方向的夹角为 $\theta$，则此处折射定律表现为：
$$\frac{\sin\theta}{v} = \text{常数}$$

由于 $v = \sqrt{2gy}$：
$$\frac{\sin\theta}{\sqrt{2gy}} = C'$$

$\sin\theta$ 是切线与竖直方向夹角的正弦。由微积分：$\sin\theta = dx/ds$（水平方向的分量），其中 $ds = \sqrt{1 + (dy/dx)^2}\,dx$。

更常用的形式：设曲线切线方向角为 $\phi$（从水平方向起算），即 $\sin\theta$ 实际上等价于某个不方便表达的量。经典方法是引入参数 $\alpha$。

**经典推导**：

由 $\sin\theta/v = \text{常数}$，即光线满足的方程：
$$\frac{\sin\theta}{\sqrt{y}} = \text{常数}$$

但这不是最方便的形式。经典处理中直接令：
$$dx = \frac{y}{\sqrt{2R - y}}\,dy$$

通过分离变量得到参数方程。实际上这样走比较复杂。我们直接利用变分法。

> 📌 折射定律类比法的核心结论是：最速降线应满足的方程可以等价于"光在 $v = \sqrt{2gy}$ 介质中的费马路径满足的斯涅尔定律"。查书本可知这直接导出摆线（旋轮线）的参数方程：
>
> $$x = R(\theta - \sin\theta), \quad y = R(1 - \cos\theta)$$
>
> 其中 $\theta$ 为摆线参数（滚轮转动角度），$R$ 为滚轮半径。

为了完整性，我们在方法二中严格推导。

---

#### 方法二：泛函的变分法（欧拉-拉格朗日方程）

这是最速降线的**严格推导方法**。

**建立泛函**：

质点从 $A(0,0)$ 到 $B(x_1, y_1)$（$y$ 轴向下为正）沿曲线 $y(x)$ 下滑。

微元弧长：
$$ds = \sqrt{1 + y'^2}\,dx$$

速度（由能量守恒）：
$$v = \sqrt{2gy}$$

时间微元：
$$dt = \frac{ds}{v} = \frac{\sqrt{1 + y'^2}}{\sqrt{2gy}}\,dx$$

总时间（目标泛函）：
$$T[y] = \int_0^{x_1} \frac{\sqrt{1 + y'^2}}{\sqrt{2gy}}\,dx = \frac{1}{\sqrt{2g}}\int_0^{x_1} F(y, y')\,dx$$

其中：
$$F(y, y') = \sqrt{\frac{1 + y'^2}{y}}$$

**欧拉-拉格朗日方程**（$F$ 不显含 $x$）：

由于 $\partial F/\partial x = 0$，存在首次积分（Beltrami恒等式）：
$$F - y'\frac{\partial F}{\partial y'} = C \quad (\text{常数})$$

计算：
$$\frac{\partial F}{\partial y'} = \frac{1}{2}\sqrt{\frac{1}{y}}\cdot\frac{2y'}{\sqrt{1+y'^2}} = \frac{y'}{\sqrt{y(1+y'^2)}}$$

代入 Beltrami 恒等式：
$$F - y'\frac{\partial F}{\partial y'} = \sqrt{\frac{1+y'^2}{y}} - y'\cdot\frac{y'}{\sqrt{y(1+y'^2)}}$$
$$= \frac{1+y'^2}{\sqrt{y(1+y'^2)}} - \frac{y'^2}{\sqrt{y(1+y'^2)}}$$
$$= \frac{1}{\sqrt{y(1+y'^2)}} = C$$

即：
$$\frac{1}{\sqrt{y(1+y'^2)}} = C$$
$$y(1 + y'^2) = \frac{1}{C^2} = 2R \quad (\text{令 } 2R = 1/C^2)$$

其中 $R$ 是待定常数。所以：
$$1 + \left(\frac{dy}{dx}\right)^2 = \frac{2R}{y}$$
$$y'^2 = \frac{2R - y}{y}$$
$$\frac{dy}{dx} = \sqrt{\frac{2R - y}{y}}$$

**分离变量**：
$$dx = \sqrt{\frac{y}{2R - y}}\,dy$$

**积分——引入参数**：

令 $y = R(1 - \cos\theta)$（$0 \leq \theta \leq 2\pi$），则 $dy = R\sin\theta\,d\theta$：
$$2R - y = 2R - R(1-\cos\theta) = R(1 + \cos\theta)$$
$$\sqrt{\frac{y}{2R - y}} = \sqrt{\frac{R(1-\cos\theta)}{R(1+\cos\theta)}} = \sqrt{\frac{1-\cos\theta}{1+\cos\theta}} = \tan\frac{\theta}{2}$$

或者用三角恒等式：
$$\frac{1-\cos\theta}{1+\cos\theta} = \frac{2\sin^2(\theta/2)}{2\cos^2(\theta/2)} = \tan^2\frac{\theta}{2}$$

所以：
$$dx = \tan\frac{\theta}{2}\cdot R\sin\theta\,d\theta$$

$$\sin\theta = 2\sin\frac{\theta}{2}\cos\frac{\theta}{2}$$

$$dx = R\tan\frac{\theta}{2}\cdot 2\sin\frac{\theta}{2}\cos\frac{\theta}{2}\,d\theta$$
$$= 2R\sin^2\frac{\theta}{2}\,d\theta$$
$$= R(1 - \cos\theta)\,d\theta$$

积分：
$$x = \int R(1 - \cos\theta)\,d\theta = R(\theta - \sin\theta) + C'$$

由边界条件 $(x=0, y=0)$：$y=0$ 时 $\theta=0$，代入 $x(0)=0$ 得 $C'=0$。

**最终结果——摆线（旋轮线）的参数方程**：

> $$\boxed{\begin{cases} x = R(\theta - \sin\theta) \\ y = R(1 - \cos\theta) \end{cases} \quad (\theta \geq 0)}$$

其中 $R$ 由终点 $B(x_1, y_1)$ 确定，$\theta$ 为滚轮转动参数。

**验证**：当 $\theta = 2\pi$ 时，$x = 2\pi R$，$y = 0$（回到出发点水平高度）。当 $\theta = \pi$ 时，$x = \pi R$，$y = 2R$（最低点）。

---

#### 最速降线的物理意义

最速降线是**摆线**（cycloid，又称旋轮线）——一个圆沿直线滚动时，圆周上一点所描绘的轨迹。

摆线的性质：
- 它是连接两点的所有光滑曲线中下降时间**最短**的曲线
- 摆线的等时性：从摆线上任意高度无初速度释放质点，到达最低点的时间相同（惠更斯发现）
- $R$ 是生成摆线的滚动圆的半径

## 题型总结

### 题型一：概念释义题
**考题特征**：对分析力学核心概念进行解释（简正坐标、哈密顿原理、正则变换、拉格朗日方程、达朗贝尔原理等）

**答题要点**：
1. 定义准确（给出数学表达式）
2. 说明物理意义（该概念在力学中的地位）
3. 简述主要性质或重要结论
4. 注意与其他概念的关联

---

### 题型二：多自由度耦合振动（双摆）
**考题特征**：给定耦合系统（双摆、耦合摆等），求小振动下的简正频率和简正模式。

**解题模板**：
1. 选定合适的广义坐标
2. 写出动能 $T$ 和势能 $V$
3. 小振动近似：$\cos\theta \approx 1-\theta^2/2$，$\sin\theta \approx \theta$，耦合项 $\cos(\theta_1-\theta_2)\approx 1$
4. 构造 $L = T - V$（写成二次型）
5. 从拉格朗日方程导出线性运动方程
6. 设解 $\theta_i = A_i e^{i\omega t}$，解本征值问题得到 $\omega$
7. 求对应的振幅比（本征矢量），分析简正模式

---

### 题型三：静平衡——牛顿力学 + 虚功原理
**考题特征**：同一静平衡问题，用两种方法（牛顿力学、虚功原理）分别求解。

**牛顿力学方法**：
1. 受力分析，画出所有力
2. 列出力平衡方程（$\sum F_x = 0, \sum F_y = 0$）和力矩平衡方程（$\sum M = 0$）
3. 解方程得到平衡条件

**虚功原理方法**：
1. 选择适当的广义坐标
2. 计算各主动力的虚功 $\delta W = \sum \vec{F}_i \cdot \delta\vec{r}_i$
3. 将虚位移用广义坐标变分表示
4. 令 $\delta W = 0$ 得到平衡方程
5. 解出平衡条件

> 📌 虚功原理的优势：不需要求约束反力，只需要知道主动力即可。对于复杂约束系统尤为方便。

---

### 题型四：作用量计算与比较
**考题特征**：计算不同运动方式的作用量，对比验证最小作用量原理。

**解题模板**：
1. 确定运动规律 $y(t)$（由运动学条件给出）
2. 写出拉格朗日量 $L = T - V$
3. 计算作用量 $S = \int_{t_1}^{t_2} L\,dt$
4. 比较不同运动的作用量大小
5. 得出结论：真实运动的 $S$ 最小（或取极值）

---

### 题型五：最速降线（变分法）
**考题特征**：用欧拉-拉格朗日方程求解泛函极值问题。

**解题模板**：
1. 建立目标泛函（如时间 $T[y] = \int F(y, y')dx$）
2. 识别 $F$ 是否显含 $x$（若不显含，用 Beltrami 恒等式简化）
3. 列出欧拉-拉格朗日方程或 Beltrami 恒等式
4. 引入参数进行换元积分
5. 利用边界条件确定积分常数
6. 写出最终曲线方程（参数形式）

**最速降线的标准结果**：
$$\boxed{x = R(\theta - \sin\theta), \quad y = R(1 - \cos\theta)}$$
即摆线（旋轮线）。

---

> 📖 以上就是《理论力学》平时大作业的全部解答（感悟题除外）。这五类题型覆盖了分析力学的核心内容：**基本概念 → 小振动 → 静平衡（含虚功原理）→ 作用量与变分原理 → 变分法应用**。
